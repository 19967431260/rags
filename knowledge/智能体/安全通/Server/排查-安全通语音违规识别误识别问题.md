## 问题：Server 安全通语音违规识别误识别问题

## 问题详情

**现象**：
语音敏感词监控出现没说敏感词但检测到敏感词的情况（误识别）。易盾返回的content与frontSegment内容不匹配

## 排查过程

1. 提供taskid给易盾分析 → 确认ASR转译错误导致误伤
2. 分析content与frontSegment关系 → frontSegment是命中断句前20s的证据信息
3. 确认策略调整流程 → 需要汇总误识别taskid反馈给易盾
4. 排查声纹检测问题 → content是声纹检测和语音识别内容，frontSegment是关联证据

## 问题原因

ASR转译错误导致误伤，声纹检测内容与实际违规内容不匹配。frontSegment是关联证据信息，不是违规内容本身

## 解决方案

1. 汇总误识别taskid反馈给易盾调整策略
2. 理解frontSegment含义：命中断句前20s的证据信息，用于辅助判断
3. 优先看evidences的action命中结果，嫌疑情况下frontSegment作为补充
4. 联系易盾技术支持调整细分策略
