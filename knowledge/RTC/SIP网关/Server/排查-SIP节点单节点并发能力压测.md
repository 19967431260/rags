## 问题：SIP节点单节点并发能力压测

## 问题详情

**现象**：
需要确认SIP节点单节点最大并发能力，为扩容提供依据

**环境信息**：
- 平台：服务端

## 排查过程

1. 设计压测方案 → 逐步增加并发数
2. 监控服务器指标 → CPU、内存、磁盘IO、网络
3. 观察画面质量 → 记录卡顿和延迟情况
4. 确定瓶颈 → 16c机器约10-15路并发

## 问题原因

需要实测确定SIP节点单节点最大并发能力

## 解决方案

16c机器单节点约支持10-15路SIP并发（20-30路流），建议纵向扩容至32c或48c提升并发能力
