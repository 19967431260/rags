## 问题：Web端接收不到视频画面

## 问题详情

**现象**：
发送端音视频都正常，接收端视频接收不到，声音断断续续

**环境信息**：
- 平台：Web

## 排查过程

1. 确认发送接收端 → 发送端UID 5317124，接收端UID 5254464
2. 检查白名单配置 → 确认正式环境已关闭云代理
3. 分析网络状况 → 重新进入房间观察服务器调度
4. 确认问题原因 → 部分用户网络带宽不够

## 问题原因

部分用户网络带宽不足导致视频接收异常

## 解决方案

确认是网络带宽问题，建议测试环境验证。正式环境和测试环境服务器配置一致
