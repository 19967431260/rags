# G2私有化部署支持H5和微信小程序接入

## 问题描述

客户咨询G2行内私有化部署是否支持H5和微信小程序接入，以及是否需要额外服务器

## 解答

G2私有化部署支持H5和微信小程序接入，只需要提供域名和对应的SSL证书即可。不需要额外服务器，也不需要单独加转码服务器。
