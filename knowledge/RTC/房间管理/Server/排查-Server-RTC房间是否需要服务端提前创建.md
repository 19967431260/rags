## 问题：Server RTC房间是否需要服务端提前创建

## 问题详情

**现象**：
客户咨询音视频房间是否需要通过服务端接口提前创建。

**排查过程**：


## 问题原因



## 解决方案

RTC房间不需要单独创建，客户端直接加入会自动创建房间。服务端只需监控房间创建销毁过程，通过抄送监控即可。
