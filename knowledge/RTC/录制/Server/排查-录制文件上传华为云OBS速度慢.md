## 问题：Server 录制文件上传华为云OBS速度慢

## 问题详情

**现象**：
客户反馈录制的视频文件上传到华为云 OBS 速度很慢（5MB 文件需要 3.5 秒，约 1.4MB/s）。客服检查日志确认 S3 挂载写入速度受限。

## 排查过程

1. 客服检查上传日志，确认上传速度确实很慢 → 速度确认
2. 确认能信侧录制服务器是本地磁盘传输，速度 5MB/s → 本地速度
3. 该客户环境上传到华为云 OBS，S3 挂载方式写入受限 → 根因
4. 客服将上传方式改为 SDK 方式后，速度显著提升 → 解决方案

**关键发现**：华为云 S3 挂载方式写入速度有限制，导致录制文件上传慢

## 问题原因

华为云 S3 挂载方式写入速度有限制，导致录制文件上传慢

## 解决方案

将录制文件上传方式从 S3 挂载方式改为 SDK 方式上传，可以显著提升上传速度。建议如上传速度有要求，优先使用 SDK 方式。
