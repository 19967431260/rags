## 问题：Web Web SDK 5.8.30变声效果失效

## 问题详情

**现象**：
升级到RTC Web SDK 5.8.30版本后，远端流变声效果失效，声音还是原声。5.6.30版本正常。

## 排查过程

1. 客户反馈变声问题 → 技术支持检查插件版本\n2. 客户提供房间信息和日志 → 分析发现后进入用户听不到先进用户的变声效果\n3. 本地调试测试 → 确认问题存在\n**关键发现**：SDK版本升级后变声插件兼容性问题

## 问题原因

RTC Web SDK 5.8.30版本变声插件兼容性问题

## 解决方案

升级到5.8.31版本修复此问题。注意变声插件版本需要与SDK版本保持一致。
