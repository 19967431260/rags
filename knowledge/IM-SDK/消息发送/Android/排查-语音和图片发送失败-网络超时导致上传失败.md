## 问题：Android 语音和图片发送失败-网络超时导致上传失败

## 问题详情

**现象**：
用户反馈发送IM语音和图片失败，日志显示aws.s3上传失败，错误为Couldn't resolve host name。

**相关日志**：
aws.s3 Error type:99, Response code:-1
curlCode: 6, Couldn't resolve host name

## 排查过程

1. 分析日志 → aws.s3 Error type:99, Response code:-1
2. 关键错误 → curlCode: 6, Couldn't resolve host name
3. 研发判断 → 网络超时导致上传文件失败

**关键发现**：curlCode: 6 表示无法解析主机名，网络超时导致上传失败

## 问题原因

网络超时导致文件上传至AWS S3失败。

## 解决方案

针对海外网络不稳定情况，建议升级到9.19.14优化版本，该版本针对海外网络环境进行了深度优化。
