## Q: IM SDK V9版本Android 5以下兼容性

**问题描述**：客户使用IM SDK V9版本(8.9.117)，咨询是否支持Android 5以下系统。

**答案**：8.9.117版本可以支持Android 5以下使用，8.10.0及以上需要Android 5.0+。

## Q: 移动端V9与服务端V8兼容性

**问题描述**：咨询移动端使用V9 SDK，服务端使用V8 API是否兼容

**答案**：V9以前版本相互兼容，V10是破坏性更新不兼容。移动端V9与服务端V8可以兼容，但客户端换到V10时服务端也建议换到V10 API
