## 问题：Web 删除本地会话后仍会重新出现在会话列表

## 问题详情

**现象**：
客户删除本地会话后，刷新应用时该会话又重新出现在会话列表中。

## 排查过程

1. 确认删除后确实不会立即出现 → 删除操作正常
2. 重新初始化后会话又出现 → 会话恢复
3. 客服确认 web 端会话依赖漫游 → 机制说明
4. 只要不删除漫游，消息同步后会重新产生会话 → 根因确认

**关键发现**：web 端会话依赖服务器漫游消息，删除本地会话不会删除云端漫游，重新登录或初始化时会从服务器重新拉回漫游数据

## 问题原因

web 端会话依赖服务器漫游消息，删除本地会话不会删除云端漫游，重新登录或初始化时会从服务器重新拉回漫游数据

## 解决方案

如需彻底删除会话，需要删除对应的漫游消息（服务端接口）。删除漫游消息后，该会话不会再重新出现。
