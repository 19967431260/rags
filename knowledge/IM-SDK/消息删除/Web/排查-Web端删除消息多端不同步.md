## Q: Web端删除消息多端不同步

**问题描述**：客户反馈一个端删除消息，其他端拉取历史记录还是能获取到，没有收到删除回调

**排查过程**：客户反馈删除不同步 → 客服确认删除接口是否多端同步 → 客服说明是同步的 → 客服建议提供双设备日志查看 → 客服说明删除是自己端删自己的，和别人无关

**根因**：删除消息接口只是删除自己看到的消息，不会同步给其他人

**解决方案**：删除消息是针对本端生效的，不会同步删除其他端的消息，其他端需要自行调用删除接口
