## 问题：Server 第三方回调出现大量502错误

## 问题详情

**现象**：
客户反馈消息第三方回调出现问题，大批量消息受影响。

## 排查过程

1. 客户提供消息ID排查 → 2. 发现客户业务服务器出现大量502报错 → 3. 确认客户在切换服务导致 → 4. 客户修复后502消失

## 问题原因

客户业务服务器在切换服务时出现异常，导致对云信回调请求的响应出现502错误。

## 解决方案

检查业务服务器状态，确保回调地址可正常访问。云信侧只是普通的post请求，没有特别的保护机制。
