## 问题：V10 SDK开启云端会话后首页消息无通知

## 问题详情

**现象**：
客户使用V10 SDK发送消息，发现首页消息列表没有通知显示，而V9版本有该功能。

## 排查过程



## 问题原因

V10 SDK默认使用云端会话，需要开启云端会话权限才能正常使用相关功能。

## 解决方案

在控制台开启云端会话权限：进入应用管理 → 功能配置 → 云端会话，开启对应开关即可。
