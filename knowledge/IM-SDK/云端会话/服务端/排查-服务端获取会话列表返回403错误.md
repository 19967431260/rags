## 问题：服务端 服务端获取会话列表返回403错误

## 问题详情

**现象**：
调用服务端获取会话接口返回code:403 forbidden错误。

**环境信息**：
- SDK 版本：
- 系统版本 / 设备：
- 平台：服务端

## 排查过程

1. 检查appkey → 标准版
2. 确认功能限制 → 云端会话需要高级版
3. 提供替代方案 → 使用客户端接口获取本地会话
4. 说明漫游机制 → 服务端会漫游最多100个会话到新设备

## 问题原因

云端会话是高级版功能，标准版无法使用服务端API获取会话列表

## 解决方案

1. 升级到高级版使用云端会话
2. 使用客户端SDK接口获取本地会话
3. 标准版不支持第三方回调，需通过其他方式实现
