# 服务端是否有单纯踢登录的API

## 问题描述

咨询是否有不封禁只踢用户下线的API接口

## 问题背景

- 查看服务端API文档，封禁接口可以踢人，但会同时封禁账号

## 解答

目前服务端没有单纯踢登录的API，只有封禁用户时会踢下线。刷token的方式也不行。如需踢人，只能使用封禁接口。
