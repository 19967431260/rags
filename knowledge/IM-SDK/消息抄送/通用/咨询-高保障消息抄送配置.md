# 高保障消息抄送配置

## 问题描述

客户咨询抄送失败是否会重试，以及项目发版期间如何处理抄送

## 解答

默认抄送不会重试。如需重试功能，可在控制台全局功能中开通「高保障消息抄送」。开通后如果超时失败，服务器会尝试重新抄送最多1000次，每5秒重试一次。支持将最多50万条消息缓存，待接口恢复后手动提交重新抄送。
