## 问题：Web Web端会话丢失与漫游机制说明

## 问题详情

**现象**：
用户反馈昨天聊天会话今天查看时丢失，onSessions获取不到该会话，且昨天聊天记录全部消失。

## 排查过程

1. 确认问题范围 → 丢个别会话，非全部
2. 核实消息记录 → 服务器无昨天消息记录，只有今天
3. 检查客户本地 → 客户本地也无昨天消息，最新是今天，再往前是12月23号
4. 分析原因 → Web端本地会话根据本地消息生成，漫游消息7天

## 问题原因

Web端本地会话基于本地消息生成，漫游消息仅保留7天；超过7天未登录导致会话漫游不到

## 解决方案

V9架构限制：本地会话根据本地消息生成，漫游消息7天，无法开发接口同步。建议升级到V10，V10使用云端会话，只要会话未主动删除就能获取
