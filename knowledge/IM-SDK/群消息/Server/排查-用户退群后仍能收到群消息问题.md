# 用户退群后仍能收到群消息问题

## 问题描述

用户已退群（nim_tlist无数据，nim_tlist2有数据），但仍能接收到该群的消息

## 问题背景

1. 核查数据库发现nim_tlist无数据，nim_tlist2有数据
2. 确认是服务器临界场景下的异常
3. 尝试修改数据库和清理redis缓存

## 根因分析

服务器临界场景下群成员数据异常，tlist2表valid字段未正确设置

## 解决方案

临时解决方案：将nim_tlist2表中对应记录的valid字段设置为0。根本解决需要服务端优化处理逻辑
