## 问题：Server 更新消息接口返回107344错误

## 问题详情

**现象**：
调用更新消息接口更新单聊消息，返回107344不允许更新消息，operator和原发送者一致且未超过七天。

**排查过程**：
1. 客户提供消息ID
2. 客户自行发现原因：消息类型写错

## 问题原因

消息类型参数填写错误

## 解决方案

检查并修正消息类型参数
