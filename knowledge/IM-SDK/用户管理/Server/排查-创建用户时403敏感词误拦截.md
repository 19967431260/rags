## 问题：Server 创建用户时403敏感词误拦截

## 问题详情

**现象**：
调用userCreate接口创建accid为1414466的用户时，使用用户名"王勇"，返回403错误：forbbiden info，被易盾敏感词策略误拦截。

**相关日志**：
403 forbbiden info
taskId: 9yjc5musj32l0iplgnzwc02i0010aje5

## 排查过程

1. 检查错误信息 → 返回403 forbbiden info
2. 确认taskId: 9yjc5musj32l0iplgnzwc02i0010aje5
3. 联系易盾策略经理 → 将"王勇"加白

**关键发现**：正常人名被敏感词策略误判

## 问题原因

易盾敏感词策略将正常人名"王勇"误判为敏感词

## 解决方案

联系易盾策略经理将该词加白，加白后几分钟即可生效
