## 问题：Web 点播视频转码后Web端立即访问失败

## 问题详情

**现象**：
管理后台上传点播视频并转码成功后，立刻在Web端访问视频加载失败，等待1-2分钟后恢复正常。iOS和Android端不存在此问题，仅Web端出现。浏览器F12显示未解析到远端CDN的IP。

## 排查过程

1. 检查视频URL → 技术支持侧访问正常
2. 对比CDN节点IP → 失败时浏览器未解析到CDN IP
3. 检查网络环境 → 怀疑本地网络限制或浏览器拦截

**关键发现**：浏览器未解析到CDN IP，更像是浏览器自身拦截请求，CDN侧已有数据

## 问题原因

浏览器自身拦截了对应请求，可能与本地网络环境或浏览器安全策略有关

## 解决方案

建议使用手机流量开热点，电脑连接热点测试是否仍有问题。技术支持侧访问正常，CDN侧数据已就绪，问题出在客户端网络环境或浏览器层面。
