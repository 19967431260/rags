## 问题：Server 超大群修改人数上限触发806错误问题

## 问题详情

**现象**：
客户调用接口调整超大群人数上限（2000->4000）时触发806错误，提示owner own team count exceed。

## 排查过程

1. 客户反馈调整超大群人数上限触发806错误
2. 技术支持确认服务器对超过2000人的超大群数量有限制
3. 默认限制：SuperTeamOwnSuperTeamCountLimit=20
4. 需要在disconf-device模块-device.properties中增加配置参数

## 问题原因

服务器默认限制单个用户拥有的超过2000人的超大群数量为20个

## 解决方案

在disconf--device模块---device.properties中增加SuperTeamOwnSuperTeamCountLimit和SuperTeamAddSuperTeamCountLimit参数，修改为目标期望值
