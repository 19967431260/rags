## 问题：Web Web端在线状态订阅断网后无法恢复在线状态

## 问题详情

**现象**：
客服端断网后用户端通过pushEvents监听到离线状态，但客服端重新连网后用户端未收到在线状态更新。

## 排查过程

1. 确认问题现象 → 用户端监听pushEvents，只收到离线回调未收到在线回调
2. 确认平台信息 → 客服端Web，用户端Web和UniApp
3. 排查客服端上线状态 → 需要确认客服端断网后是否正常重新上线

## 问题原因

待进一步排查，需要客服端日志和断网时间点分析

## 解决方案

需要提供客户端日志和断网时间点进行进一步分析，确认客服端断网后是否正常重新上线并触发状态同步。
