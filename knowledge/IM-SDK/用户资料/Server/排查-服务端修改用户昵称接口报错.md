## 问题：Server 服务端修改用户昵称接口报错

## 问题详情

**现象**：
客户调用服务端修改用户昵称接口返回invalid request body错误。

## 排查过程

1. 确认接口路径参数account_id是否正确 → 检查路径参数
2. 确认body参数格式是否为JSON → 检查body格式
3. 建议用postman测试验证 → 提供测试方案

**关键发现**：body参数格式不正确，name需要以JSON方式封装

## 问题原因

body参数格式不正确，name需要以JSON方式封装

## 解决方案

PATCH /im/v2/users/{account_id}，account_id为具体用户值。name等参数需要以JSON格式封装到body中。
