## 问题：Server 管理后台查不到用户信息原因

## 问题详情

**现象**：
新旧版管理后台都查不到用户信息，但accid在客户端正在使用

**排查过程**：

1. 检查appkey → 客户提供fa2f633c9a0c32376948a1a997431bd8
2. 查询账号 → 11890936账号未注册
3. 确认正确key → 实际应为d72cc53f39e17eee39d761c2079c5e8d

**关键发现**：使用了错误的appkey导致查询不到用户信息

## 问题原因

搞错了appkey，使用了错误的账号查询

## 解决方案

确认正确的appkey后再查询；管理后台需要选择正确的appkey才能查到对应数据
