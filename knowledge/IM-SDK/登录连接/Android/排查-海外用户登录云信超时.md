# 海外用户登录云信超时

## 问题描述

海外用户（新加坡、加拿大、日本、美国、泰国等）登录云信时出现超时问题

## 问题背景

- 使用Android SDK 9.8.0版本
- Android系统版本4.4
- 仅部分用户登录不上

## 根因分析

SDK版本较老（9.8.0），缺少网络连接优化。9.18.0和9.19.2版本做了网络链接优化，9.19.2还增加了快连机制

## 解决方案

建议升级到Android SDK 9.19.4版本，该版本包含网络连接优化和快连机制，可解决海外连接性问题。

更新日志参考：https://doc.yunxin.163.com/messaging/concept/DM1MjI5MTE?platform=client

注意：Android 4.4版本较低，建议在Android 7以上系统使用。
