# IM续费后测试环境无法登录

## 问题描述

客户IM已续费，但测试环境仍无法登录，提示功能未开通

## 问题背景

- 检查测试环境应用的功能开通状态

## 根因分析

测试环境应用的功能未开通，仅正式环境开通

## 解决方案

测试环境应用需要单独开通功能。联系技术支持人员（如王斌）在后台恢复测试环境应用的功能即可。
