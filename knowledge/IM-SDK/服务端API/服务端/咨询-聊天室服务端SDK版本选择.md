# 聊天室服务端SDK版本选择

## 问题描述

客户咨询GitHub上的服务端SDK版本0.1和0.2.0-SNAPSHOT如何选择

## 解答

服务端SDK V9（V1）未完全封装，目前需要自行将URL和param传入，没有单独的每个接口一个方法。建议使用最新版本0.2.0-SNAPSHOT，或自行封装调用。
