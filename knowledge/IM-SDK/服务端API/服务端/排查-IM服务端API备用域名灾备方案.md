## 问题：服务端 IM服务端API备用域名灾备方案

## 问题详情

**现象**：
多次出现IM服务端API请求超时和域名不通问题，客户需要备用域名作为灾备方案。

**环境信息**：
- SDK 版本：
- 系统版本 / 设备：
- 平台：服务端

## 排查过程

1. 5月3日Redis服务器宕机导致部分请求失败 → 系统自动主从切换恢复
2. 5月15日机房网络故障 → 影响七鱼在线客服
3. 5月22日机房网络设备割接异常 → 提供备用域名方案

## 问题原因

公有云环境受机房网络设备故障、Redis宕机等影响，需要备用域名作为灾备。

## 解决方案

提供备用域名api-cn-bak.yunxinapi.com作为灾备，与主域名节点线路有区别。建议客户端实现域名切换机制，主备互为灾备。如需更高可用性，可升级至专属云套餐。
