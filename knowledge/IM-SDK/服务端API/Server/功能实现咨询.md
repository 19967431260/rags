## Q: 服务端发送好友申请提示消息实现双方不同文案

**问题描述**：使用服务端接口nimserver/msg/sendMsg.action发送好友申请通过提示时，需要让A看到'对方已同意您的好友申请'，B看到'您已同意对方的好友申请'。

**答案**：使用sendMsg接口的sendersilent和receiversilent参数控制消息可见性，发送方无感知和接收方无感知参数可控制谁不要看到这条消息。参考文档：https://doc.yunxin.163.com/messaging/server-apis/DQ2NTg4ODE?platform=server

## Q: 通过后台API删除账号所有收发消息

**问题描述**：想删除一个账号的所有收发消息，但只知道账号，调用接口返回查询失败

**答案**：V9没有直接API可以删除账号所有消息。建议：1. 如果接了抄送，从抄送记录中捞取消息参数后调用删除接口；2. 利用抄送维护消息记录，每条消息以抄送形式到达业务服务器
