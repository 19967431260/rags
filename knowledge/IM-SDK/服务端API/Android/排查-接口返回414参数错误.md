**现象**：

- 平台：Android

## 问题原因

414是参数错误，非权限问题。403才是功能未开通。

## 解决方案

414为参数错误，需要获取复现时间点的日志排查具体参数问题。403才是功能未开通的返回码。参考日志获取文档：https://faq.yunxin.163.com/kb/main/#/item/KB0179
