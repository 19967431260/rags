# 聊天室获取历史消息数量限制

## 问题描述

客户使用getHistoryMsgs接口获取聊天室历史消息，发现超过100条就获取不到了

## 解答

聊天室获取历史消息接口(getHistoryMsgs)的limit参数一次最大支持100条，超过100条需要分页获取。
