## 问题：Windows 聊天室独立模式设置扩展字段不显示

## 问题详情

**现象**：
Windows聊天室独立进入时设置enterInfo.login_ext_扩展字段，但用户信息没有显示。

## 排查过程

1. 确认字段设置 → login_ext_字段
2. 检查加入成功时字段打印 → 正常
3. 确认其他端获取情况 → 网页端通过custom获取为空
4. 与研发沟通确认 → 独立模式下该接口不支持设置扩展字段
5. 客户追问election版本可以 → 确认需要走IM登录的非独立模式

## 问题原因

独立模式下，聊天室接口不支持设置扩展字段。需要使用走IM登录的非独立模式才能设置扩展字段

## 解决方案

如果需要设置扩展字段，需要使用非独立模式（先登录IM再进入聊天室），而不是独立模式
