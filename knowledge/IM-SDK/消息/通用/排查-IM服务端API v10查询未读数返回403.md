## 排查过程

1. 客服确认接口调用情况 → 接口分析
2. 确认 IM V10 未开通云端会话功能 → 根因

**关键发现**：客户使用的是IM V10但未开通云端会话功能，而该查询未读数接口依赖云端会话。

## 问题原因

客户使用的是IM V10但未开通云端会话功能，而该查询未读数接口依赖云端会话。

## 解决方案

该接口需要云端会话支持，当前IM V10版本不支持云端会话，需确认并开通对应功能模块。
