## 问题：Web IM V9 Web端多域名请求超时排查

## 问题详情

**现象**：
客户反馈多家用户无法聊天，连接超时。从截图中看到statistic数据上报等多个域名请求超时。

## 排查过程

1. 检查日志级别 → 建议开启debug日志
2. 检查网络限制 → 发现SDK域名全超时
3. 确认客户网络环境 → 客户声称无网络限制
4. 建议排查 → 让用户ping和dig测试，尝试手机5G网络

## 问题原因

客户设备所在网络存在限制，或防火墙/代理导致SDK域名无法访问

## 解决方案

检查网络限制：确认是否有防火墙、代理等限制；使用ping和dig测试域名连通性；尝试手机5G网络直接测试排除网络环境问题
