## Q: JSP项目集成IM SDK方案

**问题描述**：客户使用JSP开发前端页面，需要集成IM即时通讯功能，咨询如何集成SDK。

**答案**：云信没有针对JSP的专门接入方式，但可以使用以下方案：
1. **推荐方案**：使用无UI SDK（nim-web-sdk-ng），不限制开发框架的纯JS SDK
   - 安装：npm install nim-web-sdk-ng@">=10"
   - 将下载的js文件导入到JSP项目中即可使用
2. **备选方案**：使用带UI的UIKit SDK，但对开发框架有要求，目前支持React、Vue2或Vue3
