## Q: Web UIKit无未读会话列表专用接口

**问题描述**：客户咨询：Web端UIKit中是否有接口可以查询未读的会话列表。

**答案**：UIKit没有专门的未读会话列表查询接口。只支持按顺序增量获取所有会话列表（ConversationStore.conversations），获取全量后需在业务层自行筛选未读会话。参考：https://doc.yunxin.163.com/messaging2/references/web/typedoc/IMUIKit/Latest/classes/ConversationStore.html#conversations

## Q: Web UIKit自定义会话列表点击事件

**问题描述**：客户在Web端集成UIKit，想在会话列表点击时获取对端账号（account），自行配置的点击事件概率性触发不了。

**答案**：在调用this.$uikit.render(ConversationContainer, {...})时，在参数中定义onConversationItemClick回调：this.$uikit.render(ConversationContainer, { onConversationItemClick: (conversation) => { console.log('会话点击：'+conversation); } }, this.$refs.conversation)。onConversationItemClick应与renderConversationName同级别定义在ConversationContainer参数中。
