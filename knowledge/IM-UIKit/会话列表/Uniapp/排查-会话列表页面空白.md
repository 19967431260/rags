## 问题：Uniapp Uniapp会话列表页面空白

## 问题详情

**现象**：
客户在Uniapp项目中集成会话列表组件，页面显示空白，无任何内容。控制台报错onOnShow/onOnHide生命周期调用时无活动组件实例。

**相关日志**：
Vue warn: onOnShow is called when there is no active component instance

## 排查过程

1. 检查初始化配置 → 发现未开启云端会话
2. 建议开启enableV2CloudConversation配置
3. 检查路由配置 → 会话路由已配置
4. 检查生命周期报错 → 发现Vue warn: onOnShow is called when there is no active component instance

**关键发现**：生命周期API使用不当，且云端会话未开启

## 问题原因

未开启云端会话功能，且组件生命周期使用存在问题

## 解决方案

1. 在管理后台开启云端会话功能
2. 在UIKit初始化时设置enableV2CloudConversation=true
3. 确保路由配置正确
4. 检查onShow/onHide生命周期是否在setup()中正确调用
