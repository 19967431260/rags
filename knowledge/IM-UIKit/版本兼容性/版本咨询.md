## Q: IM UIKit V9 与 V10 版本区别及选择建议

**问题描述**：用户咨询IM UIKit V9和V10版本的区别，以及如何选择合适的版本。

**答案**：V9和V10版本主要区别：
1. 云端会话：V10支持云端会话功能，会话列表可以保留全量；V9只能保留七天内有消息的会话，最多100个。云端会话需要开通1999及以上套餐
2. 框架支持：V10 UIKit只支持Vue3；V9支持Vue2和Vue3
3. Web版本：V10 Web UIKit支持Vue2和Vue3
4. Node版本：建议使用Node 18及以上

推荐：现在都推荐用V10，目前迭代都在V10上进行。

## Q: IM UIKit Web端直接跳转到指定聊天窗口

**问题描述**：用户需要实现从自己项目点击好友，直接跳转到UIKit demo并打开对应好友的聊天窗口，而不是手动选择。

**答案**：使用gotoChat方法实现页面加载时直接定位到指定聊天窗口。关键代码：
```
const store = this.$uikit.getStateContext()?.store;
const myAccount = store.userStore.myUserInfo.accountId;
const conversationType = 1;
const receiverId = "对方account";
const conversationId = `${myAccount}|${conversationType}|${receiverId}`;

if (store.localConversationStore.conversations.get(conversationId)) {
  store.uiStore.selectConversation(conversationId);
} else {
  const isSelected = true;
  store.localConversationStore.insertConversationActive(conversationType, receiverId, isSelected);
}
```
注意：如果使用的是localConversationStore，需要用相应的方法。

## Q: API操作会话接口需要开通云端会话功能

**问题描述**：用户调用服务端API操作会话相关接口时遇到问题，咨询需要什么配置。

**答案**：API操作的会话相关接口都需要开通云端会话功能才能使用。云端会话功能只有1999及以上的套餐支持。免费版不支持这些接口。

## Q: IM UIKit Web端黑名单功能对接

**问题描述**：用户咨询如何在IM UIKit Web端对接黑名单功能。

**答案**：黑名单功能可以参考官方文档实现。UIKit本身不直接提供黑名单UI按钮，需要开发者自行实现拉黑按钮，调用底层SDK的黑名单接口。
