## Q: Web UIKit安装JSX编译依赖jsx-web-compiler

**问题描述**：客户询问图示依赖是什么，以及如何安装。

**答案**：jsx-web-compiler是需要主动安装的npm依赖。安装方式：命令行执行 npm install jsx-web-compiler，或在package.json的dependencies中增加 "jsx-web-compiler": "^1.0.2" 后执行 npm install。
