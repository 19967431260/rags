## Q: Chrome 140版本兼容问题升级建议

**问题描述**：Chrome浏览器最近发布的140版本与RTC Web SDK存在兼容性问题。

**答案**：建议尽快升级到RTC Web SDK 5.8.30版本以解决Chrome 140版本兼容问题。

## Q: Chrome 140版本兼容性问题

**问题描述**：Chrome浏览器最近发布的140版本与RTC Web SDK存在兼容性问题，不开麦克风只开摄像头时一定失败。

**答案**：建议尽快升级到RTC Web SDK 5.8.30版本，如使用Edge 140版本以上，需升级到5.8.31版本。AI变声插件wasm文件需同步更新。
