## 问题：Web 虚拟背景加载延迟及人物变黑问题

## 问题详情

**现象**：
使用虚拟背景插件后，视频播放后虚拟背景会延迟1秒左右才加载，且人物会变黑。

**环境信息**：
- SDK 版本：
- 系统版本 / 设备：
- 平台：Web

## 排查过程

1. 确认加载顺序 → 需要先加载虚拟背景再publish
2. 分析异步问题 → 加载是异步的，需要处理plugin-load事件
3. 提供示例代码 → 在plugin-load回调中开启虚拟背景并发布
4. 排查人物变黑 → 需要对比有无虚拟背景的效果

## 问题原因

虚拟背景加载是异步的，未等待加载完成就发布；人物变黑问题需进一步排查

## 解决方案

在plugin-load事件回调中开启虚拟背景，然后再publish。确保SDK版本和插件版本一致
