# 音视频通话1.0和2.0计费方式区别

## 问题描述

客户咨询音视频通话1.0和2.0的计费方式差异，以及功能开通后的收费规则

## 解答

音视频通话1.0和2.0计费方式：
- 1.0没有月功能费，纯按量计费
- 2.0包含月功能费，同时有包含部分免费的通话时长
- 测试环境功能开通同样需要计费
