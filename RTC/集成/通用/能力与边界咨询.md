## Q: IM SDK支持iOS模拟器RTC不支持

**问题描述**：客户询问云信 SDK 能否在 iOS 模拟器上运行，用于上传 App Store 需要 iPad 截图。

**答案**：IM SDK 支持 iOS 模拟器运行。RTC SDK 不支持在模拟器运行，需要真机测试。如需 iPad 截图，可使用模拟器运行 UI 部分截图，音视频功能需在真机测试。
