## 问题：SIP节点负载均衡调度优化

## 问题详情

**现象**：
SIP节点调度不均衡，部分节点负载过高，需要优化调度策略

**环境信息**：
- 平台：服务端

## 排查过程

1. 分析当前调度 → 信令连哪台SDP就回哪台
2. 发现问题 → 公网入口统一导致SDP返回混乱
3. 优化方案 → 按端口范围转发到对应节点
4. 验证效果 → 调整后负载分布更均衡

## 问题原因

SIP调度策略导致信令和媒体集中到单一节点

## 解决方案

按端口范围配置转发规则，确保信令和媒体SDP返回一致，实现负载均衡
