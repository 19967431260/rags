## 问题：Server 部分用户个人录制视频只有音频无画面

## 问题详情

**现象**：
客户反馈多个会议中部分专家的个人录制视频文件只有音频没有画面，但汇总视频中能看到该专家的头像和桌面。问题出现在特定切片中，非所有用户都有此问题。

## 排查过程

1. 客服确认是哪个 cid 和用户 ID → 定位问题用户
2. 确认仅特定切片（1、2 切片）部分用户有此问题 → 切片范围
3. 客服确认为转码问题 → 根因方向
4. 该问题是之前已定位的问题，需要服务器更新修复 → 已知问题
5. 服务器更新修复后，该问题在后续会议中未再出现 → 修复验证

**关键发现**：私有化服务器录制转码存在问题，导致部分用户的高优先级录制流在特定切片中画面丢失

## 问题原因

私有化服务器录制转码存在问题，导致部分用户的高优先级录制流在特定切片中画面丢失

## 解决方案

该问题为服务端已知问题，通过服务器更新（重启并部署修复包）解决。问题修复后需进行验证测试。
