## Q: 查询历史会议视频和文件对应

**问题描述**：客户想把公有云的历史视频文件转存到私有化环境中，需要查找之前的视频文件。客户询问公有云还保留了哪些数据、生成视频的记录表格，以及文件名是否能解析出房间名。

**答案**：如音视频功能还在，可调用服务端 API 查询：https://doc.yunxin.163.com/nertc/server-apis/TI5MTIwNDc?platform=server。但 2024 年历史数据已无法通过服务器查询。客户可依赖抄送存储自行维护 cid 与 cname 的映射关系。
