## 问题：通用 无法获取视频流问题

## 问题详情

**现象**：
无法获取到视频流

**环境信息**：
- 时间：2025-06-10

## 排查过程

1. 确认问题现象 → 无法获取视频流
2. 分析日志 → 检查配置参数
3. 定位问题 → 未设置screen和screenAudio参数

**关键发现**：需要设置screen和screenAudio参数为true才能使用自定义采集功能

## 问题原因

未设置screen和screenAudio参数为true，导致无法使用自定义采集功能

## 解决方案

设置screen和screenAudio参数为true：
```javascript
{
  screen: true,
  screenAudio: true
}
```
