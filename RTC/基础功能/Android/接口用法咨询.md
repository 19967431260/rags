## Q: 快速切换IM用户的方法

**问题描述**：
客户咨询如何快速从用户A切换到用户B（已登录IM、加入RTC、进入聊天室）。

**答案**：
切换用户的流程：
1. logoutIM（退出当前IM账号）
2. leaveChannel（离开RTC房间）
3. loginIM（登录新用户B）

## Q: RTC Maven仓库访问配置

**问题描述**：
客户反馈maven库访问不了，无法下载nertc-audio包。

**答案**：
1. **仓库配置**：
   ```gradle
   maven { url "https://oss.sonatype.org/content/repositories/snapshots/" }
   ```

2. **依赖配置**：
   ```gradle
   implementation 'com.netease.yunxin:nertc-audio:5.8.5001-SNAPSHOT'
   ```

3. **注意**：直接拼接的URL可能无法打开，需要通过maven配置访问
