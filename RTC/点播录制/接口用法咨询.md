## Q: 点播历史视频可通过接口批量删除

**问题描述**：客户历史点播视频较多，想确认除控制台页面外，是否支持通过服务端接口批量删除历史视频文件，以降低持续存储成本。

**答案**：支持。可调用点播视频删除相关 Server API 批量删除历史视频，也可以在控制台按页勾选删除。若数据量很大，建议先查询列表再按批次执行删除。
