## 问题：Android Android平板前置摄像头画面偏暗问题

## 问题详情

**现象**：
客户在私有化环境测试时，Android平板前置摄像头画面很暗，但后置摄像头和系统相机正常。设备型号较老，测试分辨率为1080p。切换到流畅模式画面亮度正常，其他分辨率偏暗。

## 排查过程

1. 检查环境光线 → 环境明亮但画面仍暗
2. 升级SDK到5.9.20暗光增强版本 → 改善不明显
3. 切换camera1 API → 仍然偏暗
4. 测试其他设备 → 其他设备正常
5. 对比公有云和私有化环境 → 需要进一步排查

**关键发现**：特定老设备前置摄像头在高分辨率下曝光不足

## 问题原因

老设备前置摄像头在高分辨率下曝光参数需要针对性调整

## 解决方案

针对老设备型号进行曝光参数调整。临时方案可使用流畅模式或降低分辨率。SDK团队需要针对该设备型号优化曝光算法。
