## 问题：Server 存储离线消息设置对API发送消息的作用及离线消息未收到问题

## 问题详情

**现象**：
咨询存储离线消息设置对通过API接口发送的消息是否有效，以及离线后再登录没收到消息的问题

## 排查过程

1. 确认设置有效性 → 存储离线消息设置对API发送的消息有作用
2. 排查未收到原因 → 需要消息ID和接收方日志进一步分析
**关键发现**：问题待进一步排查，需要日志确认

## 问题原因

待排查，需要消息ID和接收方日志

## 解决方案

存储离线消息设置对API接口发送的消息有作用；如离线后未收到消息，需提供消息ID和接收方日志文件进行排查
