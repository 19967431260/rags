## 问题：Server 群名中文显示问号

## 问题详情

**现象**：
通过API设置群名为中文，获取群信息时返回问号。

**环境信息**：
- 平台：Server
- 产品：IM V10

**相关日志**：
（从会话提取关键报错日志，无则省略）

## 排查过程

1. 检查返回数据 → 确认返回的是问号
2. 检查设置值 → 发现设置时就是问号
3. 关键发现：客户上传的参数本身就是问号，不是SDK问题

## 问题原因

客户上传的群名参数本身就是问号，需要排查上传参数逻辑

## 解决方案

检查创建/修改群组时上传的name参数，确保中文编码正确；可通过指南针查询确认设置值
