# 群主退群和解散群的相关规则

## 问题描述

客户咨询群主退群是否会解散群组，以及解散后能否恢复

## 解答

群主相关操作规则：
1. 群主退出前需要先转让群组，直接退群不会自动解散
2. 群主解散群后，群ID无法恢复，只能重新建群
3. 查询用户已加入群组列表的接口不会显示已退出的群
