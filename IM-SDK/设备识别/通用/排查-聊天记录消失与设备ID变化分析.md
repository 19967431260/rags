## 问题：通用 聊天记录消失与设备ID变化分析

## 问题详情

**现象**：
用户反馈聊天记录不见了，本地消息不漫游。

**环境信息**：
- 平台：通用

## 排查过程

1. 确认查询方式 → 本地历史查询
2. 检查云端历史 → 确认最后消息时间
3. 分析设备ID → 发现SDK上报设备ID变化
4. 确认原因 → 用户重装或清理数据

**关键发现**：设备ID变化说明用户重装APP或清理数据，导致本地记录丢失

## 问题原因

设备ID变化说明用户重装APP或清理数据，导致本地记录丢失

## 解决方案

本地消息不漫游时，重装或清理数据会导致聊天记录丢失，需引导用户确认操作
