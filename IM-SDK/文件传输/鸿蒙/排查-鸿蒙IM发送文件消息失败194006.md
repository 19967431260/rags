## 问题：鸿蒙 鸿蒙IM发送文件消息失败194006

## 问题详情

**现象**：
鸿蒙版本IM发送file类消息失败，报错net ease send message error 194006 upload file error。

## 排查过程

1. 使用API demo测试可以正常发送 → 排除服务端问题
2. 对比发现NIMHttpServiceConfig配置问题 → 定位配置
3. 修改uploadReplaceFormat和downloadReplaceFormat配置后解决 → 确认方案

**关键发现**：NIMHttpServiceConfig配置不正确

## 问题原因

NIMHttpServiceConfig配置不正确，缺少uploadReplaceFormat和downloadReplaceFormat参数。

## 解决方案

配置httpServiceConfig.chunkUploadHost、uploadReplaceFormat和storageServiceConfig.downloadHost、downloadReplaceFormat，格式为'host/{bucket}/{object}'。
