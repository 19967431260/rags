**关键发现**：Web聊天室匿名登录414错误通常是因为头像字段（iconUrl）长度超过了允许的限制，应配置为有效的在线URL而非base64 data URI

## 问题原因

Web聊天室匿名登录414错误通常是因为头像字段（iconUrl）长度超过了允许的限制，应配置为有效的在线URL而非base64 data URI。

## 解决方案

Web聊天室匿名登录414错误：检查头像字段（iconUrl）长度，确保配置的是可访问的在线URL（不是data:开头的base64数据），建议控制在合理长度内。
