# 单端登录与聊天室自动退出

## 问题描述

客户希望用户进入新聊天室时自动退出之前进入的聊天室，实现同时只在一个聊天室

## 解答

在控制台开启单端登录配置即可实现。开启后，如果用户进了A聊天室，再去B聊天室时，会自动退出A聊天室。此配置按长链接计算，可以实现用户同时只在一个聊天室的效果。
