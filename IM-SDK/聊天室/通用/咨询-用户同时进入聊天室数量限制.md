# 用户同时进入聊天室数量限制

## 问题描述

客户咨询一个用户同时能处在多少个聊天室里面

## 解答

一个用户最多同时进入10个不同的聊天室，每个聊天室建立不同的连接。如果超过10个连接，服务器会踢出最早建立的连接。不支持手动配置该限制。
