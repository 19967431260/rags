## Q: 如何通过ESM方式减小小程序包体积

**问题描述**：Uniapp小程序主包体积过大，希望通过模块化引入方式减小SDK体积。

**答案**：1. npm安装SDK后，参考官方ESM引入demo 2. 在nim.js中通过registerService注册需要的功能模块 3. 执行npm run build打包产物 4. 在项目中引入dist目录下编译出的产物，而非直接引入整个SDK
