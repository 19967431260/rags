## 问题：Server 自定义抄送环境名称配置

## 问题详情

**现象**：
客户咨询自定义抄送的环境名称在哪里配置。

## 排查过程

1. 确认需要配置routeEnvironment字段 → 确认字段
2. 发现抄送总开关关闭 → 发现问题
3. 需要先打开总开关 → 解决方案

**关键发现**：抄送总开关未打开

## 问题原因

抄送总开关未打开

## 解决方案

构造message对象后，对V2NIMMessageRouteConfig字段赋值routeEnvironment。需要先打开抄送总开关。
