## 问题：服务端 客户端收不到自定义系统通知

## 问题详情

**现象**：
服务端调用sendCustomNotification发送自定义系统通知，日志显示发送成功（isSuccess=true, code=200），但客户端003s1fcyoy2wu9b7601534618a54972c收不到消息。

## 排查过程

1. 检查服务端调用记录 → 只查到/im/v2/friends调用，未查到自定义通知发送记录
2. 客户增加日志后重新发送 → 仍只查到/im/v2/friends记录
3. 客户提供traceId：ddb226bfcaf646ed9a1c99ce92dbb3e1 → 确认是用003pqbhyu4l416e9bf53ee2a6158f258发送的
4. 需要客户端SDK日志 → 接收方离线，需要上线后提取日志

## 解决方案

需要接收方上线并提取客户端SDK日志进行排查，参考：https://faq.yunxin.163.com/#KB0179/1
