## 问题：Web端Sync_Error和事件服务解包失败问题

## 问题详情

**现象**：
Web端从3月12日开始出现异常：1、Sync_Error同步数据失败；2、事件服务解包失败；3、SYNC_DATA_ERROR服务器繁忙。

**环境信息**：
- 平台：Web

## 排查过程

1. 客户提供错误统计截图和账号详单
2. 服务器确认整体登录成功率波动不大
3. 联系服务器确认sync有关报错与偶发的同步协议超时有关
4. 分析事件服务解包失败影响范围，主要集中在3-4个用户上

## 根因分析

sync有关报错与一些偶发的同步协议超时有关；事件服务解包失败主要影响事件服务的协议接收，可能导致个别用户看其他人的在线状态出现不准。

## 解决方案

sync问题已做调整，建议观察；事件服务解包失败建议升级到最新的9.19.2版本，对这方面有做优化，或选择再观察一段时间。
