# 群主被封号后群成员无法退出

## 问题描述

群主违规被永久封号后，群成员无法正常退出群聊

## 问题背景

- 群主封号后确实无法操作群相关功能

## 解答

封号后确实没办法处理。建议改用全局禁言代替封号，全局禁言后用户仍可登录，只是不能发言，群成员可以正常退出群聊。
