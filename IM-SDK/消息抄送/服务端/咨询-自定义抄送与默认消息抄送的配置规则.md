# 自定义抄送与默认消息抄送的配置规则

## 问题描述

客户咨询在已配置消息抄送地址的情况下，配置自定义抄送/第三方回调的触发规则

## 解答

自定义抄送配置规则：
1. 发送消息时env参数和环境名称必须相同才能触发对应环境的自定义抄送
2. env不设置时走默认的消息抄送
3. 如果env匹配成功，只会给自定义抄送发一份，不会重复发给默认消息抄送地址
