## 问题：Server 消息抄送没收到

## 问题详情

**现象**：
客户反馈从昨天开始消息抄送没收到。

## 排查过程

1. 客户反馈消息抄送没收到
2. 技术支持询问是哪个appkey
3. 客户发送appkey
4. 技术支持发现回调接口一直返回502
5. 客户确认是被拦截了

## 问题原因

回调接口返回502，被客户侧服务器拦截。

## 解决方案

检查回调接口服务状态，确保接口可正常访问，不被拦截。
