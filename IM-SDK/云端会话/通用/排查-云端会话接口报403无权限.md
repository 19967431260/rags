## 问题：通用 云端会话接口报403无权限

## 问题详情

**现象**：
客户使用V10服务端接口查询云端会话列表（https://doc.yunxin.163.com/messaging2/server-apis/DAwNDQ0NTU），返回403错误。客户已有IM高级版套餐。

**复现步骤**：
（从会话提取；无明确步骤则省略此节）

**环境信息**：
- SDK 版本：V10
- 系统版本 / 设备：服务端
- 其他：（网络环境、账号类型等，无则省略）

**相关日志**：
HTTP 403错误

## 排查过程

1. 确认接口path和V9/V10版本对应关系 → 确认接口正确
2. 确认套餐版本（高级版）→ 确认套餐足够
3. 确认控制台云端会话开关 → 定位问题

**关键发现**：云端会话功能未在控制台开通；开通后客户端需切换到云端会话模式

## 问题原因

云端会话功能未在控制台开通；开通后客户端需切换到云端会话模式，否则本地会话和云端会话未读数会不一致

## 解决方案

在云信控制台开通云端会话功能；客户端需要切换到云端会话（默认是本地会话），避免本地和云端未读数不一致。
