## Q: iOS SDK敏感权限API调用问题

**问题描述**：客户反馈云信SDK内部会调用麦克风、位置等敏感权限API，而应用有一版没有这些功能，苹果审核要求提供权限描述信息。客户询问是否有只提供纯文本信息发送的SDK。

**答案**：云信SDK没有提供纯文本消息专用版本。SDK支持位置和语音消息功能，理论上会涉及相关API。建议客户与研发确认具体使用了哪些API，或提供调用堆栈以便排查。
