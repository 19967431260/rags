## 问题：Web Web端deleteServerSessions提示非法操作或没有权限

## 问题详情

**现象**：
Web端调用 nim.deleteServerSessions 删除服务器上会话列表中的会话时，提示非法操作或没有权限

## 解决方案

到网易云信控制台开启'服务端会话列表'功能
