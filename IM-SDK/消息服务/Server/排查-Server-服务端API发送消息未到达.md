## 问题：Server 服务端API发送消息未到达

## 问题详情

**现象**：
通过服务端api给用户发送消息，接口调用成功但在IM控制台查询历史消息查不到，发送方800001接收方77123668。

## 排查过程

1. 确认接口调用情况
2. 查询服务端请求记录
**关键发现**：未查到给77123668发消息的请求记录，请求未到达服务器

## 问题原因

客户请求未实际发送到云信服务器

## 解决方案

检查客户端请求链路，确认请求是否真正发出
