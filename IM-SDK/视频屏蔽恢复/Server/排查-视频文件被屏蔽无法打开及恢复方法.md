## 问题：Server 视频文件被屏蔽无法打开及恢复方法

## 问题详情

**现象**：
视频文件无法打开，点播管理显示正常，经排查是被屏蔽状态

## 排查过程

1. 确认问题 → 文件无法打开
2. 检查状态 → 发现视频被屏蔽而非删除
3. 提供恢复方案 → 可通过API或控制台恢复
4. 确认恢复效果 → 恢复后可以正常播放
**关键发现**：视频被屏蔽导致无法访问

## 问题原因

视频被屏蔽（可能是通过API或控制台操作），不是删除

## 解决方案

视频屏蔽后不会删除，可通过以下方式恢复：1. 调用视频恢复API；2. 在控制台手动恢复；恢复后地址不变，可能有缓存需要多刷新
