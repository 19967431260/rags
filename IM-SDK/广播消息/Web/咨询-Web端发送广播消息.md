# Web端发送广播消息

## 问题描述

咨询Web端广播消息收发的API接口

## 问题背景

- 确认广播消息接收使用onbroadeastmsg回调

## 解答

Web端SDK不支持发送广播消息，接收广播消息使用onbroadeastmsg回调。发送广播消息需要调用服务端API：https://doc.yunxin.163.com/messaging/server-apis/Dg1NDYxNTM?platform=server
