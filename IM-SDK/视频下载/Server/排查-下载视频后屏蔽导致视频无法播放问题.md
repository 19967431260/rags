## 问题：Server 下载视频后屏蔽导致视频无法播放问题

## 问题详情

**现象**：
业务场景是下载成功后将视频屏蔽，但多笔订单显示下载的视频无法播放（没有时长），恢复后能正常播放

## 排查过程

1. 分析问题 → 下载后屏蔽，但下载的视频无法播放
2. 排查原因 → 可能是下载未完成就屏蔽
3. 确认机制 → 视频存入点播成功后才会推送地址
4. 分析可能性 → 检查下载完成时间和屏蔽操作时机
**关键发现**：可能下载未完成就执行了屏蔽操作

## 问题原因

可能是在视频还未完全下载完成时就执行了屏蔽操作，导致下载的文件不完整

## 解决方案

确保视频完全下载完成后再执行屏蔽操作；视频存入点播成功后才会推送地址，没有读写时效问题；建议检查下载过程中的异常和文件大小
