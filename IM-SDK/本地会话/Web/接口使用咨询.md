## Q: Web SDK 删除本地会话接口

**问题描述**：用户询问 Web SDK 7-9 版本删除本地会话的接口。

**答案**：Web SDK 删除本地会话可以使用以下接口之一：
- deleteSession
- deleteSessions
- deleteLocalSession

具体使用方法参考官方文档。
