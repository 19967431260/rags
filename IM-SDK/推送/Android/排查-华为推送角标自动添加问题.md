## 问题：Android 华为推送角标自动添加问题

## 问题详情

**现象**：
客户发现华为推送自动显示角标，但并未配置badge参数，咨询角标来源。

**环境信息**：
- 平台：Android
- 产品：IM-SDK

**相关日志**：
（从会话提取关键报错日志，无则省略）

## 排查过程

1. 检查发送消息参数 → 未配置badge
2. 检查控制台证书配置 → 发现配置了Activity
3. 确认角标来源 → 证书中的activity字段自动添加
4. 删除证书activity → 缓存问题导致仍显示
5. 确认最终方案 → 使用addNum:0或删除证书activity

## 问题原因

控制台推送证书中配置的Activity字段会自动作为badgeClass添加到推送中，同时enableUnreadCount设置也会触发角标显示。

## 解决方案

华为推送角标来源：1)证书中配置的Activity字段；2)enableUnreadCount设置。如需关闭角标，删除证书中的activity配置或在payload中设置addNum:0。
