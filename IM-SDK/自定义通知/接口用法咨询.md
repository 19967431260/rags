## Q: 鸿蒙平台如何发送自定义系统通知，与Android格式保持一致？

**问题描述**：客户希望在鸿蒙平台上实现与Android端相同格式的自定义系统通知功能，但不清楚具体的API使用方法和参数格式。

**答案**：使用V10 SDK的`sendCustomNotification`方法发送自定义通知。conversationId需要使用V2NIMConversationIdUtil工具类获取，content字段传递JSON格式内容。

示例代码：
```typescript
await nim.notificationService.sendCustomNotification(
  conversationId,
  content,
  {
    "notificationConfig": {},
    "pushConfig": {},
    "antispamConfig": {},
    "routeConfig": {}
  }
)
```

注意：自定义通知不属于P2P消息，不会出现在历史消息记录中。
