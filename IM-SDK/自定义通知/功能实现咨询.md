## Q: 应用内指令通知可用自定义通知实现

**问题描述**：客户希望向指定 uid 下发数据更新提醒、业务指令或分析结果通知，咨询除普通私聊消息订阅外是否有更合适的实现方式。

**答案**：这类轻量指令型通知可使用 IM 自定义通知实现；发送端调用自定义通知接口下发，接收端监听通知回调处理。需注意服务端调用频控为 100 次/秒。

## Q: 鸿蒙平台发送自定义通知时提示conversationId参数错误如何解决？

**问题描述**：客户在使用`sendCustomNotification`方法时遇到参数错误，报错信息显示conversationId验证失败。

**答案**：需要使用`V2NIMConversationIdUtil`工具类来生成正确的conversationId，而不是直接拼接字符串。该工具类会根据会话类型和对方ID生成符合规范的conversationId。

正确示例：
```typescript
import { V2NIMConversationIdUtil } from '@nimsdk/nim'

// 使用工具类生成
const conversationId = V2NIMConversationIdUtil.p2pConversationId('userId')
```
