## 问题：Server 物理机故障导致群成员置换失败

## 问题详情

**现象**：
调用置换人员接口返回{"code":"000001","msg":"请求失败，{code: 414; desc: not team member}"}，同时发现群成员状态不一致。

## 排查过程

1. 检查服务端API调用
2. 查询群成员状态
3. 排查服务端故障
**关键发现**：北京时间2025年11月27日16:09至16:15，线上某核心服务集群中一台物理机节点突发硬件故障

## 问题原因

物理机硬件故障导致IM部分请求处理异常

## 解决方案

运维团队已将故障节点从服务注册中心强制下线并完成隔离，服务已恢复正常
