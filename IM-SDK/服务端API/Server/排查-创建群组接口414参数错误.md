## 问题：Server 创建群组接口414参数错误

## 问题详情

**现象**：
调用创建群组接口报414错误，传参包含tid、owner、members、msg。

**环境信息**：
- 平台：Server
- 产品：IM-SDK

**相关日志**：
（从会话提取关键报错日志，无则省略）

## 排查过程

（从会话提取排查过程）

## 问题原因

缺少必填参数joinMode，且不需要传tid（创建时自动生成）。

## 解决方案

加上joinMode参数，去掉tid参数。必填参数参考文档：https://doc.yunxin.163.com/messaging/server-apis/Dk4MDY3MzQ?platform=server
