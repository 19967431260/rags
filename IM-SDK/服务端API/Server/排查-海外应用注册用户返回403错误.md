## 问题：Server 海外应用注册用户返回403错误

## 问题详情

**现象**：
新创建应用调用注册用户接口返回403错误，code: 403, msg: 'access forbidden'。客户使用open.yunxinapi.com域名。

**环境信息**：
- 平台：Server
- 产品：IM-SDK

**相关日志**：
（从会话提取关键报错日志，无则省略）

## 排查过程

（从会话提取排查过程）

## 问题原因

海外应用需要使用海外域名open-sg.yunxinapi.com，而非国内域名。

## 解决方案

海外应用请将域名修改为open-sg.yunxinapi.com。
