## Q: V9和V10 API域名区别

**问题描述**：询问V9和V10后端调用的API域名是否一样，海外节点域名是什么。

**答案**：V10是open.yunxinapi.com，V9是api.netease.com。海外V10是open-sg.yunxinapi.com，V9是api-sg.netease.com。

## Q: IM V9服务端API仅支持form表单不支持JSON

**问题描述**：客户询问V9版本服务端API是否可以使用JSON格式调用，当前使用header指定JSON格式。
