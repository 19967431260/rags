## Q: 群通知消息attachment字段解析

**问题描述**：客户查询服务端历史消息接口，messageType为5的通知消息attachment字段内容根据ope类型不同而变化，需要了解各字段含义。

**答案**：ope字段表示操作类型：0-邀请入群，1-被邀请人同意，3-群信息更新。joinmode表示入群模式。
