## 问题：服务端 IM服务端API请求超时返回502错误

## 问题详情

**现象**：
调用IM服务端API(api.netease.im:443)出现超时和502错误，时间发生在2025年5月15日17:20左右。

**环境信息**：
- SDK 版本：
- 系统版本 / 设备：
- 平台：服务端

## 排查过程

1. 客户反馈云信API一直超时并返回502 → 技术支持立即响应排查
2. 确认机房网络故障影响 → 进行灾备调整
3. 服务陆续恢复，向客户发送故障公告

## 问题原因

机房网络故障导致网易云信部分服务出现请求失败及超时。

## 解决方案

云信团队已进行灾备调整，服务已恢复。如遇类似问题可联系技术支持确认服务状态。
