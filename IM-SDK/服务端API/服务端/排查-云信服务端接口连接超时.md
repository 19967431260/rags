## 问题：服务端 云信服务端接口连接超时

## 问题详情

**现象**：
多个云信服务端接口(msg/sendMsg、chatroom/create、getToken等)出现connect timed out超时错误。

**环境信息**：
- SDK 版本：
- 系统版本 / 设备：
- 平台：服务端

## 排查过程

1. 确认内部状态 → 无事故通知
2. 检查网络 → 从云信机房到客户服务器IP有丢包
3. 协同排查 → 建议联系阿里云一起定位
4. 确认恢复 → 约15分钟后恢复正常

## 问题原因

网络链路问题导致从云信机房到客户服务器出现丢包

## 解决方案

网络问题已恢复，如频繁出现建议联系阿里云排查网络链路
