## 问题：服务端 发送消息接口特殊字符转义问题

## 问题详情

**现象**：
客户调用发送消息接口时，conversation_id中包含特殊字符|，导致请求失败。

**环境信息**：
- 平台：服务端
- 产品：IM-SDK

**相关日志**：
（从会话提取关键报错日志，无则省略）

## 排查过程

1. 客户调用发送消息接口失败 → 怀疑URL中的|字符问题
2. 尝试使用URLEncoder对整个URL编码 → 失败
3. 将|转为URL编码%7C → 请求成功

## 问题原因

conversation_id中包含特殊字符|，需要对其进行URL编码（%7C）后再拼接到URL中。

## 解决方案

URL中的特殊字符（如|）需要进行URL编码，|应转为%7C。
