## 问题：Android IM登录失败错误码415

## 问题详情

**现象**：
Android SDK 6.3.0版本，用户云信登录失败，错误code是415。用户网络正常，其他接口和第三方服务都正常。

## 排查过程

1. 确认415为网络错误 → 确认错误类型
2. 查询FAQ文档 → 查询文档
3. 检查用户登录记录，最近7天无成功登录 → 确认登录状态
4. 需要SDK日志进一步分析 → 需要更多信息
5. 建议用户切换网络（4G/5G，不开VPN）再试 → 提供方案

**关键发现**：网络连接问题导致登录失败

## 问题原因

网络连接问题导致登录失败，具体原因需要SDK日志确认。

## 解决方案

1. 建议用户切换网络环境（4G/5G，不开VPN）
2. 如问题持续，需要获取SDK日志分析
3. 参考文档：https://faq.yunxin.163.com/#KB0088
