## 问题：通用 登录code=415网络连接问题排查

## 问题详情

**现象**：
用户反馈登录偶尔返回code=415，需要排查网络连接情况、稳定性及延迟。

**环境信息**：
- 网络类型：WiFi

**相关日志**：
（从会话提取关键报错日志，无则省略）

## 排查过程

1. 尝试拉取IM日志 → 无法获取用户日志
2. 询问用户网络类型 → 确认使用WiFi
3. 观察现象 → 用户反馈一会能连接，一会断开
4. 进一步确认 → 非随身WiFi，使用办公室/家里WiFi

**关键发现**：网络连接不稳定导致登录异常

## 问题原因

网络连接不稳定导致登录异常，具体原因待日志分析确认。

## 解决方案

建议用户提供云信日志以便分析；确认网络环境（是否随身WiFi）；尝试切换4G网络对比测试。
