## 问题详情

**现象**：
客户咨询服务端调用删除或撤回消息接口后，客户端集成SDK时的影响。

**排查过程**：


## 问题原因



## 解决方案

删除：只删云端，下次从云端拉取时拉不到，无回调通知。撤回：存离线通知，离线期间撤回的，下次登录可收到回调，客户端需监听撤回回调并刷新UI。
