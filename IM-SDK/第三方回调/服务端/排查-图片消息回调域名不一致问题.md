# 图片消息回调域名不一致问题

## 问题描述

文字消息回调正常，但图片消息回调的域名不正确，没有调用到配置的回调地址

## 问题背景

- 文字消息回调到https://api.gomochat.net正常
- 图片消息回调到了https://beta.gomo.wooyavip.com/neteasy/callback

## 根因分析

发消息时配置了env参数，env设置了不同的回调地址

## 解决方案

检查发消息时是否设置了env参数，env参数会导致回调地址变化。统一env配置或去掉env参数即可解决。
