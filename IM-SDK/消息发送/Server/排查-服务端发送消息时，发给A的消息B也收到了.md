## 问题：服务端发送消息时，发给A的消息B也收到了

## 问题详情

**现象**：
服务端发送消息时，发给A的消息B也收到了。

**环境信息**：
- 产品：IM V9
- 平台：Server

## 排查过程

1. 检查发消息接口的接收方参数 → 确认是否正确传递
2. 打印接口入参 → 确认接收方accid是否正确

**关键发现**：发消息接口的接收方参数可能传递错误

## 问题原因

发消息接口的接收方参数传递错误，导致消息发送给了错误的用户。

## 解决方案

1. 检查发消息接口的接收方参数是否正确传递
2. 建议在调用sendText等接口时打印入参
3. 确认接收方accid是否正确
