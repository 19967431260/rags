## 问题：Web Web端消息接收延迟3分钟

## 问题详情

**现象**：
用户20:06:17发送的消息，客户端20:09:41才收到，延迟约3分钟。

## 排查过程

1. 分析日志发现本地db同步数据很慢 → 定位问题
2. 同步到的消息几秒才能入库一条 → 确认现象
3. 消息队列排到3分钟后才入库回调 → 确认延迟原因

**关键发现**：本地DB同步数据慢，消息量大导致入库延迟

## 问题原因

本地DB同步数据慢，消息量大导致入库延迟。

## 解决方案

建议清理本地缓存，或考虑调整漫游策略，减少自动同步的消息数量，后续按需查询历史记录。迁移到新套餐可支持配置。
