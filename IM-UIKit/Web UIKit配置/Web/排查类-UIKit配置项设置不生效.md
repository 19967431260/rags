## 问题：UIKit配置项设置不生效

## 问题详情

**现象**：
客户咨询UIKit配置项如何设置，尝试隐藏某些列但设置不生效。

## 排查过程

1. 客户尝试通过style设置display:none不生效
2. 技术支持建议检查配置位置
3. 提供官方文档链接和配置示例
4. 客户确认配置位置后问题解决

## 问题原因

配置位置不正确导致设置不生效。

## 解决方案

参考官方文档的配置项说明，确保配置写在正确的位置。
