## Q: 自定义消息类型判断方法

**问题描述**：客户需要判断自定义消息的具体类型（如1002）。

**答案**：通过getAttachment()获取附件后强转为CustomAttachment，再获取content：if (msg.getAttachment() instanceof CustomAttachment) { return ((CustomAttachment) msg.getAttachment()).getContent(); }
