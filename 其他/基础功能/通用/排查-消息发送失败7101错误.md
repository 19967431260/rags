## 问题：通用 消息发送失败7101错误

## 问题详情

**现象**：
消息发送失败，返回错误码7101

**环境信息**：
- 平台：通用
- 时间：2025-06-26

## 排查过程

1. 检查日志 → 发现send message failed: response code = 7101
2. 分析错误码 → 7101表示被对方拉黑
3. 确认问题 → 消息确实没有送达

**关键发现**：7101错误码表示被对方拉黑

## 问题原因

用户被对方拉黑，消息无法送达对方

## 解决方案

1. 业务层捕获7101错误码
2. 提示用户"您被对方拉黑，消息无法送达"
3. 建议用户检查好友关系
