## 问题：会议接口时间戳单位错误

## 问题详情

**现象**：
调用会议接口返回时间戳相关错误，客户使用毫秒级时间戳。

## 排查过程

1. 客户反馈接口调用失败
2. 检查发现客户使用毫秒级UTC时间戳
3. 告知会议接口HTTP header中CurTime使用秒级时间戳
4. 只有startTime和endTime使用毫秒级

## 问题原因

HTTP header中的CurTime需要使用秒级时间戳，而非毫秒级

## 解决方案

HTTP header中的CurTime使用秒级时间戳，startTime和endTime使用毫秒级时间戳
