## <span id="400">400 实时音视频安全通审核抄送</span>

实时音视频安全审核结果抄送，其中包含抄送任务详情、审核结果、截图证据等信息。

### 触发场景

实时音视频安全通审核抄送在以下场景下会被触发：

- 针对房间内的实时音视频流进行内容安全审核时，如果检测到违规信息，会触发此抄送，其中包含触发审核规则的违规行为详细信息。
- 如果已联系网易易盾技术支持开通了全量抄送，此时除了违规内容的抄送之外，安全通服务还会根据截图频率定期发送截图对应的检测结果。
    - 全量抄送中也包含检测结果为正常的任务信息。
    - 修改截图频率请参考 [步骤 3](https://doc.yunxin.163.com/nertc/server-apis/zMzMTYzNDM#操作流程)。
- 审核任务停止时，安全通服务也会发送一条实时音视频安全通审核抄送，其中包含任务停止前的检测结果。此抄送表示检测任务结束。其中，status 为 102，表示抄送结束。
