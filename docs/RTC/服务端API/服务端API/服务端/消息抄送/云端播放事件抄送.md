## <span id="500 创建任务成功">500 创建任务成功</span>
创建输入在线媒体流任务成功。

### **字段说明**

eventType 为 500，data 中封装的字段说明如下：

|  <div style="width:60px">字段</div>  	|  <div style="width:60px">类型</div>  	| <div style="width:100px">示例</div> 	|  <div style="width:200px">说明</div>  	|
|---	|---	|---	|---	|
| streamUrl 	| String 	| http://******.vod.126.net/jdvodvk8hs8ih/d317c15b-f42a-455e-a100-f1388e6cdbfd.mp4 	| 点播或直播流地址。  	|
|  playTs 	| Number 	| 1575508644 	| 云端播放器开始播放在线媒体流时的 Unix 时间戳，单位为秒（s）。 	|
| idleTimeout 	| Number 	| 300 	| 云端播放器处于空闲状态的最大时长，单位为秒（s）。 	|
| channelId 	| Number 	| 43356618819536 	| 房间 ID。 	|
| taskId 	| String 	| stream_1 	| 任务 ID。 	|
| code 	| Number 	| 200 	| 状态码。详细说明请参考[在线媒体流状态码](https://doc.yunxin.163.com/nertc/server-apis/zE4MjQ5NTY)。 	|
| timestamp 	| Number 	| 1606963065982 	| 该事件发生的 Unix 时间戳，单位为毫秒（ms）。 	|

### **JSON 示例**

```
{
    "data": { 
            "channelId":1144880576163552,
            "code":200,
            "idleTimeout":2,
            "playTs":0,
            "streamUrl":"http://******.vod.126.net/jdvodvk8hs8ih/d317c15b-f42a-455e-a100-f1388e6cdbfd.mp4",
            "taskId":"9aad689e-ba91-4764-bed1-b2e09bc21f5d",
            "timestamp":1653470489602
            },
    "eventType": 500
}
```
