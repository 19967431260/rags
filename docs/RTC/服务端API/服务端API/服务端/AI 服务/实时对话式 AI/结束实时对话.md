调用该接口结束一个实时对话任务。

### 请求体参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| ---- | ---- | ---- | ---- | ---- |
`cid` | Number | 是 | 1144509608259296 | 房间 ID。<br> 该 ID 为创建 AI 实时对话接口调用成功后返回的房间 ID |
`taskType` | Number | 是 | 7 | 服务器任务类型。实时对话固定为 7。 |
`taskId` | String | 是 | 2a784467d647bb87b60b719f6fa56333 | 任务唯一标识符，房间内唯一。<br> 调用创建 AI 实时对话接口后，可在返回 Body 中获取 taskId。 |

### 请求示例

```JSON
{
    "cid": 1144509608259296,
    "taskType": 7,
    "taskId": "2a784467d647bb87b60b719f6fa56333"
}
```

## 响应信息

### 响应参数

| <div style="width:100px">参数名称</div> | 类型 | 示例 | 描述
| ---- | ---- | ---- |
| `code` | Number | 200 | 状态码，200 表示成功，具体请参考下文状态码。 |

### 响应示例

```JSON
{
    "code"：200,
    "errmsg":""
}
```

### 状态码

- 成功返回 `code`：返回内容中，`code` 为 `200` 表示调用正常。
- 异常返回 `code`：若 `code` 为其他值，请根据 `code` 与 `errmsg` 在 [错误码和状态码](https://doc.yunxin.163.com/nertc/server-apis/TQ3Njc1Nzg?platform=server) 查看问题原因。
