删除指定的克隆音色资源。删除后该音色将不可恢复，请谨慎操作。

:::note notice
- 删除音色前请确保该音色未被其他任务使用。
- 删除操作不可逆，请谨慎操作。
- 系统预置音色不支持删除。
:::

## 功能描述

支持删除已创建的克隆音色，包括：

- 删除指定音色 ID 的克隆音色。
- 释放音色占用的存储资源。
- 清理相关音色配置信息。

## 请求信息

### 请求 URI

```
DELETE https://rtc-ai.yunxinapi.com/v1/api/tts/voice
```

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](https://doc.yunxin.163.com/nertc/server-apis/TYyNjA3MzQ?platform=server#%E8%AF%B7%E6%B1%82%E5%A4%B4header)。

### 请求体参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| :---- | :---- | :---- | :---- | :---- |
| voiceId | String | 是 | GZ_6FXfkjpzhnZI1 | 音色 ID，创建克隆音色时返回的唯一标识符 |

### 请求体示例

```JSON
{
    "voiceId": "GZ_6FXfkjpzhnZI1"
}
```

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
| :---- | :---- | :---- | :---- |
| code | Number | 状态码，200 表示成功。出错时，错误信息在 errMsg 中 | 是 |
| errMsg | String | 错误描述信息，成功时为空 | 否 |

### 响应体示例

```JSON
{
    "code": 200
}
```
