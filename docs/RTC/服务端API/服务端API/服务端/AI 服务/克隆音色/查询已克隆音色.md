查询已克隆的 AI 语音合成音色信息，包括音色 ID、描述信息、克隆状态及可重复克隆次数等详细信息。

:::note notice
- 仅支持查询网易云信克隆的音色信息（vendorId=100）。
- 返回的音色 ID 可直接用于创建 AI 语音合成任务。
- 克隆音色的可重复克隆次数会随使用而递减。
:::

## 功能描述

- 获取当前账号下所有已克隆音色的详细信息。
- 查看音色的描述信息和试听链接。
- 了解音色的可重复克隆次数，便于资源规划。

## 请求信息

### 请求 URI

```
GET https://rtc-ai.yunxinapi.com/v1/api/tts/voice
```

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](https://doc.yunxin.163.com/nertc/server-apis/TYyNjA3MzQ?platform=server#%E8%AF%B7%E6%B1%82%E5%A4%B4header)。

### 请求查询参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| :---- | :---- | :---- | :---- | :---- |
| vendorId | Number | 是 | 100 | 供应商 ID，100 表示查询网易云信克隆音色 |

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
| :---- | :---- | :---- | :---- |
| code | Number | 状态码，200 表示成功 | 是 |
| voices | Array | 可用音色信息列表 | 是 |
    voice | String | 音色 ID，创建 AI 任务时使用 | 是 |
    desc | String | 音色描述信息 | 是 |
    cloned | Boolean | 是否为克隆音色 | 是 |
    availableCloneTimes | Number | 可重复克隆次数 | 是 |
    audioDemo | String | 音色试听链接 | 是 |

### 响应体示例

```JSON
{
    "code": 200,
    "voices": [
        {
            "voice": "GZ_6FXfkjpops1756193022",
            "desc": "复刻 0903",
            "audioDemo": "https://yunxin-vc.nos-jd.163yun.com/14665/GZ_6FXfkjpops1756193022.mp3",
            "cloned": true,
            "availableCloneTimes": 8
        },
        {
            "voice": "pddIWSUeLyW1757399218",
            "desc": "db0903",
            "audioDemo": "https://yunxin-vc.nos-jd.163yun.com/14665/pddIWSUeLyW1757399218.wav",
            "cloned": true,
            "availableCloneTimes": 5
        }
    ]
}
```
