把房间中的所有用户从房间中移除，并删除房间。

::: note note
- 服务端创建房间后 24 小时内，如果无人加入房间，则房间会自动销毁。
- 开通消息抄送服务后，若您成功调用此服务端接口删除房间，会触发 eventType=2（房间结束）抄送，若房间内有用户，则会触发 eventType=5（用户离开房间）的抄送。
:::
