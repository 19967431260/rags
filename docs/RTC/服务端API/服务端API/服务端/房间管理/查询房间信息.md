查看房间的基本信息，包括房间 ID、房间名称、房间创建者 ID、房间内在线用户总数、房间状态、房间创建时间、房间结束时间。

## 注意事项
只能查询最近 30 天的房间信息。

## <span id="URL">URL</span>

- 请求方法：GET
- URL：网易云信为该功能提供以下两个 API 请求地址，使用 V2 地址需在 URL 中指定 cid，使用 V3 地址需在 URL 中指定 cname，您可以根据业务需求调用任一接口。

    ```
    https://rtc.yunxinapi.com/v2/api/rooms/{cid}
    https://rtc.yunxinapi.com/v3/api/rooms?cname={cname}
    ```


-  Content-Type：application/json
