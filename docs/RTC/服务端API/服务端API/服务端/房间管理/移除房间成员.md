从指定房间中移除成员，即将该成员加入黑名单。

如果 URL 中未指定拉黑时长，则表示此房间内永久拉黑该成员，该成员会被移出此房间、且无法再次进入。适用于主播、房主或管理员踢人、拉黑等场景。

::: note note
开通消息抄送服务后，若您成功调用此服务端接口移除房间成员，会触发 eventType=5（用户离开房间）的抄送。
:::

- URL：
