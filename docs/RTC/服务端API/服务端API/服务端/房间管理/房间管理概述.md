网易云信提供房间管理功能，帮助您管理房间及房间内的成员和查询房间信息。

## 功能概述

房间管理功能可以通过服务端 API 实现，主要包括以下功能点：

| 功能 | 说明 |
|---|---|
| 管理房间 | 支持管理房间及用户的权限，包括创建房间、移除成员等。 |
| 查看房间信息 | 支持查看房间及房间成员信息。 |
| 事件通知 | 房间管理功能提供实时的房间状态回调及房间内成员状态回调。 |
