网易云信云端播放服务费用包括订阅费用和转码费用。本文介绍使用云端播放功能时，产生的这两种费用的计费方式。

## 费用组成

网易云信云端播放服务费用包括订阅费用和转码费用。其中对两种费用的介绍如下：
<li>订阅费用：指云端播放器作为一个虚拟主播加入 RTC 房间后产生的订阅费用，计入实时音视频的订阅费用中。
<li>转码费用：指云端播放器将在线媒体流转码输入 RTC 房间后产生的转码费用。


::: note important
按月计费，月账单最小结算单位为千分钟，不满 1 千分钟按照 1 千分钟进行费用结算。
:::
  
## 订阅费用

成功创建云端播放任务后，云端播放器会作为一个虚拟主播加入 RTC 房间，所以当房间中其他用户订阅云端播放的媒体流时，会产生订阅费用。这部分费用直接计入实时音视频的订阅费用中，详细信息请参考[实时音视频计费模式](https://doc.yunxin.163.com/nertc/server-apis/TkyNzg0NDc?platformId=50002#计费模式)。

::: note notice
房间中用户都离开后，云端播放任务不会自动停止，若要停止，请调用[销毁云端播放任务](https://doc.yunxin.163.com/nertc/server-apis/jE4MzIwNzQ?platform=server)接口。
:::

## 转码费用

云端播放器将在线媒体流转码并输入到 RTC 房间，由此会产生转码费用。
<br>转码费用包含以下两种：
<li>音频转码费用：如果向房间内输入纯音频流，则产生音频转码费用。音频转码费用 = 音频转码单价 × 音频转码时长。
<li>视频转码费用：如果向房间内输入音视频流，则产生视频转码费用。视频转码费用 = 视频转码单价 × 视频转码时长。

### 转码单价

云端播放的转码单价如下表：

| 媒体 | 规格 | 单价 |
|---|---|---|
| 音频 | 标准语音规格 | 6 元/千分钟 |
| 视频SD单价 | 集合分辨率 ≤ 307,200(640 × 480) | 12元/千分钟 |
| 视频HD单价 | 307,200(640 × 480) ＜ 集合分辨率 ≤ 921,600(1280 × 720) | 24元/千分钟 |
| 视频HD+单价 | 集合分辨率 > 921,600(1280 × 720) | 90元/千分钟 |

### 转码时长

转码输出的媒体流时长即为转码时长。其中：

- 音频转码时长：网易云信云端播放器对纯音频流进行转码的时长。
- 视频转码时长：网易云信云端播放器对视频流进行转码的时长。
