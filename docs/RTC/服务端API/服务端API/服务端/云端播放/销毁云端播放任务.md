关闭一个指定虚拟播放器或房间内所有虚拟播放器的播放任务。

## 销毁指定播放任务

### 接口地址信息

- 请求方法：POST
- URL：网易云信为该功能提供以下两个 API 请求地址，使用 V3 地址需在 URL 中指定 cname，您可以根据业务需求调用任一接口。
    - `https://rtc.yunxinapi.com/v2/api/task/close`
    - `https://rtc.yunxinapi.com/v3/api/task/close?cname={cname}`

**URL 中参数说明**：
| 参数名称 	| 类型 	| 示例 	| 描述 	|
|---	|---	|---	|---	|
| cname 	| String 	| abc 	| 房间名称。<br>仅在调用 V3 接口时需要设置。|

## 接口请求频率

默认上限为 50 次/秒，若请求频率超出限制，可能会返回 429 错误码。<br>若您需要上调上限，请参考<a href="https://doc.yunxin.163.com/nertc/server-apis/Tg3MDU1NTE?platform=server#如何处理调用服务端-restful-api-超出频率限制" target="_blank">如何处理调用服务端 RESTful API 超出频率限制</a>。

### 请求参数

- 请求中 Header 的设置请参考[请求结构](https://doc.yunxin.163.com/nertc/server-apis/TYyNjA3MzQ?platform=server#%E8%AF%B7%E6%B1%82%E7%BB%93%E6%9E%84)描述。

- 请求中 Body 须为 JSON 格式，参数说明如下：

| <div style="width:80px">参数名称</div> 	| <div style="width:80px">类型</div> 	| <div style="width:80px">是否必选</div> 	| <div style="width:100px">示例</div> 	| <div style="width:300px">描述</div> 	|
|---	|---	|---	|---	|---	|
| cid 	| Number 	| 必选，仅适用于 V2 接口 	| 1144509608259296 	| 房间 ID。 <br>该 ID 为[创建云端播放任务](https://doc.yunxin.163.com/nertc/server-apis/DgwMTQ2NzA)接口调用成功后返回的房间 ID，仅在调用 V2 接口时需要设置。 	|
| taskType 	| Number 	| 必选 	| 2 	| 服务器任务类型。固定为 2。 	|
| taskId 	| String  	| 必选|  2a784467d647bb87b60b719f6fa56333 	| 任务唯一标识符，房间内唯一。<note type="note">调用创建云端播放任务接口后，可在返回 Body 中获取 taskId。<note>|


### 返回参数

Body 中不返回任何参数。

### 示例

#### **请求示例**

```
{
        "cid": 1344332211223399,
        "taskType": 2,  // 常量 2 
        "taskId": "2a784467d647bb87b60b719f6fa56333"
}
```

#### **返回示例**
```
{
    "code"：200，
    "errmsg":"",
}
```

### 状态码

返回内容中，code 为 200 表示调用正常。
<br>若 code 为其他值，请根据 code 与 errmsg 在[在线媒体流状态码](https://doc.yunxin.163.com/nertc/server-apis/zE4MjQ5NTY)中查看问题原因。
