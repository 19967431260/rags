开通云端播放相关的消息抄送服务后，网易云信服务器将以 HTTPS POST 请求方法向你的服务器发送云端播放事件通知回调。请求 Body 的数据格式为 JSON，字符编码为 UTF-8，签名算法为 SHA1。

消息抄送的实现方法请参见[消息抄送服务概述](https://doc.yunxin.163.com/nertc/server-apis/TYzOTgzMjY)。

云端播放事件类型如下表所示。

| event_type | 事件含义 |
|---|---|
| 500 | [创建任务成功](https://doc.yunxin.163.com/nertc/server-apis/DA4NDAwMjY?platformId=50326#500%20创建任务成功) |
| 501 | [更新任务成功](https://doc.yunxin.163.com/nertc/server-apis/DA4NDAwMjY?platformId=50326#501%20更新任务成功) |
| 502 | [销毁任务成功](https://doc.yunxin.163.com/nertc/server-apis/DA4NDAwMjY?platformId=50326#502%20销毁任务成功) |
| 503 | [开始播放](https://doc.yunxin.163.com/nertc/server-apis/DA4NDAwMjY?platformId=50326#503%20开始播放) |
| 504 | [播放结束](https://doc.yunxin.163.com/nertc/server-apis/DA4NDAwMjY?platformId=50326#504%20播放结束) |
| 505 | [播放异常](https://doc.yunxin.163.com/nertc/server-apis/DA4NDAwMjY?platformId=50326#505%20播放异常) |
| 506 | [暂停播放](https://doc.yunxin.163.com/nertc/server-apis/DA4NDAwMjY?platform=server#506-暂停播放)|
| 507 | [恢复播放](https://doc.yunxin.163.com/nertc/server-apis/DA4NDAwMjY?platform=server#507-恢复播放)
