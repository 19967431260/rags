使用旁路推流服务之前，请先在网易云信控制台开通旁路推流服务。

## 开通旁路推流

1. 登录 <a href="https://app.yunxin.163.com/index#/" target="_blank">网易云信控制台</a>。

2. 在首页单击指定应用名称。
3. 在 **产品总览** 区域，单击 **音视频通话 2.0** 产品选项卡中的 **功能配置**。

    <img alt="功能配置.png" src="https://yx-web-nosdn.netease.im/common/50fdee5139f9e30204445b9386f205cd/功能配置.png" style="width:80%;border: 0px solid #BFBFBF;">

4. 单击 **服务端功能** 页签，单击 **旁路推流** 的开关按钮，开启旁路推流。

    <img alt="开启旁路推流.png" src="https://yx-web-nosdn.netease.im/common/9e288495703ded0374fc12f682df6b33/开启旁路推流.png" style="width:80%;border: 0px solid #BFBFBF;">

5. 单击 **确定**。

## 开启旁路推流录制

在互动直播、视频会议、在线教育等场景中，可以通过旁路推流录制功能，将视频录制下来，方便观看回放、内容安全审核、教学视频制作等。

开启旁路推流录制功能后，会自动进行录制。

1. 开通旁路推流后，单击 **旁路推流** 右侧的 **子功能配置**。

    <img alt="旁路推流子功能配置.png" src="https://yx-web-nosdn.netease.im/common/fb19f99dc4678db67d5eaaa2b2198f34/旁路推流子功能配置.png" style="width:80%;border: 1px solid #BFBFBF;">

2. 单击 **旁路推流录制** 的开关按钮，单击 **确定**，开启旁路推流录制。

    <img alt="开启旁路推流录制.png" src="https://yx-web-nosdn.netease.im/common/85e7562fa5f5c6d48f0e6df355bebf9a/开启旁路推流录制.png" style="width:80%;border: 0px solid #BFBFBF;">
