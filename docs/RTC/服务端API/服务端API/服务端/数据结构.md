## recordConfig

录制模式。

被以下接口引用：
- [创建云端录制任务](https://doc.yunxin.163.com/nertc/server-apis/TEyOTk2OTY?platform=server)
- [查询录制任务配置](https://doc.yunxin.163.com/nertc/server-apis/TU4ODczNDk?platform=server)

参数名称 | 类型 | 是否必选 | 示例值 | 说明 |
---- | ---- | ---- | ---- | ---- |
recordType | Number | 否 | 1 | 录制模式。默认值为 0。录制模式的详细说明，请参考 [录制模式](https://doc.yunxin.163.com/nertc/server-apis/DQ3NDIxNzg?platform=server)。 |\
    | | | | 可设置为：|\
    | | | | - 0：合流+单流录制。 |\
    | | | | - 1：合流录制。 |\
    | | | | - 2：单流录制。 |\
    | | | | - 100：模板文件集合模式。 |
modeList | JsonArray | 否 | - | 模板文件集合模式列表。`recordType` 填 `100` 时，该字段必填。 |
  mode | Number | 否 | 0 | 录制模式：|\
    | | | | - 0：单流录制。 |\
    | | | | - 1：录制带有共享屏幕的画廊视图。 |\
    | | | | - 2：录制带有共享屏幕的当前演讲者。 |\
    | | | | - 3：单独录制当前演讲者。 |\
    | | | | - 4：单独录制画廊视图。 |\
    | | | | - 5：单独录制屏幕共享。 |\
    | | | | - 6：录制单独音频文件。 |\
    | | | | - 100：单人视频+混合音频，每个用户单独生成文件。 |
  layoutType | Number | 否 | 0 | 布局类型，默认为通过控制台设置的自定义布局。录制模式的详细说明，请参考 [录制布局](https://doc.yunxin.163.com/nertc/server-apis/TA0MjE1MzE?platform=server)。可设置为：|\
    | | | | - 0：画廊模式。 |\
    | | | | - 1：主讲人模式。 |\
    | | | | - 2：自定义布局。 |\
    | | | | - 3：悬浮布局。 |\
    | | | | - 4：垂直布局。 |
  layout | Object | 否 | 0 | 自定义布局时，设置合流录制的画面布局。详细的参数说明，请参考 [使用自定义布局](https://doc.yunxin.163.com/nertc/server-apis/TA0MjE1MzE?platform=server#%E4%BD%BF%E7%94%A8%E8%87%AA%E5%AE%9A%E4%B9%89%E5%B8%83%E5%B1%80)。 |
  watermark | Object | 否 | - | 水印配置。详细信息请参见 [watermark](#watermark)。
userNameDisplay | Object | 否 | - | 用户名显示配置。 |
  enable | Boolean | 否 | true | 是否启用用户名展示。 |
  position | Number | 否 | 0 | 用户名显示位置。取值范围 [0, 8]，默认 0。<ul><li>`0`：左下角</li><li>`1`：下方居中</li><li>`2`：右下角</li><li>`3`：左边居中</li><li>`4`：正中心</li><li>`5`：右边居中</li><li>`6`：左上角</li><li>`7`：上方居中</li><li>`8`：右上角</li></ul> |
  fontSize | Number | 否 | 12 | 用户名字体大小字号。默认 12。 |
  fontColor | String | 否 | "#FFFFFF" | 用户名字体颜色（十六进制 RGB 值、或字符串）。默认为 "#FFFFFF"，表示白色。 |
  fontAlpha | Number | 否 | 255 | 用户名字体透明度。取值范围 [0, 255]，0 表示全透明，255 表示不透明。默认为 255。 |
  bgColor | String | 否 | "#000000" | 用户名背景颜色（十六进制 RGB 值、或字符串）。默认为 "#000000"，表示黑色。 |
  bgAlpha | Number | 否 | 128 | 用户名背景透明度。取值范围 [0, 255]，0 为全透明，255 为不透明。默认为 128，表示半透明。 |
  offsetX | Number | 否 | 20 | 显示位置横向偏移，相对窗口宽度百分比。取值范围 [-100, 100]，默认为 0。 |
  offsetY | Number | 否 | -20 | 显示位置纵向偏移，相对窗口高度百分比。取值范围 [-100, 100]，默认为 0。 |
maxIdleTime | Number | 否 | 30 | 数据超时时长（单位：秒）。缺省默认为 0，表示不启用。如果需要启用，取值范围为 [30,86400]。<note type=note>录制过程中若连续超过超时时长仍未收到用户的媒体数据，将立即结束当前文件的录制；当再次收到媒体数据时将自动启动新的切片，切片编号保持连续递增。</note>

| | | | | - 若此参数传空，表示音频流全不订阅。 |

| | | | | - 黑名单和白名单不可同时指定。 |\
| | | | | - 若您希望订阅视频，必须设置 videoUidList。 |\
| | | | | - 若指定 streamType 为 0，同时设置 videoUidList 时会报错 414。 |
| videoUidList.unSubscribeUids | JsonArray | 否 | 无 | - 视频订阅黑名单，即不订阅哪些用户的视频流。 |\
| | | | | - 不可与 videoUidList.subscribeUids 同时设置。 |\
| | | | | - 若此参数传空，表示 [视频流全订阅](https://doc.yunxin.163.com/nertc/server-apis/jQwNDAxNDc?platform=server#%E8%A7%86%E9%A2%91%E6%B5%81%E5%85%A8%E8%AE%A2%E9%98%85)。 |
