为满足用户在音视频通话过程录制和备案等需求，音视频通话 2.0 提供云端录制功能，将 1 对 1 音视频通话或多人音视频通话过程中的音视频内容录制并储存下来，以便后续提供给更多人观看回放或存档。

## 适用场景

云端录制广泛应用于各种行业，举例如下：

行业 | 使用场景 |
---- | ---- |
娱乐社交 | 为秀场直播、互动连麦直播、语音直播等泛娱乐直播场景提供直播片段录制，支持直播回放，内容安全审核，降低平台运营的风险。 |
金融行业 | 为视频远程面签、财产公证等业务办理提供录音录像功能，帮助金融机构实现业务留痕、业务回溯，满足金融监管要求。<br>支持录制实时叠加文字/时间戳/图片水印，防篡改，保障录像文件完整有效。 |
在线教育 | 为 1 对 1、小班课、大班课等在线课堂提供多方音视频录制能力，实现线上教学的录制存储，让学生可重复观看教学内容，提高学习效果。<br>帮助在线教育平台进行课堂教学分析，提升课堂的教学质量。 |
互联网问诊 | 为视频诊疗、手术直播教学、远程会诊等医疗过程进行在线录制，为病人后期复查、转诊提供诊疗依据，减少重复检查，提高医疗效率。 |
公开招标、邀请招标 | 招采场景是指企业或组织通过公开招标、邀请招标等方式，从多个供应商中选择最优合作伙伴，以采购所需商品、服务或工程的过程。 |

| 功能 | 说明 |
| --- | --- |
| 录制内容 | 支持实时音视频通话的过程录制，包括屏幕共享和其他自定义音视频的录制。 |
| 录制模式 | 支持以下三种录制模式：|\
| | - **合流录制模式**：房间内所有或指定 UID 的媒体流混合录制为一个录制文件。 |\
| | - **单流录制模式**：分别录制房间内每个 UID 的媒体流，每个 UID 均有其对应的录制文件。 |\
| | - **合流+单流录制**：房间内所有或指定 UID 的视频内容和音频混合录制为一个录制文件，同时每个 UID 均有其对应的录制文件。 |
| [录制布局](https://doc.yunxin.163.com/nertc/server-apis/TA0MjE1MzE?platform=server) | 支持多种录制布局：|\
| | - 画廊模式。 |\
| | - 悬浮模式。 |\
| | - 垂直模式。 |\
| | - 主讲人模式。 |\
| | - 存在辅流的模板布局。 |\
| | - 自定义模式。 |
| 录制的媒体流类型 | 支持录制指定的的媒体类型：|\
| | - 仅录制音频 |\
| | - 仅录制视频 |\
| | - 同时录制音频和视频 |
| [录制的黑白名单](https://doc.yunxin.163.com/nertc/server-apis/DYwODU5NTQ?platform=server) | 支持在创建云端录制任务时设置订阅录制的白名单或黑名单，以及在录制过程中更新订阅名单。详细信息请参考 [更新订阅名单](https://doc.yunxin.163.com/nertc/server-apis/DYwODU5NTQ?platform=server)。 |
| [录制文件管理](https://doc.yunxin.163.com/nertc/server-apis/jYwNzI3MDk?platform=server) | 云端录制生成的 `.aac` 音频和 `.mp4` 视频文件自动上传并存储在网易云信 [点播](/docs/jY3NDM4Nzc/jgyNzg2NzY) 平台中。文件地址会通过消息抄送返回，录制完成后可以通过播放器 SDK 拉流观看回放。 |
| 云端录制事件抄送 | 云端录制功能还提供实时的录制状态回调，您可以通过录制 [消息抄送服务](https://doc.yunxin.163.com/nertc/server-apis/TYzOTgzMjY) 获取到录制事件的相关通知。 |
| 云端录制文件内容审核 | 网易云信安全通服务，提供中文长音频检测、声纹检测、视频图片等多种检测类型。您可以在开启云端录制时，指定生成的录制文件是否需要进行内容审核，同时可以指定内容审核的截图频率等配置。

## 名词解释

| 类别 | 术语 | 说明 |
|------|------|------|
| **基础概念** | 云端录制 | 一种将音视频通话或直播内容存储到云端服务器的技术，可用于会议回放、课程点播等场景。 |
| | 合流录制 | 将多路音视频流混合成一路输出的录制模式，可自定义参与者视频画面的大小、位置和布局。 |
| | 布局模式 | 指定多个视频流在录制画面中的排列方式，包括模板布局和自定义布局两大类。 |
| | 布局配置参数 | 控制录制画面排列的各种参数，如 layoutType、canvas、users 等，用于精确定义视频流的位置和大小。 |
| **流类型** | 主流 | 通常指用户摄像头采集的视频流，是用户的主要视频来源。 |
| | 辅流 | 指屏幕共享、白板共享等非摄像头采集的媒体流，在会议、教学等场景中通常作为重要内容进行展示。用户关闭辅流后，录制布局会自动切换回原先设置的模板布局。 |
| **模板布局类型** | 画廊模式 | 多人画面平铺在画布上，按照具体人数平分整体画布，每个用户画面大小一致。 |
| | 主讲人模式 | 设置某用户为主讲人，其画面为画布中的主画面，其他用户画面悬浮于主画面之上。 |
| | 悬浮模式 | 主讲人画面铺满整个画布，其他用户小视窗从画布左下角开始依次水平排列，最多 2 行。 |
| | 垂直模式 | 主讲人画面在画布左侧显示为大视窗，其他用户小视窗在右侧垂直排列，最多两列。 |
| **布局元素** | 画布（`canvas`） | 混流录制的背景画面，可设置宽度、高度和背景颜色。 |
| | 用户窗口（`users`） | 参与者的视频画面，可设置位置、大小和层级关系。 |
| | 占位图片（`images`） | 当配置的用户未上线或不发送视频流时，可在对应区域显示的图片。 |
| | 图层编号（`zOrder`） | 控制元素在重叠区域的显示优先级，值越大显示越靠前。 |
| **适应模式** | 适应图片（`adaption=0`） | 保证视频内容全部显示，未覆盖区域填充背景色。 |
| | 适应区域（`adaption=1`） | 保证所有区域被填满，视频超出部分会被裁剪。 |
| **其他概念** | 主讲人 | 在某些布局模式中被特殊对待的用户，通常其视频画面会占据较大区域。 |

## 方案版本

### 新版云端录制方案（推荐）

云端录制服务于 2021 年 11 月 18 日上线完全通过服务端配置的新版云端录制方案，为您提供录制任务的启停、查询以及录制订阅列表的更新等功能。您可以将云端录制的相关配置完全交由服务端控制，例如订阅类型、录制布局等等，无需配合控制台进行布局设置。

新版云端录制方案提供更丰富的录制布局配置、支持订阅黑白名单设置，方案更加细节和灵活。

::: note note
- 若您尚未接入过网易云信的云端录制功能，建议直接使用纯服务端配置的云端录制方案。
- 同一房间中，不可同时使用新旧两套云端录制 API。
:::
