为满足用户在音视频通话过程录制和备案等需求，音视频通话 2.0 提供云端录制功能，支持 1 对 1 或多人的音视频通话过程录制。通过云端录制功能，您可以将语音聊天、视频聊天的内容录制并储存下来，以便后续提供给更多人观看回放或存档。本文介绍新版云端录制的实现流程。

## 注意事项

- 开始集成云端录制功能之前，请先参考 [云端录制概述](https://doc.yunxin.163.com/nertc/server-apis/zE4NDg0NTM)，了解新版云端录制方案。

    网易云信建议您使用新版服务端配置方案，以体验更为灵活的配置策略。旧版的服务端配置方案不再迭代更新。
- 同一房间内只能使用一种服务端配置方案，即新旧两套服务端接口不可在同一房间使用。

步骤序号 | 参考文档 | 步骤说明
---- | ---- | ----
| 1 | [开通云端录制](https://doc.yunxin.163.com/nertc/server-apis/TE3MTkxNjQ?platform=server) | - 在 [网易云信控制台](https://app.yunxin.163.com/global/home) 开通 **云端录制** 功能和网易云信 **点播** 服务。<br>该功能涉及资源计费，请确认您已了解相关的计费规则。 |\
| | | - 开启消息抄送服务，接收云端录制事件的相关回调。详细信息请参考 [消息抄送](https://doc.yunxin.163.com/nertc/server-apis/TYzOTgzMjY)。 |
| 2 | [创建云端录制任务](https://doc.yunxin.163.com/nertc/server-apis/TEyOTk2OTY?platform=server) |应用服务器调用网易云信服务端 API 创建录制任务，开始云端录制。 |
| 3 | [（可选）设置录制布局](https://doc.yunxin.163.com/nertc/server-apis/TA0MjE1MzE#设置录制布局) | 支持以下两种方式配置布局模式，如果不配置，则采用默认布局：<br><ul><li>通过服务端 API 设置布局模式（推荐），包括悬浮模式、垂直模式、画廊模式、主讲人模式和自定义布局。<li>通过控制台设置布局模式，包括画廊模式、主讲人模式和自定义布局。 |
| 4 | [（可选）查询云端录制任务](https://doc.yunxin.163.com/nertc/server-apis/TU4ODczNDk?platform=server) | 查询云端录制任务的状态和配置，包括录制模式、录制订阅名单、内容安全审核配置、录制布局、水印。 |
| 5 | [（可选）更新订阅名单](https://doc.yunxin.163.com/nertc/server-apis/DYwODU5NTQ?platform=server) | 默认情况下云端录制会订阅房间内所有发布的音视频流。您可以在录制过程中随时更新订阅名单，支持设置白名单或黑名单。 |
| 6 | [（可选）更新录制布局](https://doc.yunxin.163.com/nertc/server-apis/jg1MDk1MjY?platform=server) | 默认情况下云端录制会订阅房间内所有发布的音视频流。您可以在录制过程中随时更新订阅名单，支持设置白名单或黑名单。 |
| 7 | [停止录制任务](https://doc.yunxin.163.com/nertc/server-apis/zIwNDM4NDg?platform=server) | 在音视频通话中，App server 调用服务端 API 停止录制任务。或者所有成员离开房间、房间结束之后，录制任务自动结束。 |
| 8 | [管理录制文件](https://doc.yunxin.163.com/nertc/server-apis/jYwNzI3MDk?platform=server) | 录制文件自动上传到 [网易云信点播服务端](https://doc.yunxin.163.com/vod/server-apis?platform=server) 中，您可以通过消息抄送接收录制文件的消息事件，或通过服务端 API 查询录制文件的文件名、URL 等信息。 |
| 9 | [抄送云端录制事件](https://doc.yunxin.163.com/nertc/server-apis/TM0NTU3OTM?platform=server) | 开通云端录制相关的消息抄送服务后，网易云信服务器将以 HTTPS POST 请求方法向您的服务器发送云端录制事件通知回调。 |
