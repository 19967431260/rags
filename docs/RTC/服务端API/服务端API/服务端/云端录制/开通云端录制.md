使用云端录制服务之前，请先在网易云信控制台开通云端录制服务。本文介绍开通云端录制、配置云端录制布局、配置录制文件输出速度。

## 前提条件

已在网易云信控制台 [开通 **点播** 服务](https://doc.yunxin.163.com/console/concept/zc3NDYzNzc?platform=console)。

## 开通云端录制

1. 登录 <a href="https://app.yunxin.163.com/index#/" target="_blank">网易云信控制台</a>。

2. 在首页单击指定应用名称。
3. 在 **产品总览** 区域，单击 **音视频通话 2.0** 产品选项卡中的 **功能配置**。

    <img alt="功能配置.png" src="https://yx-web-nosdn.netease.im/common/50fdee5139f9e30204445b9386f205cd/功能配置.png" style="width:60%;border: 0px solid #BFBFBF;">

4. 单击 **服务端功能** 页签，单击 **云端录制** 的开关按钮，开启云端录制。

    <img alt="开通云端录制.png" src="https://yx-web-nosdn.netease.im/common/af0bf471471cf33fc4341a277ff4148f/开通云端录制.png" style="width:60%;border: 0px solid #BFBFBF;">

5. 单击 **确定**。

## 配置云端录制布局（可选）

::: note note
该功能将逐步废弃并下线，建议您使用服务端提供的 [RESTful API](https://doc.yunxin.163.com/nertc/server-apis/TEyOTk2OTY?platform=server) 设置合流布局，不推荐使用控制台进行相关配置。
:::

1. 开启云端录制后，单击 **云端录制** 右侧的 **子功能配置**。

    <img alt="云端录制子功能配置.png" src="https://yx-web-nosdn.netease.im/common/d20948259a4b9fd9cd97e1151036f546/云端录制子功能配置.png" style="width:60%;border: 0px solid #BFBFBF;">

2. 设置画布尺寸。

    单击 **画布尺寸** 区域框中的 **编辑**，根据需求，选择将画布设置为 **竖屏（480x720）** 或 **横屏（720x480）**，单击 **保存**。

    <img alt="画布尺寸.png" src="https://yx-web-nosdn.netease.im/common/121503e47a6bb33bccca944238696dcf/画布尺寸.png" style="width:60%;border: 0px solid #BFBFBF;">

    ::: note note
    画布尺寸设置仅对画廊模式或自定义模式生效。
    :::

4. 设置录制布局模式。

    单击 **布局模式** 区域框中的 **编辑**，根据需求，选择布局模式，单击 **保存**。

    支持设置为：
    - **画廊模式**
    - **主讲人模式**
    - **自定义布局**

    <img alt="设置录制的布局模式.png" src="https://yx-web-nosdn.netease.im/common/91f3d32957440c8a1b3f7bc0a323ff23/设置录制的布局模式.png" style="width:60%;border: 1px solid #BFBFBF;">

5. （可选）自定义布局配置。

    若录制布局模式设置为 **自定义布局**，且开通了录制功能，您可以设置自定义布局相关参数。

    1. 单击布局模式右侧的 **自定义布局配置**。

    <img alt="自定义布局配置.png" src="https://yx-web-nosdn.netease.im/common/0cf2797c770ca2e87b03e70c52054af5/自定义布局配置.png" style="width:60%;border: 0px solid #BFBFBF;">

    2. 在 **实时并发数** 右侧选择房间人数。

    <img alt="设置布局参数.png" src="https://yx-web-nosdn.netease.im/common/fce52d76eb7b0579ec05cd291d0bbfe3/设置布局参数.png" style="width:60%;border: 0px solid #BFBFBF;">

    3. 为每位成员设置 **横向坐标**、**纵向坐标**、**画面宽度** 和 **画面高度**。

        您可以实时在右侧预览显示效果，并根据效果调整设置。
    4. （可选）在页面顶端单击 **添加并发配置**。并重复以上步骤，设置其他房间人数下的布局样式。

6. 单击 **确认配置**。

## 配置录制文件输出速度

云端录制结束后，会陆续生成录制文件，当录制文件较多时，可能会需要排队，您可以配置是正常排队输出，还是高优先级输出。

1. 开启云端录制后，单击 **云端录制** 右侧的 **子功能配置**。

    <img alt="云端录制子功能配置.png" src="https://yx-web-nosdn.netease.im/common/d20948259a4b9fd9cd97e1151036f546/云端录制子功能配置.png" style="width:60%;border: 0px solid #BFBFBF;">

2. 单击 **录制文件输出速度** 区域框中的 **编辑**，选择输出速度，单击 **保存**。

    <img alt="配置录制文件输出速度.png" src="https://yx-web-nosdn.netease.im/common/13ce1e4d87f1e17597d847de2b164aba/配置录制文件输出速度.png" style="width:60%;border: 0px solid #BFBFBF;">

    参数 | 描述
    ---- | ---- |
    常速模式 | 默认为常速模式，即正常排队输出。 |
    快速模式 | 录制文件会优先高优先级输出。<note type="note">快速模式需要单独收费，1000 元/月。</note> |
