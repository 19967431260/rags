开通云端录制相关的消息抄送服务后，网易云信服务器将以 HTTPS POST 请求方法向您的服务器发送云端录制事件通知回调。请求 Body 的数据格式为 JSON，字符编码为 UTF-8，签名算法为 SHA1。

消息抄送的实现方法请参考 [消息抄送服务概述](https://doc.yunxin.163.com/nertc/server-apis/TYzOTgzMjY)。
