
实时查询视频截图信息。

<note type="note">安全通相关服务端接口的请求结构、请求头、公共参数、字符编码均与其他服务端 API 相同，但 Content-Type 不同，此接口的 Content-Type 为 application/json。</note>

## 接口请求地址

```
POST https://rtc.yunxinapi.com/livewallsolution/query/image  HTTP/1.1
Content-Type: application/json;charset=utf-8
```

## 接口请求频率

默认上限为 50 次/秒，若请求频率超出限制，可能会返回 429 错误码。<br>若您需要上调上限，请参考<a href="https://doc.yunxin.163.com/nertc/server-apis/Tg3MDU1NTE?platform=server#如何处理调用服务端-restful-api-超出频率限制" target="_blank">如何处理调用服务端 RESTful API 超出频率限制</a>。

## 请求参数

- 请求中 Header 的设置请参考[请求结构](https://doc.yunxin.163.com/nertc/server-apis/TYyNjA3MzQ?platform=server#%E8%AF%B7%E6%B1%82%E7%BB%93%E6%9E%84)描述。

- 请求中 Body 须为 JSON 格式，参数说明如下：

| <div style="width:100px">参数名称</div> | <div style="width:100px">类型</div> | <div style="width:100px">是否必选</div> | <div style="width:250px">描述</div> |
|---|---|---|---|
| taskId | String | 可选 | 音视频内容安全审核任务 ID，是其唯一标识。 |
| levels | String | 可选 | String 形式的数组。0: 正常；1: 嫌疑；2: 确定。 |
| pageSize | Number | 可选 | 每页查询数据条数，取值范围为 0~1000。 |
| pageNum | Number | 可选 | 查询页码。 |

## <span id="返回参数">返回参数</span>

<style>

    width: 12%;
}

    width: 10%;

| 参数名称 | 类型 | 描述 |
|---|---|---|
| code | Number | 接口调用状态，200 表示成功调用。 |
| msg | String | 状态描述。 |
| result | 数组 | 接口返回结果。通常包含业务相关的返回结果。 |
| channelName | String | 进行内容审核的音视频房间名称。 |

result 字段说明：

| 参数名称 | 类型 | 描述 |
|---|---|---|
| status | Number | 数据状态，0: 成功，20: taskId过期，30: taskId不存在。 |
| images | json对象 | 截图分页数据。 |

其中 images 结构如下：

| 参数名称 | 类型 | 描述 |
|---|---|---|
| count | Number | 查询截图张数总量。 |
| rows | JSONArray | 分页数据。 |

rows 结构如下：

| 参数名称 | 类型 | 描述 |
|---|---|---|
| url | String | 截图地址。 |
| label | Number | 分类信息，100：色情，200：广告，400：违禁，500：涉政，600：谩骂，700：灌水。 |
| labelLevel | Number | 分类级别，1：不确定，2：确定。 |
| beginTime | Number | 截图开始时间。 |
| endTime | Number | 截图结束时间。 |
| uid | Number | 此违规信息对应的用户 ID。 |
| callbackStatus | Number | 回调状态。0：不需要回调；1：等待回调；2：已回调；3：回调失败。 |
