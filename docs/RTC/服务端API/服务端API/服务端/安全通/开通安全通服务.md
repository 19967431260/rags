
音视频通话 2.0 安全通功能提供多种类型的音频和视频检测类型，可支撑多种业务场景的内容安全检测需求。使用安全通服务之前，请先在网易云信控制台开通安全通服务。

<note type="note">
<ul><li>实时音视频安全通服务现已开通免费试用，您可以在控制台申请试用。申请通过后，可以免费享受 15 天的实时音视频内容审核服务，在 15 天内可以享受高达 500 次的音视频内容安全审核。</li>
<li>正式版功能涉及计费，请 <a href="https://yunxin.163.com/bizQQWPA.html">联系云信商务经理</a> 沟通计费策略。</li></ul>
</note>

## 前提条件

已预留接收消息抄送的服务器公网地址。审核结果会以消息抄送方式发送至您预留的抄送地址，请在开通服务前准备抄送地址，并保证抄送地址长期有效。

## 操作流程
1. 登录<a href="https://app.yunxin.163.com/index#/" target="_blank">网易云信控制台</a>。
2. 申请试用。
    1. 在首页单击指定应用名称。
    2. 在**产品总览**区域，单击**音视频通话 2.0** 产品选项卡中的**功能配置**。


        ![功能配置.png](https://yx-web-nosdn.netease.im/common/50fdee5139f9e30204445b9386f205cd/功能配置.png)


    3. 单击**安全通**页签，单击**免费试用**，单击**确定**。

        ![开通安全通.png](https://yx-web-nosdn.netease.im/common/ffea007403c9ebbb0c6727d22e2a8abc/开通安全通.png)


3. 配置功能。
    1. 配置**内容安全抄送地址**。

        接收音视频内容审核结果的抄送地址与 IM、音视频通话 2.0 的消息抄送地址相同，若您此前从未开通抄送功能、未配置抄送地址，请单击前往配置，配置抄送地址。
        
        如果您已经开通了[消息抄送功能](https://doc.yunxin.163.com/nertc/server-apis/DExNjg2MDc?platform=server)，此处默认使用同一个抄送地址。如果需要使用其他抄送地址接收结果，可以在调用服务端 API 创建审核任务时，指定另一个抄送地址。
 
    2. 开通安全通后，在 **RTC 安全通检测** 右侧单击 **子功能配置**。

        ![安全通配置.png](https://yx-web-nosdn.netease.im/common/05aeac078a65a381142cacffd130a5b1/安全通配置.png)


    3. 在**内容安全配置**区域，单击**配置**，开启机器过检右侧的功能开关，配置相关参数，参数说明如下表所示。

        
        ![内容安全配置.png](https://yx-web-nosdn.netease.im/common/815e7ac7a7bbdb01997521248b1e1bf1/内容安全配置.png)

        ![配置安全通.png](https://yx-web-nosdn.netease.im/common/ff218ed9fcf0a35aae832129dadff33e/配置安全通.png)


        参数 | 描述
        ---- | -------------- | 
        选择审核节点 |内容审核所使用的服务器节点的位置<!--，包括：<ul><li> 国内：待审核的数据发回中国进行审核。<li>海外：如果是出海的 App，为满足不同国家的安全监管要求，您可以根据需要选择海外节点审核，数据会传到海外服务器进行审核。 -->|
        选择审核语种 |为了让安全通能够准确地理解和识别不同语言中的敏感内容，您可以设置审核的语言种类<!--，，包括： <ul><li> 单一语种：只使用一种语言进行内容安全检测，适用于只有一种语言的场景。<li>自适应语种：安全通会根据数据中的语言自动适配相应的语言，适用于多语言混合的场景。--> |
        过检类型| 过检类型支持：<ul><li>视频与音频同时过检<li>仅过检视频<li>仅过检音频 | 
        截图频率 | 截图频率默认为 5 秒，支持设置为 1~600 秒。 | 


    4. 单击**保存**。

## 常见问题

1. 为什么在控制台找不到音视频通话 2.0 对应的 **安全通** 按钮？

    安全通功能已于 2021 年 3 月底正式升级，于 4 月 1 日之前开通的安全通功能均属于旧版功能，使用过旧版功能的应用无法直接在控制台开通新版安全通功能，若您有相关需求，请[联系云信商务经理](https://yunxin.163.com/bizQQWPA.html)切换至新版。

2. 试用版的音视频通话 2.0 可以开通安全通吗？

    可以，但是试用版的音视频通话 2.0 只能申请试用安全通，建议您尽快开通正式版产品。

3. 开通安全通之后，是否需要在抄送配置中手动勾选抄送类型？

    不需要。开通安全通功能之后，默认开通相关抄送。
