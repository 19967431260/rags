
实时查询音频断句信息。

<note type="note">安全通相关服务端接口的请求结构、请求头、公共参数、字符编码均与其他服务端 API 相同，但 Content-Type 不同，此接口的 Content-Type 为 application/json。</note>

## <span id="接口请求地址">接口请求地址</span>

```
POST https://rtc.yunxinapi.com/livewallsolution/query/audio/task HTTP/1.1
Content-Type:application/json;charset=utf-8
```

## 接口请求频率

默认上限为 50 次/秒，若请求频率超出限制，可能会返回 429 错误码。<br>若您需要上调上限，请参考<a href="https://doc.yunxin.163.com/nertc/server-apis/Tg3MDU1NTE?platform=server#如何处理调用服务端-restful-api-超出频率限制" target="_blank">如何处理调用服务端 RESTful API 超出频率限制</a>。

## <span id="请求参数">请求参数</span>

- 请求中 Header 的设置请参考[请求结构](https://doc.yunxin.163.com/nertc/server-apis/TYyNjA3MzQ?platform=server#%E8%AF%B7%E6%B1%82%E7%BB%93%E6%9E%84)描述。

- 请求中 Body 须为 JSON 格式，参数说明如下：

| <div style="width:100px">参数名称</div> | <div style="width:100px">类型</div> | <div style="width:100px">是否必选</div> | <div style="width:250px">描述</div> |
|---|---|---|---|
| taskId | String | 可选 | 音视频内容审核任务ID，是其唯一标识。 |
| channelName | String | 可选 | 进行内容审核的音视频房间名称。 |
| startTime | Number | 必选 | 断句开始时间，Unix 时间戳，精确到毫秒。 |\
|        |       |       | - 指定 record 参数为 0 时，可查询 90 天内的数据。|\
|        |       |       | - 指定 record 参数为 1 时，可查询 7 天内的数据。 |
| endTime | Number | 必选 | 断句结束时间。 |
| record | Number | 可选 | 是否返回录制文件地址。<br>0：（默认）不返回<br>1：返回 |

**注意**：startTime 和 endTime 之间的时间差不可超过 10 分钟。

## <span id="返回参数">返回参数</span>

| 参数名称 | 类型 | 描述 |
|---|---|---|
| code | Number | 接口调用状态，200 表示成功调用。 |
| msg | String | 状态描述。 |
| result | 数组 | 接口返回结果。通常包含业务相关的返回结果。 |
| channelName | String | 进行内容审核的音视频房间名称。 |

result 字段说明：

| 参数名称 | 类型 | 描述 |
|---|---|---|
| segmentId | String | 直播音频断句id。 |
| id | Number | 废弃字段，固定返回值为 0。无需关注。 |
| taskId | String | 音视频安全通审核任务 ID，是其唯一标识。 |
| action | Number | 检测结果，0：通过，1：嫌疑，2：不通过。 |
| asrStatus | Number | 音频翻译结果，2：检测中，3：检测完成，4：检测失败。 |
| asrResult | Number | 音频翻译检测失败原因，1：文件格式错误，2：文件下载失败，3：解析失败，4：音频流不存在。 |
| callback | String | 调用开启安全通审核接口时，传递的 callback 字段数据。 |
| censorSource | Number | 审核来源，0：易盾人审，1：客户人审，2：易盾机审。 |
| uid | Number | 此违规信息对应的用户ID。 |
| startTime | Number | 开始时间点，单位毫秒。 |
| endTime | Number | 结束时间点，单位毫秒。 |
| segments | json数组 | 证据信息，通过时数据为空。 |
| records | json数组 | 录制信息。 |

其中 segments 结构如下：

| 参数名称 | 类型 | 描述 |
|---|---|---|
| label | Number | 分类信息，100：色情，200：广告，300：暴恐，400：违禁，500：涉政，600：谩骂。 |
| level | Number | 分类级别，0：正常，1：不确定，2：确定。 |
| evidence | String | 命中的敏感词或者声纹检测的分值。 |
| subLabels | 数组 | 分类信息。 |

records 结构如下：

| 参数名称 | 类型 | 描述 |
|---|---|---|
| startTime | Number | 开始时间点，单位为毫秒。 |
| endTime | Number | 结束时间点，单位为毫秒。 |
| url | String | 录制地址。 |
