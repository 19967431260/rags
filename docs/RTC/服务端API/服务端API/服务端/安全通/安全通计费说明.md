本文介绍 RTC 内容安全的计费说明和计费示例。 RTC 内容安全的总费用包括：**RTC 反垃圾送检费用**+ **RTC 内容安全检测费用**
## 基本概念
- 纯音频送检时长：指的是只有音频送检的时长。
- 纯视频送检时长：指的是只有视频送检的时长。
- 音视频同时送检时长：指的是既有音频送检又有视频送检的时长。
- RTC 反垃圾送检：RTC 房间的音视频流送检时（通过云信安全通或易盾的内容安全），系统会添加反垃圾机器人加入房间，该机器人会订阅需要送检的音视频流送到易盾进行审核，因此当您使用反垃圾送检服务时，会产生 RTC 反垃圾送检费用。

## 内容安全费用组成

RTC 内容安全费用包括 **RTC 反垃圾送检费用**和 **RTC 内容安全检测费用**。具体示意图如下图所示。

## RTC 反垃圾送检费用


### 计费公式
**RTC 反垃圾送检费用 = 纯音频送检费用 + 纯视频送检费用 + 音视频同时送检费用。其中每一项费用都是各自时长*单价。**

### 计费单价
反垃圾送检按送检时长计时。送检时长类型分为纯音频送检时长、纯视频送检时长、音视频同时送检时长。具体档位、分辨率和单价如下表所示。

### 计费说明

- **计费规则**：反垃圾送检按送检时长计时，送检时长为反垃圾机器人订阅房间内的音视频流时长，送检时长类型分为纯音频送检时长、纯视频送检时长、音视频同时送检时长。
- **反垃圾送检时长统计规则**：纯音频送检时，同一时间房间内多路音频流送检，只算一路音频时长；纯视频送检时，同一时间房间内多路视频流送检，只算一路视频时长；音视频同时送检时，只算视频送检时长。
- **计费周期**：按月计费，月账单最小结算单位为千分钟，不满 1 千分钟部分按照 1 千分钟计算。

::: note important
2023年9月1日起，原有的 视频HD+ 档位被拆分为 视频UHD、视频2K、视频2K+ 三个档位。
在2023年9月1日前开通音视频通话2.0安全通服务的APPKEY，其视频UHD、视频2K、视频2K+价格与其原视频HD+保持一致。
:::


### 计费示例

假设用户 A、B、C 三人一起在音视频通话 2.0 房间内持续停留了 1000 分钟，彼此订阅相互的音频和视频流。您将整个房间的音视频通话过程进行送检。控制台设置的过检类型为**视频与音频同时过检**。


用户 |推流类型| 分辨率|
---- | -------------- | 
A | 纯音频 | 不涉及|
B | 音频和视频 |640 × 480|
C | 视频 |640 × 480 |

费用计算如下：

反垃圾机器人订阅 A 和 B 的音频，一个房间内多路音频算一路音频时长，音频送检的费用为： 5.9 元 / 千分钟 × 1000分钟 / 1000 = 5.9 元。

反垃圾机器人订阅 B 和 C 的集合分辨率为 640 × 480 + 640 × 480 = 614,400，规格为视频 HD 档位，所以按照 HD 单价进行计费。

音视频送检的费用为： 25 元 / 千分钟 × 1000分钟 / 1000 = 25 元。

该房间的 RTC 反垃圾送检费用 = 25 + 5.9 = 30.9 元。

**RTC 内容安全检测费用 = 月功能费（1000元/200万次/月）+ 超出套餐的费用**

超出套餐的费用 = （检测量 x 抵扣系数）x 标准单价

### 计费说明
- **计费规则**：音频送检时，按检测时长计费。视频送检时，RTC 服务端采用智能截帧策略，截取视频图片进行送检，按视频图片条数进行计费。
- **计费周期**：按月计费，月账单最小结算单位为千分钟，不满 1 千分钟部分按照 1 千分钟计算。
- **RTC 内容安全检测套餐赠送的 200 万次只能用于标准检测场景，特殊检测场景不参与 200 万次赠送次数抵扣。**

### 计费示例

假设用户的音频（中文）检测量为 200 分钟，领导人声纹检测量为 200 分钟，视频色情检测量为 200 条。RTC 内容安全检测费用计算如下：
- 音频（中文）检测计费量 = 200 * 15.37 = 3074 分钟
- 领导人声纹检测计费量 = 200 * 5.31 = 1026 分钟
- 视频色情检测计费量 = 200 * 1.1 = 220 条

系统优先把 3074 + 220 = 3294 次计费量在 200 万次里面进行抵扣，领导人声纹检测 1026 分钟不参与 200 万次抵扣，最终费用 = 1026 / 1000 * 0.9 = 0.92 元。
