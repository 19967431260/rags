## 指南针功能简介
指南针音视频通话相关的功能如下：
### 通话调查

指南针提供了音视频通话查询功能，帮助用户监测和管理其音视频通话的状态和详情。通过该功能，您可以方便地获取通话的基本信息、质量指标以及其他相关数据，从而实现对通话质量的监控和分析。关于通话调查的具体步骤，参考[通话总览](https://doc.yunxin.163.com/campass/docs/Tc4OTc0ODU?platform=console)。

### 质量指标

音视频通话质量指标，包括进房成功率、首帧时长、网络延时和卡顿率。这些数据指标可以帮助用户监测和评估音视频通话的质量，从而提供更好的用户体验和服务。

- **进房成功率**：音视频通话参与者成功进入通话房间的比例。该指标反映了通话系统的稳定性和可靠性。较高的进房成功率表示通话系统运行良好，参与者能够顺利加入通话。
- **首帧时长**：从参与者加入通话到首帧音视频数据显示的时间。较短的首帧时长表示通话系统响应快速，参与者能够尽快看到和听到其他参与者的音视频数据。
- **网络延时**：音视频数据在网络传输过程中的延迟时间。较低的网络延时表示网络连接稳定，音视频数据能够快速传输，提供实时的通话体验。
- **卡顿率**：音视频通话过程中出现卡顿现象的频率。较低的卡顿率表示通话系统和网络质量良好，参与者能够顺畅地进行通话，减少卡顿和中断的情况。

### 实时监控

音视频通话实时监控功能可以帮助用户实时监测和评估音视频通话的状态和质量，以便及时调整和优化通话体验。您可以通过实时监控平台查询以下监控指标：

- **实时在线规模**：当前音视频通话中的在线用户数和在线房间数。可以通过监控平台查看当前通话中的参与者数量，了解通话的规模和规模的变化趋势。
- **进房体验**：参与者进入音视频通话房间的体验情况。其中，5 秒进房成功率是指参与者成功进入通话房间所需时间不超过 5 秒的比例。较高的5秒进房成功率表示通话系统响应快速，参与者能够迅速加入通话，提供即时的通话体验。
- **实时通话质量**：包括音频网络延时、视频网络延时、音频卡顿率和视频卡顿率等指标。用户可以实时监测这些指标，了解音视频数据在网络传输和展示过程中的延迟和卡顿情况。
- **房间 Top 20 排行**：提供了按通话用户数、用户平均进房时长、音频卡顿率、视频卡顿率、音频网络延时和视频网络延时等指标的房间排行。用户可以根据这些指标了解各个房间的通话情况和性能表现，以便进行有针对性的优化和改进。
- **实时地域分布**：展示了音视频通话参与者的地理位置分布情况，包括中国大陆地区和全球区域。用户可以通过地域分布信息了解通话参与者的地理分布情况。
- **多维监测**：平台还提供了对SDK版本、网络类型和操作系统等维度的监测。用户可以根据这些维度的数据了解不同环境下的通话性能和用户体验差异，以便进行适当的优化和针对性的支持。
