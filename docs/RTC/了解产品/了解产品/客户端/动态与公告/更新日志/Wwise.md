本文介绍网易云信音视频通话 NERTC SDK 适配 Wwise 设计工具的版本更新日志。具体功能请前往 [下载 SDK 体验](https://doc.yunxin.163.com/nertc/resource?platform=wwise) 和 [查看 API 文档](https://doc.yunxin.163.com/nertc/client-apis/DE5MzA2MTM?platform=wwise)。

## 5.4.129 (2024-09-20)

- 已知问题修复。
- 性能优化。

## 5.4.127 (2024-07-31)

**新增**

- SDK 针对 Wwise 的适配版本升级到 2022.1.8.8316。
- 新增支持 Unreal Engine 项目，详情请参考 [集成 Unreal Engine 项目](https://doc.yunxin.163.com/nertc/guide/zQ3NjI1NDE?platform=wwise)。
- 新增语音消息、语音消息转文字功能。

**优化**

- 优化海外地区的网络连接稳定性。
- 性能优化。

## 5.4.112 (2024-01-10)

NERTC SDK 适配 Wwise 设计工具的首版发布。

- 支持 AI 降噪。
- 支持外放和耳机模式下保持高清音质。
- 支持小队模式、范围语音、空间音效。
- 支持自定义音频路由规则设置。
- 满足 FPS\TPS、大逃杀、小队开黑、SLG 等各种类型游戏的场景。
