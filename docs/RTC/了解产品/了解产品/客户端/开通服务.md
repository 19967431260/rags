
若您之前从未开通过任何网易云信的产品服务，请先在<a href="https://yunxin.163.com/" target="_blank">云信官网</a>通过微信、在线消息或电话等方式联系云信商务经理，一对一沟通您的具体需求。

## 前提条件
[已创建应用](https://doc.yunxin.163.com/console/guide/TIzMDE4NTA?platform=console)
## 开通服务

1. 登录<a href="https://app.yunxin.163.com/index#/" target="_blank">网易云信控制台</a>。
2. 在首页中单击指定应用名称，进入该应用的详情页面。
3. 在**应用配置**页面的**产品总览**区域，单击**音视频通话2.0**产品中的**免费试用**或**正式开通**。

    ![开通RTC2.png](https://yx-web-nosdn.netease.im/common/92e0bb1b2a70aa8101077f01ca112945/开通RTC2.png)


4. 设置 RTC 服务的区域，系统会根据使用时长预估费用，请填写**充值金额**。

    ::: note important
    - 应用创建后，服务区域不支持修改，请谨慎操作。如果您需要修改数据中心，请提交工单联系网易云信技术支持。
    - 如果您的应用主要服务中国国内用户，请选择**国内**。如果您的应用需要出海，主要服务海外用户，请选择**海外**。
    - 具体的计费策略，请参见[按量计费](https://doc.yunxin.163.com/nertc/quick-start/zMyMzUyODU?platform=android)。
    :::
  
    ![开通服务.png](https://yx-web-nosdn.netease.im/common/8c07440de06c0c6add6fd27c6f52543e/开通服务.png)   

5. 勾选**我已阅读并同意《网易云信服务条款》**，并单击**提交订单**。
 
6. 在**支付台**中确认金额、选择支付方式，并单击**立即支付**。
