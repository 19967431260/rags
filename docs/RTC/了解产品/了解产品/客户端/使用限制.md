在使用音视频通话 2.0 之前，请先了解以下使用限制。

## <span id="业务限制">业务限制</span>

<table>
    <tr>
        <th width="20%"><b>限制项</b></th>
        <th width="40%"><b>说明</b></th>
    </tr>
    <tr>
        <td>单房间内人数限制</td>
        <td><ul><li>单房间内最多支持 60 名主播同时连麦进行实时音视频通话。</li><li>单房间内总人数上限可达到 10,000 人。</li></ul></td>
    </tr>
    <tr>
        <td>单房间内用户登录的 QPS（Queries Per Second）</td>
        <td>每个房间的所有用户登录 QPS 总和为 200，即每秒内最多可支持 200 个用户登录同一房间。</td>
    </tr>
    <tr>
        <td>单个 API 每秒最大请求数</td>
        <td>不高于 1200。<br>每个 API 的具体限制不同，请参考各个 API 接口文档。</td>
    </tr>
    <tr>
        <td>Web 端在 H5 平台的能力限制</td>
        <td>Web 端在 H5 平台不支持伴音、客户端录制、麦克风采集音量设置相关 API，详细接口限制请参考 <a href="/nertc/guide/Tk0MzA1ODc?platform=android#Web%E7%AB%AF%E5%9C%A8H5%E5%B9%B3%E5%8F%B0%E6%9C%89%E5%93%AA%E4%BA%9B%E6%8E%A5%E5%8F%A3%E9%99%90%E5%88%B6">Web 端在 H5
            平台有哪些接口限制</a>。</td>
    </tr>
</table>

<note type="notice">如果需要提高限制，请提前联系网易云信技术支持咨询评估，超出限制可能会引起部分业务受限。</note>
