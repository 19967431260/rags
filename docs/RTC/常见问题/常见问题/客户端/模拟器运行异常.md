## 问题描述

在运行 Android 模拟器时，出现以下类似删除 x86 相关的异常信息：

```
E RoomKit :  java.lang.UnsatisfiedLinkError: No implementation found for void com.netease.yunxin.lite.LiteEngineCenter.unInitialize()
```

## 问题原因

缺少相应的库文件。

## 解决方案

在 app 目录下的 build.gradle 文件的 defalutConfig 中添加以下代码：

```
ndk {
    abiFilters+= listOf("armeabi-v7a","arm64-v8a")
}
```

::: note note 
若已添加相应配置，请检查是否存在 x86 相关的配置。若有，请删除相关配置。
:::
