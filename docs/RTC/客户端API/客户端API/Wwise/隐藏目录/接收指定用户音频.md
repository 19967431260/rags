## SetReceiveUserIDWithGameObjectID

### 接口描述

绑定 userID 和 gameobject 对象。userID 传入 0 解除绑定关系。

::: note note
默认最多能绑定3个userID,因为服务器最多只会转发3路音频流。如果需要更多的音频转发路数，需要联系销售或技术支持同学。
:::

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| gameObjectID | ulong | 是   | -       |  Wwise gameobject 对象唯一标识。|
| userID | ulong | 是 | - | 用户唯一 ID。|

### 示例代码

```
IRtcEngine _rtcEngine = IRtcEngine.GetInstance();
private void SetReceiveUserIDWithGameObjectID(){
    GameObject gameObject;//YOUR BIND GameObject
    ulong userID = 0; //YOUR RECEIVER ID
    var gameobjectID = AkSoundEngine.GetAkGameObjectID(gameObject);
    _rtcEngine.SetReceiveUserIDWithGameObjectID(gameobjectID, userID);
}
```

### 返回值

无

## GetReceiveUserIDWithGameObjectID

### 接口描述

获取绑定 gameobject 对象的 userID。

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| gameObjectID | ulong | 是   | -       |  Wwise gameobject 对象唯一标识。|

### 示例代码

```
private void GetReceiveUserIDWithGameObjectID(){
    GameObject gameObject;//YOUR GameObject
    var gameobjectID = AkSoundEngine.GetAkGameObjectID(gameObject);
    ulong userID = _rtcEngine.GetReceiveUserIDWithGameObjectID(gameobjectID);
}
```

### 返回值

gameobject 对象绑定的用户唯一 ID。


## EnableLoopbackWithGameObjectID

### 接口描述

开启或关闭 loopback。如果开启,则仅播放，不会发送到 RTC 服务器，如果关闭，则仅发送到服务器，不播放。

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| gameObjectID | ulong | 是   | -       |  Wwise gameobject 对象唯一标识。|
| enableLoopback | Boolean | 是   | -       |  是否开启 loopback。|

### 示例代码

```
private void EnableLoopbackWithGameObjectID(){
    GameObject gameObject;//YOUR BIND GameObject
    var gameobjectID = AkSoundEngine.GetAkGameObjectID(gameObject);
    bool enable = _rtcEngine.EnableLoopbackWithGameObjectID(gameobjectID);
}
```

### 返回值

无


## GetEnableLoopbackWithGameObjectID

### 接口描述

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| gameObjectID | ulong | 是   | -       |  Wwise gameobject 对象唯一标识。|

### 示例代码

```
private void EnableLoopbackWithGameObjectID(){
    GameObject gameObject;//YOUR BIND GameObject
    var gameobjectID = AkSoundEngine.GetAkGameObjectID(gameObject);
    bool enable = _rtcEngine.GetEnableLoopbackWithGameObjectID(gameobjectID);
}
```

### 返回值

- true: 开启 Loopback。
- false: 关闭 Loopback。
