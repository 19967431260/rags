## SetAudioProfile

### 接口描述

设置音频编码属性。需要在播放或采集插件运行之前设置。非必要不需要手动设置。


### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| profile | RtcAudioProfileType | 是   | -       |  设置采样率，码率，编码模式和声道数: #WWISEPLUGINAudioProfileType。|
| scenario | RtcAudioScenarioType | 是 | - | 设置音频应用场景: #WWISEPLUGINAudioScenarioType 。。|

### 示例代码

```
private void SetAudioProfile(){
    _rtcEngine.SetAudioProfile(RtcAudioProfileType.kWWISEPLUGINAudioProfileStandard, RtcAudioScenarioType.kWWISEPLUGINAudioScenarioMusic);
}
```

### 返回值

- 0: 方法调用成功。
- 其他: 方法调用失败。
