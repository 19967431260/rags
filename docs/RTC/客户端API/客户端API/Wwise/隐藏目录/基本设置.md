## InitEngine

### 接口描述

初始化 NERTC SDK 服务。

必须先调用该方法进行初始化，才能使用其他方法。初始化成功后，默认处于音视频通话模式。

::: note note

必须使用同一个 AppKey 才能进行通话。
:::

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| param | RtcEngineContext | 是   | -       | 传入的 RtcEngineContext 对象。|

### 示例代码

```
// Rtc引擎实例
IRtcEngine _rtcEngine = IRtcEngine.GetInstance();

void Start()
{
    InitEngine();
}

void InitEngine()
{
    var context = new RtcEngineContext();
    context.logPath = Application.persistentDataPath; //YOUR LOG PATH
    context.appKey = APPKEY;
#if UNITY_ANDROID && !UNITY_EDITOR
    AndroidJavaClass playerClass = new AndroidJavaClass("com.unity3d.player.UnityPlayer");
    AndroidJavaObject activity = playerClass.GetStatic<AndroidJavaObject>("currentActivity");
    context.context = activity;
#endif
    
    _rtcEngine.InitEngine(context);
}

```

### 返回值

- 0: 方法调用成功；
- 其他: 方法调用失败。

## SetParameters

### 接口描述

设置音视频通话的相关参数。通过 JSON 配置 SDK 提供技术预览或特别定制功能。以标准化方式公开 JSON 选项。初始化之前只对最后一次设置的JSON生效，调用初始化之后的JSON即时生效。

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| parameters | String | 是   | -       |  音视频通话的相关参数。JSON 字符串形式。|

### 示例代码

```
void SetParameters(){
    //比如关闭日志加密功能
    _rtcEngine.SetParameters("{\"sdk.enable.encrypt.log\":false}");
}
```

### 返回值

无


## release

### 接口描述

销毁 RTC 引擎对象。

### 示例代码

```
//进程退出后必须要释放引擎资源
void OnApplicationQuit(){
    _rtcEngine.release();
}
```

### 返回值

无
