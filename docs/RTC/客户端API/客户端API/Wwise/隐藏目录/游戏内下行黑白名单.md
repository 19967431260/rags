

| API                              | 描述                                    |
|---------| --------------------------------- | ------------------------------------- |
| [SetSubscribeAudioAllowList]()        | 指定只订阅的音频流                             |
| [SetSubscribeAudioBlockList]()        | 指定不订阅的音频流                             |

## SetSubscribeAudioAllowList

### 接口描述

指定订阅的音频流。

::: note note

- 此接口需要在加入房间成功后调用。
- 对于调用接口时不在房间的用户 ID (uid) 不生效。
:::

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| uids  | ULong 数组 | 是 | -       |  只订阅此 用户 uid 列表 的音频。<br /> 此列表为全量列表。如果列表为空或 null，则取消订阅白名单。|

### 示例代码

```
private void SetSubscribeAudioAllowList(){
    ulong[] users = new ulong[]{111,222};
    _rtcEngine.SetSubscribeAudioAllowList(users);
    //取消s
    _rtcEngine.SetSubscribeAudioAllowList(null);
}
```

### 返回值

- 0: 方法调用成功。
- 其他: 方法调用失败。


## SetSubscribeAudioBlockList

### 接口描述

指定不订阅的音频流。

::: note note

- 此接口需要在加入房间成功后调用。
- 对于调用接口时不在房间的用户 ID (uid) 不生效。
:::

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| uids  | ULong 数组 | 是 | -       |  不订阅此用户 uid 列表的音频。。<br /> 此列表为全量列表。如果列表为空或 null，取消订阅黑名单。|

### 示例代码

```
private void SetSubscribeAudioBlockList(){
    ulong[] users = new ulong[]{111,222};
    _rtcEngine.SetSubscribeAudioBlockList(users);

    //取消
    _rtcEngine.SetSubscribeAudioBlockList(null);

}
```

### 返回值

- 0: 方法调用成功。
- 其他: 方法调用失败。
