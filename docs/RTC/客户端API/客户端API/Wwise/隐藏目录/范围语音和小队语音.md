
| API                              | 描述                                    |
| --------------------------------- | ------------------------------------- |
| [SetRangeAudioMode]()                 | 设置范围语音模式                              |
| [SetRangeAudioTeamID]()               | 设置范围语音小队                              |
| [SetAudioRecvRange]()                 | 设置范围语音接收范围                            |
| [UpdateSelfPosition]()                | 引擎3D音效算法中本人坐标方位更新接口             |

::: note note
SetRangeAudioMode、SetRangeAudioTeamID、SetAudioRecvRange必须要同时设置有效值才会生效，否则可能会导致入会失败。
:::


## SetRangeAudioMode

### 接口描述

设置范围语音模式。支持加入房间后修改参数，但必须在加入房间前要先设置有效值。
  
::: note note

- 此接口在加入房间前调用。
- 支持加入房间后修改参数。
:::

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| mode | RtcRangeAudioMode | 是   | -       |  范围语音模式。|

### 示例代码

```
private void SetRangeAudioMode(){
    _rtcEngine.SetRangeAudioMode(RtcRangeAudioMode.kWWISEPLUGINRangeAudioModeDefault);
}
```

### 返回值

- 0: 方法调用成功。
- 其他: 方法调用失败。


## SetRangeAudioTeamID

### 接口描述

设置范围语音小队。支持加入房间后修改参数，但必须在加入房间前要先设置有效值。

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| teamID | int | 是   | -       | 小队 ID。|

### 示例代码

```
private void SetRangeAudioTeamID(){
    int teamID = 1;
    _rtcEngine.SetRangeAudioTeamID(teamID);
}
```

### 返回值

- 0: 方法调用成功。
- 其他: 方法调用失败。



## SetAudioRecvRange

### 接口描述

设置范围语音接收范围。支持加入房间后修改参数，但必须在加入房间前要先设置有效值。

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| range | int | 是   | -       |  监听器能够听到扬声器并接收其文本消息的距离扬声器的最大距离。无默认值, 有效范围[1, ∞)。。|

### 示例代码

```
private void SetAudioRecvRange(){
    int range = 50;
    _rtcEngine.SetAudioRecvRange(range);
}
```

### 返回值

- 0: 方法调用成功。
- 其他: 方法调用失败。


## UpdateSelfPosition

### 接口描述

更新当前玩家的坐标方位。

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| positionX | int | 是   | -       | 当前玩家的X轴坐标。|
| positionY | int | 是   | -       | 当前玩家的Y轴坐标。|
| positionZ | int | 是   | -       | 当前玩家的Z轴坐标。|

### 示例代码

```
// player or GameObject
public GameObject selfGameObject;
IRtcEngine _rtcEngine = IRtcEngine.GetInstance();
private void EnableAudioRange(){
    int teamID = 1; //设置你的小队ID
    _rtcEngine.SetRangeAudioTeamID(teamID);
    _rtcEngine.SetRangeAudioMode(RtcRangeAudioMode.kNERtcRangeAudioModeTeam);//设置你的语音模式
    _rtcEngine.SetAudioRecvRange(50);
}

//update my position in the game world
private IEnumerator UpdateMySelfPosition()
{
    while (true)
    {
        yield return new WaitForSeconds(0.2f);
        if (selfGameObject  != null)
        {
            int result = _rtcEngine.UpdateSelfPosition(selfGameObject.transform.position.x, selfGameObject.transform.position.y, selfGameObject.transform.position.z);
            if(result != (int)RtcErrorCode.kWWISEPLUGINNoError)
            {
                Debug.Log($"UpdateSelfPosition Failed : {result}");
            }
        }
    }

}
```

### 返回值

无
