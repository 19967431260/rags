
| API                              | 描述                                    |
| --------------------------------- | ------------------------------------- |
| [PauseAudio]()                    | 暂停音频输入与输出                      |
| [ResumeAudio]()                   | 恢复音频输入与输出                      |


## PauseAudio

### 接口描述

暂停音频输入与输出。


### 返回值

- 0: 方法调用成功。
- 其他: 方法调用失败。


## ResumeAudio

### 接口描述

恢复音频输入与输出。

### 返回值

- 0: 方法调用成功。
- 其他: 方法调用失败。


### 示例代码

```
private void OnApplicationPause(bool pause)
{
    if (pause)
    {
        _rtcEngine.PauseAudio();
    }
    else
    {
        _rtcEngine.ResumeAudio();
    }
}
```
