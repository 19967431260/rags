## EnableAudioVolumeIndication

### 接口描述

启用说话者音量提示。
<br>该方法允许 SDK 定期向 App 反馈本地发流用户和瞬时音量最高的远端用户（最多 3 位）的音量相关信息，即当前谁在说话以及说话者的音量。
<br>启用该方法后，只要房间内有发流用户，无论是否有人说话，SDK 都会在加入房间后根据预设的时间间隔触发 `onRemoteAudioVolumeIndication` 和
`onRemoteAudioVolumeIndication` 回调。


### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| enable  | Boolean | 是 | -       |  是否启用说话者音量提示。|
| interval | ulong  | 是 | -       |  指定音量提示的时间间隔，单位为毫秒。必须设置为 100 毫秒的整数倍值。|
| enableVad | Boolean | 是 | -      |  是否启用本地采集人声监测。|

### 示例代码

```
private void EnableAudioVolumeIndication(){
    bool enable = true;
    ulong interval = 1000;
    bool enableVad = true;
    _rtcEngine.EnableAudioVolumeIndication(enable, interval, enableVad);
}
```

### 返回值

- 0: 方法调用成功。
- 其他: 方法调用失败。
