## OnConnectionStateChanged

### 回调描述

网络连接状态已改变回调。
<br>该回调在网络连接状态发生改变的时候触发，并告知用户当前的网络连接状态和引起网络状态改变的原因。

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| state  | int  | 是 | -       |  当前的网络连接状态。|
| reason | int  | 是 | -       |  引起当前网络连接状态发生改变的原因。|


## OnJoinChannel

### 回调描述

加入房间回调，表示客户端已经登入服务器。

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| cid  | ULong  | 是 | -       |  客户端加入的房间 ID。|
| uid | ULong  | 是 | -       |  用户 ID。 如果在 joinChannel 方法中指定了 uid，此处会返回指定的 ID; <br />如果未指定 uid，此处将返回云信服务器自动分配的 ID。|
| result  | RtcErrorCode  | 是 | -       |  返回结果。|
| elapsed | ULong  | 是 | -       |  从 joinChannel 开始到发生此事件过去的时间，单位为毫秒。|

## OnUserJoined

### 回调描述

远端用户加入当前房间回调。
<br>该回调提示有远端用户加入了房间，并返回新加入用户的 ID。
<br>如果加入之前，已经有其他用户在房间中了，新加入的用户也会收到这些已有用户加入房间的回调。

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| uid  | ULong  | 是 | -       |  新加入房间的远端用户 ID。|
| userName | String  | 是 | -       |  新加入房间的远端用户名。|


## OnUserLeft

### 回调描述

远端用户离开当前房间的回调。
<br>提示有远端用户离开了房间（或掉线）。通常情况下，用户离开房间有两个原因，即正常离开和超时掉线：
- 正常离开的时候，远端用户会收到正常离开房间的回调提醒，判断用户离开房间。
- 超时掉线的依据是，在一定时间内（40~50s），用户没有收到对方的任何数据包，则判定为对方掉线。

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| uid  | ULong  | 是 | -       |  离开房间的远端用户 ID。|
| reason | int  | 是 | -       |  远端用户离开原因。|

## OnLocalAudioVolumeIndication

### 回调描述

提示房间内本地用户瞬时音量的回调。

- 该回调默认禁用。可以通过 IRtcEngineEx::enableAudioVolumeIndication "enableAudioVolumeIndication" 方法开启。
- 开启后，本地用户说话，SDK 会按  IRtcEngineEx::enableAudioVolumeIndication "enableAudioVolumeIndication" 方法中设置的时间间隔触发该回调。
- 如果本地用户将自己静音（调用了 IRtcEngineEx::muteLocalAudioStream "muteLocalAudioStream"），SDK 将音量设置为 0 后回调给应用层。

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| volume | int  | 是 | -       |  （混音后的）音量，取值范围为 [0,100]。|
| enable_vad  | Boolean  | 是 | -       |  是否检测到人声。|

## OnRemoteAudioVolumeIndication

### 参数说明

提示房间内谁正在说话及说话者瞬时音量的回调。
<br>该回调默认为关闭状态。可以通过 enableAudioVolumeIndication 方法开启。开启后，无论房间内是否有人说话，SDK 都会按
enableAudioVolumeIndication 方法中设置的时间间隔触发该回调。 <br>在返回的 speakers 数组中:

- 如果有 uid 出现在上次返回的数组中，但不在本次返回的数组中，则默认该 uid 对应的远端用户没有说话。
- 如果 volume 为 0，表示该用户没有说话。
- 如果 speakers 数组为空，则表示此时远端没有人说话。

### 参数说明

| 参数名称 | 类型 | 是否必填 | 默认值 |   描述   |
| :------ | :------- | :-------- | :-------- | :------------------- |
| speakers | RtcAudioVolumeInfo 数组  | 是 | -       | 每个说话者的用户 ID 和音量信息的数组。|
| totalVolume  | int  | 是 | -       | （混音后的）总音量，取值范围为 [0,100]。|
