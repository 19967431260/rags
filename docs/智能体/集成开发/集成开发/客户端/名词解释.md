为了帮助您更好地理解和使用 [网易云信智能体管理平台](https://rtc-agent-console.netease.im/#/home)，本文对平台中常见的专业术语进行了通俗易懂的解释。

| 名词 | 英文全称 | 解释 |
| :--- | :--- | :--- |
| **Agent** | Agent | **智能体**。您可以把它当成一个有 **灵魂** 的虚拟机器人。它不仅能按固定套路回答问题，还能像真人一样思考、带有情绪、记住您说的话，并使用工具帮您办事。 |
| **AppKey** | Application Key | **应用密钥**。这是您在 [网易云信控制台](https://app.yunxin.163.com/global/home) 创建应用后获得的一串唯一代码，相当于您应用的 **身份证号**。平台通过它来识别您的身份并管理您名下的智能体。 |
| **LLM** | Large Language Model | **大语言模型**。您可以把它理解为智能体的 **大脑**。它负责理解您说的话，并思考如何回答。常见的 LLM 有千问、豆包、OpenAI 等。模型越强大，智能体就越聪明。 |
| **ASR** | Automatic Speech Recognition | **自动语音识别**。这是智能体的 **耳朵**。它的作用是把您说的话（语音）转换成文字，这样 **大脑** （LLM）才能看懂并进行处理。 |
| **TTS** | Text-to-Speech | **文本转语音**。这是智能体的 **嘴巴**。当 **大脑** （LLM）想好怎么回答并写出文字后，TTS 会把这些文字朗读出来，变成您能听到的声音。 |
| **流式 (Stream)** | Streaming | **流式传输**。一种数据传输方式。在智能体对话中，流式意味着智能体可以 **边想边说**，不需要等整段话都想好才开始发音，这能大大减少您等待回答的时间。 |
| **Pipeline** | Pipeline | **对话流水线/流控**。字面意思是 **流水线**。在智能体中，它是一套 **交通指挥系统**。当用户说话时，声音先经过耳朵（ASR），再传给大脑（LLM），最后通过嘴巴（TTS）说出来。Pipeline 就是负责管理这个全过程的管家，例如决定什么时候该打断智能体说话、什么时候该进入休眠。 |
| **上下文** | Context | **短期记忆**。例如您设置 **上下文大小** 为 3，智能体就能记住您们刚刚聊过的前 3 句话。这样聊天时就不会 **前言不搭后语**。 |
| **CosyVoice** | CosyVoice | **一种高质量语音合成技术**。由阿里通义实验室开源的语音模型，能够生成非常自然、带有丰富情感和语气（如笑声、呼吸声）的拟人化声音。 |
| **Latex 公式** | LaTeX | **一种数学排版系统**。常用于书写复杂的数学公式。开启 **播报 Latex 公式** 后，智能体能正确读出像 $E=mc^2$ 这样的专业数学表达式，而不是乱读一通。 |
| **VAD** | Voice Activity Detection | **语音活动检测**。这是一种判断 **有没有人在说话** 的技术。平台支持 **语义 VAD**，它不仅能判断有没有声音，还能结合上下文判断您是不是真的说完了，从而决定智能体什么时候该接话。 |
| **回声消除 (AEC)** | Acoustic Echo Cancellation | **回声消除**。当智能体通过扬声器说话时，声音可能会被麦克风再次录进去形成回声。这项技术能自动过滤掉这些回声，保证对话清晰。 |
| **AI 降噪 (ANS)** | AI Noise Suppression | **人工智能降噪**。利用 AI 技术自动识别并消除背景噪音（如风扇声、键盘敲击声、马路噪音），让智能体只听清您说话的声音。 |
| **Prompt** | Prompt | **提示词**。这是您给智能体设定的 **人设** 和 **规则**。例如，您可以告诉它：**您是一个温柔的幼教老师，请用简单的词汇和孩子聊天。** 提示词决定了智能体的性格和说话方式。 |
| **Token** | Token | **词元**。大模型处理文本的基本单位。一个 Token 可以是一个字、一个词或一个标点符号。例如 **Max Tokens** 就是用来限制智能体每次回答的最大长度。 |
| **Temperature** | Temperature | **温度值**。这是一个控制智能体回答 **创造力** 的参数。值越低（接近 0），回答越严谨、固定。值越高（接近 1 或更高），回答越发散、有创意，适合讲故事或闲聊。 |
| **Top p** | Top p | **核采样**。和 Temperature 类似，也是用来控制回答的多样性。通常建议只调整 Temperature 或 Top p 中的一个。 |
| **Pipeline** | Pipeline | **对话流控**。这就像是智能体内部的 **交通指挥员**，负责协调耳朵（ASR）、大脑（LLM）和嘴巴（TTS）之间的工作流程。例如，当您打断智能体说话时，Pipeline 会告诉它立刻停下来听您说。 |
| **MCP** | Model Context Protocol | **模型上下文协议**。您可以把它理解为智能体的 **外挂工具箱**。通过 MCP，智能体可以连接到外部世界，例如去网上查天气、搜新闻，或者控制智能家居设备。 |
| **RTC** | Real-Time Communication | **实时音视频通信**。就像打微信语音电话一样，是一种能让声音瞬间传给对方的技术，保证聊天超低延迟、不卡顿。 |
| **回调** | Callback | **自动通知**。就像是 **报警器**。当智能体遇到错误，或者网络延迟太高时，系统会自动给您的服务器发一条消息，告诉您 **出状况了**。 |
| **JSON** | JSON | **一种数据格式**。一种常用的书写格式，在配置高级参数时可能会用到，使用 JSON 配置会更高效。非技术人员一般不需要深入了解，使用平台提供的 **可视化编辑** 即可。 |
| **Header** | HTTP Header | **请求头**。就像寄快递时信封上的 **特殊备注**。在配置高级网络服务时，用来传递一些额外的验证信息（例如密码或身份标识）。 |
| **SSE** | Server-Sent Events | **服务器发送事件**。一种允许服务器向客户端单向推送数据的技术。在 MCP 配置中，它是一种较老的连接协议，现在平台更推荐使用 Streamable HTTP。 |
| **Agent 模式** | Agent Mode | **智能体模式**。这是智能体在特定场景下的 **变身** 能力。例如，平时它是一个普通助手，但当您说 **讲个故事吧** 时，它可以切换到 **讲故事模式**，使用专门为讲故事设定的规则和声音。 |
| **知识库** | Knowledge Base | **知识库**。这是智能体的 **专属图书馆**。您可以把公司的产品手册、操作指南等文档传给它。当用户提问时，智能体会优先从这个图书馆里找答案，确保回答专业且准确。 |
| **声纹** | Voiceprint | **声纹**。就像每个人的指纹是独一无二的一样，每个人的声音特征也是独特的。通过声纹识别，智能体可以 **听声辨人**，知道现在和它说话的是爸爸、妈妈还是孩子，从而提供个性化的回应。 |
| **ASR 热词** | ASR Hotwords | **语音识别热词**。这是一份 **重点词汇表**。如果您发现智能体经常把某个专业术语听错，您可以把这个词加到热词库里，这样它下次就能听得更准了。 |
| **音色克隆** | Voice Cloning | **声音克隆**。只需要提供一段真人的录音，平台就能通过 AI 技术模仿出这个人的声音，让智能体用这个专属声音和您说话。 |
| **环境变量** | Environment Variable | **环境变量**。您可以把它理解为 **填空题的答案**。在配置提示词时，您可以留出一些空（变量），然后在环境变量里统一设置这些空应该填什么。这样修改起来非常方便，不需要去改长长的提示词。 |
