导入外部消息到网易云信智能体，网易云信侧会将该消息合并到历史对话的上下文，并根据该消息生成记忆。

<!-- ## 功能描述

- 支持导入外部消息到指定 agent 的对话上下文中
- 网易云信侧自动将导入的消息与历史对话进行合并
- 根据导入的消息内容生成对话记忆
- 支持文本消息和图片 URL 消息类型
- 消息按时间戳顺序进行排序和合并 -->

## 请求信息

<style>
table th:first-of-type {
    width: 20%;
}
</style>

### 请求 URI

```
POST https://rtc-agent.yunxinapi.com/v1/messages/import
```

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](https://doc.yunxin.163.com/ai-hardware/server-apis/DYwNTk5MzM?platform=server#Header)。

### 请求体参数

 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
 :--- | :--- | :--- | :--- | :--- |
 agentId | String | 是 | 3cb16a1a0f774b299eaa921007064891 | Agent ID，用于标识特定的对话代理 |
 uid | String | 是 | user1 | 用户 ID，用于标识消息所属用户 |
 messages | Array | 是 | - | 待导入的对话消息列表 |
   role | String | 是 | user | 消息发送者角色。<ul><li>`user`：用户</li><li>`assistant`：助手</li></ul> |
   createTime | Long | 是 | 1746531559000 | 消息创建时间戳，单位毫秒 |
   type | String | 是 | text | 消息类型。<ul><li>`text`：文本消息</li><li>`image_url`：图片 URL 消息</li></ul> |
   text | String | 否 | 你好 | 消息文本内容，长度上限 5000 个字符。`type` 为 `text` 时必填 |
   attachment | Object | 否 | - | 消息附件信息，`type` 为 `image_url` 时必填，`text` 类型时忽略 |
     url | String | 否 | https://example.com/image.jpg | 图片 URL 地址 |
     description | String | 否 | 示例图片 | 图片描述信息 |

### 请求体示例

```JSON
{
  "agentId": "3cb16a1a0f774b299eaa921007064891",
  "uid": "user1",
  "messages": [
    {
      "role": "user",
      "createTime": 1746531559000,
      "type": "text",
      "text": "你是谁啊"
    },
    {
      "role": "assistant",
      "createTime": 1746531560000,
      "type": "text",
      "text": "我是网易小智，你使用网易云信遇到任何问题，都可以问我哦"
    },
    {
      "role": "user",
      "createTime": 1746531565000,
      "type": "image_url",
      "text": "这是什么图片？",
      "attachment": {
        "url": "https://example.com/image.jpg",
        "description": "用户上传的示例图片"
      }
    }
  ]
}
```

## 响应信息

### 响应参数

 参数名称 | 类型 | 说明 | 是否必返回 |
 :--- | :--- | :--- | :--- |
 code | Integer | 响应状态码，200 表示成功 | 是 |
 message | String | 响应消息描述 | 是 |

### 响应体示例

```JSON
{
  "code": 200,
  "message": "success"
}
```
