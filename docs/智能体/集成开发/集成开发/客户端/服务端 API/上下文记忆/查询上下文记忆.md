查询在网易云信智能体平台上添加的智能体的上下文记忆。

## 请求信息

<style>
table th:first-of-type {
    width: 20%;
}
</style>

### 请求 URI

```
GET https://rtc-agent.yunxinapi.com/v1/memories
```

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](https://doc.yunxin.163.com/ai-hardware/server-apis/DYwNTk5MzM?platform=server#Header)。

### 请求查询参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
|:--------|:-----|:---------|:-----|:-----|
| agentId | String | 是 | 3cb16a1a0f774b299eaa921007064891 | 智能体ID |
| uid | String | 是 | user1 | 用户ID |
| limitDays | Integer | 是 | 10 | 返回最近 limitDays 天更新/新增的记忆，最大180天 |
| voicePrintId | String | 否 | 99IOEqBml51 | 声纹ID，用于筛选特定声纹用户的记忆 |

### 请求示例

```cURL
GET https://rtc-agent.yunxinapi.com/v1/memories?agentId=3cb16a1a0f774b299eaa921007064891&uid=user1&limitDays=10&voicePrintId=99IOEqBml51
```

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
|:--------|:-----|:-----|:----------|
| code | Integer | 状态码，200表示请求成功 | 是 |
| message | String | 提示信息，请求失败时返回错误信息，请求成功时返回"操作成功" | 是 |
| requestId | String | 请求唯一标识符 | 是 |
| data | Array | 记忆数据列表，请求失败则返回空数组 | 是 |
   id | String | 记忆唯一标识符 | 是 |
   content | String | 记忆内容描述 | 是 |
   createdAt | String | 记忆首次生成的时间，ISO 8601 格式 | 是 |
   updatedAt | String | 记忆最后更新的时间，ISO 8601 格式 | 是 |
   attributes | Object | 记忆的属性信息 | 否 |
     topic | String | 记忆的主题分类 | 否 |
     subTopic | String | 记忆的子主题分类 | 否 |
