<!--keywords: 设备管理,唤醒词配置,SDK 配置,设备唤醒,音频识别-->

设置或更新指定设备的唤醒词配置，支持自定义唤醒词内容。

## 请求信息

### 请求 URI

```
POST https://rtc-agent.yunxinapi.com/v1/device/updateSdkConfig
```

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](https://doc.yunxin.163.com/ai-hardware/server-apis/DYwNTk5MzM?platform=server#Header)。

### 请求体参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| deviceId | String | 是 | yunxin-D721 | 设备唯一标识符 |
| awakens | String | 是 | 你好网易\n你好云信 | 唤醒词内容，多个唤醒词用换行符分隔 |

### 请求体示例

```JSON
{
    "deviceId": "yunxin-D721",
    "awakens": "你好网易\n你好云信"
}
```

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
| :--- | :--- | :--- | :--- |
| code | Integer | 响应状态码，200 表示成功 | 是 |
| message | String | 响应消息描述 | 是 |
| requestId | String | 请求唯一标识符 | 是 |
| data | Object | 响应数据对象 | 否 |
    awakens | Array | 处理后的唤醒词拼音数组 | 否 |
  success | Boolean | 操作是否成功 | 是 |

### 响应体示例

```JSON
{
    "code": 200,
    "message": "操作成功",
    "requestId": "aifab93440055f4234990966d453c5781a",
    "data": {
        "awakens": [
            "ni hao yun xin",
            "ni hao wang yi"
        ]
    },
    "success": true
}
```

### 错误码

本文仅列举部分业务接口错误码，完整列表请参考 [错误码](https://doc.yunxin.163.com/ai-hardware/server-apis/DY5MDEzOTQ?platform=server)。

| 错误码 | 说明 | 处理建议 |
| :--- | :--- | :--- |
| 400 | 参数错误 | 检查请求参数格式和必填项 |
| 2400 | 设备 SDK 配置更新失败 | 检查设备 ID 是否有效，重试请求 |
