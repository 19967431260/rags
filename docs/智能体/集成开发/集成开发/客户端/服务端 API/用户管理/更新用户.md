## 请求信息

### 请求 URI

```
PUT https://rtc-agent.yunxinapi.com/v1/ai_users/{uid}
```

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](https://doc.yunxin.163.com/ai-hardware/server-apis/DYwNTk5MzM?platform=server#Header)。

### 请求路径参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| :---- | :---- | :---- | :---- | :---- |
| uid | String | 是 | user123 | 用户唯一标识符 |

### 请求体参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| :---- | :---- | :---- | :---- | :---- |
| note | String | 否 | User1 | 用户备注 |
| userProperties | Object | 否 | - | 用户属性信息 |
    name | String | 否 | Kevin | 用户名称 |
    gender | String | 否 | MALE | 用户性别。MALE 或者 FEMALE |
    birthYear | Integer | 否 | 1990 | 出生年份 |
| subAccountUid | String | 否 | sub_001 | 子用户 ID，最大长度 32 字节，只允许字母、数字以及 "_@.-" |

### 请求体示例

```JSON
{
    "note": "User1",
    "userProperties": {
        "gender": "MALE",
        "name": "Kevin",
        "birthYear": 1990
    },
    "subAccountUid": "sub_001"
}
```

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
| :---- | :---- | :---- | :---- |
| code | Integer | 响应状态码 | 是 |
| message | String | 响应描述信息 | 是 |
| requestId | String | 请求唯一标识符 | 是 |
| data | Object | 响应数据 | 否 |
| success | Boolean | 操作是否成功 | 是 |

### 响应体示例

```JSON
{
    "code": 200,
    "message": "操作成功",
    "requestId": "aif8cac4e9f0ea4971aafb15bd4f68752e",
    "data": null,
    "success": true
}
```
