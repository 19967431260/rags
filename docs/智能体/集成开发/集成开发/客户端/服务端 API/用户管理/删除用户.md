<!--keywords: RTC 智能体平台, 用户管理, 删除用户 API, 声纹数据, 对话历史-->

删除在网易云信智能体平台上的一个用户。

:::note notice
- 删除操作不可逆，请确认后再执行
- 删除用户会清空所有相关数据，包括声纹、记忆、对话历史
- 建议在删除前做好数据备份
:::

## 请求信息

<style>
table th:first-of-type {
    width: 20%;
}
</style>

### 请求 URI

```
DELETE https://rtc-agent.yunxinapi.com/v1/ai_users/{uid}
```

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](https://doc.yunxin.163.com/ai-hardware/server-apis/DYwNTk5MzM?platform=server#Header)。

### 请求路径参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| :---- | :--- | :---- | :--- | :--- |
| uid | String | 是 | test_user_1 | 用户 ID，最大长度 32 字节 |

### 请求查询参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| :---- | :--- | :---- | :--- | :--- |
| subAccountUid | String | 否 | sub_user_001 | 子用户 ID，最大长度 32 字节，只允许字母、数字以及 "_@.-" |

### 请求示例

```
DELETE https://rtc-agent.yunxinapi.com/v1/ai_users/v1/ai_users/test_user_1
```

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
| :---- | :--- | :--- | :---- |
| code | Integer | 状态码，200 表示请求成功 | 是 |
| message | String | 提示信息。请求失败时返回错误信息，请求成功时返回 **操作成功** | 是 |
| success | Boolean | 操作是否成功 | 是 |

### 响应体示例

```JSON
{
    "code": 200,
    "message": "操作成功",
    "success": true
}
```
