<!--keywords: RTC,音视频通话,用户状态,状态值查询,智能体-->

查询用户状态值列表，获取指定用户在特定智能体实例下的设置的所有状态信息，例如亲密度，或者您在智能体中配置的其他状态。

:::note note
若您还没为智能体配置过 **状态信息设置**，请参考 [配置智能体](https://doc.yunxin.163.com/emotional-ai/guide/TU3MjE3NjE?platform=client#%E7%AC%AC%E4%B8%89%E6%AD%A5%E5%88%9B%E5%BB%BA%E6%88%96%E5%A4%8D%E5%88%B6%E6%99%BA%E8%83%BD%E4%BD%93) 完成配置。
:::

## 调用频率

接口调用频率限制为 50 次/秒。

## 请求信息

### 请求 URI

```
GET https://rtc-agent.netease.im/v1/user_state/list
```

### 请求头参数

请求 Header 的参数说明，请参考 [调用方式](https://doc.yunxin.163.com/emotional-ai/guide/DYwNTk5MzM?platform=client#%E8%AF%B7%E6%B1%82%E5%A4%B4header)。

### 请求查询参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| ---- | ---- | ---- | ---- | ---- |
| agentId | String | 是 | agent_001 | agent ID，用于标识特定的智能体实例 |
| userId | String | 是 | user_12345 | 用户唯一标识，最大长度 32 字符 |

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
| ---- | ---- | ---- | ---- |
| code | Integer | 响应状态码，200 表示成功 | 是 |
| message | String | 响应信息，请求失败时返回错误信息，请求成功时返回 **操作成功** | 是 |
| requestId | String | 请求的唯一标识 ID | 是 |
| success | Boolean | 请求是否成功 | 是 |
| data | Array | 用户状态列表 | 否 |
    stateId | Integer | 状态 ID | 是 |
    stateName | String | 状态名称 | 是 |
    stateValue | Integer | 状态值 | 是 |
    mtime | String | 状态更新时间，采用 ISO 8601 格式 | 是 |

### 响应体示例

```JSON
{
  "code": 200,
  "message": "操作成功",
  "requestId": "ai1df03d6d205b4f78b986cb4af54543d1",
  "data": [
    {
      "stateId": 2,
      "stateValue": 30,
      "stateName": "亲密度",
      "mtime": "2025-12-11T10:38:13.000+00:00"
    }
  ],
  "success": true
}
```
