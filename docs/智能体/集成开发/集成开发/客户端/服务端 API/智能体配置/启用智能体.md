<!--keywords: nertc, Agent, enableagent, 智能体, API, 服务端接口, 智能体管理-->

调用该接口启用您在网易云信智能体管理平台为应用配置的智能体。更多详情，请参考 [配置智能体](https://doc.yunxin.163.com/ai-hardware/guide/TU3MjE3NjE?platform=client)。

## 请求信息

### 请求 URI

```
POST https://rtc-agent.yunxinapi.com/v1/agent/enable
```

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](https://doc.yunxin.163.com/ai-hardware/server-apis/DYwNTk5MzM?platform=server#Header)。

### 请求体参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| agentId | String | 是 | "agent123" | 网易云信智能体 ID。 |

### 请求体示例

```JSON
{
  "agentId": "agent123"
}
```

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
| :--- | :--- | :--- | :--- |
| code | Integer | 响应码 | 是 |
| msg | String | 响应消息 | 是 |
| data | Object | 响应数据 | 否 |
| requestId | String | 请求的唯一标识 ID | 是 |

### 响应体示例

```JSON
{
  "code": 200,
  "msg": "success",
  "data": null,
  "requestId": "aid0171d85b9bf4667806f9f4f8cd41432"
}
```

### 错误码

本文仅列举部分业务接口错误码，完整列表请参考 [错误码](https://doc.yunxin.163.com/ai-hardware/server-apis/DY5MDEzOTQ?platform=server)。

| 错误码 | 说明 | 处理建议 |
| :--- | :--- | :--- | :--- |
| 407 | 智能体不存在 | 检查 `agentId` ID 是否正确。 |
