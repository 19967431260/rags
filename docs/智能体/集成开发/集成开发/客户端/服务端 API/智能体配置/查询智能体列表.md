## 请求信息

### 请求 URI

```
GET https://rtc-agent.yunxinapi.com/v1/agent
```

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](https://doc.yunxin.163.com/ai-hardware/server-apis/DYwNTk5MzM?platform=server#Header)。

### 请求查询参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| :---- | :---- | :- | :- | :---- |
| page | Integer | 是 | 1 | 第几页，从 1 开始 |
| pageSize | Integer | 是 | 10 | 页大小，取值范围为 1~100，默认为 10 |

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
| :---- | :---- | :---- | :- |
| code | Integer | 响应状态码 | 是 |
| message | String | 响应描述信息 | 是 |
| requestId | String | 请求 ID | 是 |
| data | Object | 响应数据 | 是 |
    page | Integer | 当前页码 | 是 |
    pageSize | Integer | 页大小 | 是 |
    total | Integer | 总数据量 | 是 |
    agents | Array | 智能体列表 | 是 |
      agentId | String | Agent 标识符 | 是 |
      agentName | String | Agent 名称 | 是 |
      llmPromptContent | String | LLM 提示词内容 | 是 |
      [llmConfig](https://doc.yunxin.163.com/ai-hardware/guide/DEzNzA1Mjg?platform=client#llmConfig) | Object | LLM 配置信息，具体请参考创建智能体的说明 | 是 |
      [ttsConfig](https://doc.yunxin.163.com/ai-hardware/guide/DEzNzA1Mjg?platform=client#ttsConfig) | Object | TTS 音色相关配置，具体请参考创建智能体的说明 | 是 |
      [asrConfig](https://doc.yunxin.163.com/ai-hardware/guide/DEzNzA1Mjg?platform=client#asrConfig) | Object | ASR 相关配置，具体请参考创建智能体的说明 | 是 |
      [agentConfigProperties](https://doc.yunxin.163.com/ai-hardware/guide/DEzNzA1Mjg?platform=client#agentConfigProperties) | Object | Pipeline 设置等配置，具体请参考创建智能体的说明 | 是 |
      createdAt | Long | 创建时间，单位为毫秒的时间戳 | 是 |
      updatedAt | Long | 修改时间，单位为毫秒的时间戳 | 是 |
      status | Integer | 状态，0 表示启用，1 表示禁用 | 是 |

### 错误码

本文仅列举部分业务接口错误码，完整列表请参考 [错误码](https://doc.yunxin.163.com/ai-hardware/server-apis/DY5MDEzOTQ?platform=server)。

| 错误码 | 说明 | 处理建议 |
| :--- | :--- | :--- |
| 400 | 请求参数错误 | 检查请求参数是否正确 |
| 401 | 身份认证失败 | 检查认证信息是否正确 |
| 403 | 权限不足 | 检查是否有相应的操作权限 |
| 404 | 资源不存在 | 检查请求的资源是否存在 |
| 500 | 服务器内部错误 | 稍后重试或欢迎 [提交工单](https://app.yunxin.163.com/global/service/ticket/create) 联系网易云信技术支持工程师 |
