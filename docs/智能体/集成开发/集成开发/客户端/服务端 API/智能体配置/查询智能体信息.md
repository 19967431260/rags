根据智能体（Agent）ID 获取 Agent 信息。

    width: 20%;

## 请求信息

### 请求 URI

```
GET https://rtc-agent.yunxinapi.com/v1/agent/{agentId}
```

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](https://doc.yunxin.163.com/ai-hardware/server-apis/DYwNTk5MzM?platform=server#Header)。

### 请求路径参数

 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
 :---- | :---- | :- | :---- | :---- |
 agentId | String | 是 | "agent123" | Agent ID |

## 响应信息

### 响应参数

 参数名称 | 类型 | 说明 | 是否必返回 |
 :---- | :---- | :---- | :---- |
 code | Integer | 状态码 | 是 |
 message | String | 状态描述 | 是 |
 requestId | String | 请求标识符 | 是 |
 data | Object | 响应数据 | 是 |
   agentId | String | Agent 标识符 | 是 |
   agentName | String | Agent 名称 | 是 |
   llmPromptContent | String | LLM 提示词 | 是 |
   [llmConfig](https://doc.yunxin.163.com/ai-hardware/guide/DEzNzA1Mjg?platform=client#llmConfig) | Object | LLM 配置，具体请参考创建智能体的说明 | 是 |
   [ttsConfig](https://doc.yunxin.163.com/ai-hardware/guide/DEzNzA1Mjg?platform=client#ttsConfig) | Object | TTS 音色相关配置，具体请参考创建智能体的说明 | 是 |
   [asrConfig](https://doc.yunxin.163.com/ai-hardware/guide/DEzNzA1Mjg?platform=client#asrConfig) | Object | ASR 相关配置，具体请参考创建智能体的说明 | 是 |
   [agentConfigProperties](https://doc.yunxin.163.com/ai-hardware/guide/DEzNzA1Mjg?platform=client#agentConfigProperties) | Object | AI Pipeline 设置等，具体请参考创建智能体的说明 | 是 |
   status | Long | 0 表示启用，1 表示禁用 | 是 |
   createdAt | Long | 创建时间，单位为毫秒的时间戳 | 是 |
   updatedAt | Long | 修改时间，单位为毫秒的时间戳 | 是 |
 success | Boolean | 操作是否成功 | 是 |
