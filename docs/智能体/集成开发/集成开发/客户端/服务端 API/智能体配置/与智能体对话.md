该接口用于创建和管理与智能体的实时对话，支持工具调用和自定义用户属性。客户端发送用户消息及上下文信息，服务端通过 SSE (Server-Sent Events) 流式返回 AI 的回复内容。

::: note notice
- 接口采用流式响应，需要客户端支持 SSE (Server-Sent Events)。
- 响应格式遵循 OpenAI `chat/completions` API 规范。
- 支持工具调用（Function Calling）功能。
- 会话 ID 需要保持唯一性，建议使用 UUID。
:::

## 功能描述

本接口支持以下功能：

- 支持多轮对话，保持会话上下文。
- 实时流式返回 AI 回复，提升用户体验。
- 支持自定义用户属性和动态 Prompt 变量。
- 支持工具调用，可扩展 AI 能力。
- 支持地理位置相关的智能推荐。

## 调用频率

- 单个应用的调用频率限制为 50000 次/分钟。
- 单个智能体的调用频率限制为 5000 次/分钟。

### 请求 URI

```
POST https://rtc-agent.netease.im/v1/agent/chat/completions
```

请求 URI 中的 `{{endpoint}}` 代表服务地址域名。
::: -->

### 请求头参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| :---- | :---- | :---- | :---- | :---- |
| Content-Type | String | 是 | application/json | 数据格式 |
| Accept | String | 是 | text/event-stream | 接收流式响应 |

其余请求 Header 的参数说明，请参考 [调用方式](https://doc.yunxin.163.com/emotional-ai/guide/DYwNTk5MzM?platform=client#%E8%AF%B7%E6%B1%82%E5%A4%B4header)。

### 请求体参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| :---- | :---- | :---- | :---- | :---- |
| chatSessionId | String | 是 | sess_890123_abc | 聊天会话 ID，唯一标识一个会话，建议使用 UUID |
| userId | String | 是 | user_1001 | 接入方用户 ID，用户标识 |
| agentId | String | 是 | agent_v2_001 | 已创建的智能体 ID |
| deviceId | String | 否 | dev_iphone_14 | 设备 ID，设备标识 |
| messages | Array | 是 | - | 用户对话内容，当前轮次的用户输入 |
    role | String | 是 | user | 角色类型，枚举值：user、assistant 或 tool |
    content | String | 是 | 今天天气怎么样 | 具体对话内容 |
    tool_calls | Array | 否 | - | 工具调用信息，当 role 为 `assistant` 时该参数有效 |
    tool_call_id | String | 否 | call_12345 | 工具调用标识，当 role 为 `tool` 时该参数有效 |
| tools | Array | 否 | - | 工具定义列表，建议限制工具数量，例如，最高不超过 20 个工具 |
    type | String | 是 | function | 工具类型，只支持 function |
    function | Object | 是 | - | 函数定义 |
      description | String | 是 | 获取指定城市的天气信息 | 函数功能描述，供模型理解何时调用 |
      name | String | 是 | getWeather | 函数名称，由 a-z、A-Z、0-9、下划线和连字符组成，最大 64 字符 |
      parameters | Object | 是 | - | 函数参数定义，JSON Schema 格式 |
        type | String | 是 | Object | 参数类型，通常为 String、Number、Boolean、Object、Array |
        properties | Object | 是 | - | 参数属性定义 |
        required | Array | 否 | ["city"] | 必填参数名列表 |
| userProperties | Object | 否 | - | 用户属性对象，用于传递用户自定义信息 |
    ip | String | 否 | 123.123.123.123 | 用户 IP 地址，支持 IPv4 或 IPv6 |
    promptKeyWord | Object | 否 | - | 动态 Prompt 变量，Key-Value 键值对，最多 30 个 Key |

### 请求体示例

```JSON
{
  "chatSessionId": "sess_890123_abc",
  "userId": "user_1001",
  "agentId": "agent_v2_001",
  "deviceId": "dev_iphone_14",
  "messages": [
    {
      "role": "user",
      "content": "音量调整到 80%"
    },
    {
        "role": "tool",
        "tool_call_id": "toolcall-1",
        "content": "volume adjusted to 80%"
    }
  ],
  "tools": [
    {
      "type": "function",
      "function": {
        "name": "getWeather",
        "description": "获取指定城市的天气信息",
        "parameters": {
          "type": "object",
          "properties": {
            "city": {
              "type": "string",
              "description": "城市名称"
            }
          },
          "required": ["city"]
        }
      }
    }
  ],
  "userProperties": {
    "ip": "123.123.123.123",
    "promptKeyWord": {
      "userName": "张三",
      "location": "上海",
      "preference": "spicy",
      "budget": "high"
    }
  }
}
```

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
| :---- | :---- | :---- | :---- |
| id | String | 消息唯一标识符 | 是 |
| object | String | 对象类型，固定为 chat.completion.chunk | 是 |
| created | Long | 创建时间戳（秒） | 是 |
| model | String | 使用的模型名称 | 是 |
| choices | Array | 选择列表，通常只有一个元素 | 是 |
    index | Integer | 选择索引，通常为 0 | 是 |
    delta | Object | 增量内容对象 | 是 |
      role | String | 角色类型，首包返回 assistant | 否 |
      content | String | 当前片段的文本内容 | 否 |
      tool_calls | Array | 工具调用信息 | 否 |
        id | String | 工具调用 ID | 否 |
        type | String | 工具类型 | 否 |
        function | Object | 函数信息 | 否 |
          name | String | 函数名称 | 否 |
          arguments | String | 函数参数 | 否 |
    finish_reason | String | 结束原因：stop、tool_calls 或 null | 否 |
| usage | Object | 使用统计信息（仅最后一包） | 否 |
| system_fingerprint | String | 系统指纹 | 否 |
