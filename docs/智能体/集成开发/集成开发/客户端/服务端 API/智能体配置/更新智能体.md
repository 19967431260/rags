更新网易云信智能体平台上创建的一个智能体 Agent 的信息。

## 请求信息

### 请求 URI

```
PATCH https://rtc-agent.yunxinapi.com/v1/agent/{agentId}
```

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](https://doc.yunxin.163.com/ai-hardware/server-apis/DYwNTk5MzM?platform=server#Header)。

### 请求路径参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| agentId | String | 是 | "99b6176cf59a4d03b4ef54c2d96a7155" | Agent 的唯一标识 ID |

### 请求体参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| agentName | String | 否 | "助手小易" | Agent 的显示名称 |
| llmPromptContent | String | 否 | "作为网易云信的智能语音助手..." | LLM 模型的提示词内容，用于指导 AI 回复风格和内容 |
| [llmConfig](https://doc.yunxin.163.com/ai-hardware/guide/DEzNzA1Mjg?platform=client#llmConfig) | Object | 否 | {} | LLM 模型相关配置 |
| [ttsConfig](https://doc.yunxin.163.com/ai-hardware/guide/DEzNzA1Mjg?platform=client#ttsConfig) | Object | 否 | {} | TTS（文本转语音）音色相关配置 |
| [asrConfig](https://doc.yunxin.163.com/ai-hardware/guide/DEzNzA1Mjg?platform=client#asrConfig) | Object | 否 | {} | ASR（语音识别）相关配置 |
| [agentConfigProperties](https://doc.yunxin.163.com/ai-hardware/guide/DEzNzA1Mjg?platform=client#agentConfigProperties) | Object | 否 | {} | Agent 高级属性配置，例如 Pipeline 设置等 |

### 请求体示例

```JSON
{
    "agentId": "99b6176cf59a4d03b4ef54c2d96a7155",
    "agentName": "助手小易",
    "llmPromptContent": "作为网易云信的智能语音助手，我遵循以下原则：我的名字叫小易，我的回答语气可爱乖巧，和真正的人类对话一样，内容清晰全面，但不超过 50 个词, 不回复表情和表情符号。"

}
```

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
| :--- | :--- | :--- | :--- |
| code | Integer | 响应状态码，200 表示成功 | 是 |
| message | String | 响应消息描述 | 是 |
| requestId | String | 请求的唯一标识 ID | 是 |
| data | Object | 响应数据，更新操作成功时为 null | 是 |
| success | Boolean | 操作是否成功 | 是 |

### 响应体示例

```JSON
{
    "code": 200,
    "message": "操作成功",
    "requestId": "aid0171d85b9bf4667806f9f4f8cd41432",
    "data": null,
    "success": true
}
```

### 错误码

本文仅列举部分业务接口错误码，完整列表请参考 [错误码](https://doc.yunxin.163.com/ai-hardware/server-apis/DY5MDEzOTQ?platform=server)。

| 错误码 | 说明 | 处理建议 |
| :--- | :--- | :--- |
| 407 | Agent 不存在 | 请检查 `agentId` 是否正确，或确认 Agent 是否已被删除 |
