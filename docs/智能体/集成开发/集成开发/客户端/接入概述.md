网易云信旨在打造低延迟、多样化的实时音视频 + AI 极致体验，配合优雅打断以及声纹锁定，广泛适用于智能助手、智能硬件、智能客服、虚拟陪伴应用中。基于优质、稳定的音视频流传输服务 [音视频通话](https://doc.yunxin.163.com/nertc/concept/zY0MjQ5NjE?platform=client)，帮助开发者通过第三方或自研的音视频编解码模块，为智能体快速实现人与人、人与物、物与物的实时互动连接。

## 适用场景

网易云信依托强大的实时传输网络 WE-CAN，结合全球全网节点、智能动态路由以及先进的端侧弱网对抗算法，提供高连通、低延时、高稳定的音视频码流传输云服务，有效减少因网络问题（如延时、丢包）对音视频体验造成的影响。

此外，网易云信适配了多种主流 IoT 平台，助力开发者在任意设备与场景中轻松开启实时互动能力。

网易云信在接入智能硬件时提供了嵌入式 SDK，SDK 本身不包含音视频的采集和编解码模块。它主要适用于仅需媒体流传输的 IoT 场景，例如实时监控、电话手表、AI 陪伴等。通过将对话式 AI 集成到智能硬件中，最终实现 **让智能硬件开口说话** 的目标。

## 应用配置

在开始接入网易云信智能体平台之前，请确保您已经完成了以下准备工作：

1. **创建应用并获取 App Key**：您需要在 [网易云信控制台](https://app.yunxin.163.com/global/home) 创建一个应用，并获取唯一的密钥（App Key）作为应用的身份标识。详细步骤请参考 [创建应用并获取 AppKey](https://doc.yunxin.163.com/console/concept/TIzMDE4NTA?platform=console)。

2. **开通服务**：为您的应用开通 **音视频通话 2.0** 服务。详细步骤请参考 [开通或试用产品](https://doc.yunxin.163.com/console/concept/zc3NDYzNzc?platform=console)。

    <img alt="image-netease" src="https://yx-web-nosdn.netease.im/common/a3c5123c2ea121670e445bd3b7f7e438/image.png" style="width:60%;border: 1px solid #BFBFBF;border-radius:0px;">

3. **开通对话式 AI**：在 **音视频通话 2.0** 产品选项卡中的功能配置区域，开通 **对话式 AI** 功能。

    <img alt="image-netease" src="https://yx-web-nosdn.netease.im/common/892d56ef8b9fd25f68e35b6104c66950/image.png" style="width:60%;border: 1px solid #BFBFBF;border-radius:0px;">

3. **配置智能体**：在 [网易云信智能体管理平台](https://rtc-agent-console.netease.im/#/home) 为应用配置智能体，以及为智能硬件设备添加绑定智能体。详情请参考 [配置智能体](https://doc.yunxin.163.com/ai-hardware/guide/TU3MjE3NjE?platform=client)。

4. **配置鉴权模式**：在测试开发阶段，建议在 [网易云信控制台](https://app.yunxin.163.com/global/home) 中为您的应用开通 **调试模式**，此时加入房间无需进行 Token 校验。应用正式上线前，请务必切换为 **安全模式**，以保障业务安全。详细说明请参考 [Token 鉴权](https://doc.yunxin.163.com/nertc/server-apis/TcxNDAxMTI?platform=server)。

## 获取 SDK

如果您需要通过嵌入式 SDK 接入网易云信智能体，请联系您的网易云信客户经理或者 [提交工单](https://app.yunxin.163.com/global/service/ticket/create) 联系网易云信技术支持工程师，获取最新的嵌入式 NERTC SDK 包。
