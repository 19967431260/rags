本文介绍如何在网易云信智能体管理平台为应用配置智能体，以及为智能硬件设备添加绑定智能体。

## 平台介绍

网易云信智能体管理平台是一个功能强大、界面友好的 Web 端管理工具，专为开发者和企业用户设计。通过这个平台，您可以：

- **便捷管理智能体**：直接在 Web 端创建、配置和管理您的 AI 智能体。
- **统一用户管理**：集中管理所有用户信息和权限设置。
- **智能记忆管理**：查看和管理智能体的长期记忆，包括块状记忆和结构化记忆。
- **对话历史追踪**：完整查看用户与智能体的对话消息历史。
- **设备绑定管理**：特别适合嵌入式设备场景的智能体应用。

无论您是初次接触 AI 智能体，还是经验丰富的开发者，网易云信的平台都能为您提供直观、高效的管理体验。

## 前提条件

在开始接入网易云信智能体管理平台之前，请确保您已经完成了以下准备工作：

- 如果您还没有注册 [网易云信控制台](https://app.yunxin.163.com/global/home) 账号，请先注册企业账号。详情请参考 [注册账号并登录控制台](https://doc.yunxin.163.com/console/concept/DU0ODE4NTA?platform=console)。
- 您需要在网易云信控制台创建您的应用，并获取唯一的密钥（App Key）作为应用的身份标识。详细步骤请参考 [创建应用并获取 AppKey](https://doc.yunxin.163.com/console/concept/TIzMDE4NTA?platform=console)。
- 请确保已为您的应用开通 **音视频通话 2.0** 服务。详细步骤请参考 [开通或试用产品](https://doc.yunxin.163.com/console/concept/zc3NDYzNzc?platform=console)。

## 管理智能体

### 第一步：登录管理平台

1. 打开 [网易云信智能体管理平台](https://rtc-agent-console.netease.im/#/)。
2. 使用您在 [网易云信控制台](https://app.yunxin.163.com/global/home) 注册的企业账号进行登录。
3. 登录成功后，您将进入智能体管理主界面。

### 第二步：选择 Appkey

1. 在页面上方找到 Appkey 选择区域。
2. 从下拉菜单中选择您要管理的 **Appkey**。

    <img alt="选择 Appkey 界面" src="https://yx-web-nosdn.netease.im/common/18072b7be0b4620d1ed8ea12e399480e/image.png" style="width:60%;border: 1px solid #BFBFBF;border-radius:0px;">
3. 选择完成后，系统将自动显示该 Appkey 下的所有智能体，包括：
   - 您创建的自定义智能体。
   - 系统预置智能体。

    :::note note
    - 必须先选择 Appkey 才能进行后续操作。
    - 如果您没有可用的 Appkey，请前往 **网易云信控制台** 创建应用。
    :::

### 第三步：创建或复制智能体

1. 单击 Appkey 右侧的 **添加智能体** 按钮。

    <img alt="添加智能体.png" src="https://yx-web-nosdn.netease.im/common/a516af5e78761be50509d6ffb2a00f9a/image.png" style="width:60%;border: 1px solid #BFBFBF;border-radius:0px;">

2. 在创建智能体表单中填写以下信息：
    - **智能体名称**：为您的智能体起一个易于识别的名称。
    - **角色设定**：详细描述智能体的角色定位和功能特点。
    - **其他配置**：可使用系统默认设置或根据需要自定义，主要包含以下模块：
        - **基础模型配置（LLM/TTS/ASR）**：设置大模型参数、选择TTS音色、配置语音识别灵敏度、断句策略以及语义 VAD（语音活动检测）。
        - **Pipeline 配置**：定义对话流控逻辑，包括打断策略、自定义开场白、休眠设置；同时可开启回声消除、AI 降噪、声纹识别及回调通知等音频处理功能。
        - **工具与扩展（Tools & MCP）**：配置端侧注册工具以实现 Function Call（函数调用），或配置 MCP Server 接入以扩展服务能力。
        - **Agent 模式配置**：设置特定场景下的子交互模式（如“讲故事”、“互动游戏”），允许智能体在特定情境下跳出主角色设定进行互动。
        - **知识库配置**：关联上传的文档，赋予智能体基于特定知识库回答问题的能力。
        - **情绪与状态设置**：
            - **情绪交互**：开启情绪识别、ASR 语气检测及 TTS 情绪控制，支持在端侧实时显示情绪状态。
            - **状态信息（亲密度）**：配置亲密度等动态变量。亲密度用于衡量智能体对用户的信任度与情感安全感，系统将根据此变量动态调整智能体自我暴露的程度（即“真实度”）。

            <img alt="创建或复制智能体.png" src="https://yx-web-nosdn.netease.im/common/208f163b98f3b1f3c8c522d4f2651bba/image.png" style="width:60%;border: 1px solid #BFBFBF;border-radius:0px;">

3. 完成填写后，单击 **确认创建** 按钮。
4. 系统将自动保存并显示新创建的智能体。

### 第四步：修改智能体配置

1. 在智能体列表中找到需要修改的智能体。
2. 单击智能体卡片中的 **配置角色** 按钮。

    <img alt="修改智能体配置.png" src="https://yx-web-nosdn.netease.im/common/b4e42611193e66148a835a05d497add7/image.png" style="width:30%;border: 1px solid #BFBFBF;border-radius:0px;">

3. 在弹出的编辑界面中修改智能体配置。
4. 完成修改后单击 **保存** 按钮。

    :::note note
    系统内置智能体不支持编辑功能，只有您创建的自定义智能体可以进行修改。
    :::

## 用户管理

1. 单击页面顶部导航栏中的 **用户管理** 选项。

    <img alt="顶部导航栏入口.png" src="https://yx-web-nosdn.netease.im/common/9b69f07d1347cb7868763fe36c5147e6/image.png" style="width:65%;border: 1px solid #BFBFBF;border-radius:0px;">
2. 进入全局用户管理界面，单击 **添加用户** 弹出添加用户弹窗，在弹窗中根据页面引导完成用户信息设置，然后单击 **确定**。

    <img alt="添加用户.png" src="https://yx-web-nosdn.netease.im/common/3078a10a7ceb625376be8549566975ae/image.png" style="width:65%;border: 1px solid #BFBFBF;border-radius:0px;">

    添加了用户后，您可以在用户列表管理用户，包括用户设备管理、为用户录入声纹、为用户录入联系方式、编辑用户、添加子用户、删除用户。

    <img alt="用户管理.png" src="https://yx-web-nosdn.netease.im/common/9579ba87addec13b8395d522e55a74d8/image.png" style="width:65%;border: 1px solid #BFBFBF;border-radius:0px;">

## 测试智能体

在创建了智能体和用户后，您可以与智能体对话测试效果。

1. 在智能体卡片上，单击 **开始对话**。

    <img alt="image.png" src="https://yx-web-nosdn.netease.im/common/6a37e8a727b4160cd88fb6225942d392/image.png" style="width:30%;border: 1px solid #BFBFBF;border-radius:0px;">

2. 在对话框中，选择您创建的用户。

    <img alt="image.png" src="https://yx-web-nosdn.netease.im/common/87e8cd5fe398acc2149b39777f63d171/image.png" style="width:60%;border: 1px solid #BFBFBF;border-radius:0px;">

3. 在对话框中，输入您的消息，然后单击发送。

至此，一个基本的智能体的创建与对话测试已经介绍完毕。因默认配置参数无法成为所有的使用场景的最佳实践，网易云信建议您结合自身业务场景进行参数调整，不断精进智能体的问答体验。

## 管理设备

设备管理功能专为通过嵌入式设备使用网易云信实时对话智能体的用户设计，帮助您轻松管理设备与智能体的绑定关系。在网易云信智能体开发体系中，一个用户（用户账号）可以对应多个设备（设备终端实体），每个设备必须唯一绑定一个用户，与一个智能体。

### 前提条件

在开始管理设备之前，请确保您已经至少 [添加了一个用户](#addUser)。

### 进入设备管理

平台提供两个设备管理入口，您可以选择任意一个：

**方式一**：通过智能体卡片

1. 在智能体列表中找到目标智能体。
2. 单击智能体卡片中的 **设备管理** 按钮。

    <img alt="设备管理入口.png" src="https://yx-web-nosdn.netease.im/common/5a935fc63ba90484d173b6d58e4b3c3e/image.png" style="width:30%;border: 1px solid #BFBFBF;border-radius:0px;">

**方式二**：通过顶部导航栏

1. 单击页面顶部导航栏中的 **用户管理** 选项。

    <img alt="顶部导航栏入口.png" src="https://yx-web-nosdn.netease.im/common/9b69f07d1347cb7868763fe36c5147e6/image.png" style="width:65%;border: 1px solid #BFBFBF;border-radius:0px;">
2. 进入全局用户管理界面，通过 **设备管理** 查看用户绑定的设备。

### 添加设备绑定

以下步骤假设您通过顶部导航栏的 **用户管理** 选项绑定设备：

1. 单击用户列表中 **设备管理** 按钮，出现 **添加设备** 弹窗。

    <img alt="添加设备绑定.png" src="https://yx-web-nosdn.netease.im/common/ec6829826a37d69cc0e43c340b23df8e/image.png" style="width:60%;border: 1px solid #BFBFBF;border-radius:0px;">
2. 在弹出的表单中填写以下信息：

    - **设备 ID**：输入要绑定的设备唯一标识符。
    - **用户 ID**：输入使用该设备的用户标识符。
    - **智能体 ID**：输入要绑定的智能体。

        :::note notice
        如果您是从智能体卡片进入设备管理，系统会自动填入对应的智能体 ID，您只需输入设备 ID 和用户 ID 即可。
        :::

3. 单击 **确认** 按钮完成设备绑定。

    <img alt="添加设备绑定.png" src="https://yx-web-nosdn.netease.im/common/b34df51b9198bfb66e45a4f289289132/image.png" style="width:100%;border: 1px solid #BFBFBF;border-radius:0px;">

4. 绑定设备后，您需要通过 [激活授权码 License](https://doc.yunxin.163.com/nertc/server-apis/zU3NDcxMTc?platform=server) 接口，为设备下发使用智能体的授权凭证。

### 硬件开发

在为设备绑定用户后，即可按照嵌入式 SDK 接入文档进行集成开发，让硬件开口说话。所需文档如下：

- [基础实现流程](https://doc.yunxin.163.com/ai-hardware/guide/jUyMDgyNTQ?platform=client)：在网易云信嵌入式 SDK（NERTC SDK）中配置智能体，完成 AI 对话功能的核心实现步骤。
- [开启实时互动](https://doc.yunxin.163.com/nertc/server-apis/jQzOTE2NTc?platform=server)：您可以在实时互动接口中直接调用已经配置的智能体。
- [激活授权码 License](https://doc.yunxin.163.com/nertc/server-apis/zU3NDcxMTc?platform=server)：绑定设备后，您可以通过激活授权码接口，为设备下发使用智能体的授权凭证。

### 休眠与唤醒

典型的智能硬件会有休眠与唤醒机制。您可在配置智能体时，在 **Pipeline 设置** 中设置休眠参数。

1. 在智能体列表中找到需要修改的智能体。
2. 单击智能体卡片中的 **配置角色** 按钮。
3. 在 **Pipeline 设置** 中设置休眠参数，然后单击 **保存设置**。如下面的配置示例为若 30 秒内自然用户无主动发言，则进行休眠模式，同时会抛出休眠播报提示词。

    <img alt="image.png" src="https://yx-web-nosdn.netease.im/common/ea249d040dea67ed649ccc87980356fc/image.png" style="width:60%;border: 1px solid #BFBFBF;border-radius:0px;">

对话进入休眠模式后，支持再度唤醒。

1. 在智能体列表中找到需要修改的智能体。
2. 单击智能体卡片中的 **设备管理** 按钮。
3. 在 **设备管理** 中找到对应的设备，选择 **更多操作** > **唤醒词配置** 中进行配置。

    <img alt="image.png" src="https://yx-web-nosdn.netease.im/common/a47ea070f13e80f33e342dc5176aed43/image.png" style="width:100%;border: 1px solid #BFBFBF;border-radius:0px;">

5. 在 **唤醒词编辑** 中输入唤醒词。如下面的配置示例为用户通过发出 **小智小智** 来唤醒对话。

    <img alt="image.png" src="https://yx-web-nosdn.netease.im/common/c5b210df92cbb33835e0a12b3135c826/image.png" style="width:45%;border: 1px solid #BFBFBF;border-radius:0px;">

### 解绑设备

1. 在设备管弹窗中，找到需要解绑的设备。
2. 单击对应设备行的 **更多操作** > **解绑设备** 按钮。

    <img alt="解绑设备.png" src="https://yx-web-nosdn.netease.im/common/539601e37c1ad4328e74e5a38b149cea/image.png" style="width:100%;border: 1px solid #BFBFBF;border-radius:0px;">

    <!-- 3. 在确认弹窗中单击 **确认解绑**。 -->
4. 系统将立即解除设备与智能体的绑定关系。

    :::note important
    - 在同一个 Appkey 下，一个设备 ID 只能绑定一个智能体。
    - 解绑设备后，该设备可以重新绑定到其他智能体。
    - 解绑操作不可撤销，请谨慎操作。但是您可以重新绑定。
    :::
