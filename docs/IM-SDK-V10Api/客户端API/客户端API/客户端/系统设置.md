## 支持平台

Android | iOS | macOS/Windows | Web/uni-app/小程序 | Node.js/Electron | HarmonyOS

✔️️ | ✔️️ | ✔️️ | ✔️️ | ✔️️ | ✔️️ |

## 接口类

`V2NIMSettingService` 类提供系统设置相关接口，包括设置单聊/群聊消息免打扰模式、获取单聊/群聊消息免打扰模式、获取单聊消息免打扰列表、获取会话消息的免打扰状态等接口。
