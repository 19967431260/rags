网易云信即时通讯 SDK（NetEase IM SDK，以下简称 NIM SDK）支持云端会话管理功能，包括创建、更新、删除云端会话等基础操作，以及置顶会话等进阶操作。

如何实现云端会话相关功能请参考 [云端会话管理](https://doc.yunxin.163.com/messaging2/client-apis/zExMDk4ODU?platform=client)。本文介绍 NIM SDK 云端会话相关 API。

## 支持平台

本文内容适用的开发平台或框架如下表所示：

Android | iOS | macOS/Windows | Web/uni-app/小程序 | Node.js/Electron | HarmonyOS

✔️️ | ✔️️ | ✔️️ | ✔️️ | ✔️️ | ✔️️ |

## <span id="on">on("EventName")</span>

**接口描述**

注册会话相关监听。

注册成功后，当事件发生时，SDK 会触发相关回调通知。

::: note note
- 建议在初始化后调用该方法。
- 全局只需注册一次。
- 该接口为同步。
:::

**参数说明**

:::::: div linked-codes
::: code Web/uni-app/小程序
```TypeScript
export interface NIMEBaseServiceInterface<I extends NIMEBaseListener> {
  /**
   * 继承自 EventEmitter3 的监听事件方法。
   */
  on<T extends keyof I>(eventName: T, fn: (...args: I[T]) => void): void;
  /**
   * 继承自 EventEmitter3 的监听事件方法。
   */
  once<T extends keyof I>(eventName: T, fn: (...args: I[T]) => void): void;
}
```
| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| `eventName` | T | 是 | 事件名称：<ul><li>`onSyncStarted`：数据同步开始回调。</li><li>`onSyncFinished`：数据同步结束回调。</li><li>`onSyncFailed`：数据同步失败回调，返回 [相关错误码](https://doc.yunxin.163.com/messaging2/client-apis/DUxNjU3MzU?platform=client)。</li><li>`onConversationCreated`：会话创建回调，返回创建的会话对象。</li><li>`onConversationDeleted`：会话删除回调，返回被删除的会话 ID 列表。</li><li>`onConversationChanged`：会话变更回调，返回变更后的会话列表。</li><li>`onTotalUnreadCountChanged`：会话消息总未读数变更，返回变更后的总未读数。</li><li>`onUnreadCountChangedByFilter`：指定过滤条件的会话消息未读数变更，根据过滤条件返回变更后的未读数。</li><li>`onConversationReadTimeUpdated`：同一账号多端登录后的会话已读时间戳标记的时间变更，返回同步标记的会话 ID 和标记的时间戳。</li></ul>
| `fn` | function | 是 | 监听事件的回调函数。当事件触发时，会调用该函数进行处理。
:::

## <span id="off">off("EventName")</span>

**接口描述**

::: note note

:::

**参数说明**

:::::: div linked-codes
::: code Web/uni-app/小程序
```TypeScript
export interface NIMEBaseServiceInterface<I extends NIMEBaseListener> {
  /**
   * 继承自 EventEmitter3 的取消监听方法。
   */
  off<T extends keyof I>(eventName: T, fn: (...args: I[T]) => void): void;
  /**
   * 继承自 EventEmitter3 的移除事件方法。
   */
  removeAllListeners<T extends keyof I>(eventName?: T): void;
}
```
| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| `eventName` | T | 是 | 事件名称：<ul><li>`onSyncStarted`：数据同步开始回调。</li><li>`onSyncFinished`：数据同步结束回调。如果数据同步已开始，建议在数据同步完成后再进行会话相关操作。</li><li>`onSyncFailed`：数据同步失败回调，返回相关错误码。</li><li>`onConversationCreated`：会话创建回调，返回创建的会话对象。当创建会话成功时会触发该回调。</li><li>`onConversationDeleted`：会话删除回调，返回被删除的会话 ID 列表。当删除会话成功时会触发该回调。</li><li>`onConversationChanged`：会话变更回调，返回变更后的会话列表。当更新会话成功时会触发该回调。</li><li>`onTotalUnreadCountChanged`：会话消息总未读数变更，返回变更后的总未读数。</li><li>`onUnreadCountChangedByFilter`：指定过滤条件的会话消息未读数变更，根据过滤条件返回变更后的未读数。</li></ul>
| `fn` | function | 否 | 监听事件的回调函数。当事件触发时，会调用该函数进行处理。

## <span id="createConversation">createConversation</span>
