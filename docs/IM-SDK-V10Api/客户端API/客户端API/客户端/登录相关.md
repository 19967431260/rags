完成 NIM SDK 的初始化后，用户需要登录 IM，登录成功后才能正常调用消息和会话等其他接口。如何实现 IM 登录请参考开发文档 [登录及登出 IM](https://doc.yunxin.163.com/messaging2/client-apis/Dk1MTY4MzA?platform=client)。

本文介绍网易云信即时通讯 SDK（NetEase IM SDK，简称 NIM SDK）登录相关 API。

## 接口类

`V2NIMLoginService` 类提供登录、登出、踢出其他设备端、注册登录连接状态监听器等接口。

## <span id="addLoginListener">addLoginListener</span>

**接口描述**

注册登录相关监听器，包括登录状态变化、登录失败、登录终端被踢、登录终端信息变更。

注册成功后，当以上事件发生时，SDK 会触发相关回调通知。

::: note note
- 建议在登录前调用该方法。

- 全局只需注册一次。

- 该方法为同步。
:::

**参数说明**

## <span id="login">login</span>

**接口描述**

登录 IM。调用该接口后，SDK 会自动连接服务器，传递用户信息，返回登录结果。

网易云信 IM 支持多种登录鉴权方式，具体请参考 [登录及登出 IM](https://doc.yunxin.163.com/messaging2/client-apis/Dk1MTY4MzA?platform=client) 开发指南文档。

::: note note
调用该接口成功后，建议将 `accountId` 和 `token` 保存到本地，方便下次应用启动时实现自动登录。
:::

**适用场景**

- 在新设备上首次登录 IM
- 同时在线的其他设备踢下线后再次登录 IM
- 切换 IM 账号后再次登录 IM
- 注销登录后再次登录 IM
