网易云信即时通讯 SDK（NetEase IM SDK，以下简称 NIM SDK）提供存储服务，支持添加、查询自定义存储场景和文件上传。

本文介绍存储服务相关 API。

## 接口类

`StorageService` 类提供存储服务相关接口。

## <span id="listen">listen</span>

**接口描述**

注册存储服务相关监听。

注册成功后，当事件发生时，SDK 会返回对应的回调。

::: note note
- 建议在初始化后调用该方法。
- 全局只需注册一次。
- 该方法为同步。
:::

**回调事件**

- `onFileDownloadProgress`：文件下载进度回调，返回文件的下载进度。
- `onFileUploadProgress`：文件上传进度回调，返回文件的上传进度。
- `onMessageAttachmentDownloadProgress`：消息附件下载进度回调，返回消息附件的下载进度。

**示例代码**

```dart
subsriptions.add(NimCore.instance.storageService.onFileDownloadProgress.listen((e) {
    // do something
  }));

  subsriptions.add(NimCore.instance.storageService.onFileUploadProgress.listen((e) {
    // do something
  }));

  subsriptions.add(NimCore.instance.storageService.onMessageAttachmentDownloadProgress.listen((e) {
    // do something
  }));
```

## <span id="downloadAttachment">downloadAttachment</span>

**接口描述**

下载消息附件。

若附件已经下载成功，SDK 直接触发 `onMessageAttachmentDownloadProgress` 回调，返回下载路径（path）。

下载路径规则如下：

- 若用户指定下载路径（`downloadParam.saveAs`），则始终以用户指定的为准；
- 若用户 **未指定** 下载路径，但附件（`attachment`）中有路径（path）信息，则以 `attachment.path` 为准；
- 若用户 **未指定** 下载路径，且附件（`attachment`）为空，则使用用户缓存目录。

**参数说明**

```dart
Future<NIMResult<String>> downloadAttachment(
    NIMDownloadMessageAttachmentParams downloadParam) {
  return _platform.downloadAttachment(downloadParam);
}
```

| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| `downloadParam` | [NIMDownloadMessageAttachmentParams](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#NIMDownloadMessageAttachmentParams) | 是 | 下载附件的配置参数。 |

**示例代码**

```dart
await NimCore.instance.storageService.downloadAttachment(downloadParam);
```
  
**返回值**

NIMResult\<String>：附件下载路径

**相关回调**

`onMessageAttachmentDownloadProgress`：消息附件下载进度回调
