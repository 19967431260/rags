网易云信即时通讯 SDK（NetEase IM SDK，以下简称 NIM SDK）支持多种消息类型，助您快速实现多样化的消息业务场景。在进行消息收发等操作前，您需要先构建消息。

本文中消息均指会话内消息，适用于单聊、群组、超大群类型会话。更多消息相关功能请参考开发文档 [消息概述](https://doc.yunxin.163.com/messaging2/client-apis/Dk0MDg4Njk?platform=client)。

## API 概览

API | 说明 | 起始版本
---- | ---- | ----
[createTextMessage](#createTextMessage) | 创建一条文本消息 | v10.3.0
[createImageMessage](#createImageMessage) | 创建一条图片消息 | v10.3.0
[createAudioMessage](#createAudioMessage) | 创建一条语音消息 | v10.3.0
[createVideoMessage](#createVideoMessage) | 创建一条视频消息 | v10.3.0
[createFileMessage](#createFileMessage) | 创建一条文件消息 | v10.3.0
[createLocationMessage](#createLocationMessage) | 创建一条地理位置消息 | v10.3.0
[createCustomMessage](#createCustomMessage) | 创建一条自定义消息 | v10.3.0
[createForwardMessage](#createForwardMessage) | 创建一条转发消息 | v10.3.0
[createTipsMessage](#createTipsMessage) | 创建一条提示消息 | v10.3.0
[createCallMessage](#createCallMessage) | 创建一条话单消息 | v10.3.0

## 接口类

`MessageCreator` 类提供消息构建接口，支持构建多种类型的消息。

## <span id="createTextMessage">createTextMessage</span>

**接口描述**

创建一条文本消息。

::: note note
- 在登录后、发送消息前调用该方法。
- 该方法为同步。
:::

**参数说明**

```dart
static Future<NIMResult<NIMMessage>> createTextMessage(String text) {
  return _platform.createTextMessage(text);
}
```

| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| `text` | String | 是 | 文本消息内容，长度上限为 5000 字符。 |

**示例代码**

```dart
await MessageCreator.createTextMessage(text);
```

**返回值**

已创建的消息对象 [`NIMMessage`](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#NIMMessage)

## <span id="createImageMessage">createImageMessage</span>

**接口描述**

创建一条图片消息。

::: note note
- 在登录后、发送消息前调用该方法。
- 该方法为同步。
:::

**参数说明**

```dart
static Future<NIMResult<NIMMessage>> createImageMessage(
  String imagePath,
  String? name,
  String? sceneName,
  int width,
  int height, {
  html.File? imageObj,
}) {
  return _platform.createImageMessage(
      imagePath, name, sceneName, width, height,
      imageObj: imageObj);
}
```

| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| `imagePath` | String | 是 | 图片文件地址。 |
| `name` | String | 否 | 图片文件显示名称，可不同于文件名。 |
| `sceneName` | String | 否| NOS 文件存储场景名。若使用自定义的存储场景，需要先调用 [`addCustomStorageScene`](https://doc.yunxin.163.com/messaging2/client-apis/TUwMzk0MDg?platform=client#addCustomStorageScene) 添加自定义存储场景。 |
| `width` | int | 否 | 图片宽度，单位为像素。 |
| `height` | int | 否 | 图片高度，单位为像素。 |
| `imageObj` | html.File | 否 | 图片对象。|

**示例代码**

```dart
await MessageCreator.createImageMessage(imagePath, name, sceneName, width, height);
```

**返回值**

已创建的消息对象 [`NIMMessage`](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#NIMMessage)

## <span id="createForwardMessage">createForwardMessage</span>

**接口描述**

创建一条转发消息。

::: note note
- 转发的消息类型 **不能为** `notification`、`robot`、`tip`，`avChat`。
- 转发的消息消息必须为发送成功的消息，消息状态必须为 `NIMMessagesendingState.succeeded`，消息内容与原消息相同。
- 在登录后、发送消息前调用该方法。
- 该方法为同步。
:::

**参数说明**

```dart
static Future<NIMResult<NIMMessage?>> createForwardMessage(
    NIMMessage message) {
  return _platform.createForwardMessage(message);
}
```

| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| `message` | [`NIMMessage`](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#NIMMessage) | 是 | 待转发的消息体。 |

**示例代码**

```dart
await MessageCreator.createForwardMessage(message);
```

**返回值**

已创建的消息对象 [`NIMMessage`](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#NIMMessage)

## <span id="createCallMessage">createCallMessage</span>

**接口描述**

创建一条话单消息。

::: note note
- 在登录后、发送消息前调用该方法。
- 该方法为同步。
:::

**参数说明**

```dart
static Future<NIMResult<NIMMessage>> createCallMessage(
    int type,
    String channelId,
    int status,
    List<NIMMessageCallDuration>? durations,
    String? text) {
  return _platform.createCallMessage(
      type, channelId, status, durations, text);
}
```

| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| `type` | int | 是 | 话单类型，业务侧可自定义，网易云信不校验内容。建议设置为：<li>1：音频<li>2：视频 |
| `channelId` | String | 是 | 话单频道 ID，网易云信不校验内容。 |
| `status` | int | 是 | 通话状态，业务侧可自定义状态，网易云信不校验内容。建议设置为：<li>1：通话完成<li>2：通话取消<li>3：通话拒绝<li>4：超时未接听<li>5：对方忙 |
| `durations` | List\<[NIMMessageCallDuration](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#NIMMessageCallDuration)> | 否 | 通话成员时长列表，网易云信不校验内容。 |
| `text` | String | 否 | 话单描述。 |

**示例代码**

```dart
await MessageCreator.createCallMessage(type, channelId, status, durations, text);
```

**返回值**

已创建的消息对象 [`NIMMessage`](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#NIMMessage)
