网易云信即时通讯 SDK（NetEase IM SDK，以下简称 NIM SDK）支持多种消息类型，助您快速实现多样化的消息业务场景。在进行消息收发等操作前，您需要先构建消息。

本文中消息指聊天室消息。更多聊天室消息相关功能请参考开发文档 [聊天室消息](https://doc.yunxin.163.com/messaging2/client-apis/DQzNjE0MDU?platform=client)。

## API 概览

API | 说明 | 起始版本
---- | ---- | ----
[createTextMessage](#createTextMessage) | 创建一条文本消息 | v10.5.0
[createImageMessage](#createImageMessage) | 创建一条图片消息 | v10.5.0
[createAudioMessage](#createAudioMessage) | 创建一条语音消息 | v10.5.0
[createVideoMessage](#createVideoMessage) | 创建一条视频消息 | v10.5.0
[createFileMessage](#createFileMessage) | 创建一条文件消息 | v10.5.0
[createLocationMessage](#createLocationMessage) | 创建一条地理位置消息 | v10.5.0
[createCustomMessage](#createCustomMessage) | 创建一条自定义消息 | v10.5.0
[createForwardMessage](#createForwardMessage) | 创建一条转发消息 | v10.5.0
[createTipsMessage](#createTipsMessage) | 创建一条提示消息 | v10.5.0

## 接口类

`V2NIMChatroomMessageCreator` 类提供聊天室消息构建接口，支持构建多种类型的聊天室消息。

## <span id="createTextMessage">createTextMessage</span>

**接口描述**

创建一条文本消息。

::: note note
- 在加入聊天室后、发送消息前调用该方法。
- 该方法为同步。
:::

**参数说明**

```dart
Future<NIMResult<V2NIMChatroomMessage>> createTextMessage(String text)
```

| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| `text` | String | 是 | 文本消息内容，长度上限为 5000 字符。 |

**示例代码**

```dart
final message = (await V2NIMChatroomMessageCreator.createTextMessage(
            'test text chatroom message'))
        .data;
```

**返回值**

已创建的消息对象 [`V2NIMChatroomMessage`](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#V2NIMChatroomMessage)

## <span id="createImageMessage">createImageMessage</span>

**接口描述**

创建一条图片消息。

::: note note
- 在加入聊天室后、发送消息前调用该方法。
- 该方法为同步。
:::

**参数说明**

```dart

String imagePath,
{String? name,
String? sceneName,

required int height})
```

| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| `imagePath` | String | 是 | 图片文件地址。 |
| `name` | String | 否 | 图片文件显示名称，可不同于文件名。 |
| `sceneName` | String | 否| NOS 文件存储场景名。若使用自定义的存储场景，需要先调用 [`addCustomStorageScene`](https://doc.yunxin.163.com/messaging2/client-apis/TUwMzk0MDg?platform=client#addCustomStorageScene) 添加自定义存储场景。 |
| `width` | int | 是 | 图片宽度，单位为像素。 |
| `height` | int | 是 | 图片高度，单位为像素。 |

**示例代码**

```dart
 final message = (await V2NIMChatroomMessageCreator.createImageMessage(
            imagePath,
            width: 50,
            height: 100))
        .data;
```

**返回值**

已创建的消息对象 [`V2NIMChatroomMessage`](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#V2NIMChatroomMessage)

## <span id="createForwardMessage">createForwardMessage</span>

**接口描述**

创建一条转发消息。

::: note note
- 转发的消息类型 **不能为** `notification`、`robot`、`tip`，`avChat`。
- 转发的消息消息必须为发送成功的消息，消息状态必须为 `NIMMessagesendingState.succeeded`，消息内容与原消息相同。
- 在加入聊天室后、发送消息前调用该方法。
- 该方法为同步。
:::

**参数说明**

```dart
Future<NIMResult<V2NIMChatroomMessage>> createForwardMessage(V2NIMChatroomMessage message)
```

| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| `message` | [`V2NIMChatroomMessage`](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#V2NIMChatroomMessage) | 是 | 待转发的消息体。 |

**示例代码**

```dart
final message = (await V2NIMChatroomMessageCreator.createForwardMessage(
        msg
       ))
        .data;
```

**返回值**

已创建的消息对象 [`V2NIMChatroomMessage`](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#V2NIMChatroomMessage)
