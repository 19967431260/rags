网易云信即时通讯 SDK（NetEase IM SDK，以下简称 NIM SDK）提供聊天室队列服务，支持用户对聊天室队列以及队列元素进行操作管理。

本文介绍聊天室队列服务相关 API。更多聊天室相关功能请参考 [聊天室概述](https://doc.yunxin.163.com/messaging2/client-apis/jYyNjkwMDA?platform=client)。

## API 概览

### 聊天室监听

API | 说明 | 起始版本
---- | ---- | ----
[addQueueListener](#addQueueListener) | 注册聊天室队列监听 | v10.6.1
[removeQueueListener](#removeQueueListener) | 取消注册聊天室队列监听 | v10.6.1

### 聊天室队列

API | 说明 | 起始版本
---- | ---- | ----
[queueOffer](#queueOffer) | 在队列中新增或更新元素 | v10.6.1
[queuePoll](#queuePoll) | 在队列中取出头元素或指定的元素 | v10.6.1
[queueList](#queueList) | 排序列出所有元素 | v10.6.1
[queuePeek](#queuePeek) | 查看队列的头元素 | v10.6.1
[queueDrop](#queueDrop) | 清空队列 | v10.6.1
[queueInit](#queueInit) | 初始化队列 | v10.6.1
[queueBatchUpdate](#queueBatchUpdate) | 批量更新队列元素 | v10.6.1 

## 接口类

`V2NIMChatroomQueueService` 类提供聊天室队列服务接口，包括注册/注销聊天室队列监听器、初始化队列、清空队列、管理队列元素等。

## 使用前提

在使用 `V2NIMChatroomQueueService` 类中的 API 前，需要先调用 [`V2NIMChatroomClient#getChatroomQueueService`](https://doc.yunxin.163.com/messaging2/client-apis/zE5NTI4MjU?platform=client#getChatroomQueueService) 方法获取 `V2NIMChatroomQueueService` 类。

## <span id="addQueueListener">addQueueListener</span>

**接口描述**

注册聊天室队列监听器。

注册成功后，当事件发生时，SDK 会返回对应的回调。 

::: note note
- 建议在初始化后调用该方法。
- 全局只需注册一次。
- 该方法为同步。
:::

**回调事件**

- `onChatroomQueueOffered`：聊天室新增队列元素回调。在聊天室队列中新增元素成功后，SDK 会返回该回调。
- `onChatroomQueuePolled`：聊天室取出（移除）队列元素回调。在聊天室队列中取出（移除）元素会触发该回调。
- `onChatroomQueueDropped`：聊天室清空队列元素回调。聊天室队列中的元素被清空会触发该回调。
- `onChatroomQueuePartCleared`：聊天室清理部分队列元素回调。聊天室队列中的部分元素被清理时会触发该回调。
- `onChatroomQueueBatchUpdated`：聊天室批量更新队列元素回调。当聊天室队列中的元素被批量更新时会触发该回调。
- `onChatroomQueueBatchOffered`：聊天室批量添加队列元素回调。当聊天室队列中批量添加元素时会触发该回调。

**示例代码**

```Dart
import 'package:nim_core_v2/nim_core_v2.dart';

void addChatroomQueueListener(V2NIMChatroomQueueService queueService) async {
  NIMResult<void> result = await queueService.addQueueListener();
  if (result.isSuccess) {
    queueService.onChatroomQueueOffered.listen((element) {
      print('New queue element offered: ${element.key} - ${element.value}');
    });

    queueService.onChatroomQueuePolled.listen((element) {
      print('Queue element polled: ${element.key} - ${element.value}');
    });

    queueService.onChatroomQueueDropped.listen((_) {
      print('Queue dropped');
    });

    queueService.onChatroomQueuePartCleared.listen((elements) {
      print('Part of queue cleared: ${elements.length} elements');
    });

    queueService.onChatroomQueueBatchUpdated.listen((elements) {
      print('Queue elements batch updated: ${elements.length} elements');
    });

    queueService.onChatroomQueueBatchOffered.listen((elements) {
      print('Queue elements batch offered: ${elements.length} elements');
    });
  } else {
    print('Failed to add queue listener: ${result.code} - ${result.errorDetails}');
  }
}
```

## <span id="queueOffer">queueOffer</span>

**接口描述**

在队列中新增或更新元素。

新增或更新元素成功后，会触发 `onChatroomQueueOffered` 回调。 

**参数说明**

```Dart
Future<NIMResult<void>> queueOffer( V2NIMChatroomQueueOfferParams offerParams )
```

| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| `offerParams` | [`V2NIMChatroomQueueOfferParams`](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#V2NIMChatroomQueueOfferParams) | 是 | 新增或更新队列元素的配置参数。 |

**示例代码**

```Dart
import 'package:nim_core_v2/nim_core_v2.dart';

void offerQueueElement(V2NIMChatroomQueueService queueService, String key, String value) async {
    final params = V2NIMChatroomQueueOfferParams(elementKey: key, elementValue: value);
    NIMResult<void> result = await queueService.queueOffer(params);
    if (result.isSuccess) {
      print('Successfully offered queue element: $key - $value');
    } else {
      print('Failed to offer queue element: ${result.code} - ${result.errorDetails}');
    }
  }
```

**返回值**

NIMResult\<void>

**相关回调**

`onChatroomQueueOffered`：聊天室新增队列元素回调。

## <span id="queuePoll">queuePoll</span>

**接口描述**

在队列中取出队列中的指定元素。若未指定元素，则默认取出队列的头元素。

取出成功后，会触发 `onChatroomQueuePolled` 回调。

::: note note 
仅聊天室创建者或管理员才能进行取出（移除）操作。
:::

**参数说明**

```Dart
Future<NIMResult<V2NIMChatroomQueueElement>> queuePoll( String elementKey )
```

| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| `elementKey` | String | 否 | <li>若为空，则表示取出队列的头元素；<li>若不为空，则表示取出指定的队列元素。 |

**示例代码**

```Dart
import 'package:nim_core_v2/nim_core_v2.dart';

void pollQueueElement(V2NIMChatroomQueueService queueService, String? elementKey) async {
    NIMResult<V2NIMChatroomQueueElement> result = await queueService.queuePoll(elementKey ?? '');
    if (result.isSuccess && result.data != null) {
      print('Successfully polled queue element: ${result.data!.key} - ${result.data!.value}');
    } else {
      print('Failed to poll queue element: ${result.code} - ${result.errorDetails}');
    }
  }
```

**返回值**

NIMResult\<[V2NIMChatroomQueueElement](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#V2NIMChatroomQueueElement)> 聊天室队列元素

**相关回调**

`onChatroomQueuePolled`：聊天室取出（移除）队列元素回调

## <span id="queueList">queueList</span>

**接口描述**

排序列出队列中的所有元素。

**参数说明**

```Dart
Future<NIMResult<List<V2NIMChatroomQueueElement>>> queueList()
```

**示例代码**

```Dart
import 'package:nim_core_v2/nim_core_v2.dart';

void listQueueElements(V2NIMChatroomQueueService queueService) async {
  NIMResult<List<V2NIMChatroomQueueElement>> result = await queueService.queueList();
  if (result.isSuccess && result.data != null) {
    print('Queue elements:');
    for (var element in result.data!) {
      print('${element.key} - ${element.value}');
    }
  } else {
    print('Failed to list queue elements: ${result.code} - ${result.errorDetails}');
  }
}
```

**返回值**

NIMResult\<List\<[V2NIMChatroomQueueElement](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#V2NIMChatroomQueueElement)>> 队列元素列表

**相关回调**

无。

## <span id="queueBatchUpdate">queueBatchUpdate</span>

**接口描述**

批量更新队列元素。

批量更新成功后，会触发 `onChatroomQueueBatchUpdated` 回调。

**参数说明**

```Dart
Future<NIMResult<List<String>?>> queueBatchUpdate(
List<V2NIMChatroomQueueElement> elements, {
bool notificationEnabled = true,
String? notificationExtension,
})
```

| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| `elements` | List\<[`V2NIMChatroomQueueElement`](https://doc.yunxin.163.com/messaging2/client-apis/zExMjk2NzY?platform=client#V2NIMChatroomQueueElement)> | 是 | 需要更新的队列元素元素列表，如果输入为空/数量为 0/数量超过 100，则返回 191004 参数错误。<br>key 和 value 字段限制分别为 128 和 4096 字节。 |
| `notificationEnabled` | bool | 否 | 是否发送广播通知。默认为 true，表示发送广播通知。 |
| `notificationExtension` | String | 否 | 本次操作生成的通知扩展字段，字段长度限制为 2048 字节。 | 

**示例代码**

```Dart
import 'package:nim_core_v2/nim_core_v2.dart';

void batchUpdateQueueElements(V2NIMChatroomQueueService queueService, List<V2NIMChatroomQueueElement> elements) async {
  NIMResult<List<String>?> result = await queueService.queueBatchUpdate(elements);
  if (result.isSuccess) {
    print('Successfully batch updated queue elements');
    if (result.data != null) {
      print('Non-existent elements: ${result.data}');
    }
  } else {
    print('Failed to batch update queue elements: ${result.code} - ${result.errorDetails}');
  }
}
```

**返回值**

NIMResult\<List\<String>?> 不存在的元素 Key 列表

**相关回调**

`onChatroomQueueBatchUpdated`：聊天室批量更新队列元素回调
