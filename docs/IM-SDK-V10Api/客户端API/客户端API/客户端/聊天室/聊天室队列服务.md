网易云信即时通讯 SDK（NetEase IM SDK，以下简称 NIM SDK）提供聊天室队列服务，支持用户对聊天室队列以及队列元素进行操作管理。

本文介绍聊天室队列服务相关 API。更多聊天室相关功能请参考开发指南文档 [聊天室概述](https://doc.yunxin.163.com/messaging2/client-apis/jYyNjkwMDA?platform=client)。

## 接口类

`V2NIMChatroomQueueService` 类提供聊天室队列服务接口，包括注册/注销聊天室队列监听器、初始化队列、清空队列、管理队列元素等。

## 使用前提

在使用 `V2NIMChatroomQueueService` 类中的 API 前，需要先调用 [`V2NIMChatroomClient#getChatroomQueueService`](https://doc.yunxin.163.com/messaging2/client-apis/DYyMTk0NjE?platform=client#getChatroomQueueService) 方法获取 `V2NIMChatroomQueueService` 类。

## <span id="addQueueListener">addQueueListener</span>

**接口描述**

注册聊天室队列监听器。

注册成功后，当事件发生时，SDK 会返回对应的回调。

- 该方法为同步。

- 全局只需注册一次。
- 该方法为同步。
:::

**参数说明**

:::::: div linked-codes
::: code Web/uni-app/小程序
```TypeScript
export interface NIMEBaseServiceInterface<I extends NIMEBaseListener> {
  /**
   * 继承自 EventEmitter3 的监听事件方法。
   */
  on<T extends keyof I>(eventName: T, fn: (...args: I[T]) => void): void;
  /**
   * 继承自 EventEmitter3 的监听事件方法。
   */
  once<T extends keyof I>(eventName: T, fn: (...args: I[T]) => void): void;
}
```
| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |

| `fn` | function | 是 | 监听事件的回调函数。当事件触发时，会调用该函数进行处理。
