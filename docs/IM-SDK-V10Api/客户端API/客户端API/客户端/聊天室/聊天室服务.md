网易云信即时通讯 SDK（NetEase IM SDK，以下简称 NIM SDK）提供聊天室服务，支持用户收发聊天室消息、管理聊天室成员、维护聊天室信息。

本文介绍聊天室服务相关 API。更多聊天室相关功能请参考开发指南文档 [聊天室概述](https://doc.yunxin.163.com/messaging2/client-apis/jYyNjkwMDA?platform=client)。

## 接口类

`V2NIMChatroomService` 类提供聊天室服务接口，包括注册/注销聊天室监听器、收发聊天室消息、管理聊天室成员、维护聊天室信息等。

## 使用前提

在使用 `V2NIMChatroomService` 类中的 API 前，需要先调用 [`V2NIMChatroomClient#getChatroomService`](https://doc.yunxin.163.com/messaging2/client-apis/DYyMTk0NjE?platform=client#getChatroomService) 方法获取 `V2NIMChatroomService` 类。

## <span id="on">on("EventName")</span>

**接口描述**

注册聊天室监听。

注册成功后，当事件发生时，SDK 会返回对应的回调。

::: note note

- 建议在初始化后调用该方法。

- 全局只需注册一次。

- 该方法为同步。
:::

**参数说明**

:::::: div linked-codes
::: code Web/uni-app/小程序
```TypeScript
export interface NIMEBaseServiceInterface<I extends NIMEBaseListener> {
  /**
   * 继承自 EventEmitter3 的监听事件方法。
   */
  on<T extends keyof I>(eventName: T, fn: (...args: I[T]) => void): void;
  /**
   * 继承自 EventEmitter3 的监听事件方法。
   */
  once<T extends keyof I>(eventName: T, fn: (...args: I[T]) => void): void;
}
```
| 参数名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| `eventName` | T | 是 | 事件名称：<li>`onSendMessage`：本端发送聊天室消息状态回调，本端发送聊天室消息或插入聊天室消息成功后，SDK 会返回该回调。<li>`onReceiveMessages`：聊天室消息接收回调。本端接收到消息，或其他端同步的消息时会触发该回调。<li>`onChatroomInfoUpdated`：聊天室资料信息更新回调。聊天室内所有成员均会收到该回调。<li>`onChatroomMemberEnter`：成员进入聊天室回调。聊天室内所有成员均会收到该回调。<li>`onChatroomMemberExit`：成员退出聊天室回调。当成员退出或被踢出聊天室，聊天室内所有成员均会收到该回调。<li>`onChatroomMemberRoleUpdated`：聊天室成员角色类型变更回调。聊天室内所有成员均会收到该回调。<li>`onChatroomMemberInfoUpdated`：聊天室成员信息变更回调。聊天室内所有成员均会收到该回调。<li>`onSelfChatBannedUpdated`：聊天室成员本人禁言状态变更回调。<li>`onSelfTempChatBannedUpdated`：聊天室成员本人临时禁言状态变更回调。<li>`onChatroomChatBannedUpdated`：聊天室整体禁言状态变更回调。<li>`onMessageRevokedNotification`：聊天室消息撤回回调。<li>`onChatroomTagsUpdated`：聊天室标签信息变更回调。
| `fn` | function | 是 | 监听事件的回调函数。当事件触发时，会调用该函数进行处理。
:::

## <span id="cancelMessageAttachmentUpload">cancelMessageAttachmentUpload</span>

::: note note
- 该方法仅支持取消文件类消息（图片、视频、文件）的附件上传。

## <span id="kickMember">kickMember</span>

**接口描述**

将聊天室成员踢出聊天室。

本地端或多端同步踢出聊天室成员成功后：

- 被踢方收到被踢出聊天室回调 [`onChatroomKicked`](https://doc.yunxin.163.com/messaging2/client-apis/DAxNjk0Mzc?platform=client#V2NIMChatroomClientListener)。
- 聊天室内的其他成员会收到成员退出聊天室回调 [`onChatroomMemberExit`](https://doc.yunxin.163.com/messaging2/client-apis/DAxNjk0Mzc?platform=client#V2NIMChatroomListener)。
- 聊天室内所有成员收到一条通知类消息，类型为 [`V2NIMChatroomMessageNotificationType.V2NIM_CHATROOM_MESSAGE_NOTIFICATION_TYPE_MEMBER_KICKED(7)`](https://doc.yunxin.163.com/messaging2/client-apis/DAxNjk0Mzc?platform=client#V2NIMChatroomMessage#V2NIMChatroomMessageNotificationType)。

::: note notice
- 仅允许聊天室创建者和管理员调用该接口，否则返回错误码 109432。
- 如果被踢方为管理员，则仅允许聊天室创建者操作，否则返回错误码 109427。
:::

## <span id="updateMemberRole">updateMemberRole</span>

**接口描述**

更新聊天室成员角色类型。

本地端或多端同步更新聊天室成员角色类型成功后，聊天室内所有成员收到成员角色类型变更回调 [`onChatroomMemberRoleUpdated`](https://doc.yunxin.163.com/messaging2/client-apis/DAxNjk0Mzc?platform=client#V2NIMChatroomListener) 及通知类消息，消息类型为 [`V2NIMChatroomMessageNotificationType.V2NIM_CHATROOM_MESSAGE_NOTIFICATION_TYPE_ROLE_UPDATE(18)`](https://doc.yunxin.163.com/messaging2/client-apis/DAxNjk0Mzc?platform=client#V2NIMChatroomMessageNotificationType)。

::: note notice
- 仅允许聊天室创建者和管理员调用该接口，否则返回错误码 109432。
- 如果变更为管理员、或将管理员变更为其他角色，则仅允许聊天室创建者操作，否则返回错误码 109427。
- 不允许操作虚构用户和匿名游客。

## <span id="setMemberBlockedStatus">setMemberBlockedStatus</span>

**接口描述**

设置聊天室成员黑名单状态。[聊天室黑名单用户](https://doc.yunxin.163.com/messaging2/client-apis/Dc4NTY1ODQ?platform=client#技术原理) 与聊天室服务器断开连接，无法收发聊天室消息。

本地端或多端同步设置聊天室成员黑名单状态成功后：

- 聊天室内所有成员收到成员信息变更回调 [`onChatroomMemberInfoUpdated`](https://doc.yunxin.163.com/messaging2/client-apis/DAxNjk0Mzc?platform=client#V2NIMChatroomListener)。
- 将成员加入黑名单后（设置 `blocked` 参数为 `true`），被拉黑方收到退出聊天室回调 [`onChatroomExited`](https://doc.yunxin.163.com/messaging2/client-apis/DAxNjk0Mzc?platform=client#V2NIMChatroomClientListener)。所有聊天室成员会收到成员退出聊天室回调 [`onChatroomMemberExit`](https://doc.yunxin.163.com/messaging2/client-apis/DAxNjk0Mzc?platform=client#V2NIMChatroomListener)，同时会收到对应的聊天室通知消息，通知消息类型为 [`V2NIMChatroomMessageNotificationType.V2NIM_CHATROOM_MESSAGE_NOTIFICATION_TYPE_MEMBER_BLOCK_ADDED(2)`](https://doc.yunxin.163.com/messaging2/client-apis/DAxNjk0Mzc?platform=client#V2NIMChatroomMessageNotificationType)。
- 将成员从黑名单移除后（设置 `blocked` 参数为 `false`），所有聊天室成员收到成员加入聊天室回调 [`onChatroomMemberEnter`](https://doc.yunxin.163.com/messaging2/client-apis/DAxNjk0Mzc?platform=client#V2NIMChatroomListener)，同时会收到对应的聊天室通知消息类型为 [`V2NIMChatroomMessageNotificationType.V2NIM_CHATROOM_MESSAGE_NOTIFICATION_TYPE_MEMBER_BLOCK_REMOVED(3)`](https://doc.yunxin.163.com/messaging2/client-apis/DAxNjk0Mzc?platform=client#V2NIMChatroomMessageNotificationType)。

::: note notice
- 仅允许聊天室创建者和管理员调用该接口，否则返回错误码 109432。
- 如果被操作方为管理员，则仅允许聊天室创建者操作，否则返回错误码 109427。
- 不允许操作虚构用户和匿名游客。

## <span id="setMemberTempChatBanned">setMemberTempChatBanned</span>

**接口描述**

设置聊天室成员临时禁言状态。

被临时禁言后，无法在聊天室发送消息。临时禁言结束后，恢复发送消息权限。

本地端或多端同步设置聊天室成员临时禁言状态成功后：

- **被禁言成员**：收到临时禁言状态变更回调 [`onSelfTempChatBannedUpdated`](https://doc.yunxin.163.com/messaging2/client-apis/DAxNjk0Mzc?platform=client#V2NIMChatroomListener)。

::: note notice
- 仅允许聊天室创建者和管理员调用该接口，否则返回错误码 109432。
