网易云信即时通讯（NIM）提供全面的跨平台支持，满足您在各种设备和系统上的即时通讯需求。本文列举了网易云信即时通讯客户端 NIM SDK 支持的平台详情。

有关 UI 组件的详细信息，请访问 [即时通讯 IM UIKit](https://doc.yunxin.163.com/messaging-uikit/concept?platform=client) 和对应平台 [更新日志](https://doc.yunxin.163.com/messaging-uikit/concept/zMzNDI4MDI?platform=client)。
