## <span id="10.9.76 - 2026-03-12">10.9.76 (2026-03-12)</span>

**网易云信 IM 即时通讯 HarmonyOS Atomic Service SDK 首次发布**，v10.9.76 版本支持以下功能：

- 支持多端登录、单聊/群聊、用户信息以及好友关系功能模块。
- 支持多样化的消息类型。
- 支持会话管理、群组管理、用户和好友管理能力。
- 支持信令功能。
