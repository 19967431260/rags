## 计费组成

即时通信 IM 的基础功能包由 **套餐包** 和 **资源包** 组成。

- **套餐包**：即时通信 IM 套餐包分为 2024 免费版、2024 标准版、2024 高级版、2024 旗舰版和 2024 专属版。创建应用后默认为免费版，2024 免费版为免费服务，同时支持试用体验各个版本的专享功能。

- **资源包**：超出版本套餐包免费额度以外的日活、文件存储、频控值等所需支付的费用。

### 资源包详情

| 模块 | 功能项 | 说明 |
| ---- | ---- | ---- |
| 存储 | 消息存储时间 | 消息存储在网易云信服务器的时间，超出部分将无法搜索和获取。 |
| ^^ | 文件存储 | 文件存储能力，默认赠送 1T，超出部分单独计费。 |
| 扩展能力 | 高级群容量 | 单个高级群容量的最大值，最高可扩展至 5000 人。 |
| ^^ | 语音转文字 | 语音消息体转文字功能项，支持中文和英文。 |
| ^^ | 翻译 | 支持 100+ 文字语言间的转换。 |
| 圈组消息费 | 下行消息量 | 圈组单月下行消息条数超出部分费用，1 万日活赠送 2 亿条下行消息。日活超过 1 万，则每万日活赠送 2000 万条下行消息。 |
| 频控 | 如单次类-消息模块 | 根据 API 进行归类，调整一个归类的频控，则归类下的所有 API 均生效。 |

:::note note
- 圈组下行消息：一条消息下发时接收过的次数，在线接收、登录拉取均计入下行消息数。
- 每项增值服务仅对单个 SDK AppKey 生效，如果您有多个 SDK AppKey 需要升级，您需分别为其购买单独的套餐包。
:::
