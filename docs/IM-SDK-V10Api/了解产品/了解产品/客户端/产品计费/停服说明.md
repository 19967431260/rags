本文介绍网易云信即时通讯 IM 账号欠费停服的相关说明。

## 判定条件

当您的账户余额不足以支付账单或可用额度为负值（可用额度=账户余额-未结清金额）时，系统将判定您的账户已欠费。

## 停服预警

进入欠费关停流程前，系统不会立即关停您的服务，而是发送欠费通知（邮件、短信、站内信等方式），并声明停服预警期。

自收到欠费通知当日起，到预警期结束时，您的账户可用额度仍小于等于零，系统将自动关停该账户的全部服务。您可以在预警期结束前及时充值。

## 停服影响

停服后（含停服当日），您将无法使用该应用的所有即时通信 IM 功能（如登录等）及 [网易云信控制台](https://app.yunxin.163.com/global/home) 中除基本配置外的其他功能（如群组管理、回调配置等）。

:::note note
停服后，只能通过充值账户余额的方式重新激活应用，不支持回退至免费版。
:::

## 资源回收

当应用因欠费或退费导致停止服务时，资源回收规则如下：

- 停服后，您的应用（AppID）资源将保留 90 天。如您在 90 天内未通过充值方式恢复服务，系统将回收您的资源且无法恢复。

- 被回收且无法恢复的数据包括但不限于：账号信息、账号下资源信息、用户信息、群组信息、聊天记录等。

    :::note note
    手动关停的应用同样适用上述资源回收规则。
    :::

## 恢复服务

如需继续使用原应用，您可在停服后 90 天内充值账户余额以恢复服务。详细操作步骤请参考 [线上充值](https://doc.yunxin.163.com/console/concept/DkzNzI4MTE?platform=console)。

:::note note
手动关停的应用同样可以在 90 天内恢复服务，从而避免资源被回收。
:::

## 特殊情况

- 应用从免费版套餐版本升级至正式套餐版本后，不支持重新降级为免费版。如您需要重新使用免费版套餐，请创建新的应用。
- 因内容违规或其他不符合 IM 协议而导致停服的网易云信账号，将无法恢复使用。
