本文主要介绍 IM 消息结构体信息和消息响应体信息。

## 消息结构体

参数名称 | 类型 | 是否必选 | 描述 | 示例 |
--- | --- | --- | --- | --- |
`message` | Object | 是 | 消息结构体。 | -
 `message_type` | Integer | 是 | 消息类型。<br>0：文本；1：图片；2：语音；3：视频；4：地址位置；6：文件；10：提示；11：机器人；12：音视频话单，100：自定义。<note type=note>对于未开通安全通（即网易易盾反垃圾）功能的应用，自定义消息不会过内容审核。 | 0
 `sub_type` | Integer | 否 | 消息子类型。用于对消息进行二级分类，取值范围为大于 0 的正整数。推荐自定义消息类型使用此字段区分不同业务场景，其他消息类型可根据实际需求选择性使用。 | 1
 `text` | String | 否 | 对于文本消息和提示消息，该字段必填，值为消息内容，长度上限 5000 位字符。<br>对于非文本/提示消息，该字段非必填，值为描述信息，可用于 [全文关键字搜索历史消息](https://doc.yunxin.163.com/messaging2/guide/jA4MDk5MzA?platform=client#全文搜索历史消息)，长度上限 500 位字符。 | a text message
 `attachment` | Object | 否 | 非文本消息/提示消息的属性或自定义消息内容。<li>对于非文本消息/提示消息，该字段必填，每种消息的属性参数见：[消息格式示例](https://doc.yunxin.163.com/messaging2/server-apis/TEzMDQwODc?platform=server#消息格式示例)。<li>该字段长度上限以使用的 IM 套餐为准；IM 旗舰版及以上套餐才支持配置字段上限。 | {<br>"name":"tupian",<br>"md5":"9894907e4f7",<br>"url":"http://nimtest.com/cbc50",<br>"ext":"jpg",<br>"width":6814,  <br>"height":2332,<br>"size":388245<br>} |
 `location_x` | Double | 否 | 空间坐标 x。定向消息不支持该参数。 | 89
 `location_y` | Double | 否 | 空间坐标 y。定向消息不支持该参数。 | 64
 `location_z` | Double | 否 | 空间坐标 。定向消息不支持该参数。 | 23   


## 消息响应体

参数名称 | 类型 | 描述 | 是否必返回 | 示例
--- | --- | --- | --- | --- |
`code` | Integer | 状态码。 | 是 | 200，表示请求成功。 |
`msg` | String | 提示信息。| 是 | 请求失败时返回错误信息，请求成功时返回 "success"。 | 
`data` | Object | 返回的 JSON 消息对象，请求失败则返回空对象。|　是　| - |
 `message_server_id` | Long | 服务端消息 ID。 | 是 | 502714177945601
 `message_client_id` | String | 客户端消息 ID。 | 是 | 4268bca6d00e0fd44f
 `sender_id` | String | 消息发送方帐号 ID。 | 是 | wqt
 `conversation_type` | Integer | 会话类型，1：单聊会话；2：高级群会话；3：超大群会话。 | 是 | 1
 `receiver_id` | String | 消息接收方账号 ID。 | 否 | apiv2test
 `team_id` | Long | 群组 ID。`team_id` 与 `receiver_id` 只会返回其一。 | 否 | 7491025
 `message_type` | Integer | 消息类型。<br>0：文本；1：图片；2：语音；3：视频；4：地址位置；6：文件；10：提示；12：音视频话单；100：自定义。 | 是 | 0
 `sub_type` | Integer | 消息子类型。用于对消息进行二级分类，取值范围为大于 0 的正整数。推荐自定义消息类型使用此字段区分不同业务场景，其他消息类型可根据实际需求选择性使用。 | 否 | 1
 `create_time` | Long | 消息发送时间戳。 | 是 | 123456123111 
 `text` | String | 文本/提示消息内容或多媒体消息的描述文本（该描述信息可用于云端历史消息关键词检索）。 | 否 | msg 
 `attachment` | Object | 多媒体消息的属性或自定义消息内容。 | 否 | ["name", "image"]
 `extension` | String | 消息发送时传入的第三方扩展字段。 | 否 | laboris enim aliqua sint nostrud
 `sender_client_type` | Integer | 发送者客户端类型。1：AOS；2：iOS；4：PC；16：Web；32：REST；64：MAC；65：HARMONY。| 是 | 1
