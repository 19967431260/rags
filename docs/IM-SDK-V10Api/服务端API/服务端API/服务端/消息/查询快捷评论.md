网易云信服务端支持查询指定消息（包括单聊消息、高级群消息和超大群消息）的快捷评论。

::: note note
调用添加消息快捷评论接口前，请至网易云信控制台 [开通消息快捷评论](https://doc.yunxin.163.com/messaging2/guide/jU0Mzg0MTU?platform=client#全局功能)。
:::
