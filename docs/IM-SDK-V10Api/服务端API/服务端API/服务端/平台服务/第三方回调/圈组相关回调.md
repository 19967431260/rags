通过圈组相关事件回调功能，可以实现您的应用服务器干预圈组相关事件的处理结果（具体干预流程请参见
[第三方回调概述](https://doc.yunxin.163.com/messaging/server-apis/jI3ODc2ODE?platform=server)），满足您的应用对于第三方内容审核、安全策略等方面的业务需求。

本文主要为您提供如下信息：
- 通过 HTTP 协议 POST 方法实现相关回调的示例
- 通过 cURL 命令行实现相关回调的示例
- 消息体中的 JSON 字段说明 

::: note note :::
圈组相关的基本概念和功能介绍，请参见[什么是圈组](https://doc.yunxin.163.com/messaging/server-apis/jUyODc1NDM?platform=server)。
:::
