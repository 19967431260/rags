聊天室创建后，可以通过获取到的聊天室地址进入聊天室。该接口用于请求获取聊天室地址。

::: note notice
如果您使用的是 10.9.20+ 的安卓和 iOS NIM SDK，可以在 [登录聊天室](https://doc.yunxin.163.com/messaging2/guide/DI2NDc1NzQ?platform=client#%E7%AC%AC%E5%85%AD%E6%AD%A5%E7%99%BB%E5%BD%95%E8%81%8A%E5%A4%A9%E5%AE%A4) 时自动获取最优服务器地址。其他客户端 NIM SDK 接口也可以获取聊天室地址，但推荐通过服务端 API 获取。
:::

## 请求信息

### 请求 URL

```
GET https://{endpoint}/im/v2/chatrooms/{room_id}/actions/address
```

:::note note
请求 URL 中的 `{endpoint}` 代表服务地址域名，您可以根据用户服务区域选择中国大陆和海外服务地址，并支持搭建高可用主备域名机制。详情请参考 [调用方式](https://doc.yunxin.163.com/messaging2/server-apis/zcwODA3MTU?platform=server#%E6%9C%8D%E5%8A%A1%E5%9C%B0%E5%9D%80) 服务地址章节。
:::

### 请求头参数

请求 Header 的参数说明请参考 [请求 Header](https://doc.yunxin.163.com/messaging2/server-apis/zcwODA3MTU?platform=server#请求-header)。

### 路径参数

参数名称 | 类型 | 是否必选 | 描述
--- | --- | --- | ---
`room_id` | Long | 是 | 需要获取地址的聊天室 ID，聊天室创建成功后，网易云信会返回聊天室 ID。

### 查询参数

参数名称 | 类型 | 是否必选 | 描述
--- | --- | --- | ---
`account_id` | String | 是 | 进入聊天室的用户账号 ID。
`client_type` | Integer | 否 | 进入聊天室的用户的客户端类型。<ul><li>1（默认）：weblink（客户端为 Web/uni-app/HarmonyOS 端时使用）</li><li>2：commonlink（客户端为非 Web/uni-app/HarmonyOS 端时使用）</li><li>3：wechatlink（微信小程序使用）</li></ul>
`client_ip` | String | 是 | 进入聊天室的用户的客户端 IP。<br>网易云信服务器会根据用户 IP 的所在地区，返回合适的地址。
`ip_type` | Integer | 否 | 支持的 IP 协议类型。0（默认）：IPv4；1：IPv6；2：全部支持。

## 响应信息

### 响应头参数

响应 Header 的参数说明请参考 [响应 Header](https://doc.yunxin.163.com/messaging2/server-apis/zcwODA3MTU?platform=server#响应-header)。

### 响应体参数

参数名称 | 类型 | 说明 | 是否必返回
--- | --- | --- | ---
`code` | Integer | 状态码，200 表示请求成功。 | 是
`msg` | String | 提示信息。请求失败时返回错误信息，请求成功时返回 "success"。 | 是
`data` | Object | 返回的 JSON 数据对象，请求失败则返回空对象。 | 是
 `items` | Array of strings | 返回的聊天室地址列表。 | 否

### 响应体示例

```JSON
{
    "code": 200,
    "msg": "success",
    "data": {
        "items": [
            "testchat.yunxinapi.com:8090"
        ]
    }
}
```
