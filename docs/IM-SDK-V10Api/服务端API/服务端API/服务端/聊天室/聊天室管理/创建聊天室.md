该接口用于创建聊天室。聊天室人数无上限，默认可以自由进出（可以设置黑名单不允许进入），人员进出聊天室默认触发事件通知。

## 功能描述

网易云信 IM 聊天室采用多层架构设计，可以实现真正意义上的大型聊天室，参与人数无上限，又可满足消息到达的实时性要求，主要应用于娱乐直播、教育直播等场景。

::: note notice
- 聊天室功能单独开通后才能使用，具体请参考 [开通和配置聊天室功能](https://doc.yunxin.163.com/messaging2/server-apis/jg4MzMyNTY?platform=server)。
- 聊天室的创建只能通过服务端 API 实现，暂不支持在客户端创建。
:::

创建聊天室完成后，若需要修改进出聊天室事件的通知状态，可以参考 [开启/关闭进出聊天室事件通知](https://doc.yunxin.163.com/messaging2/server-apis/DEyMDExMTU?platform=server)。

### 请求体参数

参数名称 | 类型 | 是否必选 | 描述
---- | ---- | ---- | ----
`creator` | String | 是 | 聊天室创建者的 IM 账号 ID。
`room_name` | String | 是 | 聊天室名称。<li>长度上限 128 位字符。<li>需要通过反垃圾审核。
`announcement` | String | 否 | 聊天室公告。<li>长度上限 4096 位字符。<li>需要通过反垃圾审核。
`live_url` | String | 否 | 直播地址，长度上限 1024 位字符。
`extension` | String | 否 | 自定义扩展字段，第三方可以跟据此属性自定义扩展，建议封装成 JSON 格式，{key:value}。<li>长度上限 4096 位字符。<li>需要通过反垃圾审核。
`queue_level` | Integer | 否 | 聊天室队列管理权限。<br>0（默认）：所有人都有权限变更队列；1：只有管理员才能变更队列。
`delay_close_policy` | Integer | 否 | 聊天室定时关闭方式。<br>0（默认）：不开启定时关闭；1：固定时间关闭（不管聊天室中是否还有用户）；2：空闲关闭（等聊天室中没有用户后固定时间关闭）。<note type=notice><ul><li>如果需要使用聊天室定时关闭功能（聊天室自动销毁机制），需要先在 [网易云信控制台](https://app.yunxin.163.com/global/home) 开启（具体请参考 [开通和配置聊天室功能](https://doc.yunxin.163.com/messaging2/server-apis/jg4MzMyNTY?platform=server)）。</li><li>开启该功能后，自动销毁机制的策略可以直接通过该字段进行配置，也可以在 [网易云信控制台](https://app.yunxin.163.com/global/home) 配置（优先以该参数传入的策略为准，若未传入该参数，则以控制台配置的策略为准）。</li><li>仅该功能开通后创建的聊天室支持定时关闭，开通前创建的聊天室不支持定时关闭。</li></ul></note>
`delay_seconds` | Long | 否 | 聊天室定时关闭的时间，单位秒，最大值为 7x24x3600 秒。<note type=note><ul><li>当不开启定时关闭功能（delay_close_policy = 0）时，该参数不生效。</li><li>当选择固定时间关闭或空闲关闭策略时（delay_close_policy = 1 或 2）：<ul><li>若该参数不为空，则以传入的定时时间为准；</li><li>若未传入该参数，则以控制台的定时时间为准。</li></ul></li></ul></note>
`in_out_notification` | Boolean | 否 | 是否关闭人员进出聊天室事件通知。false：关闭；true（默认）：开启。
`antispam_configuration` | Object | 否 | 安全通相关配置项。
 `enabled` | Boolean | 否 | 是否开启安全通，默认为 `true`，即默认启用 [安全通](https://doc.yunxin.163.com/messaging/server-apis/Dg0NDY2NzY?platform=server) 功能。如果需要关闭安全通检测，请设置为 false。<note type="notice">该字段只针对在 [网易云信控制台](https://app.yunxin.163.com/global/home) 开通了安全通业务的应用生效。若未开启安全通业务，该字段无效。</note>
 `business_id_map` | Array of objects | 否 | 安全通业务列表。
  `type` | Integer | 是 | 检测类型。1：文本；2：图片。
  `business_id` | String | 是 | 安全通业务 ID，可以指定当前群组信息过安全通某个检测策略。<br>默认情况下网易云信控制台会生成默认业务，开通安全通后，客户端不需要配置业务 ID 就能默认走该策略，若需要自定义检测策略，请 [提交工单](https://app.yunxin.163.com/global/service/ticket/create) 联系网易云信技术支持工程师进行配置，配置好后传入对应的安全通业务 ID，表示当前群组信息过安全通的指定检测策略。
