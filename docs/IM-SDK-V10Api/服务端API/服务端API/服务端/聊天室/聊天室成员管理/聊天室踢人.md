该接口用于将聊天室中的指定成员踢出聊天室。

::: note notice
- 只有聊天室创建者和管理员才能踢人出聊天室。
- 只有聊天室创建者才能踢管理员，管理员之间不能互踢。
:::

## 请求信息

### 请求 URL

```
PATCH https://{endpoint}/im/v2.1/room_members/{account_id}/actions/kicked
```

:::note note
请求 URL 中的 `{endpoint}` 代表服务地址域名，您可以根据用户服务区域选择中国大陆和海外服务地址，并支持搭建高可用主备域名机制。详情请参考 [调用方式](https://doc.yunxin.163.com/messaging2/server-apis/zcwODA3MTU?platform=server#%E6%9C%8D%E5%8A%A1%E5%9C%B0%E5%9D%80) 服务地址章节。

参数名称 | 类型 | 是否必选 | 描述
---- | ---- | ---- | ----

`notification_extension` | String | 否 | 通知扩展字段，JSON 格式，长度上限为 2048 字符。
