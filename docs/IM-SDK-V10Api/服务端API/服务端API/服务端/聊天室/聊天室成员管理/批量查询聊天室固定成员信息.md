
该接口用于批量查询指定的聊天室固定成员信息。

## 调用频率

单个应用默认最高调用频率请参考 [频控说明](https://doc.yunxin.163.com/messaging2/server-apis/DUzNjAzMTc?platform=server)。

## 请求信息

### 请求 URL

```
GET https://{endpoint}/im/v2/room_members/{room_id}/actions/batch
```

:::note note
请求 URL 中的 `{endpoint}` 代表服务地址域名，您可以根据用户服务区域选择中国大陆和海外服务地址，并支持搭建高可用主备域名机制。详情请参考 [调用方式](https://doc.yunxin.163.com/messaging2/server-apis/zcwODA3MTU?platform=server#%E6%9C%8D%E5%8A%A1%E5%9C%B0%E5%9D%80) 服务地址章节。
:::

### 请求头参数

请求 Header 的参数说明请参考 [请求 Header](https://doc.yunxin.163.com/messaging2/server-apis/zcwODA3MTU?platform=server#请求-header)。

### 路径参数

参数名称 | 类型 | 是否必选 | 描述
---- | ---- | ---- | ----
`room_id` | Long | 是 | 聊天室 ID。

### 查询参数

参数名称 | 类型 | 是否必选 | 描述
---- | ---- | ---- | ----
`account_ids` | Array of strings | 是 | 需要查询的聊天室成员列表，一次最多查询 200 个账号。

## 响应信息

### 响应头参数

响应 Header 的参数说明请参考 [响应 Header](https://doc.yunxin.163.com/messaging2/server-apis/zcwODA3MTU?platform=server#响应-header)。

### 响应体参数

参数名称 | 类型 | 说明 | 是否必返回 |
---- | ---- | ---- | ----
`code` | Integer | 状态码，200 表示请求成功。 | 是
`msg` | String | 提示信息。请求失败时返回错误信息，请求成功时返回 "success"。 | 是
`data` | Object | 返回的 JSON 数据对象，请求失败则返回空对象。 | 是
 `failed_list` | Array of objects | 查询失败的账号列表。 | 否
  `account_id` | String | 用户账号 ID。 | 否
  `error_code` | Integer | 查询失败的错误码。 | 否
  `error_msg` | String | 查询失败的错误信息。 | 否
 `success_list` | Array of objects | 查询成功的成员信息列表。 | 否
  `account_id` | String | 在线成员的账号 ID。 | 是
  `room_nick` | String | 成员在聊天室中的昵称。 | 否
  `room_avatar` | String | 成员在聊天室中的头像。 | 否
  `extension` | String | 成员信息扩展字段。 | 否
  `member_role` | Integer | 成员角色类型。<br>0：普通用户；1：创建者；2：管理员；3：普通游客；4：匿名游客。 | 是
  `member_level` | Integer | 成员级别。 | 否
  `is_online` | Boolean | 是否在线，true：在线；false：不在线。 | 是
  `enter_time` | Long | 进入聊天室的时间戳。 | 否
  `blocked` | Boolean | 是否在聊天室黑名单中，true：在黑名单；false：不在。 | 是
  `chat_banned` | Boolean | 是否被禁言，true：被禁言；false：未被禁言。 | 是
  `temp_chat_banned` | Boolean | 是否被临时禁言，true：被临时禁言；false：未被临时禁言。 | 是
  `temp_chat_banned_time` | Long | 临时禁言的时长，单位秒。 | 否
  `tags` | String | 用户标签，只有在线成员且登录时指定了标签才会有此字段 | 否
  `notify_target_tags` | String | 通知标签，只有在线成员且登录时指定了标签才会有此字段 | 否
  `online_count` | Integer | 在线设备数量。 | 是
  `online_info_list` | Array of objects | 在线设备信息列表。 | 否
   `robot` | Boolean | 是否为机器人。 | 否
   `room_nick` | String | 成员在聊天室中的昵称。 | 否
   `room_avatar` | String | 成员在聊天室中的头像。 | 否
   `enter_time` | Long | 进入聊天室的时间戳。 | 否
   `client_type` | Integer | 客户端类型。 | 否

## 错误码

本文仅列举部分业务接口错误码，完整列表请参考客户端 API [错误码](https://doc.yunxin.163.com/messaging2/client-apis/DUxNjU3MzU?platform=client)。 <!-- IM V10 -->

| 错误码 | 错误码描述 | 错误信息示例
| ---- | ---- | ----
| 200 | 请求成功 | success
| 414 | 参数错误 | parameter error
| 416 | 频率超限 | rate limit exceeded
| 113404 | 聊天室不存在 | chatroom not exist
| 113410 | 聊天室服务未开通 | chatroom service disabled
| 113406 | 聊天室已关闭 | chatroom closed
| 500 | 服务端内部错误 | internal server error
| 503 | 服务器繁忙 | server busy
