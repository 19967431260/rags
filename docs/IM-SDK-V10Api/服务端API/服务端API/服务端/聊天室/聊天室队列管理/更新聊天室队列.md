该接口用于更新聊天室队列，主要表现在聊天室队列中添加或更新元素。

## 调用频率

单个应用默认最高调用频率请参考 [频控说明](https://doc.yunxin.163.com/messaging2/server-apis/DUzNjAzMTc?platform=server)。

## 请求信息

### 请求 URL

```
PATCH https://{endpoint}/im/v2/room_queues/{room_id}
```

:::note note
请求 URL 中的 `{endpoint}` 代表服务地址域名，您可以根据用户服务区域选择中国大陆和海外服务地址，并支持搭建高可用主备域名机制。详情请参考 [调用方式](https://doc.yunxin.163.com/messaging2/server-apis/zcwODA3MTU?platform=server#%E6%9C%8D%E5%8A%A1%E5%9C%B0%E5%9D%80) 服务地址章节。
:::

### 请求头参数

请求 Header 的参数说明请参考 [请求 Header](https://doc.yunxin.163.com/messaging2/server-apis/zcwODA3MTU?platform=server#请求头)。

### 路径参数

参数名称 | 类型 | 是否必选 | 说明
---- | ---- | ---- | ----
`room_id` | Long | 是 | 聊天室 ID。聊天室创建成功后，网易云信会返回聊天室 ID。

### 请求体参数

参数名称 | 类型 | 是否必选 | 说明
---- | ---- | ---- | ----
`operator_account_id` | String | 是 | 操作者账号 ID。
`element_list` | Array of objects | 是 | 队列元素列表。
 `element_key` | String | 是 | 队列元素的 key。
 `element_value` | String | 是 | 队列元素的 value。
 `element_account_id` | String | 否 | 该队列元素的所属用户的账号 ID，若不填，则默认为 `operator_account_id`。
 `element_transient` | Boolean | 否 | 该队列元素是否为临时元素。默认为 false，即为持久化元素。<li>当 `element_transient` 为 true（临时元素）时，只有该元素的所属者在线，才能添加成功；元素所属者一旦离线，该元素会立刻被删除。<li>当 `element_transient` 为 false（持久化元素）时，即使元素所属者离线，该元素也不会被删除。
 `element_add_policy` | Integer | 否 | 该队列元素的更新策略，默认为 2。<li>0：只能新增该元素。若新增元素的 key 已存在，那么该元素新增无效（对已存在的元素不做处理），并将该 key 放在 `fail_element_list` 中返回。<li>1：只能更新该元素。若更新元素的 key 不存在，那么该元素更新无效，并将不存在的 key 放在 `fail_element_list` 中返回。<li>2（默认）：可以新增，也可以更新该元素。若元素的 key 存在，则更新该元素；若不存在，则新增该元素。
`notification_config` | Object | 否 | 聊天室队列变更通知的相关配置信息。
 `notification_enabled` | Boolean | 否 | 是否需要发送变更通知，默认为 true，需要发送。
 `notification_extension` | String | 否 | 通知事件的扩展字段，最大长度为 2048 位字符。
 `high_priority` | Integer | 否 | 队列变更通知是否需要设置为高优先级消息（网易云信会优先保障投递此类消息）。<li>0（默认）：不需要（变更通知为普通消息）。<li>1：需要（变更通知为高优先级消息）。<note type=note>建议恰当使用该参数，以便在必要时，优先保障应用内的高优先级消息的投递。若全部设置为高优先级，则等于没有设置，单个聊天室最多支持每秒 10 条高优先级消息。
 `high_priority_policy` | Integer | 否 | 高优先级消息策略。针对高优先级消息，超过流控（单个聊天室最多支持每秒 10 条高优先级消息）后的处理策略。<li>0（默认）：自动降级为普通消息。<li>1：返回 416 错误码。
`route_config` | Object | 否 | 抄送相关配置项。配置前需要提前为应用开通消息抄送功能并配置抄送环境，具体请参考 [开通和配置消息抄送](https://doc.yunxin.163.com/messaging/server-apis/jY5MDk1NTQ?platform=server)。
 `route_enabled` | Boolean | 否 | 是否需要抄送至指定的应用服务器，默认为 true，需要抄送。
 `route_environment` | String | 否 | 需要抄送到的环境，对应您在 [网易云信控制台](https://app.yunxin.163.com/global/home) 中配置的自定义抄送的环境名称。
