该接口用于转让群主身份，包括高级群和超大群。

::: note notice
群主身份转让后可以选择退出此群，也可以留下来转为普通成员。
:::

## 请求信息

### 请求 URL

```
PATCH https://{endpoint}/im/v2/teams/{team_id}/actions/transfer_owner
```

:::note note
请求 URL 中的 `{endpoint}` 代表服务地址域名，您可以根据用户服务区域选择中国大陆和海外服务地址，并支持搭建高可用主备域名机制。详情请参考 [调用方式](https://doc.yunxin.163.com/messaging2/server-apis/zcwODA3MTU?platform=server#%E6%9C%8D%E5%8A%A1%E5%9C%B0%E5%9D%80) 服务地址章节。
:::

### 请求头参数

请求 Header 的参数说明请参考 [请求 Header](https://doc.yunxin.163.com/messaging2/server-apis/zcwODA3MTU?platform=server#请求-header)。

### 请求路径参数

参数名称 | 类型 | 是否必选 | 描述
---- | ---- | ---- | ----
`team_id` | Long | 是 | 群组 ID，创建群组成功时，网易云信服务器会自动生成并返回。

### 请求体参数

参数名称 | 类型 | 是否必选 | 描述
---- | ---- | ---- | ----
`team_type` | Integer | 是 | 需要转让的群组类型。1：高级群。2：超大群。
`new_owner_account_id` | String | 是 | 新群主的 IM 账号 ID。
`leave` | Integer | 是 | 转让群主后，原群主是否退出该群组。<li>1：退出。<li>2：不退出，成为普通成员。
`extension` | String | 否 | 自定义扩展字段，即自定义的通知字段，JSON 格式，不会持久化。长度上限 512 位字符。对应客户端 SDK 通知消息附件 `V2NIMMessageNotificationAttachment` 的 `serverExtension` 字段。<note type=notice>该字段仅针对高级群，对超大群无效。

### 请求体示例

```JSON
{
    "team_type": 1,
    "new_owner_account_id": "accid2",
    "leave": 1
}
```
