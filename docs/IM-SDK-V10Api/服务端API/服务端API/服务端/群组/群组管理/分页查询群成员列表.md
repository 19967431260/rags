该接口用于分页查询群组的所有成员列表，也可以根据查询类型查询群组中的禁言成员列表。

## 调用频率

单个应用默认最高调用频率请参考 [频控说明](https://doc.yunxin.163.com/messaging2/server-apis/DUzNjAzMTc?platform=server)。

## 请求信息

### 请求 URL

```
GET https://{endpoint}/im/v2.1/teams/{team_id}/actions/list_members
```

:::note note
请求 URL 中的 `{endpoint}` 代表服务地址域名，您可以根据用户服务区域选择中国大陆和海外服务地址，并支持搭建高可用主备域名机制。详情请参考 [调用方式](https://doc.yunxin.163.com/messaging2/server-apis/zcwODA3MTU?platform=server#服务地址) 服务地址章节。
:::

### 请求头参数

请求 Header 的参数说明请参考 [请求 Header](https://doc.yunxin.163.com/messaging2/server-apis/zcwODA3MTU?platform=server#请求头)。

### 请求路径参数

参数名称 | 类型 | 是否必选 | 描述
---- | ---- | ---- | ----
`team_id` | Long | 是 | 需要查询的群组 ID，创建群组成功时，网易云信服务器会自动生成并返回。

### 请求查询参数

参数名称 | 类型 | 是否必选 | 描述
---- | ---- | ---- | ----
`team_type` | Integer | 是 | 群组类型。1：高级群。2：超大群。
`member_type` | Integer | 否 | 指定需要查询的成员类型。<br>0（默认）：查询全部成员。<br>1：仅查询群主和管理员。<br>2：仅查询普通成员。
`chat_banned_type` | Integer | 否 | 指定仅查询禁言成员。<br>0（默认）：查询全部成员。<br>1：仅查询禁言成员。<br>2：仅查询非禁言成员。
`descending` | Boolean | 否 | 是否按照成员入群时间降序排列。默认为 fasle，即按照入群时间升序排序。
`page_token` | String | 否 | 分页标识符，如果为空，则从第一页开始查询。
`limit` | Integer | 否 | 每页返回的成员数上限，最大为 100。小于等于 0，或者大于 100，会报 414 错误。

## 响应信息

### 响应头参数

响应 Header 的参数说明请参考 [响应 Header](https://doc.yunxin.163.com/messaging2/server-apis/zcwODA3MTU?platform=server#响应头)。

### 响应体参数

参数名称 | 类型 | 说明 | 是否必返回 |
---- | ---- | ---- | ----
`code` | Integer | 状态码，200 表示请求成功。 | 是
`msg` | String | 提示信息。请求失败时返回错误信息，请求成功时返回 "success"。 | 是
`data` | Object | 返回的 JSON 数据对象，请求失败则返回空对象。 | 是
 `has_more` | Boolean | 是否还有更多数据。 | 是
 `next_token` | String | 分页标识符。当 `has_more` 为 `true` 时返回。 | 否
 `items` | Array of objects | 返回的群成员列表。 | 是
  `team_id` | Long | 群组 ID。 | 是
  `account_id` | String | 群成员的 IM 账号 ID。 | 是
  `member_role` | Integer | 群成员类型。0：普通成员。1：群主。2：群管理员。 | 是
  `team_nick` | String | 成员在本群内的昵称。 | 是
  `invitor_account_id` | String | 邀请该成员加入群组的账号 ID。 | 否
  `server_extension` | String | 自定义扩展字段，JSON 格式。 | 否
  `chat_banned` | Boolean | 成员在本群中是否被禁言。 | 否
  `join_time` | Long | 成员加入群的时间戳。 | 是
  `update_time` | Long | 成员更新的时间戳。 | 是
