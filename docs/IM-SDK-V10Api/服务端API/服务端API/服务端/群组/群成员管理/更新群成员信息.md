该接口用于更新群组指定成员的群昵称、禁言状态、群消息免打扰方式以及特别关注等信息，包括高级群成员和超大群成员。

::: note notice
- 只有群主和管理员可以更新普通群成员的信息。
- 管理员信息只有群主可以更新，管理员不能修改其他管理员的信息。
:::
