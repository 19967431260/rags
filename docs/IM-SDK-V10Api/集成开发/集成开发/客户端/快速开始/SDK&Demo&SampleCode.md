本文介绍如何下载网易云信即时通讯 NIM SDK、Demo 和 Sample Code。

## SDK 下载

请通过 [SDK 下载中心](https://doc.yunxin.163.com/messaging2/resource?platform=all#sdk_IM_即时通讯) 下载各平台最新稳定版 SDK 包。

即时通讯 V10 系列是基于 [V9 系列（版本号低于 10.x.x）稳定版](https://doc.yunxin.163.com/messaging/concept/DMyNjIxMTQ?platform=client) 的升级版本，于 2024-01-26 发布。它不仅兼容 V9 系列稳定的接口和功能，提供了一整套即时通讯技术解决方案，包括客户端 IM 组件、客户端 IM 基础库、全平台客户端 SDK 以及服务端 RESTful API 等，还支持云端会话、会话分组、IM 圈组融合登录、聊天室独立登录等功能。同时，全新提供适配鸿蒙系统的客户端 NIM SDK。

## Demo

即时通信 IM Demo（UIKit）演示了 IM 的重要功能，您可 [Demo 及源码](https://doc.yunxin.163.com/messaging2/resource?platform=all#demo) 体验不同平台下的 Demo。
