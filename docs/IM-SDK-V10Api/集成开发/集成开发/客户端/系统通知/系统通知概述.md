系统通知指网易云信系统下发的通知，不属于会话内消息，例如单聊场景中的**对方正在输入**功能，及群聊场景中的入群申请、入群邀请等。

## 系统通知概述

系统通知包括自定义系统通和广播通知。

### 自定义系统通知

自定义系统通知由开发者根据其业务场景自行定义并下发，SDK 仅透传该自定义系统通知，不负责解析和存储，例如实现单聊场景中的对方 **正在输入...** 的功能。

:::note note
自定义系统通知仅支持单聊和群聊场景，不支持聊天室场景。
:::

**自定义系统通知与自定义消息的区别：**

自定义消息属于会话内的一种消息类型，主要提供给开发者定制消息使用，有在线、离线、漫游、通知栏提醒，一般与普通文本，语音消息相同，位于聊天界面的左右两侧。例如，猜拳、贴图、阅后即焚均可以采用自定义消息来实现。

### 广播通知

广播通知是网易云信系统向应用内所有用户下发的全员通知，例如企业通过内部即时通讯软件向全体员工发送内部通知。

广播消息通过新版服务端 API [发送广播消息](https://doc.yunxin.163.com/messaging2/guide/zMzMzUxODQ?platform=server) 发起，SDK 收到广播通知之后通知应用层，并对应用内所有用户发送一条广播通知。
