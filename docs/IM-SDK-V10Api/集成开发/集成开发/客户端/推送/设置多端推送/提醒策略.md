网易云信 NIM SDK 支持配置多端推送策略，即当桌面端（包括 Windows，Web，macOS 等）在线时，SDK 支持设置是否需要推送给移动端。若移动端处于离线，则离线推送通知用户。若处于在线，则在线提醒用户。

## 支持平台

本文内容适用的开发平台或框架如下表所示，涉及的接口请参考下文 [相关接口](#相关接口) 章节：

安卓 | iOS | macOS/Windows | Web/uni-app/小程序 | Node.js/Electron | 鸿蒙 | Flutter
:----: | :----: | :----: | :----: | :----: | :----:
✔️️️️️ | ✔️️️️️ | ✔️️️️️ | ✔️️️️️ | ✔️️️️️ | ✔️️️️️ | ✔️️

## 前提条件

已实现第三方离线推送或在线提醒。

## 实现方法

1. 注册系统设置事件观察者，监听多端推送/提醒配置的更新。

    示例代码如下：

    :::::: div linked-codes
    ::: code 安卓
    ```Java
    NIMClient.getService(V2NIMSettingService.class).addSettingListener(new V2NIMSettingListener() {
        @Override
        public void onTeamMessageMuteModeChanged(String teamId, V2NIMTeamType teamType, V2NIMTeamMessageMuteMode muteMode) {
        }

        @Override
        public void onP2PMessageMuteModeChanged(String accountId, V2NIMP2PMessageMuteMode muteMode) {
        }

        @Override
        public void onPushMobileOnDesktopOnline(boolean need) {
        }
    });
    ```
    :::
    ::: code iOS
    ```Objective-C
    [[[NIMSDK sharedSDK] v2SettingService] addSettingListener:self];
    ```
    :::
    ::: code macOS/Windows
    ```C++
    V2NIMTeamListener listener;
    listener.onTeamMessageMuteModeChanged = [](std::string teamId, V2NIMTeamType teamType, V2NIMTeamMessageMuteMode muteMode) {
        // handle team message mute mode changed
    };
    listener.onP2PMessageMuteModeChanged = [](std::string accountId, V2NIMP2PMessageMuteMode muteMode) {
        // handle p2p message mute mode changed
    };
    listener.onPushMobileOnDesktopOnline = [](nstd::function< void(bool need)>) {
        // handle pushMobileOnDesktopOnline
    };
    settingService.addSettingListener(listener);
    ```
    :::
    ::: code Web/uni-app/小程序
    ```TypeScript
    nim.V2NIMSettingService.on("onTeamMessageMuteModeChanged", function (teamId: string, teamType: V2NIMTeamType, muteMode: V2NIMTeamMessageMuteMode)
    nim.V2NIMSettingService.on("onP2PMessageMuteModeChanged", function (accountId: string, muteMode: V2NIMP2PMessageMuteMode)
    nim.V2NIMSettingService.on("onPushMobileOnDesktopOnline", function (need: boolean)
    ```
    :::
    ::: code Node.js/Electron
    ```TypeScript
    v2.settingService.on("teamMessageMuteModeChanged", function (teamId: string, teamType: V2NIMTeamType, muteMode: V2NIMTeamMessageMuteMode)
    v2.settingService.on("p2PMessageMuteModeChanged", function (accountId: string, muteMode: V2NIMP2PMessageMuteMode)
    v2.settingService.on("pushMobileOnDesktopOnline", function (need: boolean)
    ```
    :::
    ::: code 鸿蒙
    ```TypeScript
    nim.settingService.on('onTeamMessageMuteModeChanged', (teamId: string, teamType: V2NIMTeamType, muteMode: V2NIMTeamMessageMuteMode) => {})
    nim.settingService.on('onP2PMessageMuteModeChanged', (accountId: string, muteMode: V2NIMP2PMessageMuteMode) => {})
    nim.settingService.on('onPushMobileOnDesktopOnline', (need: boolean) => {})
    ```
    :::
    ::: code Flutter
    ```Dart
    subsriptions.add(NimCore.instance.settingsService.onP2PMessageMuteModeChanged.listen((e) {
        // do something
    }));

    subsriptions.add(NimCore.instance.settingsService.onTeamMessageMuteModeChanged.listen((e) {
        // do something
    }));

    subsriptions.add(NimCore.instance.settingsService.onPushMobileOnDesktopOnline.listen((e) {
    // do something
    }));
    ```

2. 通过调用 `setPushMobileOnDesktopOnline` 方法来设置当桌面端在线时，是否需要推送给移动端。

    入参 `need` 为 true 表示桌面端在线时移动端需要推送。

    示例代码如下：

    :::::: div linked-codes
    ::: code 安卓
    ```Java
    NIMClient.getService(V2NIMSettingService.class).setPushMobileOnDesktopOnline(needPushMobileOnDesktopOnline,
    new V2NIMSuccessCallback<Void>() {
        @Override
        public void onSuccess(Void unused) {

        }
    },
    new V2NIMFailureCallback() {
        @Override
        public void onFailure(V2NIMError error) {

        }
    });
    ```
    :::
    ::: code iOS
    ```Objective-C
    [[[NIMSDK sharedSDK] v2SettingService] setPushMobileOnDesktopOnline:YES success:^{
        // 成功回调
    } failure:^(V2NIMError * _Nonnull error) {
        // 失败回调
    }];
    ```
    :::
    ::: code macOS/Windows
    ```C++
    settingService.setPushMobileOnDesktopOnline(
        true,
        []() {
            // set push mobile on desktop online succeeded
        },
        [](V2NIMError error) {
            // set push mobile on desktop online failed, handle error
        });
    ```
    :::
    ::: code Web/uni-app/小程序
    ```TypeScript
    nim.V2NIMSettingService.setPushMobileOnDesktopOnline(true)
    ```
    :::
    ::: code Node.js/Electron
    ```TypeScript
    await v2.settingService.setPushMobileOnDesktopOnline(true)
    ```
    :::
    ::: code 鸿蒙
    ```TypeScript
    await nim.settingService.setPushMobileOnDesktopOnline(true)
    ```
    :::
    ::: code Flutter
    ```Dart
    await NimCore.instance.settingsService.setPushMobileOnDesktopOnline(need);
    ```
    :::
    ::::::

::: note notice
如果 SDK 已配置 [多端登录](https://doc.yunxin.163.com/messaging2/guide/DUyNjI2MjQ?platform=client)，且已设置多端推送，则满足推送条件的各个客户端都会收到离线推送或在线提醒。如果不希望多个客户端同时收到推送或提醒，请关闭多端登录或者关闭多端推送/提醒功能。
:::

3. （可选）调用 `getPushMobileOnDesktopOnline` 方法获取当桌面端在线时，是否需要推送给移动端的配置。

    示例代码如下：

    :::::: div linked-codes
    ::: code Android
    ```Java
    // 获取当桌面端在线时，移动端是否需要推送的配置
    boolean needPushOnMobile = NIMClient.getService(V2NIMSettingService.class).getPushMobileOnDesktopOnline();

    if (needPushOnMobile) {
        System.out.println("桌面端在线时，移动端需要推送");
    } else {
        System.out.println("桌面端在线时，移动端不需要推送");
    }
    ```
    :::
    ::: code iOS
    ```Objective-C
    - (void)onPushMobileOnDesktopOnlineChanged:(BOOL)need;
    ```
    :::
    ::: code macOS/Windows
    ```C++
    settingService.getPushMobileOnDesktopOnline(
        [](bool need) {
            // get push mobile on desktop online succeeded, handle need
        },
        [](V2NIMError error) {
            // get push mobile on desktop online failed, handle error
        });
    ```
    :::
    ::: code Web/uni-app/小程序
    ```TypeScript
    nim.V2NIMSettingService.getPushMobileOnDesktopOnline()
    ```
    :::
    ::: code Node.js/Electron
    ```TypeScript
    const need = await v2.settingService.getPushMobileOnDesktopOnline()
    ```
    :::
    ::: code 鸿蒙
    ```TypeScript
    const res = await nim.settingService.getPushMobileOnDesktopOnline()
    ```
    :::
    ::: code Flutter
    ```dart
    final isPush = (await NimCore.instance.settingsService.getPushMobileOnDesktopOnline()).data;
    ```
