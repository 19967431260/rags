本文介绍如何为加速域名开启 HTTPS 安全加速，实现客户端访问加速域名时采用 HTTPS 加密，保障数据传输的安全性。

## 背景信息

只支持 PEM 格式的证书。
公钥证书格式要求如下：

- 以`-----BEGIN CERTIFICATE-----`开头，以`-----END CERTIFICATE-----`结尾。
- 每行64个字符，最后一行可以不足64个字符。

私钥 PEM 证书格式要求如下：
- 以`-----BEGIN RSA PRIVATE KEY-----`开头，以`-----END RSA PRIVATE KEY-----`结尾。
- 每行64个字符，最后一行可以不足64个字符。

## 前提条件

全网 CDN 节点已部署加速域名对应的证书。


## 操作步骤


1. 在<a href="https://app.yunxin.163.com/index#/" target="_blank">网易云信控制台</a>的首页单击指定应用名称。

2. 在左侧导航栏选择**产品功能** > **点播**，单击**分发加速配置** 页签。

    ![域名管理.png](https://yx-web-nosdn.netease.im/common/4451af7e1fc6f8de653a353c9919c903/域名管理.png)


3. 单击**域名管理**右侧的**子功能配置**。

4. 在对应加速域名的**操作**列中单击**配置**。

    ![配置CDN缓存.png](https://yx-web-nosdn.netease.im/common/8f7cd0e5373a8b85d2546cac3c48e4c1/配置CDN缓存.png)

5. 单击**HTTPS配置**页签。
6. 单击**HTTPS配置**旁边的开关，开启 HTTPS。配置相关参数，参数说明如下表所示。

    ![HTTPS配置.png](https://yx-web-nosdn.netease.im/common/02aaf63f28fc8f176e6515120b328b9d/HTTPS配置.png)

    配置 | 描述 | 
    ---- | -------------- |
    证书类型 | <ul><li>自有证书：该证书是您本地的证书。</li><li>云信托管证书：该证书上传到云信。</li></ul> | 
    证书名称 | 加速域名对应的证书名称。 |
    证书（公钥） | 单击**证书（公钥）** 文本框下面的**PEM编码参考样例** 获取配置样例。| 
    证书（私钥） | 单击**证书（私钥）** 文本框下面的**PEM编码参考样例** 获取配置样例。 |

7. 单击**保存**。

开启后，客户端访问加速域名时，采用 HTTPS 加密协议。
