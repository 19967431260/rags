当源站的内容更新后，您可以通过刷新，将缓存在 CDN 节点上的资源进行清除，客户端访问资源时，CDN 节点回源到源站获取最新资源。预热是指提前将资源缓存到 CDN 节点，以便提升访问速度。

## 按文件刷新缓存

文件刷新是以文件为单位，将缓存在 CDN 节点上的对应资源进行清除。当客户端访问资源时，CDN 回源到源站获取对应资源，并将资源缓存到节点上，从而保证用户获取到最新资源。


1. 在<a href="https://app.yunxin.163.com/index#/" target="_blank">网易云信控制台</a>的首页单击指定应用名称。

2. 在左侧导航栏选择**产品功能** > **点播**，单击**分发加速配置** 页签。

  
    ![刷新预热.png](https://yx-web-nosdn.netease.im/common/cc7d78366f37e8b8fa71025f12520d68/刷新预热.png)



3. 单击**刷新预热**右侧的**子功能配置**。


4. 在**文件刷新**页签中，配置 CDN 缓存中待清除文件的URL地址。


    ![配置文件预热.png](https://yx-web-nosdn.netease.im/common/42a76796abea63c6eb93b75d295e74ab/配置文件预热.png)


5. 单击**提交**。



## 按目录刷新缓存

目录刷新是以目录为单位，将该目录缓存在 CDN 节点上的所有文件进行清除。当客户端访问资源时，CDN 回源到源站获取对应资源，并将资源缓存到节点上，从而保证用户获取到最新资源。

1. 在<a href="https://app.yunxin.163.com/index#/" target="_blank">网易云信控制台</a>的首页单击指定应用名称。

2. 在左侧导航栏选择**产品功能** > **点播**，单击**分发加速配置** 页签。

  
    ![刷新预热.png](https://yx-web-nosdn.netease.im/common/cc7d78366f37e8b8fa71025f12520d68/刷新预热.png)



3. 单击**刷新预热**右侧的**子功能配置**。
4. 单击**目录刷新**页签，配置待清除的目录的URL地址。

    ![配置目录刷新.png](https://yx-web-nosdn.netease.im/common/8556ea89730e3e8001d011152b419a6c/配置目录刷新.png)

5. 单击**提交**。

## 文件预热

文件预热是指将文件提前缓存到 CDN 节点。当客户端访问资源时，CDN 无需再从源站拉取该资源，可以提升访问速度。

1. 在<a href="https://app.yunxin.163.com/index#/" target="_blank">网易云信控制台</a>的首页单击指定应用名称。

2. 在左侧导航栏选择**产品功能** > **点播**，单击**分发加速配置** 页签。

  
    ![刷新预热.png](https://yx-web-nosdn.netease.im/common/cc7d78366f37e8b8fa71025f12520d68/刷新预热.png)



3. 单击**刷新预热**右侧的**子功能配置**。
4. 单击**文件预热**页签，配置待预热的文件的 URL 地址。


    ![配置文件预热.png](https://yx-web-nosdn.netease.im/common/9a8af94143e8d27ef288fbb66c037ca3/配置文件预热.png)


5. 单击**提交**。
