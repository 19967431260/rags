 
## 功能概述
云信点播产品针对短视频/长视频场景中可能发生的内容违规风险，提供了视频、音频内容安全审核服务，通过对音视频内容智能识别和筛选，极大降低色情、暴恐、涉政等内容违规风险，降低人工审核成本。

云信内容安全审核服务基于海量标注数据和深度学习算法实现，从语音、视觉等多维度精准识别直播流媒体信息中包含的违禁内容，支持涉黄识别、暴恐涉政识别、广告识别、无意义直播识别、音频审核等多个功能模块。

- 视频审核：对点播视频截帧画面进行审核，自动检测视频直播的内容是否涉嫌违规，违规的内容将会通过回调的方式返回给用户，用户可对违规内容进行处理。
- 语音审核：对点播音频内容进行审核，自动检测语音内容是否涉嫌违规，违规的内容将会通过回调的方式返回给用户，用户可对违规内容进行处理。


## 功能架构

![](https://yx-web-nosdn.netease.im/quickhtml%2Fassets%2Fyunxin%2Flive%2F20210419%2Flive-img-2-1.png)


**开发者调用接口提交审核任务**：
- 云信提供了点播视频/音频内容安全审核服务端API，开发者可以通过调用接口对已上传的视频内容发起内容安全审核任务。

**内容安全审核服务**：
- 发起内容安全审核任务后，点播服务会将视频URL作为视频源送到内容安全检测引擎，内容安全检测引擎会分别对视频或音频内容进行检测。
- 内容安全检测引擎发现疑似或确定的违规内容，则将当前视频或云盘的检测结果通知给客户应用服务器。

**接收内容安全审核结果**：
  - 客户应用服务器接收到检测结果后，可以根据检测结果情况进行相关的业务处理逻辑，例如检测到违规视频或音频时，可以对当前视频进行删除或屏蔽处理。
