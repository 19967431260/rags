 
使用点播内容安全审核功能之前，请先在网易云信控制台进行开通。

::: note note
点播内容安全审核服务现已开通免费试用，您可以在控制台申请试用。申请通过后，可以免费享受 15 天的点播内容安全审核服务，在 15 天内可以享受高达 500次 的点播内容安全审核。
:::


## <span id="前置条件">前置条件</span>

需要开通点播服务，准备一个可以接收抄送的HTTP服务地址，并确保地址长期有效。云信会将点播内容安全审核结果抄送到该地址。

## <span id="开通配置说明">开通配置说明</span>

1. 在<a href="https://app.yunxin.163.com/index#/" target="_blank">网易云信控制台</a>的首页单击指定应用名称。

2. 在左侧导航栏选择**产品功能** > **点播**，单击**安全通** 页签。

   ![开通安全通.png](https://yx-web-nosdn.netease.im/common/78f9974c3deae3c0cfee2595a76a2a23/开通安全通.png)

3. 单击**免费试用**或**开通正式版**。

   ::: note note
   正式版功能涉及计费，请单击**开通正式版**，提交工单联系云信商务经理沟通计费策略。
   :::

  

4. 申请试用后，单击**子功能配置**。

   ![安全通子功能配置.png](https://yx-web-nosdn.netease.im/common/d2089051a3f11b9d736d7c67db34a05a/安全通子功能配置.png)




5. 在**点播安全通检测 子功能配置**页面，单击**抄送地址**右侧的**编辑**按钮，设置**内容安全服务配置**的结果抄送地址。

   内容审核结果会以消息抄送方式发送至配置的抄送地址。

   ![配置点播安全通.png](https://yx-web-nosdn.netease.im/common/89ceabf3988900c1cc7ff59010b1e7bb/配置点播安全通.png)

6. 单击机器过检开关，开启机器过检，并设置相关参数。


   参数 | 描述
   ---- | -------------- | 
   过检类型| 过检类型支持：<ul><li>视频与音频同时过检<li>仅过检视频 | 
   截图频率 | 截图频率默认为 5 秒，支持设置为 1~60 秒。 | 



7. 单击**保存**。



## <span id="提交内容审核">提交内容审核</span>

- 控制台开通和配置后，开发者每次上传完视频时，可以通过调用提交内容审核接口发起对该视频的内容审核任务。
- 具体接口说明请参见[提交内容安全审核](https://doc.yunxin.163.com/vod/docs/zkzNTQ1ODc?platform=server#提交内容安全审核)。



## <span id="接收审核结果">接收审核结果</span>

- 控制台配置抄送地址后，云信点播服务会将点播视频内容审核结果通过该地址通知给客户。
- 点播内容审核结果信息请参见[查询内容审核结果](https://doc.yunxin.163.com/vod/docs/zkzNTQ1ODc?platform=server#查询内容审核结果)。



## <span id="点播视频处理">点播视频处理</span>

- 开发者可以通过使用提供了丰富的点播视频管理功能，根据接收到的内容审核结果进行处理。
- 若发现违规内容，开发者可以上传的音视频文件进行删除或屏蔽用户访问。
- 具体操作可以通过控制台对文件进行处理，也可调用[服务端API](/docs/jY3NDM4Nzc/jI0Njk0NzU?#4%20%E5%AA%92%E8%B5%84%E7%AE%A1%E7%90%86)进行处理。
