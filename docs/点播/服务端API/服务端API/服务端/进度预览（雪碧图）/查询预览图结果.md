调用本接口可查询视频生成的预览快照图（雪碧图，Sprite Images）的信息，方便客户端在播放视频的同时拖动进度条预览其他时间区间的视频截图。

## 请求信息

### 请求频率

请参考 [接口概述](https://doc.yunxin.163.com/vod/server-apis/TU3MzkwNDk?platform=server#%E9%A2%91%E7%8E%87%E8%AF%B4%E6%98%8E) 频率说明章节。

### 请求 URL

```
POST  http://vcloud.163.com/app/vod/previewSnapshot/get
```

### 请求头参数

请参考 [接口概述](https://doc.yunxin.163.com/vod/server-apis/TU3MzkwNDk?platform=server#%E5%85%AC%E5%85%B1%E5%8F%82%E6%95%B0) 公共参数章节。

### 请求体参数

| 参数 | 类型 | 是否必需 | 说明 |
|---|---|---|---|
| vid | Long | 是 | 点播视频的 ID。 |

## 响应信息

### 响应体参数

参数 | 类型 | 说明 |
--- | --- | --- |
ret | Object | 返回的具体业务数据。 |
  previewsInfo | Object | 预览图布局信息。 |
    columns | Integer | 列数。 |
    interval | Integer | 截图间隔，单位为秒。 |
    rows | Integer | 行数。 |
    width | Integer | 预览图宽。 |
    height | Integer | 预览图高。 |
  previewList | List\<JSON> | 预览图列表。 |
    imageId | Integer | 预览图 ID。 |
    objectKey | String | 预览图的文件名称。 |
    beginTime | Integer | 预览图位于视频进度的具体时间点。 |
    url | String | 预览图存储的地址。 |
code | Integer | 返回的 HTTP 状态码，请参考 [状态码](https://doc.yunxin.163.com/vod/server-apis/jg5MjczMzg?platform=server)。 |
requestId | String | 请求 ID。 |
