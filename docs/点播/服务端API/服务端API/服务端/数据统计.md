该接口用于查询点播的存储使用统计数据（查询粒度为天，查询的开始时间不能超过当前时刻 90 天，且不能查询当天数据）。<!-- 数据统计模块主要展示存储统计的查询功能。 -->

## 请求信息

### 请求频率

请参考 [接口概述](https://doc.yunxin.163.com/vod/server-apis/TU3MzkwNDk?platform=server#%E9%A2%91%E7%8E%87%E8%AF%B4%E6%98%8E) 频率说明章节。

### 请求 URL

```
POST  http://vcloud.163.com/app/vod/stats/storage
```

### **<span id="输入参数">请求体参数</span>**

| 参数 | 类型 | 是否必选 | 说明 |
| :---- | :---- | :---- | :---- |
| startDate | Long | 是 | 需要查询的开始时间戳（单位：毫秒）。 |
| endDate | Long | 是 | 需要查询的结束时间戳（单位：毫秒）。 |
| currentPage | Int | 是 | 获取存储数据列表分页后的索引。 |
| pageSize | Int | 是 | 获取存储数据列表一页的记录数，取值范围：[1,500]。 |

### **<span id="示例代码">请求示例</span>**

```cURL
curl -X POST 
    -H "Content-Type: application/json;charset=utf-8" 
    -H "AppKey: 027338bf05cc4a65b5d98bc9d6********" 
    -H "Nonce: 1" 
    -H "CurTime: 1465723418" 
    -H "CheckSum: 61bbfd88c51028a2a28b3628584e65a2abe7ae13" 
    -d '{"startDate":1490976000000,"endDate":1491062400000,"type":5,"currentPage":1,"pageSize":10}'
https://vcloud.163.com/app/vod/stats/storage
```

## 响应信息

### **<span id="输出参数">响应参数</span>**

参数 | 类型 | 说明 |
:---- | :---- | :---- |
code | Int | 状态码。 |
msg | String | 错误信息。 |
ret | Object | 响应体。 |
   totalStorage | Double | 查询时段的总存储量（单位：GB）。 |
   startDate | Long | 需要查询的开始时间戳（单位：毫秒）。 |
   endDate | Long | 需要查询的结束时间戳（单位：毫秒）。 |
   currentPage | Int | 当前取得页数。 |
   pageSize | Int | 一页的记录数目。 |
   pageNum | Int | 总页数。 |
   totalRecords | Int | 获取存储数据的数目。 |
   type | Int | 统计粒度（2：每天）。 |
   storageList | List | 获取的存储列表。 |
     begintime | Long | 起始时间（单位：毫秒）。 |
     endtime | Long | 结束时间（单位：毫秒）。 |
     storage | Double | 存储量（单位：GB）。 |
