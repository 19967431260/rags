## <span id="基本描述">基本描述</span>

网易云信视频点播（VOD）提供在线或离线视频上传、视频存储、视频转码服务，视频经过CDN全网分发和加速，实现视频在PC（Web）、iOS以及Android端的多终端播放。云视频点播（VOD）提供Http接口上传和SDK上传视频，以及iOS、Android端的播放器SDK，并提供Web端的管理控制台及服务端API接口，帮助用户快速搭建安全、稳定、流畅的点播平台和应用。

目前，网易云信视频点播面向注册用户开放。

## <span id="核心能力">核心能力</span>

### <span id="海量音视频文件存储">海量音视频文件存储</span>
提供海量的文件存储服务，服务可用性99.9%，数据可靠性达99.999999999%，并通过同城热备和异地灾备方案，使海量音视频永久存储，安全无忧。
### <span id="极致加速体验">极致加速体验</span>

具备大规模全网分发能力，支持千万级并发直播观看需求，为音视频点播提供加速服务，带来更快速、更流畅的视听体验。
### <span id="强悍的转码能力">强悍的转码能力</span>

提供多路、高效率转码服务， 支持多种输入输出格式。可设定不同类型的转码模板，支持MP4、FLV和HLS等转码格式。
### <span id="内容安全保护方案">内容安全保护方案</span>

支持播放鉴权、视频加密等安全保护以及涉黄、暴恐、涉政、广告等内容智能识别。
### <span id="支持多平台播放">支持多平台播放</span>

提供多平台播放器SDK，迅速接入点播服务，支持Andriod/iOS/Windows/Web等平台。
