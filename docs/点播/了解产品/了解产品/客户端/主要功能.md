## <span id="视频上传">视频上传</span>

| **功能名称** | **功能介绍** |
| ---- | ---- |
| 批量上传 | 支持多文件批量上传 |
| Web 端视频上传 | 进入点播控制台页面，可选择本地视频文件进行上传 |
| 断点续传 | 支持文件断点续传，提供后台接口查询断点 |
| 文件分片上传 | 上传文件大小无限制，大文件需分片后上传，分片大小不超过 4MB |
| 上传 SDK | 提供多种语言的视频上传 SDK，包括 Android、CSharp、Golang、JavaScript、Java、Node.js、Objective、PHP |
| 移动端 HTTPS 上传 | 移动端支持 HTTPS 协议上传 |

## <span id="视频转码">视频转码</span>

| **功能名称** | **功能介绍** |
| ---- | ---- |
| 多视频转码 | 选择模板进行转码，支持九种转码输出格式：流畅（MP4、FLV、HLS）、标清（MP4、FLV、HLS）、高清（MP4、FLV、HLS）<br>支持批量转码、转码去重 |
| 智码超清转码 | 使用网易云信自研编码技术进行转码，实现同等画质比普通转码节省 30% 左右码率 |
| 极速转码 | 适用于 30 分钟以上的长视频，通过对视频分片并行转码，可将转码速度提升 5 倍 |
| 自定义水印 | 支持转码后输出视频设置自定义的水印（图片、文字），包括静态水印和动态水印 |
| 外挂字幕 | 转码支持导入外部字幕文件并指定字幕编码格式 |
| 转码后回调 | 可设置回调地址，供转码完成后回调 |
| 视频加密 | 将视频文件转成 FLV 或 HLS 格式的加密视频，播放器对加密视频进行解密播放，保障视频安全 |
| 多码率自适应 | 根据网络环境，播放器选择最适合的码流进行播放，确保视频播放流畅性（仅支持 HLS 协议） |
| 视频截图 | 截取视频某一帧图片，支持单张截图、多张截图、平均截图 |
| 转短视频/动图 | 支持截取某一段视频转存为短视频或 GIF 动图 |

## <span id="视频安全">视频安全</span>

| **功能名称** | **功能介绍** |
| ---- | ---- |
| URL 鉴权 | 支持播放 URL 防盗链，可自定义鉴权 Key 及过期时间 |
| 视频加密 | 支持对视频数据加密，防止视频泄露和盗链问题 |
| 内容安全审核 | 支持对视频文件进行内容审核，自动检测视频和语音内容是否涉嫌违规 |
| 视频屏蔽 | 支持对视频文件进行不删除仅屏蔽访问的处理 |

## <span id="视频管理">视频管理</span>

| **功能名称** | **功能介绍** |
| ---- | ---- |
| 视频查询 | 支持单视频查询、批量查询、用户所有视频列表查询 |
| 视频编辑 | 可对视频文件名称、视频分类 ID、视频描述信息进行修改 |
| 视频删除 | 支持单视频删除、批量删除 |
| 视频屏蔽、恢复 | 支持对所选视频进行屏蔽处理，屏蔽后的视频将无法播放及下载，视频恢复后可正常播放及下载 |
| 视频截图 | 可对视频按指定的偏移位置截取不同尺寸的图片<br>可对视频按指定的时间间隔一次性截取多张图片并生成一张图片 |
| 封面设置 | 可将截图生成的图片或本地上传的图片设置为视频封面 |
| 视频分类 | 可为视频设置所属分类、支持对分类的查询、创建、删除等操作 |

## <span id="数据统计">数据统计</span>

| **功能名称** | **功能介绍** |
| ---- | ---- |
| 存储统计 | 对存储按天为粒度进行统计 |
| 流量统计 | 对流量按天为粒度进行统计 |
| 带宽统计 | 对带宽峰值按不同时间粒度进行统计 |
| 转码时长统计 | 按天粒度对用户转码输出文件时长进行统计 |
| 分类数量统计 | 对不同分类下的文件数量进行统计 |
| 请求频率统计 | 对 API 接口的请求频率进行统计监控 |
