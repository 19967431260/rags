本文介绍了从创建网易云信开发者账号到正式使用短信服务的整个流程，适用于首次试用短信产品的客户。

## 第一步：注册账号

- 如果您还没有网易云信账号，请访问 [注册](https://app.netease.im/regist)。

- 如果您已经有网易云信账号，请直接 [登录](https://app.netease.im/login)。

详细步骤请参考网易云信控制台的 [注册账号并登录控制台](https://doc.yunxin.163.com/console/concept/DU0ODE4NTA?platform=console)。

## 第二步：创建应用

创建应用是体验或使用网易云信各款产品和服务的首要前提，您可以参考 [创建应用并获取 AppKey](https://doc.yunxin.163.com/console/concept/TIzMDE4NTA?platform=console) 在网易云信控制台创建一个应用，并查看应用密钥（App Key）。

## 第三步：开通短信并购买资源包

1. 在 [网易云信控制台](https://app.yunxin.163.com/global/home) 首页 **应用管理** 中选择应用进入应用配置页面，然后单击 **短信** 下的 **正式开通** 按钮。

    <img alt="开通短信.png" src="https://yx-web-nosdn.netease.im/common/515ceb1231f1bdf9d5f807b68959f32f/开通短信.png" style="width:30%;border: 0px solid #BFBFBF;">

2. 勾选 **我已阅读并同意《网易云信服务条款》和《禁止从事任何违法犯罪活动的声明》**，并单击 **开通**。

    <img alt="短信开通.png" src="https://yx-web-nosdn.netease.im/common/472c725d8167c6e83295fc6d6187a4d1/短信开通.png" style="width:100%;border: 0px solid #BFBFBF;">

3. 开通后，单击 **立即购买资源包**。

    :::note note
    由于短信为资源型产品，开通功能后还需要购买相应的资源包短信条数才可正常使用，否则短信将会由于无资源包而无法正常使用。请购买充足的资源包短信条数，以保障短信业务的稳定使用量。
    :::

4. 按应用的实际需求配置服务，控制台会生成预估费用。

    - 关于具体的计费策略，请参考 [计费方式](https://yunxin.163.com/price/sms) 或咨询商务经理。
    - 网易云信短信资源包购买成功后不支持退款，建议您按需购买。

        <!-- <img alt="短信资源包订单.png" src="https://yx-web-nosdn.netease.im/common/83c3c60e6015e165d9e038c9d132ede8/短信资源包订单.png" style="width:100%;border: 0px solid #BFBFBF;"> -->

5. 勾选 **我已阅读并同意《网易云信服务条款》和《禁止从事任何违法犯罪活动的声明》**，并单击 **提交订单**。

6. 单击 **立即支付** 完成订单支付。

    - 支持线上（支付宝）和线下支付。
    - 订单有效期 7 天，过期将自动关闭，还请尽快支付。
    - 充值后的账户余额，可用于网易云信所有产品后续消耗扣费。

        :::note note
        若在开通后，未购买资源包，后续可以在 **短信** 的 **功能配置** 页可以查看 **短信资源包概览** 信息，并在相应的短信资源包操作列，单击 **立即购买**。

        <img style="width:100%" src="https://yx-web-nosdn.netease.im/common/f6f043061a1bc106631cfa65fa379699/短信资源包概览.png" />
        :::

## 第四步：申请短信签名

1. 在 [网易云信控制台](https://app.yunxin.163.com/global/home) 首页 **应用管理** 中选择应用进入应用配置页面，然后单击 **短信** 下的 **功能配置** 按钮。

    <img alt="短信功能配置.png" src="https://yx-web-nosdn.netease.im/common/68374f6fd59bba36fe65041b23a89435/短信功能配置.png" style="width:30%;border: 0px solid #BFBFBF;">

2. 单击 **国内短信** 或 **国际/港澳台短信** 右侧的 **签名/正文模板管理**。

    <img alt="签名模板.png" src="https://yx-web-nosdn.netease.im/common/8c8a6dbf4b15142c7bb289fd9bfd582a/签名模板.png" style="width:100%;border: 0px solid #BFBFBF;">

3. 在 **签名管理** 页签中，单击 **创建签名** 按钮。

    <img alt="创建签名.png" src="https://yx-web-nosdn.netease.im/common/6724d757912ce2b6c1207f66d405a7b0/短信创建签名.png" style="width:100%;border: 0px solid #BFBFBF;">

4. 根据界面提示填写签名信息，然后单击 **提交审核**。

5. 等待网易云信审核通过。

## 第五步：创建短信正文模板

网易云信提供验证码短信模板（含文本验证码短信和语音验证码短信）、通知类短信模板和营销类模板供您选择。语音短信目前只支持语音验证类，故配置语音短信模板需要在验证码模板中配置。

1. 在 [网易云信控制台](https://app.yunxin.163.com/global/home) 首页 **应用管理** 中选择应用进入应用配置页面，然后单击 **短信** 下的 **功能配置** 按钮。

    <img alt="短信功能配置.png" src="https://yx-web-nosdn.netease.im/common/68374f6fd59bba36fe65041b23a89435/短信功能配置.png" style="width:40%;border: 0px solid #BFBFBF;">

2. 单击 **国内短信** 或 **国际/港澳台短信** 右侧的 **签名/正文模板管理**。

    <img alt="签名模板.png" src="https://yx-web-nosdn.netease.im/common/8c8a6dbf4b15142c7bb289fd9bfd582a/签名模板.png" style="width:100%;border: 0px solid #BFBFBF;">

3. 在 **正文模板管理** 页签中，单击 **创建正文模板** 按钮。

    <img alt="创建正文模板.png" src="https://yx-web-nosdn.netease.im/common/6a09a22bfa06e5f0b33ebdf71dfb2677/短信创建正文模板.png" style="width:100%;border: 0px solid #BFBFBF;">

4. 根据界面提示填写正文模板，然后单击 **提交审核**。填写模板之前，您需要提前提前设置 [短信签名](#signature)。

    <img alt="短信正文模板.png" src="https://yx-web-nosdn.netease.im/common/189807fff6e8d42366425a972ec5e432/正文模板.png" style="width:100%;border: 0px solid #BFBFBF;">

5. 等待网易云信审核通过。

## 第六步：调用接口正式使用

网易云信审核通过后，您可以调用发送模板短信接口，调试成功后即可使用短信功能。接口调用具体请参考 [接口指南](https://doc.yunxin.163.com/sms/server-apis/jg2NDEyMzI?platform=server)。

::: note notice
接口调用成功后，网易云信会将短信内容提交到运营商，由运营商识别短信内容并确定是否下发。仅当用户终端收到短信回执消息中包含 `DELIVRD` 时，表示短信发送成功。请参考 [短信回执抄送](https://doc.yunxin.163.com/sms/server-apis/zMxMTQ4MDA?platform=server#短信回执下行消息)。
:::

## 第七步：（可选）设置短信回执抄送

网易云信提供短信回执抄送功能，即在调用接口发送短信后，服务器会将短信发送结果或用户回复的短信内容同步至您指定的本地服务器，帮助您掌握短信的发送成功率或其他业务。

- 有关短信回执抄送服务的技术逻辑，请参考 [消息抄送服务概述](https://doc.yunxin.163.com/messaging/server-apis/jQ4ODE0NTY?platform=server)。

- 有关如何设置短信回执抄送，请参考 [短信回执抄送](https://doc.yunxin.163.com/sms/server-apis/zMxMTQ4MDA?platform=server)。
