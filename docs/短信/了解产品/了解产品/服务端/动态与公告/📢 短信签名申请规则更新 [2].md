尊敬的网易云信短信用户：

为了进一步规范短信市场秩序、保障用户权益及短信下发的成功率，网易云信将依据工信部对短信签名实名制管理的最新要求进一步加强了对签名审核的管控。

为确保合规，自即日起短信签名需符合以下任一条件，不再支持网站、公众号、小程序等其他签名类型：

| 签名类型 | 签名内容 | 要求及申请说明 |
| --- | --- | --- |
| 公司 | 公司全称 | 与企业营业执照上的名称完全一致。 |
| ^^ | 简称 | 简称需包含在营业执照全称中，且须与其他企业无混淆、重名。例如 **北京 XX 科技有限公司** 对应简称可为 **北京 XX 科技**。 |
| App | App 名称 | 需在 [工信部 ICP/IP 地址/域名信息备案管理系统](https://beian.miit.gov.cn/) 可查。申请说明：需填写任一平台应用商店的展示页链接。 |
| 商标 | 商标名称 | 商标需已在 [中国商标网](https://sbj.cnipa.gov.cn/sbj/index.html) 完成注册。 |
| 政府/机关事业单位/其他机构 | 组织全称 | 与组织/机构证照上的名称完全一致。 |
| ^^ | 组织简称 | 简称需包含在证照上的全称中，且须与其他组织/机构无混淆、重名。 |

如果您正在使用的签名不符合以上规范，建议停用并更换为满足上述最新要求且已在运营商完成实名报备的签名进行短信下发，以保障短信下发的成功率。如需协助，请联系您的网易云信专属客户经理。
