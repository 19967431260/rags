短信服务（Short Message Service）是网易云信为用户提供的一种通信服务的能力，分为国内短信服务和国际/港澳台短信服务。网易云信目前提供验证码短信，通知短信，运营短信、国际短信、语音验证码、视频短信等服务。更多详情，请访问 [短信服务产品详情页](https://yunxin.163.com/sms)。

## 产品优势

- **网易云信短信**：具体有全网覆盖、3-5 秒可达、超高到达率、7*24 小时服务监控等优势。按量付费、阶梯定价，发送越多单价越低。

- **短信接口 API**：调用简单，加快接入速度。详情请参考 [短信接口列表](https://doc.yunxin.163.com/sms/server-apis/jg2NDEyMzI?platform=server)。

## 功能亮点

- **全网覆盖**：移动、联通、电信三网，普通、170 号段全覆盖。

- **99% 高达到率**：企业专属短信通道，多通道智能切换，保障高到达率。

- **发送状态可知**：实时监控发送状态，及时发送反馈信息。

- **主备通道**：完备的监控和资源配置系统，主备通道规避风险。

## 短信服务术语表

本章节介绍网易云信短信服务中的核心概念和术语。

名词 | 说明
--- | ---
短信服务 | 短信服务（Short Message Service）是广大企业客户快速触达手机用户所优选使用的通信能力。支持国内和国际快速发送验证码、短信通知和推广短信，服务范围覆盖全球 200 多个国家和地区。国内短信支持三网合一专属通道，与工信部携号转网平台实时互联。电信级运维保障，实时监控自动切换，到达率高达 99%。
国内短信 | 国内短信是网易云信为国内企业用户提供的短信发送服务，通过 API 调用短信发送能力，将指定信息发送至国内手机号码，用于企业向手机号码发送验证码、短信通知、推广短信等短信。
国际/港澳台短信 | 国际/港澳台短信是网易云信为全球企业用户提供的短信发送服务，通过 API 或 SDK、群发助手方式调用短信发送能力，将指定信息发送至境外手机号码，用于企业向注册用户发送验证码、短信通知、推广短信等短信。支持从中国内地向中国香港、中国澳门和中国台湾及其他国家和地区手机号码发送短信。
文本短信 | 文本短信是网易云信为企业用户提供的纯文本短信发送服务，通过 API 方式调用短信发送功能，将指定信息发送至全球手机号码，用于企业向用户发送验证码、短信通知、推广短信等短信。
验证码 | 验证码是网易云信为企业用户提供的验证码发送服务，国内短信验证码支持三网合一专属通道，秒级可达。网易云信短信支持发送国际短信验证码，支持全球语言，适用于应用或网站注册、安全登录、支付认证、身份认证、密码找回、账号绑定等应用场景。
短信通知 | 短信通知是网易云信为企业用户提供的短信通知发送服务，支持国际通知短信，快速触达全球用户，助力企业海外拓展，适用于订单通知、支付通知、物流通知、会议通知、政府通知、生活服务类通知、跨境订单通知、跨境物流通知等应用场景。
推广短信 | 推广短信是网易云信为企业用户提供的短信通知发送服务，国内推广短信支持回复，双向连接，帮助企业与客户建立有效即时互动。支持全球多种场景的短信发送服务，适用于新品宣传、会员关怀、商品促销、活动邀请、跨境营销等应用场景。
短信模板 | 短信模板是具体发送的短信内容，由变量和模板内容构成，需要审核通过后才可以使用。
短信签名 | 短信签名是一种快捷、方便的个性化签名方式。当发送短信时，短信平台会根据设置，在短信内容里附加个性化签名，再发送给被叫手机号码。
