网易云信 IM 服务端支持用户登录/登出事件记录的查询功能。

## <span id='登录/登出事件记录查询'>登录/登出事件记录查询</span>

### 功能描述

- 根据时间段查询用户的登录登出记录，每次最多返回 100 条。  
- 不提供分页支持，第三方需要根据时间段来查询。  


### 调用前提

该 API 需要开通**登录登出事件记录查询**功能才能正常使用，且不支持查询开通前的登录登出事件记录。

若未开通，请前往[云信控制台](https://id.grow.163.com/?h=media&t=media&clueFrom=nim&from=nim&locale=zh_CN&referrer=https%3A%2F%2Fapp.yunxin.163.com&i18nEnable=true)开通。配置路径：选择待开通该功能的应用，然后进入**IM 即时通讯 > 功能配置 > 全局功能 > 登录登出事件记录查询**。


![登录登出事件记录查询.png](https://yx-web-nosdn.netease.im/common/d1cb28f50d1d8db2584a0a580114fb4f/登录登出事件记录查询.png)


### API 使用限制

单个应用默认最高调用频率请参考 [频控说明](https://doc.yunxin.163.com/messaging/server-apis/zk4Mzg0MjE?platform=server)。

### URL
```
POST  https://api.yunxinapi.com/nimserver/history/queryUserEvents.action HTTP/1.1
Content-Type: application/x-www-form-urlencoded;charset=utf-8
```

### 请求参数
- POST 请求中 Headers 的设置请参考[接口概述](/docs/TM5MzM5Njk/jk3MzY2MTI)。
- POST 请求中 Body 的设置如下：
<table>
<thead><tr><th>参数</th><th><div style="width: 30pt">类型</div></th><th>必须</th><th>说明</th></tr></thead>
  <tr><td> accid </td><td>String</td><td>是</td><td>要查询用户的accid</td></tr>
<tr><td> begintime</td><td>String</td><td>是</td><td>开始时间，毫秒级
</td></tr>
<tr><td> endtime </td><td>String</td><td>是</td><td>截止时间，毫秒级</td></tr>
<tr><td> limit </td><td>int</td><td>是</td><td>本次查询的记录数量上限(最多100条),小于等于0，或者大于100，会提示参数错误</td></tr>
<tr><td> reverse </td><td>int</td><td>否</td><td>1按时间正序排列，2按时间降序排列。其它返回参数414错误。默认是按降序排列</td></tr>
</table>

### 示例
#### **请求示例（curl）**
```curl
curl -X POST -H "AppKey: go9dnk***mgq3" -H "Nonce: 4tggg***3t23t" -H "CurTime: 1443592222" -H "CheckSum: 9e9db***3f86" -H "Content-Type: application/x-www-form-urlencoded" -d 'accid=11111&begintime=1448948957018&endtime=1448955895518&limit=50' 'https://api.yunxinapi.com/nimserver/history/queryUserEvents.action'
```
