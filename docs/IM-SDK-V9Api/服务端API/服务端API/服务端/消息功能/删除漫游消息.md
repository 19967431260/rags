<!--keywords: 漫游消息,删除漫游消息 -->
本文介绍如何通过云信服务端 API 删除漫游消息。


## 功能描述

网易云信服务端支持按会话类型（单聊、高级群、超大群）删除漫游消息。

## 使用场景

漫游消息中包含涉黄、涉政等敏感信息时：


- 应用管理员调用本接口删除违规消息。
- 应用服务器通过本接口自动触发违规消息删除。

## API 使用限制

单个应用默认最高调用频率请参考 [频控说明](https://doc.yunxin.163.com/messaging/server-apis/zk4Mzg0MjE?platform=server)。


## URL 

```http
POST https://api.yunxinapi.com/nimserver/msg/delRoamSession.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```



## 请求参数

- POST 请求中 Headers 的设置请参考[API调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。

- POST 请求中 Body 的设置如下：



<table>
<tr> <th style="width:160px">参数</th><th style="width:80px">类型</th><th>必填</th><th>说明</th></tr></thead>
<tr> <td> type </td><td> int </td><td>是</td><td>
           会话类型，1-单聊，2-高级群，3-超大群，其他返回 414</td></tr>
<tr> <td> from </td><td>String</td><td>是</td><td>
           发送者的云信 IM 账号（accid），最大长度 32 个字符</td></tr>
<tr> <td> to </td><td>String</td><td>是</td><td>
           消息接收方。<li>如果 type=1 则需传入消息接收者的云信 IM 账号（accid）。</li><li>如果 type=2 则需传入高级群 ID （tid）。</li><li>如果 type=3 则需传入超大群 ID （tid）。</td></tr>
</table>


## 示例


### **cURL请求示例**

```curl
curl -X POST -H "AppKey: go9dnk4**0803mgq3" -H "Nonce: 4tggge**23t23t" -H "CurTime: 1443592222" -H "CheckSum: 9e9db3b6c9abb2e1962cf3e6f7316fcc55583f86" -H "Content-Type: application/x-www-form-urlencoded" -d 'type=2&from=cs4&to=2775725369' 'https://api.yunxinapi.com/nimserver/msg/delRoamSession.action'
```

### **请求成功返回示例**

http 响应:json

```json
"Content-Type": "application/json; charset=utf-8"
{
    "code": 200
}

```

### **请求失败返回示例**

``` 
"Content-Type": "application/json; charset=utf-8"
{
    "code":414,
    "desc":"type must be 1 or 2"  // type值不合法
}
```
