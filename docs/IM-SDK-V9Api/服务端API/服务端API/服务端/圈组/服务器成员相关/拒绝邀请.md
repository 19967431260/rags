

云信服务端提供 API，支持拒绝“加入服务器”的邀请。


## 功能介绍

被邀请后，受邀方将收到邀请加入服务器的系统通知。受邀方可调用服务端 API 拒绝邀请。受邀方拒绝邀请后，邀请方会收到一条“拒绝邀请”（`SERVER_MEMBER_INVITE_REJECT(2)`）的系统通知。

## 调用时机


收到“邀请服务器成员”（SERVER_MEMBER_INVITE(1)）的系统通知时。



## URL 
```http
POST https://api.yunxinapi.com/nimserver/qchat/rejectServerInvite.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```



## 请求参数


- POST 请求中 Headers 的设置请参考[API调用方式](https://doc.yunxin.163.com/TM5MzM5Njk/docs/jk3MzY2MTI?platform=server)。
- POST 请求中 Body 的设置如下：


<table>
    <thead><tr> <th style="width:160px">参数</th><th style="width:80px">类型</th><th>必须</th><th>说明</th></tr></thead>
        <tr> <td>accid</td><td>String</td><td>是</td><td>受邀方的 IM 账号</td></tr>
        <tr> <td>serverId</td><td>Long</td><td>是</td><td>服务器 ID</td></tr>
        <tr> <td>inviteAccid</td><td>String</td><td>是</td><td>邀请者的 IM 帐号</td></tr>
        <tr> <td>msg</td><td>String</td><td>是</td><td>拒绝邀请的消息内容，最大长度 150 字符</td></tr>
        <tr> <td>requestId</td><td>Long</td><td>否</td><td>邀请 ID</td></tr>
</table>


## 示例代码
### cURL 请求示例
```curl
curl -X POST -H "AppKey: go9dnk49bk**lw0803mgq3" -H "Nonce: 4tggg**3t23t" -H "CurTime: 1443592222" -H "CheckSum: 9e9db3b6c9abb2e1962cf3e6f7316fcc55583f86" -H "Content-Type: application/x-www-form-urlencoded" -d 'serverId=1513535&accid=zhangsan&inviteAccid=lisi&msg=reject' 'https://api.yunxinapi.com/nimserver/qchat/rejectServerInvite.action'
```

### 返回示例
HTTP 响应：JSON
```json
"Content-Type": "application/json; charset=utf-8"
{
    "code":200
}

```
