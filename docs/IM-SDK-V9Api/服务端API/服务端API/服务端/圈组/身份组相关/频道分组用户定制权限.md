## <span id='创建频道分组下某人的定制权限'>创建频道分组下某人的定制权限</span>

在频道分组下为某个人定制权限。


### URL 
```http
POST http://api.yunxinapi.com/nimserver/qchat/createChannelCategoryUserIdentify.action  HTTP/1.1
Content-Type: application/x-www-form-urlencoded;charset=utf-8
```


### 请求参数


- POST 请求中 Headers 的设置请参考[API调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。

- POST 请求中 Body 的设置如下：

<table>
    <thead><tr> <th><div style="width:160px">参数</div></th><th><div style="width:80px">类型</div></th><th><div style="width:30px">必须</div></th><th>说明</th></tr></thead>
        <tr> <td>accid</td><td>String</td><td>是</td><td>操作者accid</td></tr>
        <tr> <td>serverId</td><td>String</td><td>是</td><td>服务器唯一标识</td></tr>
	    <tr> <td>categoryId</td><td>String</td><td>是</td><td>频道分组的唯一标识</td></tr>
	    <tr> <td>targetAccid</td><td>String</td><td>是</td><td>被操作的用户accid</td></tr>
</table>


### 示例
#### curl请求示例
```curl
curl -X POST -H "AppKey: go9d**w0803mgq3" -H "Nonce: 4tgg**t23t" -H "CurTime: 1443592222" -H "CheckSum: 9e9db3b6c9abb2e1962cf3e6f7316fcc55583f86" -H "Content-Type: application/x-www-form-urlencoded" -d 'serverId=1513535&accid="accid"&categoryId=10010&targetAccid="faccid"' 'http://api.yunxinapi.com/nimserver/qchat/createChannelCategoryUserIdentify.action'
```

#### 返回示例
http 响应:json
```json
"Content-Type": "application/json; charset=utf-8"
{
    "code": 200,
    "identify": {
        "serverId": 2,
        "categoryId": 1,
        "createtime": 1,
        "accid": "accid",
        "auths": "{\"1\":-1, \"2\":1}",
        "avatar": "name",
        "inviter": "icon",
        "joinTime": 1,
        "nick": "icon",
        "memberType": 1,
        "custom": "icon",
        "updatetime": 1
    }
}

```

### 状态码
200、403、414、416、431、500

具体请参考[code状态表](/docs/TM5MzM5Njk/Tk5ODIzNjk)

## <span id='修改频道分组下某人的定制权限'>修改频道分组下某人的定制权限</span>


修改频道分组下某人的定制权限。



### URL 
```http
POST http://api.yunxinapi.com/nimserver/qchat/updateChannelCategoryUserIdentify.action  HTTP/1.1
Content-Type: application/x-www-form-urlencoded;charset=utf-8
```



### 请求参数


- POST 请求中 Headers 的设置请参考[API调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。

- POST 请求中 Body 的设置如下：

<table>
    <thead><tr> <th><div style="width:160px">参数</div></th><th><div style="width:80px">类型</div></th><th><div style="width:30px">必须</div></th><th>说明</th></tr></thead>
        <tr> <td>accid</td><td>String</td><td>是</td><td>操作者accid</td></tr>
        <tr> <td>serverId</td><td>String</td><td>是</td><td>服务器唯一标识</td></tr>
	    <tr> <td>categoryId</td><td>String</td><td>是</td><td>频道分组的唯一标识</td></tr>
	    <tr> <td>targetAccid</td><td>String</td><td>是</td><td>被修改的用户的accid</td></tr>
	    <tr> <td>auths</td><td>String</td><td>是</td><td> <p>用于修改权限的 JSON 字段, 其格式为 {resource1:type1,resource2:type2,resource3:type3}。</p>
         <ul>
          <li>resource 表示权限, 其取值代表的具体权限如下：
            <ul>
<li>2：管理频道的权限，拥有该权限可创建、修改或删除频道 </li>
                <li>3：管理角色的权限，拥有该权限可创建、修改或删除身份组</li>
                <li>4：发送消息的权限</li>
                <li>9：撤回他人消息的权限</li>
                <li>10：删除他人消息的权限</li>
                <li>11：@ 他人的权限，拥有该权限可在发送消息时@他人</li>
                <li>12：@ 所有人的权限，拥有该权限可在发送消息时@所有人</li>
                <li>13：管理频道黑白名单的权限，拥有该权限可将服务器成员加入或移出频道黑名单</li>
                <li>15：实时互动频道：建立自己连接的权限 </li>
                <li>16：实时互动频道：断开他人连接的权限 </li>
                <li>17：实时互动频道：开启自己麦克风的权限 </li>
                <li>18：实时互动频道：开启自己摄像头的权限 </li>
                <li>19：实时互动频道：开启/关闭他人麦克风的权限 </li>
                <li>20：实时互动频道：开启/关闭他人摄像头的权限 </li>
                <li>21：实时互动频道：开启/关闭全员麦克风的权限 </li>
                <li>22：实时互动频道：开启/关闭全员摄像头的权限 </li>
                <li>23：实时互动频道：打开自己屏幕共享的权限 </li>
                <li>24：实时互动频道：关闭他人屏幕共享的权限 </li>
                <li>27: @身份组的权限，拥有该权限可在发送消息时@指定几个身份组的所有成员
            </ul>
        </li>
        <li> 
        <p>type 表示操作类型，其取值代表的操作如下：
        <ul><li>-1：关闭权限（deny）</li>
        <li>1：开启权限（allow）</li>
        <li>0: 继承服务器身份组的权限的开启/关闭状态</li></ul></p>
        </li>
        </ul>
                <p>注：单次最多可批量修改权限数为12。</p></td></tr>
	        <tr> <td>
</table>
