
云信服务端支持分页查询不在频道分组中的频道信息列表。

## URL

```http
POST https://api.yunxinapi.com/nimserver/qchat/queryHasNoCategoryChannelPage.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```

## 请求参数


- POST 请求中 Headers 的设置请参考 [API 调用方式](https://doc.yunxin.163.com/messaging/server-apis/jk3MzY2MTI?platform=server)。

- POST 请求中 Body 的设置如下：


<table>
    <thead><tr> <th style="width:160px">参数</th><th style="width:90px">类型</th><th>必填</th><th>说明</th></tr></thead>
        <tr> <td>serverId</td><td>Long</td><td>是</td><td>服务器 ID</td></tr>
        <tr> <td>accountId</td><td>Long</td><td>是</td><td>用户帐号 ID，返回服务器下用户可查看的频道分组</td></tr>
        <tr> <td>limit</td><td>Integer</td><td>否</td><td>本次查询的数量上限，最多返回 200 条结果</td></tr>
        <tr> <td>pageToken</td><td>String</td><td>否</td><td>分页参数，使用上一次请求成功返回的分页标识</td></tr>
</table>



## 返回参数

参数 |  说明
:-----|:--|:---
code  | 状态码  
pageInfo |   分页信息，具体见下表
data | 频道列表信息，具体见下表

pageInfo 参数说明如下：

参数| 类型| 说明
:-----|:--|:---
hasMore | Boolean | 是否有下一页
nextPageToken|String|分页标识

data 参数说明如下

参数| 类型| 说明
:-----|:--|:---
channelId | Long | 频道 ID
serverId| Long  | 服务器 ID 
name | String  | 频道名称
topic|String|频道主题
custom | String | 频道自定义扩展字段
owner |  String | 服务器所有者
type | Integer|频道类型，0-消息频道，1-实时互动频道，100-自定义频道
viewMode | Integer | 频道的查看模式，0-公开模式(默认)，1-私密模式
validFlag | Integer | 有效标志：0-无效，1-有效
createTime | Long | 创建时间
updateTime  | Long  | 更新时间
