

云信服务端支持批量查询频道黑白名单身份组。被加入黑名单或白名单的身份组即为**黑白名单身份组**。


## 功能介绍


如果传入的身份组，最终在查询到的 roleids 身份组列表中，则表示该身份组已被加入了频道的白名单或黑名单。


## URL 
```http
POST https://api.yunxinapi.com/nimserver/qchat/batchGetWhiteBlackIdentify.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```


## 请求参数

- POST 请求中 Headers 的设置请参考[API调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。

- POST 请求中 Body 的设置如下：




<table>
    <thead><tr> <th style="width:160px">参数</th><th  style="width:80px">类型</th><th>必须</th><th>说明</th></tr></thead>
        <tr> <td>serverId</td><td>Long</td><td>是</td><td>服务器 ID </td></tr>
        <tr> <td>channelId</td><td>Long</td><td>是</td><td>频道 ID </td></tr>
        <tr> <td>accid</td><td>String</td><td>是</td><td>操作者的 IM 账号</td></tr>
        <tr> <td>type</td><td>int</td><td>是</td><td>1 表示白名单，2 表示黑名单</td></tr>
        <tr> <td>roleids</td><td>String</td><td>是</td><td>服务器身份组 ID 列表，JSON Array格式，最多 100 个</td></tr>
</table>


## 返回参数


参数 | 说明
---- | --------------
code  | 状态码
identities  | 查询到的身份组列表，单个身份组所包含的字段信息参见[修复服务器身份组的返回参数中 identify 的字段说明](https://doc.yunxin.163.com/messaging/docs/DI4NzcyMTc?platform=server#修复服务器身份组的返回参数)
  



## 示例

### cURL 请求示例
```curl
curl -X POST -H "AppKey: go9dnk49**0803mgq3" -H "Nonce: 4tggg**23t23t" -H "CurTime: 1443592222" -H "CheckSum: 9e9db3b6c9abb2e1962cf3e6f7316fcc55583f86" -H "Content-Type: application/x-www-form-urlencoded" -d 'serverId=1513535&channelId=12222&type=1&timetag=0&roleids=["10"]' 'https://api.yunxinapi.com/nimserver/qchat/batchGetWhiteBlackIdentify.action'
```

## 状态码


该接口在 HTTPS Body 中返回请求的状态码，以下仅列出与接口业务相关的状态码。完整状态码请参见[状态码](https://doc.yunxin.163.com/messaging/server-apis/TM5NTk2Mzc?platform=server)。


状态码 | 说明   | 处理建议
:---- | :---------|:-----
200 | 请求成功  | -
403 | 非法操作或没有权限 |   <ul><li>检查是否已开通圈组功能</li><li>检查传入的 accid 是否为当前服务器和频道的成员，如非成员则无权限查询</li></ul>
404 | 对象不存在  | <ul><li>检查是否存在必传参数为空的问题</li><li>检查传入的服务器 ID 、频道 ID 、身份组 ID 或账号是否存在</li></ul>
414 | 参数错误 | 根据提示信息，检查传入参数的格式和限制条件
416 | 调用过于频道 | 降低调用频率 
419   |  数量超限  | 检查 roleids 所包含的账号数量是否超过了上限 100
