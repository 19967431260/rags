



修改频道分组黑白名单身份组。


## URL 
```http
POST http://api.yunxinapi.com/nimserver/qchat/updateCategoryWhiteBlackIdentify.action  HTTP/1.1
Content-Type: application/x-www-form-urlencoded;charset=utf-8
```


## 请求参数


- POST 请求中 Headers 的设置请参考[API 调用方式](https://doc.yunxin.163.com/messaging/docs/jk3MzY2MTI?platform=server)。

- POST 请求中 Body 的设置如下：


<table>
    <thead><tr> <th>参数</th><th>类型</th><th>必须</th><th>说明</th></tr></thead>
        <tr> <td>serverId</td><td>Long</td><td>是</td><td>服务器唯一标识</td></tr>
        <tr> <td>categoryId</td><td>Long</td><td>是</td><td>频道分组唯一标识</td></tr>
        <tr> <td>type</td><td>Long</td><td>是</td><td>1表示白名单，2表示黑名单</td></tr>
        <tr> <td>opeType</td><td>Long</td><td>是</td><td>1表示增加，2表示移除</td></tr>
        <tr> <td>accid</td><td>String</td><td>是</td><td>操作者accid</td></tr>
        <tr> <td>roleid</td><td>Long</td><td>是</td><td>被操作的服务器身份组id</td></tr>
</table>
