

删除频道分组。 


## URL
```http
POST http://api.yunxinapi.com/nimserver/qchat/removeChannelCategory.action  HTTP/1.1
Content-Type: application/x-www-form-urlencoded;charset=utf-8
```



## 请求参数

- POST 请求中 Headers 的设置请参考[API 调用方式](https://doc.yunxin.163.com/messaging/docs/jk3MzY2MTI?platform=server)。

- POST 请求中 Body 的设置如下：

<table>
    <thead><tr> <th>参数</th><th>类型</th><th>必须</th><th>说明</th></tr></thead>
        <tr> <td>accid</td><td>String</td><td>是</td><td>操作者accid</td></tr>
        <tr> <td>categoryId</td><td>String</td><td>是</td><td>频道分组唯一标识</td></tr>
</table>



## 示例

### cURL 请求示例

```curl
curl -X POST -H "AppKey: go9dnk**803mgq3" -H "Nonce: 4tggger**3t" -H "CurTime: 1443592222" -H "CheckSum: 9e9db3b6c9abb2e1962cf3e6f7316fcc55583f86" -H "Content-Type: application/x-www-form-urlencoded" -d 'categoryId=1513535&accid=zhangsan' 'http://api.yunxinapi.com/nimserver/qchat/removeChannelCategory.action'
```

### 返回示例

http 响应:json
```json
"Content-Type": "application/json; charset=utf-8"
{
  "code":200
}

```
