网易云信支持批量查询账号的在线状态，一次最多可查询 100 个账号。

## 使用限制

- 单个应用默认最高调用频率请参考 [频控说明](https://doc.yunxin.163.com/messaging/server-apis/zk4Mzg0MjE?platform=server)。
- 仅 **IM 旗舰版及以上套餐** 支持使用该接口。

## 请求信息

### 请求 URL

```
POST https://api.yunxinapi.com/nimserver/user/userOnlineStatus.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```

### 请求头参数

请参考 [API 调用方式](https://doc.yunxin.163.com/messaging/docs/jk3MzY2MTI?platform=server)。

### 请求体参数

参数名称 | 类型 | 是否必选 | 说明 |
---- | ---- | ---- | ----
accids | String | 是 | 需要查询的用户账号 ID 列表，一次最多查询 100 个账号。<br>格式为 JSONArray，例如：["id1", "id2", "id3"]，格式错误会返回 414 参数错误。

### 请求示例

```cURL
curl -X POST -H "AppKey: go9dnk49bk****0803mgq3" -H "Nonce: 4tgggerg **3t23t" -H "CurTime: 1443592222" -H "CheckSum: 9e9db3b***55583f86" -H "Content-Type: application/x-www-form-urlencoded" -d 'accids=%5B%22user1%22%2C%22user2%22%5D' 'http://api.yunxinapi.com/nimserver/user/userOnlineStatus.action'
```

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 |
---- | ---- | ----
code | Integer | 状态码，200 表示调用成功。
data | Object | 查询的数据结果。
 invalidAccids | String | 无效的账号列表，格式为 JSONArray。
 status | Object | 有效账号的在线状态。
  loginTime | Long | 登录时间。
  clientType | Integer | 登录的客户端类型。1:AOS，2:IOS，4:PC，16:WEB，64:MAC，65:HARMONY

### 响应成功示例

```JSON
"Content-Type": "application/json; charset=utf-8"
{
  "code": 200,
  "data": {
    "invalidAccids": [
      "user2"
    ],
    "status": {
      "user1": [
        {
          "loginTime": 1571296813350,
          "clientType": 16
        }
      ]
    }
  }
}
```

### 响应失败示例

```JSON
"Content-Type": "application/json; charset=utf-8"
{
  "code": 414, // 参数错误
  "desc": ""
}
```
