网易云信 IM 服务端支持订阅/取消订阅在线状态事件。

## <span id='订阅在线状态事件'>订阅在线状态事件</span>
### 功能描述
订阅指定用户的在线状态事件，每个账号的有效订阅账号最多为 3000 个。

### API 使用限制

单个应用默认最高调用频率请参考 [频控说明](https://doc.yunxin.163.com/messaging/server-apis/zk4Mzg0MjE?platform=server)。

### URL
```
POST https://api.yunxinapi.com/nimserver/event/subscribe/add.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```
### 请求参数

- POST 请求中 Headers 的设置请参考 [接口概述](https://doc.yunxin.163.com/messaging/server-apis/jk3MzY2MTI?platform=server)。
- POST 请求中 Body 的设置如下：
<table>
            <thead>
        <tr> <th>参数</th><th>类型</th><th>必须</th><th>说明</th></tr></thead>
           <tr> <td> accid </td><td>String</td><td>是</td><td>
           事件订阅人账号</td></tr>
           <tr> <td> eventType </td><td>int</td><td>是</td><td>
           事件类型，固定设置为1，即 eventType=1</td></tr>
           <tr> <td> publisherAccids </td><td>String</td><td>是</td><td>被订阅人的账号列表，最多100个账号，JSONArray格式。示例：["pub_user1","pub_user2"]</td></tr>
           <tr> <td> ttl </td><td>long</td><td>是</td><td>
           订阅的有效时间，单位：秒。取值范围：60～2592000（即60秒到30天）</td></tr>
</table>

### 示例
#### **请求示例（curl）**
```curl
curl -X POST -H "CheckSum: fc04024***230155c" -H "AppKey: f123***aaaa" -H "Nonce: 1" -H "CurTime: 1451200147" -H "Content-Type: application/x-www-form-urlencoded" -d 'accid=user1&eventType=1&publisherAccids=["pub_user1","pub_user2"]&ttl=86400' "https://api.yunxinapi.com/nimserver/event/subscribe/add.action"
```

#### **返回示例**

```json

"Content‐type":"application/json;charset=utf‐8"
{
  "code": 200,
  "failedAccid":[] //订阅失败的账号数组
}

```
