网易云信 IM 服务端支持设置/取消黑名单和免打扰功能。

## <span id='设置黑名单/免打扰'>设置黑名单/免打扰</span>

### 功能描述

网易云信 IM 服务端支持拉黑/取消拉黑用户，以及设置免打扰/取消免打扰功能。

设置黑名单后，不会接收到黑名单中的用户发给自己的消息。

设置免打扰用户后，当被设为免打扰的用户发送消息给自己时，不会触发在线消息提醒和离线推送的通知栏。

每个账号的黑名单和免打扰用户上限都为 3000。

### API 使用限制

单个应用默认最高调用频率请参考 [频控说明](https://doc.yunxin.163.com/messaging/server-apis/zk4Mzg0MjE?platform=server)。

### URL

```
POST https://api.yunxinapi.com/nimserver/user/setSpecialRelation.action  HTTP/1.1
Content-Type: application/x-www-form-urlencoded;charset=utf-8
```

### 请求参数

- POST 请求中 Headers 的设置请参考 [接口概述](https://doc.yunxin.163.com/messaging/server-apis/jk3MzY2MTI?platform=server)。
- POST 请求中 Body 的设置如下：

    <tr><th>参数名称</th><th><div style="width: 30pt">类型</div></th><th>字符串长度上限</th><th>是否必填</th><th>示例</th><th>描述</th></tr></thead>
    <tr> <td>accid</td><td>String</td><td>32</td><td>必填</td><td>zhangsan</td><td>用户账号 ID</td></tr>
    <tr> <td>targetAcc</td><td>String</td><td>32</td><td>必填</td><td>lisi</td><td>被拉黑或被设置免打扰的账号 ID</td></tr>

    <tr> <td>value</td><td>int</td><td>/</td><td>必填</td><td>0</td><td>操作值<br/>0 取消拉黑或取消免打扰；1 加入黑名单或设置免打扰</td></tr>
</table>

## <span id='查看指定用户的黑名单和免打扰列表'>查看指定用户的黑名单和免打扰列表</span>

```
POST https://api.yunxinapi.com/nimserver/user/listBlackAndMuteList.action  HTTP/1.1
Content-Type: application/x-www-form-urlencoded;charset=utf-8
```

- POST 请求中 Headers 的设置请参考 [接口概述](https://doc.yunxin.163.com/messaging/server-apis/jk3MzY2MTI?platform=server)。
