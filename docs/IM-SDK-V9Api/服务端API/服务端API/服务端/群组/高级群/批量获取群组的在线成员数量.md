批量查询群组的在线成员数量，一次最多查询 100 个群组。

## API 使用限制

- 单个应用中 1 秒内所有的高级群操作 API 合计最多可调用 100 次，超过后限制调用，会返回 416 错误码。

    :::note note
    除发送群消息 API 外，其他所有高级群 API 都属于高级群操作 API。
    :::

- 仅 **IM 旗舰版及以上套餐** 支持使用该接口。

## URL
```
POST https://api.yunxinapi.com/nimserver/team/listOnlineUserCount.action  HTTP/1.1
Content-Type: application/x-www-form-urlencoded;charset=utf-8
```
## 请求参数

- POST 请求中 Headers 的设置请参考[API 调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。
- POST 请求中 Body 的设置如下：
<table>
    <thead>
        <tr> <th>参数</th><th>类型</th><th>必填</th><th>说明</th></tr></thead>
           <tr> <td>tids</td><td>String</td><td>是</td><td>群 ID 列表，JSONArray，每一项表示一个 tid，最多查询 100 个群组，单个 tid 最大长度 64 位长整型</td></tr>
</table>

## 返回参数

参数|类型|说明
:----|:----|:----
code|Integer|状态码
data|String|在线群成员数量

data 中的参数说明
参数|类型|说明
:----|:----|:----
tid|Long|群 ID
onlineUserCount|Integer|在线成员数量
offlineUserCount|Integer|非在线成员数量
