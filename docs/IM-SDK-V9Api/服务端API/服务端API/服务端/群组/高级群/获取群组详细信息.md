查询指定群的详细信息（群信息+成员详细信息）。



## API 使用限制
单个应用中 1 秒内所有的高级群操作 API 合计最多可调用 100 次，超过后限制调用，会返回 416 错误码。

:::note note
除发送群消息 API 外，其他所有高级群 API 都属于高级群操作 API。
:::

## URL
```
POST https://api.yunxinapi.com/nimserver/team/queryDetail.action  HTTP/1.1
Content-Type: application/x-www-form-urlencoded;charset=utf-8
```
## 请求参数

- POST 请求中 Headers 的设置请参考[API 调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。
- POST 请求中 Body 的设置如下：
<table>
    <thead>
        <tr> <th>参数</th><th>类型</th><th>必填</th><th>说明</th></tr></thead>
           <tr> <td>tid</td><td>Long</td><td>是</td><td>服务器产生，群唯一标识，创建群时会返回，最大长度 64 位长整型</td></tr>
</table>

## 返回参数



参数|类型|说明
:----|:----|:----
code|Integer|状态码
tinfo|String|群组的详细信息

    <thead>
        <tr> <th style="width:180px">参数</th><th style="width:100px">类型</th><th>说明</th></tr></thead>
            <tr> <td>icon</td><td>String</td><td>群头像</td></tr>

            <tr> <td>intro</td><td>String</td><td>群介绍</td></tr>
            <tr> <td>maxusers</td><td>Integer</td><td>群成员最大数量</td></tr>
            <tr> <td>upcustommode</td><td>Integer</td><td>修客户端改群自定义属性的权限<br/>0，群主和管理员（默认）；1，所有人</td></tr>
            <tr> <td>tname</td><td>String</td><td>群名称</td></tr>
            <tr> <td>beinvitemode</td><td>Integer</td><td>邀请入群的验证方式，即邀请入群后，是否需要被邀请人的同意<br/>0，需要同意（默认）；1，不需要同意</td></tr>
            <tr> <td>joinmode</td><td>Integer</td><td>申请入群的验证方式<br/>0，不用验证；1，需要群主或管理员的验证；2，不允许任何人加入</td></tr>

            <tr> <td>muteType</td><td>Integer</td><td>禁言类型<br/>0，解除禁言；1，禁言普通成员；3，禁言整个群（包括群主）</td></tr>

            <tr> <td>clientCustom</td><td>String</td><td>客户端自定义字段</td></tr>
            <tr> <td>createtime</td><td>Long</td><td>创建时间</td></tr>

            <tr> <td>members</td><td>String</td><td>群成员信息，JSON，包含群昵称（nick）、用户 ID（accid）、是否被禁言（mute）等信息</td></tr>
</table>
