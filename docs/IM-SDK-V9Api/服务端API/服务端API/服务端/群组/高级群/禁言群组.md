使用该接口对群聊进行禁言或者解除禁言。群主可以禁言整个群组，只对普通成员生效，即普通成员不能发送消息，群主和管理员可以发送消息。同时，也支持禁言整个群，包括群主。

## 功能介绍

网易云信 IM 支持三种禁言方式：

- **[账号全局禁言](https://doc.yunxin.163.com/messaging/docs/zY0MzIxNDE?platform=server)**：账号被设置为全局禁言后，不能发送任何消息(包含点对点、群组（高级群和超大群）、聊天室消息)。
- **[群整体禁言](https://doc.yunxin.163.com/messaging/docs/zc0MTQyNzA?platform=server)**：可选择禁言整个群（包括群主），或者只禁言普通成员。
- **[指定群成员禁言](https://doc.yunxin.163.com/messaging/docs/DU0MzE3MjE?platform=server)**：可以禁言指定群成员，包括管理员和普通成员。

上述三种禁言方式相互独立。例如，某账号加入某高级群后，先后被全局禁言、群整体禁言、单个群成员禁言，那么需要分别解除三种禁言后才能恢复在该高级群发言。

## 使用限制

单个应用中 1 秒内所有的高级群操作 API 合计最多可调用 100 次，超过后限制调用，会返回 416 错误码。

:::note note
除 [发送群消息](https://doc.yunxin.163.com/messaging/server-apis/jgzMjIzNzc?platform=server) API 外，其他所有高级群 API 都属于高级群操作 API。
:::

## 请求信息

### 请求 URL

```
POST https://api.yunxinapi.com/nimserver/team/muteTlistAll.action HTTP/1.1
Content-Type: application/x-www-form-urlencoded;charset=utf-8
```

### 请求参数

- POST 请求中 Headers 的设置请参考 [API 调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。
- **POST 请求中 Body 的设置如下**：
