网易云信 IM 服务端支持设置群消息免打扰，设置群消息免打扰后，该群组的消息不会触发在线消息提醒和离线推送的通知栏。

## API 使用限制

单个应用中 1 秒内所有的高级群操作 API 合计最多可调用 100 次，超过后限制调用，会返回 416 错误码。

:::note note
除发送群消息 API 外，其他所有高级群 API 都属于高级群操作 API。
:::

## URL

```
POST https://api.yunxinapi.com/nimserver/team/muteTeam.action  HTTP/1.1
Content-Type: application/x-www-form-urlencoded;charset=utf-8
```

## 请求参数

- POST 请求中 Headers 的设置请参考 [API 调用方式](https://doc.yunxin.163.com/messaging/server-apis/jk3MzY2MTI?platform=server)。
- POST 请求中 Body 的设置如下：

<table>
  <thead>
    <tr><th>参数</th><th>类型</th><th>必填</th><th>说明</th></tr></thead>
      <tr><td>tid</td><td>Long</td><td>是</td><td>云信服务端产生，群唯一标识，创建群时会返回，最大长度 64 位长整型</td></tr>
      <tr><td>accid</td><td>String</td><td>是</td><td>需要设置群消息免打扰的账号 ID，最大长度 32 位字符<note type= notice>该用户必须为群组成员。</note></td></tr>
      <tr><td>ope </td><td>Integer</td><td>是</td><td>1，群消息免打扰；2，开启群消息提醒</td></tr>
</table>

## 示例

### 请求示例（curl）

```curl
curl -X POST -H "CheckSum: e8c47bc1ede283cc4c382ba741de33f1f7e7e8f2" -H "AppKey: fe416640***47ad2547" -H "Nonce: 1" -H "CurTime: 1452010720" -H "Content-Type: application/x-www-form-urlencoded" -d 'accid=zhouliangwei&tid=4104&ope=2' 'https://api.yunxinapi.com/nimserver/team/muteTeam.action'
```

### 请求成功返回示例

```json
"Content-Type": "application/json; charset=utf-8"
{
    "code":200
}
```

### 请求失败返回示例

```
"Content-Type": "application/json; charset=utf-8"
{
"code": 414,  // 参数错误
"desc": "xxxx  is empty" 
}
```
