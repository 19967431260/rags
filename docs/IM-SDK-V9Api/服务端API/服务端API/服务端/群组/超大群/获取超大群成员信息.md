
查询指定超大群的成员信息。


## API 使用限制

单个应用中 1 秒内所有的超大群操作 API 合计最多可调用 100 次，超过后限制调用，会返回 416 错误码。

:::note note
- 除超大群消息相关 API 外，其他所有超大群 API 都属于超大群操作 API。
- 超大群消息相关 API 包括：发送超大群消息，发送超大群自定义系统通知，撤回超大群消息，根据用户 ID/消息 ID 查询超大群历史消息。
:::

## URL
```
POST https://api.yunxinapi.com/nimserver/superteam/getTlists.action  HTTP/1.1
Content-Type: application/x-www-form-urlencoded;charset=utf-8
```
## 请求参数

- POST 请求中 Headers 的设置请参考[接口概述](/docs/TM5MzM5Njk/jk3MzY2MTI)。
- POST 请求中 Body 的设置如下：
<table>
       <thead>
       <tr> <th>参数</th><th>类型</th><th>必填</th><th>说明</th></tr></thead>
       <tr> <td>tid</td><td>Long</td><td>是</td><td>云信服务器产生，超大群唯一标识，创建时会返回，最大长度 64 位长整型</td></tr>
       <tr> <td>timetag</td><td>Long</td><td>是</td><td>毫秒级别时间戳，查询的时间起点</td></tr>
       <tr> <td>limit</td><td>Integer</td><td>是</td><td>本次查询的条数上限，(0,2000]</td></tr>
       <tr> <td>reverse</td><td>Integer</td><td>否</td><td>查询方向<br/>1，按时间正序排列（默认）；2，按时间降序排列</td></tr>
</table>

## 返回参数

参数|类型|说明
:----|:----|:----
code|Integer|状态码
tlists|String|群成员信息

tlists 中的参数说明

参数|类型|说明
:----|:---|:----
tid|Long|超大群 ID
uid|Long|云信分配的用户 ID
accid|String|用户账号
type|Integer|群成员类型<br>0，普通成员；1，群主；2，管理员
nick|String|群昵称
jointime|String|加入群的时间
mute|Integer|是否被禁言<br>1，禁言；0，解禁
invitoruid|Long| 云信分配的邀请者的 ID
invitor|String| 邀请者的账号
validflag|Integer| 有效标志<br>0，无效；1，有效
custom|String|自定义扩展属性
state|Integer|激活状态<br>0，未激活；1，已激活
bits|Long|群成员配置
createtime|Long|创建时间
updatetime|Long|更新时间
