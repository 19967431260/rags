
<!--keywords: 聊天室机器人,添加机器人,删除机器人,清空机器人 -->


网易云信 IM 服务端支持在指定聊天室添加、删除或清空机器人账号。



## <span id='往聊天室内添加机器人'>添加聊天室机器人</span>

### **功能描述**

在指定聊天室内批量添加机器人，机器人过期时间为24小时。



### **URL**

```http
POST https://api.yunxinapi.com/nimserver/chatroom/addRobot.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```



### **请求参数**


- POST 请求中 Headers 的设置请参考[API调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。

- POST 请求中 Body 的设置如下：

<table>
            <thead>
        <tr> <th>参数</th><th>类型</th><th>必须</th><th>说明</th></tr></thead>
           <tr> <td> roomid </td><td>long</td><td>是</td><td>
           聊天室id</td></tr>
           <tr> <td> accids </td><td> String </td><td> 是 </td><td>
           机器人账号accid列表，必须是有效账号，账号数量上限100个，格式为 JSONArray
</td></tr>
<tr> <td> roleExt </td><td> String </td><td>否</td><td>
           机器人信息扩展字段，请使用json格式，长度4096字符
</td></tr>
<tr> <td> notifyExt </td><td> String </td><td>否</td><td>
           	机器人进入聊天室通知的扩展字段，请使用json格式，长度2048字符
</td></tr>
</table>


### **示例**

#### **cURL请求示例**

```curl
curl -X POST -H "CheckSum: 51eb13ea5**8e61c65c7866c366" -H "AppKey: f541664055e**866ad7799" -H "Nonce: 1" -H "CurTime: 1451207708" -H "Content-Type: application/x-www-form-urlencoded" -d 'roomid=70&accids=["zhangsan","lisi"]' 'https://api.yunxinapi.com/nimserver/chatroom/addRobot.action'
```

#### **请求成功返回示例**

```json

"Content-Type": "application/json; charset=utf-8"
{
  "desc": {
    "failAccids": "[\"hzzhangsan\"]",
    "successAccids": "[\"hzlisi\"]",
    "oldAccids": "[\"hzwangwu\"]"
  },
  "code": 200
}

```

## <span id='从聊天室内删除机器人'>删除机器人</span>

### **功能描述**

批量删除指定聊天室内的机器人，最多删除 100 个机器人账号。


```http
POST https://api.yunxinapi.com/nimserver/chatroom/removeRobot.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```



### **请求参数**



- POST 请求中 Headers 的设置请参考[API调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。

- POST 请求中 Body 的设置如下：

<table>
            <thead>
        <tr> <th>参数</th><th>类型</th><th>必须</th><th>说明</th></tr></thead>
           <tr> <td> roomid </td><td>long</td><td>是</td><td>
           聊天室id</td></tr>
           <tr> <td> accids </td><td> String </td><td> 是 </td><td>
           机器人账号accid列表，必须是有效账号，账号数量上限100个，格式为 JSONArray
</td></tr>
</table>

### **示例**

#### **cURL请求示例**

#### **请求成功返回示例**

## <span id='清空聊天室机器人'>清空机器人</span>

### **功能描述**

删除聊天室内全部机器人。

### **URL**

```http
POST https://api.yunxinapi.com/nimserver/chatroom/cleanRobot.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```


### **请求参数**


- POST 请求中 Headers 的设置请参考[API调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。

- POST 请求中 Body 的设置如下：

<table>
            <thead>
        <tr> <th>参数</th><th>类型</th><th>必须</th><th>说明</th></tr></thead>
           <tr> <td> roomid </td><td>long</td><td>是</td><td>
           聊天室id</td></tr>
           <tr> <td> notify </td><td> boolean </td><td> 否 </td><td>
           是否发送退出聊天室通知消息，默认为false
</td></tr>
</table>

### **示例**

#### **cURL请求示例**
```curl
curl -X POST -H "CheckSum: 51eb13ea**61c65c7866c366" -H "AppKey: f5416640**66ad7799" -H "Nonce: 1" -H "CurTime: 1451207708" -H "Content-Type: application/x-www-form-urlencoded" -d 'roomid=70&notify=false' 'https://api.yunxinapi.com/nimserver/chatroom/cleanRobot.action'
```

#### **请求成功返回示例**

```json

"Content-Type": "application/json; charset=utf-8"
{
    "code": 200,
    "desc": {
        "size": 2
    }
}

```
