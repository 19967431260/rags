网易云信 IM 服务端支持设置和变更用户角色。

## <span id='设置聊天室内用户角色'>设置角色</span>

### **功能描述**

将聊天室内指定用户设置为管理员、普通等级用户、黑名单用户或禁言用户。各设置操作均有相应的权限要求，具体参见下文的**请求参数**。


### **URL**




```http
POST https://api.yunxinapi.com/nimserver/chatroom/setMemberRole.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```



### **请求参数**

- POST 请求中 Headers 的设置请参考[API调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。

- POST 请求中 Body 的设置如下：

<table>
            <thead>
    <thead><tr> <th style="width:160px">参数</th><th style="width:90px">类型</th><th>必填</th><th>说明</th></tr></thead>
           <tr> <td> roomid </td><td>long</td><td>是</td><td>
           聊天室id</td></tr>
           <tr> <td> operator </td><td> String </td><td> 是 </td><td>
           操作者账号accid
</td></tr>
<tr> <td> target </td><td> String </td><td>是</td><td>
           被操作者账号accid
</td></tr>
<tr> <td> opt </td><td> int </td><td>是</td><td>
           操作：<br>
1: 设置为管理员，operator必须是创建者 <br>
2:设置普通等级用户，operator必须是创建者或管理员 <br>
-1:设为黑名单用户，operator必须是创建者或管理员 <br>
-2:设为禁言用户，operator必须是创建者或管理员
</td></tr>
<tr> <td> optvalue </td><td> boolean </td><td>是</td><td>
           true或false，true:设置；false:取消设置；<br/>
           执行“取消”设置后，若成员非禁言且非黑名单，则变成游客
</td></tr>
<tr> <td> notifyExt </td><td> String </td><td>否</td><td>
           通知扩展字段，长度限制2048，请使用json格式
</td></tr>
</table>

### **示例**


#### **cURL请求示例**

```curl
curl -X POST -H "CheckSum: 51eb13e***61c65c7866c366" -H "AppKey: fe416640c8e**847ad2547" -H "Nonce: 1" -H "CurTime: 1451207708" -H "Content-Type: application/x-www-form-urlencoded" -d 'roomid=66&operator=zhangsan&target=lisi&opt=2&optvalue=true' 'https://api.yunxinapi.com/nimserver/chatroom/setMemberRole.action'
```

#### **请求成功返回示例**

```json
"Content-Type": "application/json; charset=utf-8"
{
  "desc": {
    "roomid": 16,
    "level": 10,
    "accid": "zhangsan",
    "type": "COMMON"
  },
  "code": 200
}

备注：
返回的type字段可能为：
        LIMITED,          //受限用户,黑名单+禁言 
        COMMON,           //普通固定成员
        CREATOR,          //创建者 
        MANAGER,          //管理员 
        TEMPORARY,        //临时用户,非固定成员
```




## <span id='变更聊天室内的角色信息'>变更角色</span>



### **功能描述**

变更聊天室内指定用户的角色信息。

### **URL**

```http
POST https://api.yunxinapi.com/nimserver/chatroom/updateMyRoomRole.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```



### **请求参数**



- POST 请求中 Headers 的设置请参考[API调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。

- POST 请求中 Body 的设置如下：

<table>
            <thead>
    <thead><tr> <th style="width:160px">参数</th><th style="width:90px">类型</th><th>必填</th><th>说明</th></tr></thead>
<tr> <td> roomid </td><td> long</td><td>是</td><td>
           聊天室 ID</td></tr>
<tr> <td> accid </td><td> String </td><td>是</td><td>
           需要变更角色信息的 IM 账号</td></tr>
<tr> <td> save </td><td> boolean </td><td>否</td><td>
           变更的信息是否需要持久化，默认false，仅对聊天室固定成员生效</td></tr>
<tr> <td> needNotify </td><td> boolean </td><td>否</td><td>
           是否需要下发通知消息<note type=notice>如果变更角色的人不在聊天室，不会下发通知消息。</td></tr>
<tr> <td> notifyExt </td><td> String </td><td>否</td><td>
           通知消息的内容，长度限制 2048</td></tr>
<tr> <td> nick </td><td> String </td><td>否</td><td>
           聊天室室内的角色信息：昵称，不超过 64 个字符</td></tr>
<tr> <td> avator </td><td> String </td><td>否</td><td>
           聊天室室内的角色信息：头像</td></tr>
<tr> <td> ext </td><td> String </td><td>否</td><td>
           聊天室室内的角色信息：开发者扩展字段</td></tr>
<tr> <td> bid </td><td>String</td><td>否</td><td>
          安全通的自定义反垃圾（即内容审核）业务的 ID。自定义反垃圾业务主要用来针对单条消息或单次操作的资料信息进行除了默认反垃圾业务以外的内容审核。如需配置自定义反垃圾，请通过云信官网首页提供的微信、在线聊天和电话等方式联系商务经理进行配置，并获取对应的业务 ID。JSON字符串，形式如 {"textbid":"","picbid":""}。若不填则使用默认的反垃圾配置</td></tr>
</table>


### **示例**

#### **cURL请求示例**
```curl
curl -X POST -H "CheckSum: 51eb13ea5**c65c7866c366" -H "AppKey: he516640c8e**47ad2547" -H "Nonce: 1" -H "CurTime: 1451207708" -H "Content-Type: application/x-www-form-urlencoded" -d 'roomid=36&accid=hzyangfeng201403&save=true&needNotify=true&nick=myNick' "https://api.yunxinapi.com/nimserver/chatroom/updateMyRoomRole.action"
```

#### **请求成功返回示例**

```json

"Content-Type": "application/json; charset=utf-8"
{
  "code": 200
}

```
