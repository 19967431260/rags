<!--keywords: 请求聊天室地址,聊天室地址 -->

创建聊天室后，您可通过调用 IM 服务端 API 请求聊天室地址（推荐），也可通过调用 SDK 相关接口建立聊天室长连接获取聊天室地址。 

## URL 
```http
POST https://api.yunxinapi.com/nimserver/chatroom/requestAddr.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```



## 请求参数


- POST 请求中 Headers 的设置请参考[API调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。

- POST 请求中 Body 的设置如下：

<table>
            <thead>
        <tr> <th>参数</th><th>类型</th><th>必须</th><th>说明</th></tr></thead>
           <tr> <td> roomid </td><td>long</td><td>是</td><td>
           聊天室id</td></tr>
           <tr> <td> accid </td><td> String </td><td> 是 </td><td>
           进入聊天室的账号
</td></tr>
<tr> <td> clienttype </td><td> int </td><td>否</td><td>
           进入聊天室的用户的客户端类型。<ul><li>1：weblink（默认，客户端为 web/uni-app 端时使用）</li><li>2：commonlink（客户端为非 web/uni-app 端时使用）</li><li>3：wechatlink（微信小程序使用）</li></ul>
</td></tr>
<tr> <td> clientip </td><td>String</td><td>否</td><td>客户端ip，传此参数时，会根据用户ip所在地区，返回合适的地址</td></tr>
</table>


## 示例

### **cURL请求示例**
```curl
curl -X POST -H "CheckSum: 51eb13ea5ee3**1c65c7866c366" -H "AppKey: f1234540c12**3456847aaaaaa" -H "Nonce: 1" -H "CurTime: 1451217708" -H "Content-Type: application/x-www-form-urlencoded" -d 'roomid=66&accid=zhangsan&clienttype=1' 'https://api.yunxinapi.com/nimserver/chatroom/requestAddr.action'
```

### **返回示例**


请求成功的返回示例如下：

```json
"Content-Type": "application/json; charset=utf-8"
{
  "addr": [
    "testchat.yunxinapi.com:6666",
    "testchat.yunxinapi.com:8888"
  ],
  "code": 200
}

```
## 状态码


该接口在 HTTPS Body 中返回请求的状态码，状态码详情请参见[状态码](https://doc.yunxin.163.com/messaging/server-apis/TM5NTk2Mzc?platform=server)。
