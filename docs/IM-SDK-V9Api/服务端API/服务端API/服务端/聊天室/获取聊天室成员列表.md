网易云信 IM 服务端支持分页获取成员列表、根据角色获取固定成员列表以及获取在线成员列表。

## <span id='分页获取成员列表'>分页获取</span>

### **功能描述**

分页获取在某时间段内指定聊天室、指定成员类型的成员列表。

### **URL**

```http
POST https://api.yunxinapi.com/nimserver/chatroom/membersByPage.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```



### **请求参数**

- POST 请求中 Headers 的设置请参考[API调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。

- POST 请求中 Body 的设置如下：

<table>
            <thead>
        <tr> <th>参数</th><th>类型</th><th>必须</th><th>说明</th></tr></thead>
<tr> <td> roomid </td><td>long</td><td>是</td><td>
           聊天室id</td></tr>
<tr> <td> type </td><td>int</td><td>是</td><td>
           需要查询的成员类型,0:固定成员;1:非固定成员;2:仅返回在线的固定成员</td></tr>
<tr> <td> endtime </td><td> long</td><td>是</td><td>
           单位毫秒，按时间倒序最后一个成员的时间戳,0表示系统当前时间</td></tr>
<tr> <td> limit </td><td> long</td><td>是</td><td>
           返回条数，<=100</td></tr>
</table>

### **返回参数**

<table>
            <thead>
        <tr> <th>参数</th><th>类型</th><th>说明</th></tr></thead>
          <tr> <td> roomid </td><td>long</td><td>
           聊天室id</td></tr>
           <tr> <td> accid </td><td>String</td><td>
           用户accid</td></tr>
           <tr> <td> nick </td><td>String</td><td>
           聊天室内的昵称</td></tr>
           <tr> <td> avator </td><td>String</td><td>
           聊天室内的头像</td></tr>
           <tr> <td> ext </td><td>String</td><td>
           开发者扩展字段</td></tr>
           <tr> <td> type </td><td>String</td><td>
           角色类型：<br>UNSET（未设置），<br>LIMITED（受限用户，黑名单或禁言），<br>COMMON（普通固定成员），<br>CREATOR（创建者），<br>MANAGER（管理员），<br>TEMPORARY（临时用户,非固定成员）</td></tr>
           <tr> <td> level </td><td>int</td><td>
           成员级别（若未设置成员级别，则无此字段）</td></tr>
           <tr> <td> onlineStat </td><td>Boolean</td><td>
           是否在线</td></tr>
           <tr> <td> enterTime </td><td>long</td><td>
           进入聊天室的时间点</td></tr>
           <tr> <td> blacklisted </td><td>Boolean</td><td>
           是否在黑名单中（若未被拉黑，则无此字段）</td></tr>
           <tr> <td> muted </td><td>Boolean</td><td>
           是否被禁言（若未被禁言，则无此字段）</td></tr>
           <tr> <td> tempMuted </td><td>Boolean</td><td>
           是否被临时禁言（若未被临时禁言，则无此字段）</td></tr>
           <tr> <td> tempMuteTtl </td><td>long</td><td>
           临时禁言的解除时长,单位秒（若未被临时禁言，则无此字段）</td></tr>
           <tr> <td> isRobot </td><td>Boolean</td><td>
           是否是聊天室机器人（若不是机器人，则无此字段）</td></tr>
           <tr> <td> robotExpirAt </td><td>int</td><td>
           机器人失效的时长，单位秒（若不是机器人，则无此字段）</td></tr>

</table>

## <span id='根据角色获取固定成员列表'>根据角色获取</span>

### **功能描述**

根据角色（创建者、管理员、黑名单成员或禁言成员）获取固定成员列表。

### **URL**

```http
POST https://api.yunxinapi.com/nimserver/chatroom/queryMembersByRole.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```



### **请求参数**

- POST 请求中 Headers 的设置请参考[API调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。

- POST 请求中 Body 的设置如下：

<table>
      <thead>
        <tr> <th>参数</th><th>类型</th><th>必须</th><th>说明</th></tr>
      </thead>
      <tbody>
          <tr> 
            <td> roomid </td>
            <td>long</td>
            <td>是</td>
            <td>聊天室id</td>
          </tr>
          <tr> 
            <td> roles </td>
            <td> String </td>
            <td> 是 </td>
            <td>
              设置需要获取的角色,格式示例：<br>
              {"creator": true,"manager": true,"blacklist": false,"mute": false} <br> <br>
              字段说明： <br>
              1、creator：聊天室创建者 <br>

## 批量获取成员信息


### **功能描述**

批量获取聊天室成员信息。

### **URL**

```http
POST https://api.yunxinapi.com/nimserver/chatroom/queryMembers.action HTTP/1.1
Content-Type:application/x-www-form-urlencoded;charset=utf-8
```



### **请求参数**


- POST 请求中 Headers 的设置请参考[API调用方式](/docs/TM5MzM5Njk/jk3MzY2MTI)。

- POST 请求中 Body 的设置如下：


<table>
            <thead>
        <tr> <th>参数</th><th>类型</th><th>必须</th><th>说明</th></tr></thead>
<tr> <td> roomid </td><td> long</td><td>是</td><td>
           聊天室id</td></tr>
<tr> <td> accids </td><td> String </td><td>是</td><td>
           \["abc","def"\], 账号列表，最多200条，格式为 JSONArray</td></tr>
</table>

### **示例**

#### **cURL请求示例**
```curl
curl -X POST -H "CheckSum: 32dc17d019**f049c9367e7" -H "AppKey: fe416640c8e**1847ad2547" -H "Nonce: 1" -H "CurTime: 1451207708" -H "Content-Type: application/x-www-form-urlencoded" -d 'roomid=4235&accids=["abc","def"]' "https://api.yunxinapi.com/nimserver/chatroom/queryMembers.action"
```

#### **请求成功返回示例**

```json

"Content-Type": "application/json; charset=utf-8"
{
  "desc": {
    "data": [
      {
        "roomid": 111,
        "accid": "abc",
        "nick": "cba",
        "type": "CREATOR", //COMMON:普通成员(固定成员)；CREATOR:聊天室创建者；MANAGER:聊天室管理员；TEMPORARY:临时用户(非聊天室固定成员)；ANONYMOUS:匿名用户(未注册账号)；LIMITED:受限用户(黑名单+禁言)
        "onlineStat": true
      }
    ]
  },
  "code": 200
}

```
