本文介绍网易云信即时通讯 IM SDK（简称 NIM SDK）稳定版 Unity 端的版本更新日志。

## 2.10.2 (2025.10.30)

内部优化。

## 2.9.0 (2025.10.21)

适配鸿蒙终端。

## 2.8.1 (2025.09.25)

Android SDK 适配 16KB 内存页大小（Page Size）。

## 2.6.0 (2024.11.08)

**新增功能**

- 新增聊天室撤回消息通知类型字段解析，新增 **323 号通知**（消息撤回），用于上层判断是否是撤回消息通知。
- 新增 macOS 平台支持，开发者能在 macOS Editor 中调试及发布 macOS 产物。

**接口变更**

[`Login`](https://doc.yunxin.163.com/messaging/references/unity/doxygen/Latest/zh/class_n_i_m_1_1_client_a_p_i.html#a240c5cb1e6f93f04f887bb58d281ed16) 接口新增参数 `LoginExtensionParams`，允许使用 [动态 Token 或第三方回调登录](https://doc.yunxin.163.com/messaging/concept/jE0NjA3NTU?platform=client#%E7%99%BB%E5%BD%95%E6%96%B9%E5%BC%8F%E6%A6%82%E8%A7%88)，若不需要则指定为 null。

## 2.4.2 (2022.10.12)

修复 iOS 16 兼容性问题。

## 2.4.1 (2022.09.09)

- 移除音视频功能，如需要，请使用 Unity RTC SDK。
- 新增部分示例代码片段。
- 稳定性优化。

## 2.4.0 (2018.12.11)

- 音视频独立信令。
- 新增 Android 本地推送。

## 2.3.0 (2018.11.07)

- 语音消息优化。
- 新增聊天室游客登录模式。

## 2.2.0 (2018.09.13)

增加事件订阅(在线状态)功能。

## 2.1.0 (2018.08.21)

- 文件上传增加标签信息。
- 上传和下载文件失败时自动进行重试。
- 语音模块采集增加能量值上报。

## 1.6.0 (2018.06.14)

- 升级 OpenSSL 和 libcurl 库，解决开发者发布应用时可能存在的安全问题。
- 解决断网重连时可能存在连接结果无法通知到上层。
- 按类型批量设置系统消息状态时将不更新已被删除的消息。
- 录制增强稳定性。
- 优化 AAC 文件时长计算。
- 解决 SDK 内存泄漏缺陷。

## 1.5.0 (2018.02.02)

支持华为与小米消息推送。

## 1.4.0 (2018.01.04)

- 增加客户端反垃圾功能。
- 增加广播消息。
- 支持缩略图下载。
- 修复已知缺陷。

## 1.3.0 (2017.11.20)

- 增加登录超时时间设置。
- 增加首次登录连接服务器失败时的重试机制。
- 语音消息模块优化。
- 日志模块优化。
- 支持 Unity 桌面端实时语音，其他端暂不支持。
- 修复已知缺陷。

## 1.2.0 (2017.05.25)

- 修复缺陷，提高 SDK 稳定性。
- 支持 Android x86 架构。

## 1.1.0 (2017.04.11)

**聊天室**

- 登录聊天室。
- 发送消息。
- 断线重连。
- 聊天室网络状态通知。
- 聊天室队列服务（针对直播连麦场景使用）。

## 1.0.0 (2017.03.02)

**连接管理**

- 登录与退出。
- 断线重连。

**用户信息管理**

- 查看、更新用户信息。
- 用户关系（消息静音，黑名单等）。

**好友管理**

- 获取好友列表。
- 添加/删除好友。

**会话消息**

- 发送与接收消息。
- 多媒体信息下载（图片/文件/语音等）。
- 消息已读状态。
- 最近联系人列表显示与管理。
- 消息记录查询与多种删除方式。

**高级群（类似企业群）**

- 创建及解散高级群。
- 添加成员。
- 退出高级群。
- 成员权限配置。
- 邀请入群。
- 入群验证配置。
- 踢人出群。
- 获取群成员。
- 群列表变更事件。
- 获取群资料。
- 群成员/资料变更事件。
- 获取某账号已加入的群列表。
- 更新、发送、接收群消息。

**语音**

- 语音录制。
- 语音播放。
