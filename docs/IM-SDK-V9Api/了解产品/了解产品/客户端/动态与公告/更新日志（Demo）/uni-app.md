
## [1.5.0] - 2024-04-29

支持呼叫组件，实现音视频通话功能，具体请参考[快速实现音视频通话](https://doc.yunxin.163.com/messaging/quick-start/DQyNzQ0MzU?platform=uniapp)。

## [1.4.0] - 2024-03-27

- 支持语音消息播放。
- 支持视频消息发送。
- 支持群设置和群管理能力。

## [1.3.0] - 2024-03-04

- 适配 Vue 2 和 Vue 3 框架。
- 支持离线推送。
- 优化断网重连。

## [1.2.0] - 2024-01-05

兼容 Vue 2 框架。

如需要适配 Vue 2 框架，请参考[快速集成源码（Vue 2）](https://doc.yunxin.163.com/messaging/quick-start/TAzNDUyODY?platform=uniapp)集成。

## [1.1.0] - 2023-11-24

- 支持微信小程序和 h5。
- 新增@功能，支持在群组中@群成员，在会话列表中收到@消息会高亮展示。
- 新增回复消息功能，采用非 Thread 模式实现。
- 新增转发消息功能，目前仅支持单条消息的转发。
- 新增好友删除功能，删除后，双方列表中都不再显示。（删除好友时不会自动解除拉黑。）
- 优化好友添加的逻辑。添加好友时自动解除拉黑。
- 支持群成员列表的排序。
- 支持会话置顶功能。
- 支持单聊会话免打扰功能。
