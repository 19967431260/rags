超大群操作后所有群成员会收到一条相应的超大群群通知消息，超大群群通知消息对应的消息对象有一个字段`attach`包含了额外的信息, `attach`有一个字段`type`来标识超大群群通知消息的类型

## 相关系统通知



超大群的如下通知属于系统通知，并非通知消息。通知消息与系统通知的区别，参见[通知消息和系统通知](https://doc.yunxin.163.com/messaging/guide/zg3NzA3NTA?platform=web#通知消息和系统通知)。



<div style="width:150px">系统通知类型</div>  | 说明  | <div style="width:200px">系统通知字段</div>
---- |------| -----
`superTeamInvite`  |  邀请加入超大群   | <ul><li>`from`：邀请方的帐号</li><li>`to`：对应的群ID</li><li>`attach`：<ul><li>`team`：被邀请加入的超大群对象</li></ul></li>
`rejectSuperTeamInvite`|  拒绝“加入超大群”的邀请  | <ul><li>`from`：拒绝入群邀请的用户的帐号</li><li>`to`：对应的群 ID</li></ul>
`applySuperTeam`| 用户主动申请加入超大群, 目标群的群主和管理员会收到一条类型为 `applySuperTeam` 的系统通知 | <ul><li>`from` ：申请方的帐号</li><li> `to` ：对应的群 ID</li></ul> 
`rejectSuperTeamApply` |   如果拒绝入群申请，那么申请人会收到一条类型为 `rejectSuperTeamApply` 的系统通知 |<ul><li>`from` ：拒绝方的帐号</li><li>`to` ：对应的群 ID</li><li> `attach`：<ul><li>`team` ：对应的超大群</li></ul></li></ul>
