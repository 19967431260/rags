<!-- keywords: 超大群,管理,查询,转让,更新,退出 -->


网易云信 NIM SDK 提供超大群功能，支持用户加入、退出、转让、修改、查询超大群。

超大群无法在客户端侧创建和解散，只能通过服务端 API 处理，具体请参见[创建超大群](https://doc.yunxin.163.com/messaging/guide/jk1ODEwNDY?platform=server)。


## 技术原理

网易云信 NIM SDK 的 [`SuperTeamService`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamService.html) 提供超大群操作相关接口， [`SuperTeamServiceObserver`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamServiceObserver.html) 提供超大群相关观察者通知接口，帮助您快速实现和使用超大群的管理功能。 

超大群相关 API 都挂载在 `superteam` 模块中，使用 [`com.netease.nimlib.sdk.superteam`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/package-summary.html) 程序包。



## 超大群相关事件监听

在进行超大群操作前，您可以提前注册监听超大群相关事件。监听后，在进行超大群管理相关操作时，会收到对应的通知。

可以根据用户需求，调用以下方法进行监听。

- [`observeMemberRemove`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamServiceObserver.html#observeMemberRemove-com.netease.nimlib.sdk.Observer-boolean-)：移除超大群成员的观察者通知
- [`observeMemberUpdate`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamServiceObserver.html#observeMemberUpdate-com.netease.nimlib.sdk.Observer-boolean-)：超大群成员信息变化观察者通知
- [`observeTeamRemove`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamServiceObserver.html#observeTeamRemove-com.netease.nimlib.sdk.Observer-boolean-)：移除超大群的观察者通知
- [`observeTeamUpdate`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamServiceObserver.html#observeTeamUpdate-com.netease.nimlib.sdk.Observer-boolean-)：超大群信息变动观察者通知
- [`observeReceiveMessage`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamServiceObserver.html#observeReceiveMessage-com.netease.nimlib.sdk.Observer-boolean-)：超大群消息接收观察者通知

示例代码如下：
```
// 移除群成员的观察者通知。
private Observer<SuperTeamMember> memberRemoveObserver = new Observer<SuperTeamMember>() {
	@Override
	public void onEvent(SuperTeamMember member) {
	}
};
// 注册/注销观察者
NIMClient.getService(SuperTeamServiceObserver.class).observeMemberRemove(memberRemoveObserver, register);

// 群成员信息变化观察者通知。超大群添加新成员，成员信息变化会收到该通知。
// 返回的参数为有更新的群成员信息列表。
Observer<List<SuperTeamMember>> memberUpdateObserver = new Observer<List<SuperTeamMember>>() {
	@Override
	public void onEvent(List<SuperTeamMember> members) {
	}
};
// 注册/注销观察者
NIMClient.getService(SuperTeamServiceObserver.class).observeMemberUpdate(memberUpdateObserver, register);

// 移除超大群的观察者通知。在退群，被踢时，会收到该通知。
Observer<SuperTeam> teamRemoveObserver = new Observer<SuperTeam>() {
    @Override
    public void onEvent(SuperTeam team) {
    // 由于界面上可能仍需要显示群名等信息，因此参数中会返回 Team 对象。
    // 该对象的 isMyTeam 接口返回为 false。
    }
};
// 注册/注销观察者
NIMClient.getService(SuperTeamServiceObserver.class).observeTeamRemove(teamRemoveObserver, register);

// 超大群信息变动观察者。超大群更新的通知通过该接口传递。
Observer<List<SuperTeam>> teamUpdateObserver = new Observer<List<SuperTeam>>() {
    @Override
    public void onEvent(List<SuperTeam> teams) {
    }
};
// 注册/注销观察者
NIMClient.getService(SuperTeamServiceObserver.class).observeTeamUpdate(teamUpdateObserver, register);

//超大群消息接收的观察者通知。
Observer<List<IMMessage>> teamMessageObserver = new Observer<List<IMMessage>>() {
    @Override
        public void onEvent(List<IMMessage> messages) {
        }
};
// 注册/注销观察者
NIMClient.getService(SuperTeamServiceObserver.class).observeReceiveMessage(teamMessageObserver, register);
```

:::note note
由于获取超大群信息和群成员信息需要跨进程异步调用，开发者最好能在第三方 APP 中做好超大群和群成员信息缓存，查询超大群和群成员信息时都从本地缓存中访问。在超大群或者群成员信息有变化时，SDK 会告诉注册的观察者，此时，第三方 APP 可更新缓存，并刷新界面。
:::

## 实现流程

本章节通过群主、管理员、普通成员之间的交互为例，介绍超大群管理的实现流程。

## 加入超大群

加入超大群可以通过以下两种方式：
- 用户接受邀请入群。
- 用户主动申请入群。

### 邀请入群

::: note note
邀请入群的权限可以通过 `InviteMode` 来定义，设为 `Manager`，那么仅限群主和管理员可以邀请人进群；设为 `All` ，那么超大群内的所有人都可以邀请人进群。
:::

通过调用 [`addMembers`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamService.html#addMembers-java.lang.String-java.util.List-java.lang.String-) 方法邀请其他用户加入超大群。
  - 若超大群的被邀请模式 `BeInviteMode` 为 `NoAuth`，那么无需验证，其他用户可直接加入超大群。
  - 若超大群的被邀请模式 `BeInviteMode` 为 `NeedAuth`，那么需要被邀请用户同意才能加入超大群。

如果在被邀请成员中存在成员拥有的超大群数量已达上限，则会返回失败成员的账号列表。  

**参数说明：**

| 参数   | 说明     |
|  ----    | --------- |
|teamId  | 超大群ID |
|accounts| 邀请入群的用户账号列表|
|postscript|邀请附言，长度不得超过 5000|

 
- 发起邀请后，被邀请用户会收到 [`SystemMessage`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/msg/model/SystemMessage.html) 系统通知，其通知类型为`SuperTeamInvite`。
- 被邀请用户可以调用 [`acceptInvite`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamService.html#acceptInvite-java.lang.String-java.lang.String-) 方法接受入群邀请，接受即入群。所有超大群成员会收到超大群通知消息（消息类型为 `MsgTypeEnum.notification`），触发事件为`SUPER_TEAM_INVITE_ACCEPT`。
- 也可以调用 [`declineInvite`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamService.html#declineInvite-java.lang.String-java.lang.String-java.lang.String-) 方法拒绝入群邀请。拒绝后，邀请者会收到 [`SystemMessage`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/msg/model/SystemMessage.html) 系统通知，其通知类型为 `SuperTeamInviteReject`。


**示例代码：**

```
NIMClient.getService(SuperTeamService).addMembers(teamId, accounts, postcrip)
    .setCallback(new RequestCallback<Void>() {
            @Override
            public void onSuccess(Void param) {
				// 返回onSuccess，表示拉人不需要对方同意，且对方已经入群成功了
            }

            @Override
            public void onFailed(int code) {
	            // 返回onFailed
            }

            @Override
            public void onException(Throwable exception) {
				...
            }
        });


//接受邀请
NIMClient.getService(SuperTeamService.class).acceptInvite(teamId, inviter).setCallback(...);

//拒绝邀请
NIMClient.getService(SuperTeamService.class).declineInvite(teamId, inviter, "").setCallback(callback);
```  

### 申请入群

通过调用 [`applyJoinTeam`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamService.html#applyJoinTeam-java.lang.String-java.lang.String-) 方法申请加入超大群。
  - 若超大群的加入模式 `VerifyType` 为 `Free`，那么无需验证，其他用户可直接加入超大群。
  - 若超大群的加入模式 `VerifyType` 为 `Apply`，那么需要群主或者群管理员同意才能加入超大群。
  - 若超大群的加入模式 `VerifyType` 为 `Private`，那么该超大群不接受入群申请，仅能通过邀请方式入群。

:::note note
直接加入超大群或者进入等待验证状态时，返回超大群信息。
:::

**参数说明：**

| 参数 | 说明     |
|  ----  | --------- |
|tid | 超大群ID |
|postscript|申请附言，长度不得超过 5000|


- 当用户发起入群申请后，该群群主和管理员会收到 [`SystemMessage`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/msg/model/SystemMessage.html) 系统通知，其通知类型为`SuperTeamApply`。
- 群主和群管理员可以调用 [`passApply`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamService.html#passApply-java.lang.String-java.lang.String-) 方法接受入群申请，接受即入群。所有超大群成员会收到通知消息（消息类型为 `MsgTypeEnum.notification`），触发事件为`SUPER_TEAM_APPLY_PASS`。
- 群主和群管理员也可以调用 [`rejectApply`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamService.html#rejectApply-java.lang.String-java.lang.String-java.lang.String-) 方法拒绝入群申请。拒绝后，申请者会收到 [`SystemMessage`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/msg/model/SystemMessage.html) 系统通知，其通知类型为 `SuperTeamApplyReject`。任意一管理员操作后，其他管理员再操作都会失败。

**示例代码：**

```
NIMClient.getService(SuperTeamService.class).applyJoinTeam(teamId, postscript).setCallback(new RequestCallback<SuperTeam>() {
    @Override
    public void onSuccess(SuperTeam team) {
        //发送加入超大群请求成功
    }

    @Override
    public void onFailed(int code) {
      	//仅仅是申请成功
        if(code == ResponseCode.RES_TEAM_APPLY_SUCCESS){
            Toast.makeText(SuperTeamMsgActivity.this, "等待验证", Toast.LENGTH_LONG).show();
        }else{
            Toast.makeText(SuperTeamMsgActivity.this, "申请失败, code=" + code, Toast.LENGTH_LONG).show();
        }
    }
    
    @Override
    public void onException(Throwable throwable) {
        Log.e(TAG, "apply join team failed:" + throwable);
    }
});

//同意入群申请
NIMClient.getService(SuperTeamService.class).passApply(tid, account).setCallback(...);

//拒绝入群申请
NIMClient.getService(SuperTeamService.class).rejectApply(tid, account, "您已被拒绝").setCallback(...);
```

## 转让超大群

::: note note
只有群主才有转让超大群的权限。
:::

通过调用 [`transferTeam`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamService.html#transferTeam-java.lang.String-java.lang.String-boolean-) 方法将超大群转让给其他成员。

- 转让群后, 群主身份转移，所有超大群成员会收到通知消息（消息类型为 `MsgTypeEnum.notification`），触发事件为`SUPER_TEAM_CHANGE_OWNER`。
- 如果转让群的同时离开群, 那么相当于同时调用[`quitTeam`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamService.html#quitTeam-java.lang.String-)主动退群。所有超大群成员会收到通知消息（消息类型为 `MsgTypeEnum.notification`），触发事件为`SUPER_TEAM_LEAVE`。


**参数说明：**

| 参数  | 说明     |
|  ----   | --------- |
|tid | 超大群ID |
|account|转让后的群主账号|
|quit|转让群的同时是否退出该群<br/>true：退出<br/>false：不退出，身份变为普通群成员|

**示例代码：**

```
NIMClient.getService(SuperTeamService.class).transferTeam(teamId, account, false)
.setCallback(new RequestCallback<List<SuperTeamMember>>() {
    @Override
    public void onSuccess(List<SuperTeamMember> members) {
        // 群转移成功
    }

    @Override
    public void onFailed(int code) {
        // 群转移失败
    }

    @Override
    public void onException(Throwable exception) {
		// 错误
    }
});
```

## 退出超大群

退出超大群可以通过以下两种方式：
- 群主或群管理员将用户踢出超大群。
- 用户主动退群。

### 踢人出群
::: note note
- 只有群主和管理员才能将成员踢出超大群。
- 管理员不能踢群主和其他管理员。
:::

通过调用 [`removeMember`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamService.html#removeMember-java.lang.String-java.lang.String-) 方法将成员踢出超大群。也可以调用[`removeMembers`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamService.html#removeMembers-java.lang.String-java.util.List-) 方法批量移除群成员。

移除成员后，所有超大群成员会收到通知消息（消息类型为 `MsgTypeEnum.notification`），触发事件为`SUPER_TEAM_KICK`。


**参数说明：**

| 参数 | 说明     |
|  ----  | --------- |
|teamId  | 超大群ID |
|members|被踢出的群成员账号列表|


**示例代码：**

```
NIMClient.getService(SuperSuperTeamService.class).removeMember(teamId, account).setCallback(new RequestCallback<Void>() {
    @Override
    public void onSuccess(Void param) {
       // 成功
    }

    @Override
    public void onFailed(int code) {
        // 失败
    }

    @Override
    public void onException(Throwable exception) {
        // 错误
    }
});
```

### 主动退群


通过调用 [`quitTeam`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/superteam/SuperTeamService.html#quitTeam-java.lang.String-) 方法主动退出超大群。

除群主（需先转让群主）外，其他用户均可以直接主动退群。主动退群后, 所有超大群成员会收到通知消息（消息类型为 `MsgTypeEnum.notification`），触发事件为`SUPER_TEAM_LEAVE`。


**示例代码：**

```
NIMClient.getService(SuperTeamService.class).quitTeam(teamId).setCallback(new RequestCallback<Void>() {
    @Override
    public void onSuccess(Void param) {
       // 退群成功
    }

    @Override
    public void onFailed(int code) {
        // 退群失败
    }

    @Override
    public void onException(Throwable exception) {
        // 错误
    }
});
```

## 修改超大群信息

::: note note
修改超大群信息需要权限。若该超大群的群信息修改权限（[`TeamUpdateModeEnum`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/team/constant/TeamUpdateModeEnum.html)）为 `Manager`，那么只有群主和管理员才能修改超大群信息；若为 `All`，则超大群内的所有人都可以修改超大群信息。
:::

可更新的超大群属性请参见[`TeamFieldEnum`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/team/constant/TeamFieldEnum.html)。

| 参数  |类型| 说明     |
|---|----|---|
|AllMute|[`TeamAllMuteModeEnum`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/team/constant/TeamAllMuteModeEnum.html)	|超大群禁言（群全员禁言），该字段只读，使用“群资料更新”接口更新该字段无效
|Announcement|String|	超大群公告	
|BeInviteMode|[`TeamBeInviteModeEnum`](https://doc.yunxin.163.com/docs/interface/%E5%8D%B3%E6%97%B6%E9%80%9A%E8%AE%AFAndroid%E7%AB%AF/IM_Android/com/netease/nimlib/sdk/team/constant/TeamBeInviteModeEnum.html)|	超大群被邀请模式：被邀请人的同意方式
