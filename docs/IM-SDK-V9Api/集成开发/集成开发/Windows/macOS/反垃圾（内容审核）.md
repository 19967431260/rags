
目前云信支持三类内容反垃圾功能，分别为客户端本地反垃圾、安全通和第三方反垃圾。您可按需开通和配置上述反垃圾功能，满足您的业务内容安全审核需求，避免内容违规导致的业务风险。


## <span id="客户端本地反垃圾">客户端本地反垃圾</span>


### 功能介绍

客户端本地反垃圾是指在发送消息前，由客户端先对内容进行审查，并根据审查结果，决定是否投递、替换消息。


::: note notice
客户端本地反垃圾可用于对单聊、群聊和聊天室的文本消息进行内容安全检测。如需检测图片、音视频等其他类型的消息，请使用[安全通](#安全通)。
:::

### 实现流程

#### 准备阶段

1. 在控制台首页**应用管理**中选择应用，然后单击 **IM 即时通讯**下的**功能配置**按钮进入功能配置页。

    <img alt="image.png" src="https://yx-web-nosdn.netease.im/common/b4c08cb9c2fa97f2bcc99523ffae697a/image.png" style="width:60%">

2. 顶部选择**全局功能**页签，开通**客户端反垃圾**功能。

    ![客户端反垃圾.png](https://yx-web-nosdn.netease.im/common/a7c10176c56f5bd9c435983b0e7341c9/客户端反垃圾.png)


3. 阅读并确认信息后，单击**确认**。


4. 开通后，单击**子功能配置**配置反垃圾词库。

    
    ![客户端反垃圾子功能配置.png](https://yx-web-nosdn.netease.im/common/86bf84c7b191b6021da3162ff4720ade/客户端反垃圾子功能配置.png)


    ![配置词库.png](https://yx-web-nosdn.netease.im/common/3df76180856e1720f3edb840fc4ae3e5/配置词库.png)

    
    替换/拦截规则| 说明
    ---- | -------------- 
    本地拦截库 | 客户端检测到文本消息存在敏感词汇后，直接拦截违规文本消息。发送者将收到对应的警告或提示，一般对应等级最高的违规，如消息存在涉政、反动内容。
    本地替换库 | 客户端将检测到的敏感词汇替换成指定的文本，再将替换后的消息发送给服务器。
    服务端拦截库 | 检测到敏感词汇后，云信服务器拦截涉及的违规消息。违规消息仅对发送者可见，接收方无法收到违规消息。一般对应消息存在广告等违规场景。


5. 完成配置后单击**发布词库**将反垃圾词库下发到客户端 SDK。


#### 运行阶段

##### 步骤1：获取反垃圾词库

调用[`Init`](https://doc.yunxin.163.com/docs/interface/messaging/pc/doxygen/Latest/zh/classnim_1_1_client.html#ad107cfd26954bb8d7d1f9e1951c19b67)方法进行初始化时，将[`client_antispam_`](https://doc.yunxin.163.com/docs/interface/messaging/pc/doxygen/Latest/zh/structnim_1_1_s_d_k_config.html#a3b9508b76a2a67dbe1a55164ff9aab26)开启，开启后 SDK 将从云信服务端获取反垃圾词库。   



##### 步骤2：审查待发送内容


调用[`Tool::FilterClientAntispam`](https://doc.yunxin.163.com/docs/interface/messaging/pc/doxygen/Latest/zh/classnim_1_1_tool.html#a635c934bba60ad00d21f3658d56db0c8)方法判断某文本消息是否存在违规内容。


  












## <span id="安全通">安全通</span>


### 功能介绍

安全通（又称“易盾反垃圾”）是网易云信提供的内容安全增值业务，为您提供全方位的内容安全检测服务。开通安全通功能并配置安全通检测规则后，指定类型的消息都会先经由安全通进行内容安全检测，之后才会转发给接收端的应用用户。安全通目前支持单聊、群聊、聊天室的文本消息、图片消息、自定义消息以及用户头像和用户资料等类型的内容安全检测。

更多 IM 安全通的功能介绍，请参见[安全通概述](/docs/TM5MzM5Njk/Dg0NDY2NzY)。

### 技术原理


![安全通技术原理.png](https://yx-web-nosdn.netease.im/common/9d2031ae1ad1873e93330a9f5be66c9a/安全通技术原理.png)


::: note note
若您需要云信服务器将审核结果抄送到您的应用服务器，即上图中的流程 6，请先开通[易盾异步反垃圾抄送](https://doc.yunxin.163.com/docs/TM5MzM5Njk/TcxNzU4NzU?platformId=60353#易盾异步反垃圾抄送)。
:::

### 实现流程


#### 准备阶段

前往云信控制台[开通安全通](https://doc.yunxin.163.com/messaging/docs/jYxOTcyNzY?platform=server)。 


#### 运行阶段



开通安全通服务后，调用相关方法发送消息时，传入下列参数可实现消息内容过安全通审核。匹配消息体命中敏感词，会返回审核结果`anti_spam_res`。`anti_spam_res`为 JSON 字符串格式，请自行解析或者反转成 JSON 对象使用。

- 内容审核入参说明
| <div style="width:140px" align="left">参数</div> |<div align="left">是否必传</div>|<div style="width:100px" align="left">类型</div>| <div align="left">说明</div> |
| --- |---|---| ---- |
| `anti_apam_using_yidun_`  |  否   | int    |  是否使用安全通对当前消息进行内容安全检测，默认检测。如不需要检测，可设置为 0 |
| `anti_spam_enable_` | 否 |[`BoolStatus`](https://doc.yunxin.163.com/docs/interface/messaging/pc/doxygen/Latest/zh/nim__sdk__util_8h.html#a2f8bd3848595e6c5bcc2f9fd9637a58c) | 是否需要对自定义消息的指定内容（`anti_spam_content_`）进行安全检测，默认不检测 |
| `anti_spam_content_`| 否 |std::string | 自定义消息中需额外进行审核的内容，`anti_spam_enable_` 开启后才能配置该参数，且仅对自定义消息有效。必须传递 JSON 格式字符串，长度不超过 5,000 字节<br><br>格式如下：{"type": 1, "data": "custom content"} <br>字段说明：<ul><li>type: 1-文本、2-图片、3-视频</li><li>data：内容可以是文本内容、图片地址、视频地址</li></ul>|

## <span id="第三方反垃圾">第三方反垃圾</span>

如果您的应用已接入第三方内容安全审核服务（第三方反垃圾服务），您可通过云信服务端的[第三方回调](https://doc.yunxin.163.com/docs/TM5MzM5Njk/jI3ODc2ODE?platformId=60353)服务来实现发送方发送消息后先过第三方内容安全审核服务再判断是否投递至接收方。
