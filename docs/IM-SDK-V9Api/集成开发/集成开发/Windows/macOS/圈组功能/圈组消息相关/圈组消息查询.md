## 功能介绍
调用`GetMessages`查询历史消息。

## 示例代码

```cpp
QChatGetMessagesParam param;
param.server_id = 123456;
param.channel_id = 123456;
param.from_time = 0;
param.to_time = 0;
param.exclude_msg_server_id = 123456;
param.limit = 20;
param.reverse = false;
param.cb = [this](const QChatGetMessagesResp& resp) {
    if (resp.res_code != NIMResCode::kNIMResSuccess) {
        // error handling
        return;
    }
    // process response
    // ...
};
Message::GetMessages(param);
```
