NIM SDK 的[`MessageService`](https://doc.yunxin.163.com/messaging/references/flutter/dartdoc/Latest/zh/nim_core/MessageService-class.html)类，提供相关方法，支持更新消息的扩展字段。

::: note notice
请不要对通知消息（即[`NIMMessageType `](https://doc.yunxin.163.com/messaging/references/flutter/dartdoc/Latest/zh/nim_core/NIMMessageType.html)为`notification`的消息）进行更新。
:::

## 前提条件

已监听消息状态变化（注册[`onMessageStatus`](https://doc.yunxin.163.com/messaging/references/flutter/dartdoc/Latest/zh/nim_core/MessageService/onMessageStatus.html)事件流），且已发送消息。


## <span id="扩展字段更新">实现方法</span>

NIM SDK 支持更新客户端数据库内的消息的客户端扩展字段。调用[`updateMessage`](https://doc.yunxin.163.com/messaging/references/flutter/dartdoc/Latest/zh/nim_core/MessageService/updateMessage.html)即可进行扩展字段更新。 

```
NimCore.instance.messageService.updateMessage(message).then((value){
        if(value.isSuccess){
          //todo success
        }
      });
```
