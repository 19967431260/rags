圈组是网易云信IM即时通讯服务的全新能力，可用来帮助您构建“类 Discord 即时通讯社群”。圈组与单聊（点对点聊天）、群聊、聊天室一起构成网易云信IM即时通讯服务的四大能力模块。


::: note note
圈组总体功能及优势介绍，请参见[圈组功能](https://doc.yunxin.163.com/messaging/docs/zMyNjEyOTk?platform=flutter)。
:::
