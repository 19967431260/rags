云信 NIM SDK 支持全局的推送属性设置，用于设置离线推送免打扰。

当用户不需要在离线时接收第三方推送服务的推送，或者只需要接收某段时间的离线推送时，用户可以通过配置推送属性来实现。

## 前提条件

已[实现第三方离线推送](https://doc.yunxin.163.com/messaging/docs/zUyMjE5NTE?platform=flutter)。
 

## 实现方法

- 方式一：调用 [`enablePushServiceAndroid`](https://doc.yunxin.163.com/messaging/references/flutter/dartdoc/Latest/zh/nim_core/SettingsService/enablePushServiceAndroid.html) 方法关闭第三方推送服务，关闭后，将不再接收离线消息。

    ```
    /// 关闭推送服务
    NimCore.instance.settingsService.enablePushService(false);
    ```


- 方式二：在开启第三方推送服务的情况下，通过 [`setPushNoDisturbConfig`](https://doc.yunxin.163.com/messaging/references/flutter/dartdoc/Latest/zh/nim_core/SettingsService/setPushNoDisturbConfig.html) 方法设置推送服务的免打扰（不接收推送消息）时间段。

    具体参数如下：

    <table>
    <tr><th>参数</th><th>类型</th><th>说明</th></tr>
    <tr><td>enable</td><td>BOOL</td><td>送消息是否开启免打扰，YES 表示开启免打扰</td></tr>
    <tr><td>endTime</td><td>String </td><td>免打扰结束时间，时间参数为北京时间的 24 小时计数 HH:mm</td></tr>
    <tr><td>startTime </td><td>String </td><td>免打扰开始时间，时间参数为北京时间的 24 小时计数 HH:mm <br/>如果结束时间小于开始时间，免打扰时间为开始时间-24:00-结束时间</td></tr>
    </table>

    示例代码如下：

    ```
    NIMPushNoDisturbConfig config = NIMPushNoDisturbConfig(
      enable: true,
      startTime: '22:00',
      endTime: '08:00'
    );
    NimCore.instance.settingsService.setPushNoDisturbConfig(config);
    ```

    :::note note
    - 请同时设置免打扰开启时间和结束时间。
    - 设置完成后，可以调用 [`getPushNoDisturbConfig`](https://doc.yunxin.163.com/messaging/references/flutter/dartdoc/Latest/zh/nim_core/SettingsService/getPushNoDisturbConfig.html) 方法获取推送服务免打扰的时间。
    :::
