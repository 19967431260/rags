<!--keywords: 未读数,监听,获取,标记,已读,内置系统通知-->

内置系统通知未读数，指在云信系统中，用户接收到的内置系统通知未读总数。本文介绍如何获取内置系统通知未读数和标记已读并提供相应的示例代码。

## 技术原理

网易云信 NIM SDK 提供内置系统通知未读数变更事件的监听接口。用户在处理系统通知后，SDK 会触发该事件，收到未读数变更信息。您可以调用[`NIMSystemNotificationManagerDelegate`](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/d2/d52/protocol_n_i_m_system_notification_manager_delegate-p.html)
接口的 `onSystemNotificationCountChanged:` 方法监听该事件。

内置系统通知对象 [`NIMSystemNotification`](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/d0/db5/interface_n_i_m_system_notification.html) 中通过属性 `read` 标志该条内置系统通知是否未读。


## 监听内置系统通知未读数

只有在注册监听内置系统通知未读数变化事件后，用户才会收到对应的系统通知。

通过调用 [`addDelegate:`](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/de/d93/protocol_n_i_m_system_notification_manager-p.html#a479b58fc3467d4ab121585fc2521ae93) 方法添加系统通知通知委托，再调用 [`onSystemNotificationCountChanged:`](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/d2/d52/protocol_n_i_m_system_notification_manager_delegate-p.html#a3919375bd7440625d8e05d3c99e5b34f) 方法监听内置系统通知未读数变化。


**示例代码：**

```objc
/// 自定义类实现 NIMSystemNotificationManagerDelegate 接口

/// 系统通知委托接口调用类声明
/// NIMSystemNotificationAdapter.h
@interface NIMSystemNotificationAdapter :NSObject<NIMSystemNotificationManagerDelegate>
@end
/// 系统通知委托接口调用类实现
/// NIMSystemNotificationAdapter.m
@implementation NIMSystemNotificationAdapter
/// 收到系统通知回调
- (void)onSystemNotificationCountChanged:(NSInteger)unreadCount {
...
}
@end

/// main.m
    /// 实例化接口调用的类
    static NIMSystemNotificationAdapter *adpater;
    adpater = [[NIMSystemNotificationAdapter alloc] init];
    /// 添加系统通知委托
    [[[NIMSDK sharedSDK] systemNotificationManager] addDelegate:adpater];
    /// 当收到系统通知时，调用h onSystemNotificationCountChanged 回调
```

## <span id="查询内置系统通知未读数">查询内置系统通知未读数</span>

### <span id="查询内置系统通知未读总数">查询内置系统通知未读总数</span>

通过调用 [`allUnreadCount`](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/de/d93/protocol_n_i_m_system_notification_manager-p.html#aa2efe3f6cd6ec19158e441bda5463006) 方法查询本地存储的内置系统通知未读总数。

**示例代码：**

```objc
  /// 获取 未读系统消息数量
    NSInteger unreadCount = [[[NIMSDK sharedSDK] systemNotificationManager] allUnreadCount];
    NSLog(@"All unread count: %lu", unreadCount);
```

### <span id="查询指定类型的内置系统通知未读数">查询指定类型的内置系统通知未读数</span>

通过调用 [`allUnreadCount:`](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/de/d93/protocol_n_i_m_system_notification_manager-p.html#afa0e434a3b5dd129f061558cc5cb0095) 方法查询指定类型的内置系统通知未读数。

其中入参需要传入 `filter`，通过 [`NIMSystemNotificationFilter`](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/dc/ddb/interface_n_i_m_system_notification_filter.html) 筛选出需要查询未读数的通知类型。

**示例代码：**

```objc
 /// 系统消息过滤器，仅返回符合类型的系统通知
    NIMSystemNotificationFilter *filter = [[NIMSystemNotificationFilter alloc] init];
    /// 过滤器设置需要过滤得到的系统通知类型，其他通知类型详见 NIMSystemNotificationType
    /// 例：
    ///     申请入群: NIMSystemNotificationTypeTeamApply = 0
    ///     拒绝入群: NIMSystemNotificationTypeTeamApplyReject = 1
    ///     邀请入群: NIMSystemNotificationTypeTeamInvite = 2
    ///     拒绝邀请: NIMSystemNotificationTypeTeamIviteReject = 3
    [filter setNotificationTypes: [[NSArray alloc]initWithObjects:
                                   @(NIMSystemNotificationTypeTeamApply),
                                   @(NIMSystemNotificationTypeTeamApplyReject),
                                   nil]];
    /// 获取 过滤后的未读系统消息数量
    NSInteger unreadCount = [[[NIMSDK sharedSDK] systemNotificationManager] allUnreadCount:filter];
    NSLog(@"[All unread count after filter: %lu]", unreadCount);
```


## <span id="标记内置系统通知已读">标记内置系统通知已读</span>

### <span id="标记所有内置系统通知为已读">标记所有内置系统通知为已读</span>

通过调用 [`markAllNotificationsAsRead`](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/de/d93/protocol_n_i_m_system_notification_manager-p.html#a8a57ad947f6fd5da3d275d0a53d2a601) 方法将所有本地存储的内置系统通知标记为已读，即调用后系统通知未读数为零。

**示例代码：**

```objc
[[[NIMSDK sharedSDK] systemNotificationManager] markAllNotificationsAsRead];
```

### <span id="标记指定类型的内置系统通知为已读">标记指定类型的内置系统通知为已读</span>

通过调用 [`markAllNotificationsAsRead:`](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/de/d93/protocol_n_i_m_system_notification_manager-p.html#a5d16c823f06a36086d1495701c9f79ac) 方法将指定类型的内置系统通知标记为已读。

其中入参需要传入 `filter`，通过 [`NIMSystemNotificationFilter`](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/dc/ddb/interface_n_i_m_system_notification_filter.html) 筛选出需要标记已读的通知类型。

**示例代码：**

```objc
/// 系统消息过滤器，仅返回符合类型的系统通知
    NIMSystemNotificationFilter *filter = [[NIMSystemNotificationFilter alloc] init];
    /// 过滤器设置需要过滤得到的系统通知类型，其他通知类型详见 NIMSystemNotificationType
    /// 例：
    ///     申请入群: NIMSystemNotificationTypeTeamApply = 0
    ///     拒绝入群: NIMSystemNotificationTypeTeamApplyReject = 1
    ///     邀请入群: NIMSystemNotificationTypeTeamInvite = 2
    ///     拒绝邀请: NIMSystemNotificationTypeTeamIviteReject = 3
    [filter setNotificationTypes: [[NSArray alloc] initWithObjects:
                                   @(NIMSystemNotificationTypeTeamApply),
                                   @(NIMSystemNotificationTypeTeamInvite),
                                   nil]];
    /// 标记符合过滤器的系统消息为已读
    [[[NIMSDK sharedSDK] systemNotificationManager] markAllNotificationsAsRead: filter];
```

### <span id="标记单条内置系统通知为已读">标记单条内置系统通知为已读</span>

通过调用 [`markNotificationsAsRead:`](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/de/d93/protocol_n_i_m_system_notification_manager-p.html#a2e49d42bc851cc509caa6862d4315792) 方法将指定的某条内置系统通知标记为已读。

其中入参 `notification` 需要传入待标记已读的内置系统通知（`NIMSystemNotification`）。
    
**示例代码：**

```objc
 /// 获取本地存储的系统通知
    NSArray<NIMSystemNotification *> *sysNotifications =
    [[[NIMSDK sharedSDK] systemNotificationManager] fetchSystemNotifications:nil limit:10];
    if ([sysNotifications count] == 0) return;
    /// 待标记已读的系统消息，此示例处为最新的一条系统通知
    NIMSystemNotification *toBeReadNotification = sysNotifications[0];
    /// 标记单条系统消息为已读
    [[[NIMSDK sharedSDK] systemNotificationManager] markNotificationsAsRead:toBeReadNotification];
```
