
圈组支持安全通（内容审核）服务。本文介绍如何实现圈组的内容安全检测。 



## **背景信息**


“类 Discord” 的大型线上社群场景中，充斥着各种不可控的风险因素，例如色情、暴恐和涉政信息等。随着国家监管的日趋严格，各平台需要对线上平台的内容进行管控，然而自行进行内容审核需要投入大量人力进行甄别。网易云信融合网易易盾的内容审核能力，为开发者提供安全通（内容审核）服务（又称“易盾反垃圾”），对内容进行有效的判别和筛选，降低人力投入。

在云信控制台开通安全通功能并配置安全通检测规则后，指定类型的消息都会先经由安全通进行内容安全检测，之后才会转发给接收端的用户。圈组的消息、用户头像、用户资料和自定义消息的内容安全检测均可通过安全通进行审核。




## **前提条件**
- 已[开通圈组功能](https://doc.yunxin.163.com/messaging/guide/TM1OTU0MTM?platform=iOS)。
- 在进行安全通能力的集成前，请确保您已开通 **IM 安全通**，且已在**子功能配置**中配置圈组相关的内容审核功能，具体请参见<a href="https://doc.yunxin.163.com/messaging/guide/jYxOTcyNzY?platform=server" target="_blank">开通 IM 安全通</a>。

## 实现内容审核


### **实现消息内容审核**

SDK 内定义圈组消息内容审核相关配置参数的结构为`NIMQChatMessageAntispamSetting`类。调用<a href="https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/d2/db1/protocol_n_i_m_q_chat_message_manager-p.html#a24452d81b713fb77b6704dcf55521b7b" target="_blank">`sendMessage:toSession:error:`</a>或<a href="https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/d2/db1/protocol_n_i_m_q_chat_message_manager-p.html#a15fb23ede995f9695a134a6d38895a02" target="_blank">`sendMessage:toSession:completion:`</a>方法<a href="https://doc.yunxin.163.com/messaging/guide/zM0OTk2NDM?platform=iOS#实现消息收发" target="_blank">发送圈组消息</a>或调用<a href="https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/d2/db1/protocol_n_i_m_q_chat_message_manager-p.html#ae95e6009f2c88a7f123c9f6219e4419c" target="_blank">`updateMessage:completion:`</a>方法<a href="https://doc.yunxin.163.com/messaging/guide/TU4OTU0Njk?platform=iOS" target="_blank">更新圈组消息</a>时，您可通过该类的成员参数进行内容审核配置。




<a href="https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/d0/d1a/interface_n_i_m_q_chat_message_antispam_setting.html" target="_blank">`NIMQChatMessageAntiSpamSetting`</a>类的 **部分** 内置方法说明如下：


<div style="width:120px">方法</div> | <div style="width:100px">参数类型</div> |说明
---- | -------------- | --------- |------
`antiSpamUsingYidun` | BOOL | 设置是否对当前消息使用安全通进行内容审核的信息。安全通开通后，默认对消息进行内容审核
`antiSpamBusinessId` | NSString* | 设置安全通的自定义反垃圾（即内容审核）业务的 ID。自定义反垃圾业务主要用来针对单条消息进行除了默认反垃圾业务以外的内容审核。 <note type=note>开通安全通后，云信控制台会自动生成默认反垃圾业务。客户端不需要配置业务 ID 就能默认采用安全通的默认反垃圾业务，如果需要自定义反垃圾业务，请先通过云信官网首页提供的微信、在线聊天和电话等方式跟商务经理发出申请，获取对应的业务 ID。</note>
`yidunAntiCheating`|  NSDictionary*  |  设置透传给易盾的反作弊检测参数，格式为 JSON，长度限制1024 ，具体请参见<a href="https://support.dun.163.com/documents/2018041901?docId=420802794453520384#%E6%AD%A5%E9%AA%A42%EF%BC%9A%E6%8E%A5%E5%85%A5%E6%96%87%E6%9C%AC%E6%A3%80%E6%B5%8Bapi,%E4%BC%A0%E5%85%A5%E5%8F%8D%E4%BD%9C%E5%BC%8Atoken%E5%AD%97%E6%AE%B5" target="_blank">文本防刷版开发文档</a> 
`yidunAntiSpamExt` | NSString* | 设置透传给易盾的反垃圾扩展字段，具体请参见<a href="https://support.dun.163.com/documents/588434200783982592?docId=476559002902757376#/%E7%94%A8%E6%88%B7%E6%89%A9%E5%B1%95%E5%8F%82%E6%95%B0" target="_blank">易盾的反垃圾增强版用户可扩展参数</a>，格式为 JSON，长度限制1024



文本消息和图片消息发送后，SDK 支持通过<a href="https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/d6/d29/interface_n_i_m_q_chat_message_antispam_result.html" target="_blank">`NIMQChatMessageAntiSpamResult`</a>类所包含的字段通知发送者消息内容的审核结果，具体字段信息参见[审核结果字段说明](https://doc.yunxin.163.com/messaging/guide/Dg0NDY2NzY?platform=server#审核结果字段说明)。消息发送者可通过注册<a href="https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/de/da7/protocol_n_i_m_chat_manager_delegate-p.html#ad65c6bf33fc6fca06268a526782cd362" target="_blank">`sendMessage:didCompleteWithError:`</a>回调函数获取文本消息和图片消息的内容审核结果。 


::: note notice
其他类型消息的内容审核结果，不支持通知消息发送者。
:::



### **实现资料内容审核**

圈组资料包括服务器资料、频道资料、身份组资料和成员资料。SDK 支持通过设置<a href="" target="_blank">`antiSpamBusinessId`</a>对上述资料类型进行内容审核。


<div style="width:120px">方法</div> | <div style="width:100px">参数类型</div> |  说明
---- | -------------- | --------- |------
`antiSpamBusinessId` | NSString*	 | 获取安全通的自定义反垃圾（即内容审核）业务的 ID。自定义反垃圾业务可用来针对圈组的资料信息进行自定义的内容审核。<note type=note>开通安全通后，云信控制台会自动生成默认反垃圾业务。客户端不需要配置业务 ID 就能默认采用安全通的默认反垃圾业务，如果需要自定义反垃圾业务，请先通过云信官网首页提供的微信、在线聊天和电话等方式跟商务经理发出申请，获取对应的业务 ID。</note>

#### 服务器资料内容审核

调用如下方法创建或修改服务器时，可通过`antiSpamBusinessId`对服务器资料进行内容审核。
