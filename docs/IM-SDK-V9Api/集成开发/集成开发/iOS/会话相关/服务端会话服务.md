 
服务端会话列表由服务器维护：

- 支持存储用户全量的会话历史列表以及保存最近一条消息

- 支持客户端从服务端拉取会话列表，不支持同步到本地会话列表

- 不支持未读数

## 获取服务端会话

### 分页获取服务端会话列表

- 接口：[fetchServerSessions:completion:](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/d5/d94/protocol_n_i_m_conversation_manager-p.html#a834c8b81ddc338b7acdd75dc4ad9f9ee)

- 参数说明：

| 参数         | 说明                                             |
| :----------- | :----------------------------------------------- |
| `option`      | 分页查询选项，可设置为空，代表全量获取。详见 [NIMFetchServerSessionOption](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/d6/ddd/interface_n_i_m_fetch_server_session_option.html)。|
| `completion`      | 获取服务端历史会话异步回调，详见 [NIMFetchRecentSessionsHistoryBlock](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/de/d36/_n_i_m_conversation_manager_protocol_8h.html#afb399f7d010bcc5cd6b504e6ea363644)。|



- 示例代码：

```objc
// 下一页
NIMFetchServerSessionOption * option = [[NIMFetchServerSessionOption alloc] init];
option.needLastMessage = needLastMsg;
option.minTimestamp = min;
option.maxTimestamp = currentMin - 1;
option.limit = limit;
[[NIMSDK sharedSDK].conversationManager fetchServerSessions:option completion:^(NSError * _Nullable error, NSArray<NIMRecentSession *> * _Nullable recentSessions, BOOL hasMore) {}];
```


::: note note
- 首次调用该方法无需传入 `minTimestamp`，返回的时间戳需要在下一次调用时传入 `minTimestamp`。
- 回调参数 `hasMore` 如果为 `YES`，表示未获取完所有服务端会话。
:::
  

### <span id="获取指定会话">获取服务端指定会话</span> 

- 接口：[fetchServerSessionBySession:completion:](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/d5/d94/protocol_n_i_m_conversation_manager-p.html#a5275314757df18fdc5c6a34aecfdea17)

- 参数说明：

| 参数         | 说明                                             |
| :----------- | :----------------------------------------------- |
| `session`      | 会话对象，详见 [NIMSession](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/d3/de1/interface_n_i_m_session.html)。|
| `completion`      | 获取服务端历史会话异步回调，详见 [NIMFetchRecentSessionHistoryBlock](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/de/d36/_n_i_m_conversation_manager_protocol_8h.html#aca9214c1bec0b217990e295cce78c2b9)。|

- 示例代码：

```objc
[[NIMSDK sharedSDK].conversationManager fetchServerSessionBySession:recent.session completion:^(NSError * _Nullable error, NIMRecentSession * _Nullable recentSession) {}];
```

- 参数说明：

| 参数         | 说明                                             |
| :----------- | :----------------------------------------------- |
| `ext`      | 扩展信息|
| `session`      | 会话对象，详见 [NIMSession](https://doc.yunxin.163.com/docs/interface/messaging/iOS/doxygen/Latest/zh/d3/de1/interface_n_i_m_session.html)。|

```objc
 [[NIMSDK sharedSDK].conversationManager updateServerSessionExt:@"newServerExt" session:recentSession.session completion:^(NSError * _Nullable error) { }];
```
