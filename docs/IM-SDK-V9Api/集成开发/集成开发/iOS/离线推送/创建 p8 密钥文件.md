
本文主要介绍如何创建 .p8 推送密钥，后续用于授权云信服务端与 APNs 实现通信。



1. 在[Apple 开发者网站](https://developer.apple.com/)的**Certificates, IDs & Profiles**（证书、标识符和描述文件）中，单击**Keys**（密钥）。

    ![Keys.png](https://yx-web-nosdn.netease.im/common/c7bb859bd486d8c31b403cea2296787d/Keys.png)

2. 填写 **Key Name** ，并勾选 **Apple Push Notifications service (APNs)**，启用 APNs 服务，再点击 **Continue**。
 
    ![Keyname.png](https://yx-web-nosdn.netease.im/common/f569ae630ab5af4321ee330ad6b518ad/Keyname.png)


3. 确认密钥配置后，单击**Confirm**。

4. 单击**Download**，生成并下载密钥文件。密钥会以文件扩展名为 .p8 的文件下载到本地。将推送证书（.cer 文件）下载到本地。
    :::note note
    - 密钥不会存储在您的开发者帐户中，也无法再次下载（只能下载一次），因此请将该文件存储在安全的位置。
    - 如果**Download**按钮处于停用状态，则表示您已经下载过该密钥文件。
    - 记录 Key ID，后续上传至云信控制台需要使用。
    :::
5. 单击 **Done** 。
