## 问题描述

系统出现崩溃，出现以下报错：

`Context.startForegroundService() did not then call Service.startForeground():ServiceRecord{……/com.netease.nimlib.service.ResponseService} `

## 问题原因

主要是由于自启动导致的。

## 解决方案


请关闭自启动，具体请参考[去除自启动](https://doc.yunxin.163.com/messaging/guide/Tg3NDExMDA?platform=android#去除自启动)。

另外，如果在代码中写入了主动启动 SDK 中的 Service，请去掉此类调用代码，SDK 内部会维护自身 Service 的启动，外部代码动启动 SDK 中的 Service，可能导致运行异常。
