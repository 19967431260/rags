## 问题描述

为了维护平台的安全性和稳定性，自 2024 年 5 月 8 日起，云信侧将不再允许上传 text/html 和 image/svg+xml 类型的文件。

## 问题原因

- 出于安全性考虑

    近期我方发现部分用户利用 text/html 和 image/svg+xml 文件执行页面跳转或进行 XSS（跨站脚本攻击）等违规操作，严重威胁了平台和用户的安全。

- 防止滥用

    限制这些文件类型可以减少恶意攻击的可能性，保护所有用户的在线体验。

## 解决方案

我们理解该措施可能会对确实需要上传上述文件类型的用户造成不便。

如果您有**特殊需求**必须上传 text/html 或 image/svg+xml 文件，**请联系我们的技术支持团队，我们将为您提供一对一的解决方案**。

我们重视每一位用户的需求，并承诺将持续改进我们的服务以满足您的合法需求，同时确保平台的安全。
