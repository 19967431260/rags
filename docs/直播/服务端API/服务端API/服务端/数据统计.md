### **上行推流质量数据**

**URL**
```
POST https://data.yunxinapi.com/live/queryPushStreamStats HTTP/1.1
Content-Type: application/json;charset=utf-8
```

**接口说明**
- 描述：上行推送到网易srs端的数据查询
- 时间粒度：1 秒
- 时效性：数据延迟不超过 1 分钟
- 最大时间跨度为 1 小时
- 最早支持 7 天前的查询

**入参说明**

| 参数名称   | 类型   | 是否必选 | 描述                          |
| ---------- | ------ | -------- | ----------------------------- |
| domain     | String | 是       | 域名                                  |
| streamName | String | 是       | 流名                          |
| beginTime  | Long | 是       | 开始时间，单位：ms，示例：1633017600000 |
