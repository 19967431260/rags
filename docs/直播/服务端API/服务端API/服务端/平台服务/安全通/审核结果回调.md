对直播流的视频和音频进行审核，自动检测视频和语音内容是否涉嫌违规，满足客户的直播内容安全需求。开发者在云信控制台配置结果回调地址后，直播服务会在直播过程中将内容审核结果发送给客户应用服务。

## 审核结果回调信息

详细抄送数据信息如下表所示。

| 参数名称        | 类型     | 描述                                            |
| :-------------- | :------- | :---------------------------------------------- |
| cid             | String   | 频道ID，32位字符串。                              |
| identifier      | String   | 业务标识，固定为live。                            |
| type            | String   | 回调类型，固定值live_anti_trash。                  |
| taskId          | String   | 审核任务唯一标识。                                |
| dataId          | String   | 数据唯一标识。                                    |
| status          | Number   | 检测状态。 101为检测中，102为检测结束。具体请参见[直播音视频解决方案结果获取](https://support.dun.163.com/documents/292111666669342720?docId=341039220029313024) 。                        |
| censorSource    | Number   | 审核来源。<ul><li>0：易盾人审<li>1：客户人审<li>2：易盾机审 |
| evidences       | json对象 | 机审证据信息。                                    |
| reviewEvidences | json对象 | 人审证据信息。                                    |

## 机审证据信息

### evidences 数据结构

| 参数名称 | 类型     | 描述             |
| :------- | :------- | :--------------- |
| audio    | json对象 | 直播语音证据信息。 |
| video    | json对象 | 直播截图证据信息。 |

### audio 数据结构

| <div style="width:80px">参数名称</div>  |  <div style="width:80px">类型</div>    | 描述                                                         |
| :-------- | :------- | :----------------------------------------------------------- |
| action    | Number   | 检测结果。<ul><li>0：通过<li>1：嫌疑<li>2：不通过                       |
| asrStatus | Number   | 音频翻译结果。<ul><li>2：检测中<li>3：检测完成<li>4：检测失败           |
| asrResult | Number   | 音频翻译检测失败原因。<ul><li>1：文件下载失败<li>2：音频流不存在<li>3：解析失败<li>4：文件格式错误 |
| speakerId | String   | 针对接入SDK监听客户，返回说话人的 ID 信息。                        |
| startTime | Number   | 最近开始时间点，单位：毫秒。                                     |
| endTime   | Number   | 最近结束时间点，单位：毫秒。                                     |
| segments  | json数组 | 分类信息详细数据 [直播音频结果获取](http://support.dun.163.com/documents/2018082201?docId=274670761104560128#响应结果) segments数据结构一致。|

### video 数据结构

| <div style="width:80px">参数名称</div>  |  <div style="width:80px">类型</div>    | 描述                                                         |
| :------- | :------- | :----------------------------------------------------------- |
| evidence | json对象 | 证据信息。                                                     |
| labels   | json数组 | 分类信息详细数据 [直播视频结果获取](http://support.dun.163.com/documents/2018041903?docId=150440627406032896#响应结果) labels数据结构一致。 |

### evidence 数据结构

| <div style="width:80px">参数名称</div>  |  <div style="width:80px">类型</div>    | 描述                                                         |
| :-------- | :----- | :----------------------------------------------------------- |
| beginTime | Number | 视频直播当前时间点，单位为毫秒，调用方获取后可自行格式化为可视化时间，如：1599205126000 转换为"2020-09-04 15:38:46"。 |
| endTime   | Number | 视频直播当前时间点，单位为毫秒，调用方获取后可自行格式化为可视化时间，如：1599205126000 转换为"2020-09-04 15:38:46"。 |
| type      | Number | <ul><li>1：图片<li>2：视频                                            |
| url       | String | 证据信息。                                                     |
| speakerId | String | 针对接入SDK监听客户，返回说话人ID信息。                        |
