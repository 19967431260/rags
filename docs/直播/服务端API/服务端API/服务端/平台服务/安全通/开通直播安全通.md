
使用直播内容安全审核功能之前，请先在网易云信控制台进行开通。  

> 注意：直播内容安全审核服务现已开通免费试用，您可以在控制台申请试用。申请通过后，可以免费享受 15 天的直播内容安全审核服务，在 15 天内可以享受高达 500次 的直播内容安全审核。

## <span id="前置条件">前置条件</span>

需要开通直播服务，准备一个可以接收抄送的HTTP服务地址，并确保地址长期有效。云信会将直播内容安全审核结果抄送到该地址。

## <span id="开通配置说明">开通配置说明</span>

1. 登录<a href="https://app.yunxin.163.com/index#/" target="_blank">网易云信控制台</a>。

2. 在首页单击指定应用名称。
3. 在左侧导航栏选择**产品功能** > **直播**，在**直播安全通检测** 区域，单击**免费试用**或**开通正式版**。


    ![直播安全通.png](https://yx-web-nosdn.netease.im/common/dc1453b5ccc34089c489f5523b9e3e40/直播安全通.png)

  ::: note note
   正式版功能涉及计费，请单击**开通正式版**，提交工单联系云信商务经理沟通计费策略。
   :::

  

4. 申请试用后，单击**子功能配置**。


5. 在**直播播安全通检测 子功能配置**页面，单击**抄送地址**右侧的**编辑**按钮，设置**内容安全服务配置**的结果抄送地址。

   内容审核结果会以消息抄送方式发送至配置的抄送地址。

   ![](https://yx-web-nosdn.netease.im/quickhtml%2Fassets%2Fyunxin%2Fdoc%2Flive_ContentSecurityArchitecture_setting_.png)

6. 单击机器过检开关，开启机器过检，并设置相关参数。


   参数 | 描述
   ---- | -------------- | 
   过检类型| 过检类型支持：<ul><li>视频与音频同时过检<li>仅过检视频 | 
   截图频率 | 截图频率默认为 5 秒，支持设置为 1~60 秒。 | 


7. 设置频道免过检名单。

   
    若将直播频道添加该名单内，则该直播服务不会对该频道提交内容内容审核。
 

8. 单击**保存**。





## <span id="接收审核结果">接收审核结果</span>

- 开发者控制台配置抄送地址后，云信直播服务会在直播过程中实时将内容审核结果通过该地址通知给客户。
- 直播内容审核结果数据详见：[地址](https://dev.yunxin.163.com/docs/product/%E7%9B%B4%E6%92%AD/%E6%9C%8D%E5%8A%A1%E7%AB%AFAPI%E6%96%87%E6%A1%A3/3.%E7%9B%B4%E6%92%AD%E5%AE%89%E5%85%A8?#3.3.3%20%E5%AE%A1%E6%A0%B8%E7%BB%93%E6%9E%9C%E5%9B%9E%E8%B0%83%E4%BF%A1%E6%81%AF)


## <span id="禁用直播流">禁用直播流</span>

- 开发者可以通过使用提供了丰富的直播流管理功能，根据接收到的内容审核结果进行处理。
- 若发现违规内容，可以在控制台直播频道下操作禁用直播流，或调用[服务端API](https://dev.yunxin.163.com/docs/product/%E7%9B%B4%E6%92%AD/%E6%9C%8D%E5%8A%A1%E7%AB%AFAPI%E6%96%87%E6%A1%A3/2.%E7%9B%B4%E6%92%AD%E7%AE%A1%E7%90%86?#2.4%20%E7%A6%81%E7%94%A8%E9%A2%91%E9%81%93)进行禁推处理。
