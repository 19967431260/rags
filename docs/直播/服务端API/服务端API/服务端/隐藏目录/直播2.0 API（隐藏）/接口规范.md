## <span id="接口规范">接口规范</span>  

### <span id="格式">格式</span>
- 接口名称
- 接口说明
- 请求方式
- 请求参数
- 返回参数
- 请求示例

### <span id="公共参数">公共参数</span>

所有接口均需要放置以下公共参数在请求头中，用于标识用户和接口鉴权。后续的接口说明不再对这些参数进行说明，但每次发起请求均需要携带。

| 参数	| 类型 |	必须 |	说明 |
| :--- | :----- | :-----| :------- |
|AppKey	| String	|	是	|	开发者平台分配的AppKey |
|Nonce	 |String	|	是	|	随机数（随机数，最大长度128个字符）|
|CurTime |	String	|	是	|	当前UTC时间戳，从1970年1月1日0点0分0秒开始到现在的秒数 |
|CheckSum |	String	|	是	|	服务器认证需要，SHA1(AppSecret+Nonce+CurTime)，16进制字符小写 |

### <span id="接口鉴权">接口鉴权</span>

接口通过请求头中的公共参数进行鉴权。用户在“云信管理控制台-应用详情页-AppKey管理”获取到的安全凭证进行SHA1(AppSecret+Nonce+CurTime)计算。

重要提示: 本文档中提供的所有接口均面向开发者服务器端调用，用于计算CheckSum的AppSecret开发者应妥善保管，可在应用的服务器端存储和使用，但不应存储或传递到客户端，也不应在网页等前端代码中嵌入。

### <span id="返回说明">返回说明</span>
| 参数   | 类型     | 说明          |
| :--- | :----- | :---------- |
| requestId | String | 请求标识 |
| code | Int | 返回结果的状态码 |
| msg | String | 当返回结果的状态码不为200时，包含的错误信息 |
| ret | String | 返回的结果集，json格式 |

> 以下接口的请求或返回参数如果为空，说明此接口除了公共的请求和返回参数外，不再需要输入或输出额外的参数。

### <span id="返回状态码">返回状态码</span>

| 状态码 | 说明                |
| ---- | ------------------- |
| 200  | 成功                |
| 409  | 用户登录认证失败    |
| 500  | 内部错误            |
| 631  | 请求参数错误        |
| 638  | 访问频率超限        |
| 2101 | 请先开通直播服务2.0 |

### <span id="频控">频控</span>

如无特殊说明，频控规则约定如下：

* 直播相关接口，60秒内最多500次请求，无惩罚时间
* 录制相关接口，60秒内最多100次请求，触碰频限后的惩罚时间为100s
