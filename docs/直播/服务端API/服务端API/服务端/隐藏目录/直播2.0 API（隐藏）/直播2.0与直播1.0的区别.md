## <span id="直播2.0与直播1.0的区别">直播2.0与直播1.0的区别</span>  

为区分原直播服务，新版本的直播服务我们称为直播2.0，原直播服务称为直播1.0。

云信直播2.0与直播1.0区别主要是对于流名的管理：
- 在直播1.0中，流名即频道，需要预先创建才可以使用进行推拉流；
- 在直播2.0中不再做这样的限制，可以直接使用符合条件的字符串进行推流（长度不大于128字节，0-9a-z）。
- 当需要提前在流名上进行录制配置时，需要通过录制配置相关接口进行提前配置。
- 直播2.0中不再提供获取流名对应推拉流地址的接口，用户根据文档中的相关规则描述生成推拉流地址，推拉流地址的生成方式见下文。

**注意：**
如果需要使用直播2.0相关功能和接口，需要联系客户经理开通权限。需要注意的是，直播2.0接口不兼容原直播1.0相关接口，即用户开通直播2.0功能后，需要全链路使用直播2.0相关接口，有相关不满足的需求可以与我们沟通反馈。
