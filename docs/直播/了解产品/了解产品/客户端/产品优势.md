网易云信直播融合主流CDN厂商资源，节点覆盖电信、联通、移动、广电等国内大中小十几家运营商、五十多个国家和地区，超 2000 多个 CDN 节点，支持业务无缝出海。全球千万级并发直播观看需求。

## <span id="高并发、低延时">高并发、低延时</span>
- 搭载网易云信自研的全球智能路由网络（WE-CAN），在全球数百个城市铺设了节点。终端登录时，直播 SDK可以访问就近的节点。

- 采用最优寻址算法辅以全局实时调度能力，全球范围内保证端到端**平均时延 < 1s**。

## <span id="强大的流媒体处理能力">强大的流媒体处理能力</span>
提供同步录制、截取缩略图、实时水印和转码等强大的流媒体处理能力。

## <span id="智码超清">智码超清</span>
智码超清技术，在高码率、高清的主流直播场景中（例如游戏直播、线上演唱会、产品发布会等），保证人眼主观画质不变的情况下，**与普通转码相比，平均节省30%~40%的码率**，降低下行流量带来的带宽成本。

## <span id="视频安全性">视频安全性</span>

- 支持IP 黑白名单、HTTPS 协议等安全访问机制，防劫持、防篡改。
- 支持推流拉流URL 防盗链，可自定义鉴权 Key 及过期时间。
- 提供添加水印、直播加密等安全防护技术，减少盗播、盗链风险。

## <span id="内容安全防护方案">内容安全防护方案</span>
- 网易云信安全通解决方案融合网易易盾的内容审核能力，支持对直播流媒体进行反垃圾检测，例如色情低俗检测、涉政检测、暴恐检测、违禁品检测、宗教服饰检测等。另外，针对国内及海外内容运营的痛点，提供全球化的智能AI检测能力。
- 针对不同国家和地区，网易云信提供专属的安全通解决方案，包括泰国、印尼等东南亚地区，以及中东、欧美国家。
