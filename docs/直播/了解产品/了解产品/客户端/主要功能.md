  
## <span id="直播推流">直播推流</span>
| **功能名称** | **功能介绍** |
|------|------|
|推流协议|支持RTMP协议推流|
|推流方式|支持网易云信iOS、Android 推流SDK客户端，也支持常见的第三方如OBS/XSplit/FMLE等推流软件|
|推流设备|支持常见的基于RTMP推流协议的第三方编解码器或盒子|

## <span id="直播拉流">直播拉流</span>
| **功能名称** | **功能介绍** |
|------|------|
|拉流协议|支持RTMP、FLV、HLS标准拉流协议以及nertc低延时拉流协议|
|拉流方式|支持网易云信iOS、Android、及Web等拉流SDK客户端，也支持常见的第三方如VLC等拉流软件|
|实时转码|支持720p、540p、360p、180p多种分辨率格式、转码视频宽高比不变，支持RTMP、FLV及HLS三种拉流协议的实时转码|

## <span id="直播管理">直播管理</span>
| **功能名称** | **功能介绍** |
|------|------|
|管理方式|支持Web端的管理控制台图形化管理及API管理|
|管理控制台|新建、修改、删除、启用、禁用直播频道以及统计查询|
|API管理|新建、修改、删除、启用、禁用直播频道以及统计查询，水印，截图等|
|直播录制|支持通过管理控制或API接口将直播录制并保存到点播平台|

## <span id="直播安全">直播安全</span>
| **功能名称** | **功能介绍** |
|------|------|
|访问限制|支持 IP 黑白名单、HTTPS协议等安全访问机制|
|URL鉴权|支持推流拉流 URL 防盗链，可自定义鉴权 Key 及过期时间|
|直播加密|直播视频加密解决方案，采用私有加密算法对视频流加密，确保视频流安全传输|
|直播审核|支持对直播流进行审核，自动检测视频和语音内容是否涉嫌违规|
|直播禁流|支持对直播频道进行禁止推流的处理|

## <span id="数据统计">数据统计</span>
| **功能名称** | **功能介绍** |
|------|------|
|流量统计|对流量按天为粒度进行统计|
|带宽统计|对带宽峰值按不同时间粒度进行统计|
|频道统计|实时统计直播频道总数和正在直播的频道数|

## <span id="SDK特性">SDK特性</span>
| **功能名称** | **功能介绍** |
|------|------|
|美颜滤镜|iOS和Android推流端支持多种实时美颜滤镜效果|
|纯音频直播|iOS和Android推流端支持纯音频直播模式|
|混音|iOS和Android推流端支持直播中播放背景音乐|
