
## <span id="基本描述">基本描述</span>  

网易云信直播源自网易多年视频技术沉淀，基于专业的跨平台视频编解码技术和大规模视频内容分发网络，提供稳定流畅、低延时、高并发的实时音视频服务，有视频直播需求的用户可调用网易云信直播提供的API或SDK即可将视频直播无缝对接到自身的业务应用系统。

本服务适用于各类音视频直播场景，比如教育行业的在线教育、网络课堂等，娱乐行业的美女直播、游戏直播等，广电行业的电视直播、赛事直播等，另外还适用于企业协作、远程医疗、云监控、庭审直播等其他视频相关行业。

目前，网易云信直播面向注册用户开放。

## <span id="功能亮点">功能亮点</span>

### <span id="码流自适应技术">码流自适应技术</span>
实时监测全链路的网络情况，自适应调整直播流，满足复杂网络条件下高清流畅直播需求。

### <span id="高并发、低延时">高并发、低延时</span>
具备大规模全网分发能力，支持千万级并发直播观看需求，端到端延时可低至1-3秒。

### <span id="强大的流媒体处理能力">强大的流媒体处理能力</span>
提供同步录制、截取缩略图、实时水印和转码等强大的流媒体处理能力。

### <span id="内容安全防护方案">内容安全防护方案</span>
多重安全措施，推流、播放认证、防盗链以及内容涉黄涉政审核等保障直播安全。

### <span id="支持多平台直播">支持多平台直播</span>
从直播推流、服务端到播放器提供全套SDK，迅速接入直播服务，支持Andriod/iOS/Web/PC等平台。
