# 本地帝王蟹 Windows/Mac 安装报错处理
## 问题
本地帝王蟹在 Windows / Mac 电脑上安装失败或者遇到报错，要怎么处理？

## 处理步骤
1. 官网下载最新版：https://skills.netease.im/weclawhelper/local
2. 最新版本仍然安装失败的话：
   - 截图提供下安装界面具体错误现象
   - 收集安装日志（路径如下）

## 日志路径
### Windows
- .weclawhelper 日志：`C:\Users\您的用户名\.weclawhelper\logs`（整个logs压缩）
- ClawHive 日志：`C:\Users\您的用户名\AppData\Roaming\clawhive\logs`（整个logs压缩）

### Mac
- .weclawhelper 日志：`/Users/您的用户名/.weclawhelper/logs`（整个logs压缩）
- ClawHive 日志：`/Users/您的用户名/Library/Logs/clawhive`（整个clawhive压缩）
