本文介绍网易云信含 UI 即时通讯（简称 IM UIKit）Electron 10.x.x 及以上版本的 IM UIKit 更新日志。IM UIKit 兼容的网易云信即时通讯 SDK（简称 NIM SDK）的变更记录请参考 [NIM SDK Electron 更新日志](https://doc.yunxin.163.com/messaging2/concept/zg5MzA5ODI?platform=client)。

<a id="10.1.0 (2026-02-12)"></a>

## 10.1.0 (2025-02-12)

**新增特性**

- 支持合并消息转发功能。
- 支持全文搜索历史聊天记录功能。
- 文件、图片、视频消息支持取消上传和查看上传进度。
- 输入框支持粘贴发送图片、视频、文件。
- 支持转让群、入群审核、入群邀请相关的群管理功能。
- 支持在线/离线状态功能。
- 支持新消息通知管理功能。

**优化**

- 优化回复消息功能。
- 优化群设置页性能。
- 优化包管理。
- 优化日志打印。

**兼容性**

兼容 NIM Electron SDK 10.9.72 版本。

<a id="10.0.0 (2025-11-28)"></a>

## 10.0.0 (2025-11-28)

全新 IM Electron UIKit 组件 **10.0.0 版本** 发布。

底层适配 NIM Electron SDK v10.9.53 版本。如何集成，请参考 [集成 UIKit（Vue 3）](https://doc.yunxin.163.com/messaging2/guide/DM1NjMzNjA?platform=electron)。

主要支持注册与登录、消息、会话、个人信息、好友、通讯录、设置等能力。
