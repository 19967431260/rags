## 10.1.0 (2025-10-31)

**新增特性**

- 收藏消息：用户现可标记和管理重要消息。
- 群组管理：增强群组控制和组织能力。
- 讨论组：支持创建专题讨论组，提升团队协作体验。
- 多行文本输入：输入框支持更灵活的文本编辑。
