本文介绍了网易云信即时通讯（NIM）提供的 UI 组件（UI Kit）适配鸿蒙系统应用的更新日志，后续简称 IM UIKit。IM UIKit 兼容的网易云信即时通讯 SDK（简称 NIM SDK）的变更记录请参考 [NIM SDK 鸿蒙版更新日志](https://doc.yunxin.163.com/messaging2/concept/zg4MjA3ODQ?platform=client)。

## 10.5.0 (2026-03-06)

**新增特性**

- 支持历史聊天记录的全文检索，按关键字、图片、视频、文件、日期、群成员查询消息。
- 支持聊天页面新消息提醒的快速定位新消息功能。

    当聊天页面有新消息时，右下角会显示 “N条新消息” 提示，点击提示或输入框即可一键跳到最新消息。
- 修复已知问题。

## 10.4.10 (2026-01-13)

**新增特性**

- 消息发送添加安全通检测。命中反垃圾后，底部插入违规提示信息（可配置开关），并提示灰色发送失败。
- 会话页面、聊天页面新增悬浮安全提示条。

## 10.4.0 (2025-01-07)

**新增特性**

UIKit 引入呼叫组件（CallKit），P2P 聊天支持音视频通话。

**依赖更新**

- 兼容 [NIM SDK（鸿蒙）10.9.72](https://doc.yunxin.163.com/messaging2/concept/zg4MjA3ODQ?platform=client) 版本。
- 兼容 [云信呼叫组件](https://doc.yunxin.163.com/nertccallkit/concept/jg2MzA3OTU?platform=client) 以下版本：

    - `@yunxin/callkit` V4.0.2 版本。
    - `@yunxin/callkit_ui` V4.0.3 版本。
    - `@nertc/nertc_sdk"` V5.9.11 版本。

## 10.3.2 (2025-06-25)

**新增特性**

- 支持会话列表左滑操作，包含删除、置顶、取消置顶功能。
- 新增 @列表显示动画效果。
- 支持 UIKit 在线状态效果展示。

**功能优化**

- 优化会话列表加载与滚动性能。
- 提升大量消息场景下的渲染效率。
- 其他性能优化。

**依赖更新**

兼容 [NIM SDK（鸿蒙）10.9.10](https://doc.yunxin.163.com/messaging2/concept/zg4MjA3ODQ?platform=client#10910-2025-06-18) 版本。

## 10.3.1 (2025-05-14)

**新增特性**

支持在初始化 SDK 时配置是否开启 **云端会话功能**，默认不开启，即默认使用本地会话功能。详情请参考 [集成 IM UIKit（V10）](https://doc.yunxin.163.com/messaging-uikit/guide/TU4MTkzNTk?platform=harmonyos)。

**功能优化**

- 优化本地会话列表刷新逻辑。
- 好友人数超出上限后，提示用户好友已达上限。
- 其他性能优化。

**依赖更新**

兼容 [NIM SDK（鸿蒙）10.8.30](https://doc.yunxin.163.com/messaging2/concept/zg4MjA3ODQ?platform=client#10830-2025-04-25) 版本。

## 10.3.0 (2025-03-07)

**新增特性**

新增本地会话组件。集成方式请参考 [集成 IM UIKit（V10）](https://doc.yunxin.163.com/messaging-uikit/guide/TU4MTkzNTk?platform=harmonyos)。

**依赖更新**

兼容 [NIM SDK（鸿蒙）10.8.5](https://doc.yunxin.163.com/messaging2/concept/zg4MjA3ODQ?platform=client#1085-2025-02-20) 版本。

## 10.0.0 (2024-09-28)

全新 IM UIKit 组件 v10.0.0 版本发布。

网易云信鸿蒙版本 IM UIKit 是基于 [网易云信即时通讯（NIM）](https://doc.yunxin.163.com/messaging2/concept/zg4MjA3ODQ?platform=client) 开发的一款 <a href="https://doc.yunxin.163.com/messaging-uikit/concept/TI3NTgyNDA?platform=client">UI 组件库</a>，包括聊天、会话列表、搜索、群管理等组件。通过 IM UIKit，可快速集成包含 UI 界面的即时通讯应用。

更多详情，请参考 [集成 IM UIKit](https://doc.yunxin.163.com/messaging-uikit/guide/TU4MTkzNTk)。
