## A

| 术语 | 说明 |
|------|------|
| AI 助聊 | 对应前端 UI 的**回复灵感**。一项基于场景智能推荐的聊天辅助功能，为用户提供多种风格的对话建议，提升用户的社交体验和聊天效率。 |

## G

| 术语 | 说明 |
|------|------|
| 个人中心 | 用于展示和设置个人信息的功能模块。 |

## H

| 术语 | 说明 |
|------|------|
| 会话列表 | 展示与联系人之间的会话，分为云端会话和本地会话。应用中需选择其一使用。 |
| 会话消息 | 指聊天消息，包括消息列表、聊天设置等功能。 |
| 回复灵感 | 同 AI 助聊。智能聊天辅助功能，提供多样化对话建议。 |
| 好友关系 | 用户之间建立的社交连接。 |
| 黑名单 | 被用户拉黑的其他用户列表。 |

## J

| 术语 | 说明 |
|------|------|
| 基础版 | 网易云信 IM UIKit UI 组件的两种风格之一（另一种为通用版）。一个应用只能选择其中一种风格。 |

## L

| 术语 | 说明 |
|------|------|
| 联系人 | 主要指通讯录中的好友。 |
| 聊天 | 同 "消息"，包括消息列表、聊天设置等功能。 |

## P

| 术语 | 说明 |
|------|------|
| 圈组（QChat） | 网易云信 IM 即时通讯服务的新能力模块，与单聊、群聊、聊天室共同构成四大能力模块。 |

## T

| 术语 | 说明 |
|------|------|
| 通用版 | 网易云信 IM UIKit UI 组件的两种风格之一（另一种为基础版）。一个应用只能选择其中一种风格。 |
| 通讯录 | 包括联系人（好友）列表、群组列表、黑名单以及好友验证消息功能。 |
