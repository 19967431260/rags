## `SettingRepo` 类

`SettingRepo` 类提供一些全局设置的获取、设置等操作，内容保存到本地 `UserDefaults` 中。

## 方法列表

| <div style="width:240px">接口</div> | 接口原型 | 接口描述 |
| --- | --- | --- |
| `getHandsetMode` | `getHandsetMode() -> Bool` | 获取音频播放模式。 |
| `getP2PMessageMuteMode` | `getP2PMessageMuteMode(accountId: String) -> V2NIMP2PMessageMuteMode` | 设置单聊消息免打扰模式。 |
| `getPushDetailEnable` | `getPushDetailEnable() -> Bool` | 新消息通知是否展示消息详情。 |
| `getRecentForward` | `getRecentForward() -> [String]?` | 获取最近转发的会话。 |
| `setPushEnable` | `setPushEnable(_ open: Bool)` | 新消息通知时，是否响铃。 |
| `getShowReadStatus` | `getShowReadStatus() -> Bool` | 获取群组聊天页面是否展示已读未读状态，同步方法，从 `userDefault` 中读取。 |
| `getPushEnable` | `getPushEnable() -> Bool` | 收到新消息是否通知栏通知。 |
| `updateRecentForward` | `updateRecentForward(_ recentList: [String])` | 更新最近转发的会话。 |
| `setHandsetMode` | `setHandsetMode(_ value: Bool)` | 设置音频播放模式。 |
| `setP2PMessageMuteMode` | `setP2PMessageMuteMode(accountId: String, muteMode: V2NIMP2PMessageMuteMode, _ completion: @escaping (NSError?) -> Void) ` | 设置单聊消息消息免打扰。 |
| `setPushShowDetail` | `setPushShowDetail(_ value: Bool, _ completion: @escaping (NSError?) -> Void)` | 设置通知消息是否展示消息详情。 |
| `setShowReadStatus` | `setShowReadStatus(_ value: Bool)` | 设置群组聊天页面是否展示已读未读状态。 |
