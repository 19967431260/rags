通讯录好友相关接口提供通讯录相关操作的功能。

## ContactRepo 模块

通讯录模块（[`NEContactUIKit`](https://github.com/netease-kit/nim-uikit-ios/tree/master/NEContactUIKit)）主要包含好友、用户、黑名单等相关接口。

## 方法列表

| <div style="width:240px">接口</div> | 接口原型 | 接口描述 |
| --- | --- | --- |
| `addBlockList` | `addBlockList`(accountId: String, _ completion: @escaping (NSError?) -> Void) | 添加到黑名单。 |
| `addContactListener` | `addContactListener`(_ listener: NEContactListener) | 添加一个好友状态变化监听。 |
| `addFriend` | `addFriend`(accountId: String, params: V2NIMFriendAddParams, _ completion: @escaping (NSError?) -> Void) | 添加好友。 |
| `clearNotification` | `clearNotification`(_ completion: @escaping (NSError?) -> Void) | 清空通知。 |
| `deleteFriend` | `deleteFriend`(account: String, params: V2NIMFriendDeleteParams, _ completion: @escaping (NSError?) -> Void) | 删除好友。 |
| `getBlockList` | `getBlockList`(_ completion: @escaping ([NEUserWithFriend]?, NSError?) -> Void) | 获取黑名单列表。 |
| `getContactList` | `getContactList`(_ completion: @escaping ([NEUserWithFriend]?, NSError?) -> Void) | 获取通讯录列表数据，我的好友信息（去除黑名单中好友）。 |
| `getUserWithFriend` | `getUserWithFriend`(accountIds: [String], _ completion: @escaping ([NEUserWithFriend]?, NSError?) -> Void) | 根据 AccountId 获取好友信息 如果不是好友，NEUserWithFriend 中的 friend 为 nil |
| `getFriendList` | `getFriendList`(accountIds: [String], _ completion: @escaping ([NEUserWithFriend]?, NSError?) -> Void) | 根据账号 ID，查询账号 ID 对应的好友信息，包括昵称、头像等数据。 |
| `getAddApplicationList` | `getAddApplicationList`(option: V2NIMFriendAddApplicationQueryOption, _ completion: @escaping (V2NIMFriendAddApplicationResult?, NSError?) -> Void) | 获取好友申请列表。 |
| `getUnreadApplicationCount` | `getUnreadApplicationCount`(_ completion: @escaping (Int, NSError?) -> Void) | 获取未读好友申请数量。 |
| `getUserInfo` | `getUserInfo`(_ accountIds: [String]) -> [NEUserWithFriend]? | 根据 AccountId 获取用户信息，本地查询。 |
| `getUserListFromCloud` | `getUserListFromCloud`(accountIds: [String], _ completion: @escaping ([NEUserWithFriend]?, NSError?) -> Void) | 根据 AccountId 获取用户信息，远端拉取。 |
| `removeBlockList` | `removeBlockList`(accountId: String, _ completion: @escaping (NSError?) -> Void) | 从黑名单中移除。 |
| `removeContactListener` | `removeContactListener`(_ listener: NEContactListener) | 删除一个好友状态变化监听。 |
| `setAddApplicationRead` | `setAddApplicationRead`(_ completion: @escaping (Bool, NSError?) -> Void) | 重置未读数。 |
| `setFriendInfo` | `setFriendInfo`(accountId: String, params: V2NIMFriendSetParams, _ completion: @escaping (NSError?) -> Void) | 更新好友信息，如备注等。 |
| `updateSelfUserProfile` | `updateSelfUserProfile`(_ updateParams: V2NIMUserUpdateParams, _ completion: @escaping (NSError?) -> Void) | 更新当前账号的用户信息。 |
