网易云信即时通讯 IM UIKit（简称 NIM UIKit）杂项业务接口。

## `MiscRepo` 类

`MiscRepo` 类根据 UI 层业务逻辑，提供 SDK 杂项相关的数据获取、设置等操作，通过 NIM SDK [`MiscService`](https://doc.yunxin.163.com/messaging2/references/android/doxygen/Latest/zh/interfacecom_1_1netease_1_1nimlib_1_1sdk_1_1misc_1_1_misc_service.html) 提供的接口实现。

## 方法列表

| 接口 | 接口原型 | 接口描述 |
| --- | --- | --- |
| `clearCacheSize` | `clearCacheSize(fileType: List<DirCacheFileType>, callback: FetchCallback<Void>?)` | 清理 SDK 缓存。 |
| `clearMessageCache` | `clearMessageCache()` | 清理消息缓存。 |
| `getCacheSize` | `getCacheSize(fileType: List<DirCacheFileType>, callback: FetchCallback<Long>?)` | 获取 SDK 缓存大小。 |
