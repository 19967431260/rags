本文主要介绍如何解决工程不兼容的报错问题。

## 问题描述

项目工程不兼容?.语法，导致运行报错。报错信息如下：

```
ERROR  Failed to compile with 5 errors                                                            17:32:52

error  in ./src/components/NEUIKit/Chat/message/message-input.vue?vue&type=template&id=9e2aee0e&scoped=true

Syntax Error: Unexpected token (11:95)

@ ./src/components/NEUIKit/Chat/message/message-input.vue?vue&type=template&id=9e2aee0e&scoped=true 1:0-465 1:0-465

@ ./src/components/NEUIKit/Chat/message/message-input.vue
```

## 问题原因

缺少相关依赖以及部分配置信息。

## 解决方案

1. 安装 **vue-template-babel-compiler** 插件。

    ```
    npm i vue-template-babel-compiler
    ```

2. 在 **vue.config.js** 进行如下配置。

    ```
    chainWebpack(config) {
        config.plugins.delete("preload"); // TODO: need test
        config.plugins.delete("prefetch"); // TODO: need test


        config.module
        .rule("vue")
        .use("vue-loader")
        .tap((options) => {
            options.compiler = require("vue-template-babel-compiler");
            return options;
        });
    //.....
    }
    ```
