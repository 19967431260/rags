## 问题描述

如果您对云信 UIKit node 版本进行了锁定，在引入云信 IM UIKit 后无法实现自定义需求。

## 解决方案

1. 下载云信 IM UIKit。

    ```
    npm install @xkit-yx/im-kit-ui
    ```

2. 在 `module` 中将云信 IM UIKit 中的 lib 文件都复制到您的项目中。

3. 在您的代码中直接引用 lib 下的 index.js。
