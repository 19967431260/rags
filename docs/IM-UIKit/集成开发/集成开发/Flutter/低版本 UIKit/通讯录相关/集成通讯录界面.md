## 集成通讯录界面

通讯录页面在`ContactPage`类中实现,接入之前请确保已经引入`nim_contactkit_ui`依赖。

```
dependencies:
  nim_contactkit_ui: ^1.0.0-rc.0
```



### **初始化 nim_contactkit_ui**

使用`nim_contactkit_ui`之前，需调用`ContactKitClient.init`方法对该模块进行初始化，以保证该模块功能的正常使用。初始化方法全局只需要调用一次，建议您在main.dart 文件中调用。

::: note notice 
初始化需要在 IM UIKit 界面加载之前进行。
:::

示例代码如下：

```dart
void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({Key? key}) : super(key: key);

  // This widget is the root of your application.
  @override
  Widget build(BuildContext context) {
    ContactKitClient.init();
    // ...
  }
}
```

### **命名路由直接跳转**

您可以使用命名路由直接跳转到`ContactPage`：

```dart
Navigator.pushNamed(context, RouterConstants.PATH_CONTACT_PAGE);
```

### **实例化ContactPage跳转**

```dart
Navigator.push(context, MaterialPageRoute(builder: (context) => ContactPage()));
```

- **ContactPage实例化参数**

```dart
ContactPage({Key? key, this.config})
```

- **ContactPage参数说明**  

| 参数     | <div style="width:100px">类型</div> | 说明            |
| :------- | :---------------------------------- | :-------------- |
| `config` | ContactUIConfig                     | 页面配置 (可选) |
