如果默认的搜索组件（`SearchContainer`）无法满足您的业务需求，您可通过本文介绍的参数进行搜索界面自定义。

## 自定义参数

`SearchContainer` 提供的自定义参数如下：

<div style="width:180px">参数</div> | 类型 | 说明 | 子组件
:---- | :---- | :--- | :----
`renderEmpty` | `(() => Element)` | 没有好友和群组时的状态展示，具体示例参考下文 [自定义好友和群组为空时的内容](#自定义好友和群组为空时的内容) | SearchContainer 组件
`renderSearchResultEmpty` | `(() => Element)` | 没有搜索结果时的自定义渲染，默认的展示内容参考 [搜索结果为空时的内容](#搜索结果为空时的内容) | ^^
`onClickChat` | `(() => void)` | 单击去聊天回调 | AddContainer 组件
`prefix` | String | 样式前缀，默认值：`"search"` | ^^
`commonPrefix` | String | 公共样式前缀，默认值：`"common"` | ^^
