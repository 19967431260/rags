会话列表组件（`ConversationContainer`）模块用于显示当前用户的会话（包括单聊和群聊）列表，同时提供会话单击和右键菜单功能，方便对会话进行操作。

## 界面集成

在您的项目中导入会话列表组件，并设置会话列表界面尺寸，即可集成 IM UIKit 默认的会话列表界面。

示例代码如下：

:::::: div linked-codes
::: code React

```JSX/TSX
import React from 'react'
import { ConversationContainer } from '@xkit-yx/im-kit-ui'

import '@xkit-yx/im-kit-ui/es/style'

const Page1 = () => {
  return (
    <div style={{ width: '30vw', height: '100%' }}>
      <ConversationContainer />
    </div>
  )
}
```
:::
::: code Vue

```JSX/TSX
<template>
   <div :style="{ width: '10vw', height: '100vh'}" ref="conversation" />
</template>
<script>
import {
  ConversationContainer, // 会话列表组件
} from '@xkit-yx/im-kit-ui';
import '@xkit-yx/im-kit-ui/es/style/css';
export default {
  // ...
  mounted() {
      this.$uikit.render(ConversationContainer, null, this.$refs.conversation);  
  }
}
</script>
```
:::
::::::

## 功能介绍
IM UIKit 默认的会话列表界面，支持如下功能：

- 会话单击事件：
  - 选中会话
  - 重置会话未读数
- 右键菜单事件：
  - 设置会话免打扰
  - 删除会话
  - 置顶会话

## 定制界面

如果默认的界面 UI 无法满足您的业务需求，请根据自定义参数进行自定义。会话列表组件（`conversation-kit`）提供的自定义参数，请参考 [定制会话列表界面](https://doc.yunxin.163.com/messaging-uikit/guide/DI1Mzk4ODg?platform=web#%E8%87%AA%E5%AE%9A%E4%B9%89%E5%8F%82%E6%95%B0)。
