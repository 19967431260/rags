如果会话列表（`ConversationContainer`）模块提供的默认界面无法满足您的业务需求，您可通过会话列表组件的自定义参数，自定义界面。

## 自定义参数

会话列表组件（`ConversationContainer`）模块提供的自定义参数如下：

:::details 单击展开查看参数类型详解。

<span id="renderCustomP2pConversation">`renderCustomP2pConversation`</span>

```JavaScript
renderCustomP2pConversation?: (
    options: {
      conversation: V2NIMConversation
    } & ConversationCallbackProps
  ) => JSX.Element | null | undefined
```

<span id="renderCustomTeamConversation">`renderCustomTeamConversation`</span>

```JavaScript
renderCustomTeamConversation?: (
    options: {
      conversation: V2NIMConversation
    } & Omit<ConversationCallbackProps, 'onConversationItemMuteChange'>
  ) => JSX.Element | null | undefined
```

<span id="options">`options`</span>

```JavaScript
(options: {
    conversation: V2NIMConversation
  }) => JSX.Element | null | undefined
```

:::

| 参数 | 参数类型 | 说明 |
| --- | --- | --- |
| `renderConversationListEmpty` | () => JSX.Element \| null \| undefined | 自定义渲染会话列表为空时的内容，具体示例参考 [自定义 **会话列表为空时的内容**](#自定义会话列表为空时的内容)。
| `renderCustomP2pConversation` | [`renderCustomP2pConversation`](#renderCustomP2pConversation) | 自定义渲染会话类型是单聊的内容，具体示例参考 [自定义会话的展示内容](#自定义会话的展示内容)。
| `renderCustomTeamConversation` | [`renderCustomTeamConversation`](#renderCustomTeamConversation) | 自定义渲染会话类型是群聊的内容，具体示例参考 [自定义会话的展示内容](#自定义会话的展示内容)。 |
| `renderConversationName` | [`options`](#options) | 自定义渲染会话名称。如果单聊会话定义了 `renderCustomP2pConversation` 或群组会话定义了 `renderCustomTeamConversation` 则不生效。具体示例参考 [自定义会话的展示内容](#自定义会话的展示内容)。
| `renderConversationMsg` | [`options`](#options) | 自定义渲染会话消息。如果单聊会话定义了 `renderCustomP2pConversation` 或群组会话定义了 `renderCustomTeamConversation` 则不生效。 |
| `renderP2pConversationAvatar` | [`options`](#options) | 自定义渲染单聊会话头像。如果定义了 `renderCustomP2pConversation` 则不生效。 |
| `renderTeamConversationAvatar` | [`options`](#options) | 自定义渲染群聊会话头像。如果定义了 `renderCustomTeamConversation` 则不生效。 |
| `prefix` | String | 样式前缀，默认值为 `conversation`。 |
| `commonPrefix` | String | 公共样式前缀，默认值为 `common`。 |
| `onConversationItemClick` | ((id: string) => void) | 会话单击事件。 |
| `onConversationItemDeleteClick` | ((id: string) => void) | 会话删除事件。 |
| `onConversationItemStickTopChange` | ((id: string, isTop: boolean) => void) | 会话置顶状态改变事件。 |

### 会话的展示内容

您可自定义会话列表中某个会话对象的头像、昵称以及最后一条消息的展示。

- 示例代码

    :::::: div linked-codes
    ::: code React

    ```JSX/TSX
    import { ComplexAvatarContainer } from '@xkit-yx/im-kit-ui'
    //全局上下文
    const { store } = useStateContext()
    const renderCustomP2pConversation = (options) => {
            // 对话方 account
            const conversationPartnerId = nim.V2NIMConversationIdUtil.parseConversationTargetId(
            options.conversation.conversationId
            )
            return (
            <div
                onClick={() =>
                options.onConversationItemClick(options.conversation)
                }
            >
                <ComplexAvatarContainer
                account={conversationPartnerId}
                />
                <div>
                <div className="nick">
                    {options.conversation.name}
                </div>
                <div className="msg-body">
                    {options.conversation.lastMessage.text}
                </div>
                </div>
            </div>
            );
        }

    <ConversationContainer
    renderCustomP2pConversation={renderCustomP2pConversation}
    ></ConversationContainer>
    ```
    :::
    ::: code Vue2/Vue3

    ```JavaScript/TypeScript
    <template>
    <div ref="conversation" />
    </template>

    <script>
    import { ComplexAvatarContainer } from "@xkit-yx/im-kit-ui";
    import { compile } from 'jsx-web-compiler';
    import { Observer } from 'mobx-react';

    ...
        mounted() {
        this.$uikit.render(
        ConversationContainer,
        {
            renderCustomP2pConversation: (options) => {
            // 对话方 account
            const conversationPartnerId =
                nim.V2NIMConversationIdUtil.parseConversationTargetId(
                options.conversation.conversationId,
                );
            const store = this.$uikit.context.store;
            return compile(
                `<div
                    onClick={() =>
                        context.options.onConversationItemClick(context.options.conversation)
                    }
                    >
                    <context.ComplexAvatarContainer account={context.conversationPartnerId} />
                    <div>
                        <div className="nick">
                        {context.options.conversation.name}
                        </div>
                        <div className="msg-body">
                        {context.options.conversation.lastMessage.text}
                        </div>
                    </div>
                    </div>
                `,
                {
                ComplexAvatarContainer,
                options,
                store,
                conversationPartnerId,
                },
            );
            },
        },
        this.$refs.conversation,
        );
        );
    }
    </script>
    ```
    ::::::

- 示例代码解读

    - `ComplexAvatarContainer` 是 UIKit 的头像组件，传入 `account` 即可展示头像，如用户没有头像就展示昵称后两位。

    - 上述示例代码的 `const renderCustomP2pConversation = (options) => {}` 中，`options` 的定义为

        ```JavaScript
         (
            options: {
            conversation: V2NIMConversation
            } & ConversationCallbackProps
        ) => JSX.Element | null | undefined
        ```

        其中 `V2NIMConversation` 和 `ConversationCallbackProps` 的定义如下：

        ```JavaScript
        type V2NIMConversation = {
            /**
            * 会话 ID
            */
            conversationId: string;
            /**
            * 会话类型
            */
            type: V2NIMConversationType;
            /**
            * @computed
            *
            * 会话名称. 拼接字段
            */
            name?: string;
            /**
            * @computed
            *
            * 头像. 拼接字段
            */
            avatar?: string;
            /**
            * @computed
            *
            * 是否免打扰. 拼接字段
            */
            mute?: boolean;
            /**
            * 是否置顶
            */
            stickTop: boolean;
            /**
            * 会话分组
            */
            groupIds?: string[];
            /**
            * 本地扩展
            *
            * 注: web 端没有持久化存储, 内存中存储这个字段. 登出或者重新初始化后这个值都会再次成为空字符串.
            */
            localExtension: string;
            /**
            * 服务端扩展信息
            */
            serverExtension: string;
            /**
            * 会话中最新的消息
            */
            lastMessage?: V2NIMLastMessage | undefined;
            /**
            * 撤回通知.
            *
            * 注: 当消息状态为撤回时, lastMessage 为空, 可以参考这个撤回的通知.
            */
            /**
            * 会话的未读消息计数
            */
            unreadCount: number;
            /**
            * 排序时间戳
            */
            sortOrder: number;
            /**
            * 会话创建时间戳
            */
            createTime: number;
            /**
            * 更新时间戳
            */
            updateTime: number;
        };

        type ConversationCallbackProps = {
            /* 单击事件 */
            onConversationItemClick: (conversation: NimKitCoreTypes.IConversation) => void;
            /* 右键删除事件 */
            onConversationItemDeleteClick: (conversation: NimKitCoreTypes.IConversation) => void;
            /* 右键置顶事件 */
            onConversationItemStickTopChange: (conversation: NimKitCoreTypes.IConversation, isTop: boolean) => void;
            /* 右键静音事件 */
            onConversationItemMuteChange: (conversation: NimKitCoreTypes.IConversation, mute: boolean) => void;
        }
        ```

### 会话元素

可自定义的会话元素包括会话名称、会话消息、单聊会话头像和群聊会话头像。

下文以自定义渲染会话名称为例，介绍的如何在所有会话的名称前都加上 **网易云信** 前缀。

- 示例代码

    :::::: div linked-codes
    ::: code React
    ```JSX/TSX
    const renderConversationName = (options) => {
        return (
        <div style={{ fontWeight: "bold" }}>
            网易云信-{options.conversation.name}
        </div>
        );
    }
    }

    <ConversationContainer
    renderConversationName={renderConversationName}
    ></ConversationContainer>
    ```
    :::
    ::: code Vue2/Vue3
    ```JavaScript/TypeScript
    <template>
    <div ref="conversation" />
    </template>

    <script>
    ...
        mounted() {
        this.$uikit.render(ConversationContainer, {
            renderConversationName: (options) => {
            const onClick = () => {
                // 自行实现
                this.onConversationItemClick(options);
            };
            return compile(`<div style={{fontWeight: 'bold'}} onClick={context.onClick}>网易云信-${options.conversation.name}</div>`,{
                onClick: onClick,
            }
            }
        }, this.$refs.conversation);
        }
    </script>
    ```
