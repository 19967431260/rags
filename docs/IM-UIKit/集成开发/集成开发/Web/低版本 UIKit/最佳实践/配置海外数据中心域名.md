

本文主要介绍如何在 UIKit 中设置海外 link 和 lbs 域名。


## React

在 `initOptions` 中传入海外 link 域名和 lbs 域名。

```
import React from 'react'
import { Provider } from '@xkit-yx/im-kit-ui'

const App = () => {
  const initOptions = {
    appkey: '',  // 传入您的App Key
    account: '', // 传入您的云信 IM 账号
    token: '',   // 传入您的 Token 
    lbsUrls: [''], // lbs 地址，非必传。默认为云信公网提供的链接，SDK 连接时会向 lbs 地址请求得到 socket 连接地址。为防止 lbs 链接被网络运营商劫持，可传入自己代理的地址做备份，格式如 ['公网地址', '代理地址']
    linkUrl: '', // socket 备用地址，非必传。当 lbs 请求失败时，尝试直接连接 socket 备用地址。如不传，SDK 会有内部默认的备用地址
  }

  return (
    <Provider initOptions={initOptions} sdkVersion={1}>      <div className="app">……</div>    </Provider>
  )
}
```

## Vue

```
const uikit = new IMUIKit({

        account: '',  // 传入 accid，必传

        lbsUrls: [''], // lbs 地址，非必传。默认为云信公网提供的链接，SDK 连接时会向 lbs 地址请求得到 socket 连接地址。为防止 lbs 链接被网络运营商劫持，可传入自己代理的地址做备份，格式如 ['公网地址', '代理地址']
        linkUrl: '', // socket 备用地址，非必传。当 lbs 请求失败时，尝试直接连接 socket 备用地址。如不传，SDK 会有内部默认的备用地址
    },
   // ...
})
```
