IM UIKit 的默认语言为中文，您可以通过修改配置参数，增加应用中的语系种类，实现本地化能力。

## 前提条件

根据本文操作前，请确保您已 [集成 IM UIKit](https://doc.yunxin.163.com/messaging-uikit/guide/jMzMzQ0MDk?platform=web)。

## 修改配置参数

IM UIKit 将调用 `setLanguage` 方法进行语言配置，目前支持中文、英文。

您可以基于 [**/NEUIKit/utils/i18n.ts**](https://github.com/netease-kit/nim-uikit-web/blob/main/vue3-demo/src/components/NEUIKit/utils/i18n.ts) 配置更多您需要的语言。

::: note note 
**i18nData** 中内置表情（emoji）使用固定的取值，无需根据语系修改。
:::

以中文语系为例，代码如下：

```JavaScript
// App.vue
initIMUiKit(opts: { account: string; token: string }) {
  // ...
  setLanguage("zh");
  // ...
},
```
