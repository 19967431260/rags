网易云信为您提供基于 H5 开发的即时通讯 Demo。您可以安装 Demo，快速了解和体验网易云信 IM 即时通讯服务的多项能力，如会话、会话列表和通讯录。

## 下载 Demo

- 单击前往 [H5-Vue Demo 在线体验页面](https://yiyong.netease.im/yiyong-static/statics/IMUIKit-h5/index.html#/login)。
- 单击前往 [H5-React Demo 在线体验页面](https://yiyong.netease.im/yiyong-static/statics/IMUIKit-h5-react/index.html#/login)。

## 源码地址

请前往 [网易云信 GitHub 开源代码仓库](https://github.com/netease-kit/nim-uikit-H5) 获取源码。

::: note note
您可以选择自主使用 Vue 框架（nim-kit-vue-ui-h5）或者 React 框架（nim-kit-react-ui-h5）。
:::
