本文介绍网易云信 IM UIKit uni-app 版如何实现自定义消息收发。

## 前提条件

根据本文操作前，请确保您已经完成 [IM UIKit 初始化](https://doc.yunxin.163.com/messaging-uikit/guide/jQyNzk0Nzk?platform=uniapp#%E7%AC%AC%E5%9B%9B%E6%AD%A5%E5%BC%95%E5%85%A5%E5%B9%B6%E5%88%9D%E5%A7%8B%E5%8C%96%E7%BB%84%E4%BB%B6)。

## 发送自定义消息

以 **发送一条评价消息** 为例，发送方可以在会话界面的消息发送按钮上绑定发送消息的函数。

发送时，可将 **是否为评价消息** 标记在附件信息 `attach` 字段或者其他字段中，然后在消息接收时进行判断，如果是评价消息，则使用自定义渲染。您也可 [提交工单](https://app.yunxin.163.com/global/service/ticket/create) 联系网易云信技术支持工程师获取最贴近您业务的字段。

您可以根据业务需要，决定调用 store 层的 [`sendCustomMsgActive`](https://doc.yunxin.163.com/messaging/references/web/typedoc/UIKit/Latest/zh/classes/MsgStore.html#sendCustomMsgActive) 或 SDK 原生方法 [`sendCustomMsg`](https://doc.yunxin.163.com/docs/interface/messaging/web/typedoc/Latest/zh/NIM/classes/nim.NIM.html#sendCustomMsg)（可以传递更丰富的参数）发送函数实现自定义消息的发送。

- 通过调用 `sendCustomMsgActive`

    ```JavaScript
    // 发送评价消息
        const sendEvaluationMessage = () => {
        // store 文档参考 https://doc.yunxin.163.com/docs/interface/messaging/web/typedoc/UIKit/Latest/zh/modules.html
        const [scene, to] = store.uiStore.selectedSession.split('-')
        const from = store.userStore.myUserInfo.account
        //用户自定义参数，这里以 isEvaluation 为例
        const attach = {
            isEvaluation: true
        }
        // sence 代表发送场景 form 发送方 to 接收方
        uni.$UIKitStore.msgStore.sendCustomMsgActive({
            scene,
            from,
            to,
            attach: JSON.stringify(attach)
        })
        }
    ```

- 通过调用 SDK 原生方法 `sendCustomMsg`

    ```JavaScript
    const sendEvaluationMessageBySdk = () => {
        // store 文档参考 https://doc.yunxin.163.com/docs/interface/messaging/web/typedoc/UIKit/Latest/zh/modules.html
        const [scene, to] = store.uiStore.selectedSession.split('-')
        const from = store.userStore.myUserInfo.account
        //用户自定义参数
        const ext = {
            isEvaluation: true
        }
        uni.$UIKitNIM.nim.sendCustomMsg({
            scene,
            to,
            ext: JSON.stringify(ext),
        }).then((msg) => {
            // 将返回的消息体插入到当前消息列表
            uni.$UIKitStore.msgStore.addMsg(msg.sessionId, [msg])
        })
        }
    ```

## 接收自定义消息

同样以 **接收评价消息** 为例，接收方可根据消息体上的 `ext` 字段，判断当前消息是否为评价消息，从而自行实现任意的 UI 展示。

在 **NEUIKit 的/NEUIKit/pages/Chat/message/message-item.vue** 下，添加自定义消息渲染的代码。

```XML
<div
      // todo 自行判断自定义消息
      v-else-if="msg.ext.isEvaluation"
    >
      // todo 自行实现任意的 自定义消息 UI 展示
       <div className='text'>欢迎您进行评价</div>
```
