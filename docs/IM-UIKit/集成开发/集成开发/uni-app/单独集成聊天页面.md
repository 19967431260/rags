若您在集成 IM UIKit 时，只需要集成聊天页面（会话消息组件），不需要会话列表、通讯录、搜索等其他页面，可以参考本文实现。

## 前提条件

已完成 IM 初始化以及 UIKit 相关配置，具体请参考 [快速开始](https://doc.yunxin.163.com/messaging-uikit/guide/jgwMDI2NTU?platform=uniapp#%E7%AC%AC%E5%9B%9B%E6%AD%A5%E5%BC%95%E5%85%A5%E5%B9%B6%E5%88%9D%E5%A7%8B%E5%8C%96%E7%BB%84%E4%BB%B6)。

## 操作步骤

1. 参考组件导入和初始化文档，在 UIKit 组件导入并初始化后，完成聊天（会话消息）页面相关路由配置。

    ```JSON
    {
    "pages": [
        {
            "path": "pages/NEUIKit/pages/Chat/index",
            "style": {
                "navigationBarBackgroundColor": "#F6F8FA",
                "navigationBarTextStyle": "black",
                "navigationStyle": "custom",
                "enablePullDownRefresh": false,
                "app-plus": {
                "softinputNavBar": "none",
                "bounce": "none"
                }
            }
        }
        ]
    }
    ```

2. 将项目的其他页面跳转至聊天页面。

    在路由跳转前，您需要调用 `uni.$UIKitStore.uiStore.selectConversation` 接口，并传入 `conversationId`。

    完整示例代码如下：

    ```JavaScript
    const navigateToChat = (account) => {
        /**
    * from 为发送方account
    * V2NIMConversationType 为会话类型 单聊 1 群聊 2
    * to 为接收方account 如果是群 则是群id
    */
        const conversationId = `${from}|${V2NIMConversationType}|${to}`
        await uni.$UIKitStore.uiStore.selectConversation(
        conversationId
        )
        uni.navigateTo({ url })
    }
    ```
