## 跳转通讯录页面

IM UIKit 提供默认通讯录页面的类名为 `ContactPage`。您可以使用鸿蒙提供的路由 [`Navigation`](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/arkts-navigation-navigation-V5) 跳转到默认会话列表界面 `ContactPage`。

### 跳转通讯录页面

**Navigation 原型**

```TypeScript
//跳转到通讯录界面
this.pathStack.pushPath({ name:'ContactPage' })
```

**参数说明**

<div style="width:100px">参数 | <div style="width:100px">类型</div> | 说明
---- | ---- | ----
`pathStack` | NavPathStack | 路由相关的操作都是基于页面栈 `NavPathStack` 提供的方法进行，每个 `Navigation` 都需要创建并传入一个 `NavPathStack` 对象，用于管理页面。主要涉及页面跳转、页面返回、页面替换、页面删除、参数获取、路由拦截等功能。<br>更多详情，请参考《华为官方文档》[`Navigation`](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/arkts-navigation-navigation-V5)。
`name` | string | IM UIKit 默认注册的通讯录界面地址 `ContactPage`。

### 跳转好友信息页面

**Navigation 原型**

```TypeScript
//跳转到好友信息页面
this.pathStack.pushPath({ name: "PersonInfoPage", param: '*****' })
```

**参数说明**

<div style="width:100px">参数 | <div style="width:100px">类型</div> | 说明
---- | ---- | ----
`pathStack` | NavPathStack | 路由相关的操作都是基于页面栈 `NavPathStack` 提供的方法进行，每个 `Navigation` 都需要创建并传入一个 `NavPathStack` 对象，用于管理页面。主要涉及页面跳转、页面返回、页面替换、页面删除、参数获取、路由拦截等功能。<br>更多详情，请参考《华为官方文档》[`Navigation`](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/arkts-navigation-navigation-V5)。
`name` | string | IM UIKit 默认注册的好友信息界面地址 `PersonInfoPage`。
`param` | string | 好友 ID。

### 跳转好友选择器页面

**Navigation 原型**

```TypeScript
//跳转到好友选择器页面
this.pathStack.pushPath({
      name: 'PersonSelectPage',
      param: new PersonSelectParam(this.onClickSureButton, [], 10)
    })
```

**Navigation 参数说明**

<div style="width:100px">参数 | <div style="width:100px">类型</div> | 说明
---- | ---- | ----
`pathStack` | NavPathStack | 路由相关的操作都是基于页面栈 `NavPathStack` 提供的方法进行，每个 `Navigation` 都需要创建并传入一个 `NavPathStack` 对象，用于管理页面。主要涉及页面跳转、页面返回、页面替换、页面删除、参数获取、路由拦截等功能。<br>更多详情，请参考《华为官方文档》[`Navigation`](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/arkts-navigation-navigation-V5)。
`name` | string | IM UIKit 默认注册的好友选择界面地址 `PersonSelectPage`。
`param` | PersonSelectParam | 好友选择参数。

**`PersonSelectParam` 参数说明**

<div style="width:100px">参数 | <div style="width:100px">类型</div> | 说明
---- | ---- | ----
`filterList` | string[] | 过滤列表（不展示在好友选择列表中）。
`selectLimit` | number | 单次最大选择人数。
`onClickSureButton` | (selectedList: NEUserWithFriend[]) => void | 确认按钮单击回调。

**`NEUserWithFriend` 参数说明**

<div style="width:100px">参数 | <div style="width:100px">类型</div> | 说明
---- | ---- | ----
`user` | V2NIMUser | 用户资料。
`friend` | V2NIMFriend | 好友资料。
