网易云信 IM UIKit 通过会话消息模块（`chatkit-ui`）提供单聊和群聊两种会话界面。会话中支持文本消息、表情（EMoji）、图片、视频、语音等消息类型，同时支持撤回消息、删除消息等管理功能。

:::note note
会话界面的部分功能需要在 [网易云信控制台](https://app.yunxin.163.com/global/home) 开通后才能使用。具体请参考 [控制台开通](https://doc.yunxin.163.com/messaging-uikit/concept/zMzMDQ2MTg?platform=client#%E6%8E%A7%E5%88%B6%E5%8F%B0%E5%BC%80%E9%80%9A)。
:::

## 跳转会话页面

IM UIKit 提供的默认单聊和群聊会话页面（`Page`）的类名为 `ChatP2PPage` 和 `ChatTeamPage`。您可以使用鸿蒙提供的路由 [`Navigation`](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/arkts-navigation-navigation-V5) 跳转到默认会话界面 `ChatP2PPage` 或 `ChatTeamPage`。

### 跳转单聊页面

**Navigation 原型**

```TypeScript
//跳转到单聊界面
this.pathStack.pushPath({ name:'ChatP2PPage', param: '*****' })
```

**参数说明**

<div style="width:100px">参数 | <div style="width:100px">类型</div> | 说明
---- | ---- | ----
`pathStack` | NavPathStack | 路由相关的操作都是基于页面栈 `NavPathStack` 提供的方法进行，每个 `Navigation` 都需要创建并传入一个 `NavPathStack` 对象，用于管理页面。主要涉及页面跳转、页面返回、页面替换、页面删除、参数获取、路由拦截等功能。<br>更多详情，请参考《华为官方文档》[`Navigation`](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/arkts-navigation-navigation-V5)。
`name` | string | IM UIKit 默认注册的会话界面地址 `ChatP2PPage`。
`param` | string | 会话 ID，通过调用 [`V2NIMConversationIdUtil`](https://doc.yunxin.163.com/messaging2/client-apis/DAxNjk0Mzc?platform=client#V2NIMConversationIdUtil) 的对应函数创建。组成方式：用户账号（accountId）\| 会话类型（V2NIMConversationType）\| 聊天对象账号（accountId）或群组 ID。
