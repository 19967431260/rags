
您可设置会话消息的已读未读状态和语音消息播放模式。这两个功能设置后，在 IM UIKit 全局生效。


## 已读未读状态设置

- 调用`SettingRepo.setShowReadStatus`方法设置是否全局展示会话消息（包含单聊消息和群聊消息）的已读和未读状态。入参配置为 true 则展示下图中表示已读状态和未读状态的图标，为 false 则不展示。 

    示例代码如下：
    
    ```java
    //入参为true则代表展示，为false不展示
    SettingRepo.setShowReadStatus(true);
    ```

    <img style="width:40%" src="https://yx-web-nosdn.netease.im/common/c1bcd1e6ebb5db819d14b9791e5e41cd/全局配置.png"/>


- 调用`SettingRepo.getShowReadStatus`方法可查询当前是否全局展示会话消息的已读未读状态。

    示例代码如下：

    ```
    //获取当前是否展示已读未读状态值
    SettingRepo.getShowReadStatus();
    ```
## 语音消息播放模式

调用`SettingRepo.setHandsetMode`方法可设置语音消息的播放模式（听筒模式或外放模式）。

```java
//入参value值，true代表听筒，false代表外放，ConfigRepo.AUDIO_PLAY_EARPIECE和ConfigRepo.AUDIO_PLAY_OUTSIDE
SettingRepo.setHandsetMode(value);

```
