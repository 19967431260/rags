## 概述

用户在线状态功能可以让您的应用显示好友的实时在线情况，提升用户互动体验。

- 开启此功能后，在会话列表和好友列表的用户头像上将显示用户的在线状态。当绿圈出现时表示对方在线，灰色圆圈则表示对方当前离线。
- 关闭此功能时，不再显示好友的用户在线状态。

## 开启用户状态显示

在 Application 的 onCreate 方法中将 `enableOnlineStatus` 的默认值改为 true 。

```Java
public class MyApplication extends Application {
    @Override
    public void onCreate() {
        super.onCreate();

        // 初始化 SDK 前配置 NIM UIKit
        IMKitConfigCenter.setEnableOnlineStatus(true);        // 打开用户在线状态功能

        // 之后再初始化 SDK
        // SDKInitializer.initialize(...);
    }
}
```
