**正在输入** 状态提示是现代即时通讯应用的标准功能，它能够增强用户间的实时互动感，提供更自然的对话体验。当对话方正在编辑消息时，接收方会在会话界面看到 **对方正在输入...** 的提示，增强通讯的实时感与自然度。

<div id="main-content">

本功能通过简单的配置开关来启用和控制。

## 启用 **对方正在输入** 状态

IM UIKit 默认启用 **对方正在输入** 状态，UIKit 会自动处理输入状态的发送和显示。

## 关闭 **对方正在输入** 状态

如需全局禁言此功能，可在 Application 的 `onCreate` 方法中进行配置。请确保该配置在应用启动时就已完成，适用于全局一致的配置需求。

```Java
public class MyApplication extends Application {
    @Override
    public void onCreate() {
        super.onCreate();

        // 初始化 SDK 前配置 NIM UIKit
        IMKitConfigCenter.enableTypingStatus(false);        // 禁用"正在输入"状态

        // 之后再初始化 SDK
        // SDKInitializer.initialize(...);
    }
}
```
