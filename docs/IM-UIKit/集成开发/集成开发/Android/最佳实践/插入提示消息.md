## 实现步骤

1. 插入本地提示消息。

    在触发您的消息提醒相关逻辑时，可以直接使用 UIKit 提供插入本地消息的功能来实现。

    ```java
    V2NIMMessage tipMessage =
        V2NIMMessageCreator.createTipsMessage("提示消息内容");
    ChatRepo.insertMessageToLocal(
        tipMessage, conversationId, sendId, tipMessage.getCreateTime(), null);
    ```

2. 提示消息展示。

    当您成功插入本地消息后，UIKit 会通过消息回调自动进行刷新 UI，展示提示消息。

    如果 UIKit 当前默认的提示消息 UI 效果不符合您的需求，您可以任选一下其中一种方式实现您的需求：

    - 如果您是通过源码的方式集成 `chatkit-ui`，那么可以直接修改 `ChatTipsMessageViewHolder` 文件实现您的需求。
    - 如果您是通过远端依赖的方式集成 `chatkit-ui`，那么可以通过自定义消息的方式实现提示消息，实现自定义消息 `ViewHolder` 的相关代码，可以参考`ChatTipsMessageViewHolder` 文件。
