## 开启消息收藏功能

在 Application 的 onCreate 方法中将 `setEnableCollectionMessage` 设置为 true 。

```Java
public class MyApplication extends Application {
    @Override
    public void onCreate() {
        super.onCreate();

        // 初始化 SDK 前配置 NIM UIKit
        IMKitConfigCenter.setEnableCollectionMessage(true);        // 启用消息收藏

        // 之后再初始化 SDK
        // SDKInitializer.initialize(...);
    }
}
```
