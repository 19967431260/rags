网易会议桌面端支持 Windows 和 macOS 平台，本文介绍桌面端网易会议的操作指导。

## 下载安装

- 可前往 [网易会议官网](https://meeting.163.com/#toDown) 下载最新安装包。
- 网页版（Web）入口：打开 [网易会议官网](https://meeting.163.com/)，找到 **会议网页版**。

## 功能介绍

### 登录

1. 安装成功并打开 App 后，在登录界面上输入企业代码，单击 **下一步**，进入账号密码登录页。在单击 **下一步** 前，请同意服务协议与隐私政策。

    :::note note
    企业代码可联系企业管理员获取。账号和密码由企业管理员在会议管理后台创建。详情请参考 [设置企业代码](https://doc.yunxin.163.com/meeting/concept/DI1MDY1ODg?platform=console)。
    :::

2. 输入正确的账号密码后，单击 **登录** 即可完成登录。

    首次登录后，建议您设置新密码。

### 发起会议

单击主界面中的 **发起会议** 按钮，将打开发起会议界面，如下图所示。网易会议默认创建会议使用随机会议 ID（9 位数字），同时网易会议 App 会为每个用户生成一个位数为 10 位的个人会议 ID，用户可以选择使用个人会议号创建会议。用户还可以选择入会时是否打开摄像头和麦克风、设置会议密码。单击 **发起会议** 按钮，将立即创建一场会议。

### 加入会议

单击主界面中的 **加入会议** 按钮，将打开加入会议界面，如下图所示。用户需要输入正确的 9 位/10 位会议 ID 方可加入会议。用户可以选择入会时是否打开摄像头和麦克风。单击 **加入会议** 按钮，将加入该会议 ID 房间。

如果该会议设置了会议密码，则需要输入会议密码，校验通过后才可进入会议。

如果会议已被锁定，则无法加入会议，请联系会议主持人。

如果会议开启了等候室，则会进入等候室。

### 参会功能

作为普通参会者，有以下功能：

- **静音：**

    关闭/打开麦克风。当用户为静音状态时，其他参会者听不到用户的讲话。

    支持单击△切换音频设备

- **停止视频：**

    关闭/打开摄像头。当用户为停止视频状态时，其他参会者看不到用户的画面。

    支持单击△切换视频设备

- **共享屏幕：**

    桌面端&WEB 端支持共享屏幕，开始共享屏幕后，其他参会者可看到用户的屏幕。

    支持选择是否共享电脑声音。WEB 端仅支持在共享浏览器标签页时选择是否支持声音。

    支持选择共享应用窗口。

    <img alt="image48.png" src="https://yx-web-nosdn.netease.im/common/ccc27ba4d34c4540f957367570e2cab1/image48.png" style="width:80%;border: 0px solid #BFBFBF;">

- **共享白板：**

    支持共享白板，打开 **共享白板** 可打开白板，如下图所示。网易会议的白板支持笔锋、画笔设置等。可单击 **退出白板** 回到视频画面。白板支持授权批注，获得授权的参会者可以进行白板协作。

    <img alt="image49.png" src="https://yx-web-nosdn.netease.im/common/eb29f724c1eecb7e0942648942a3515e/image49.png" style="width:80%;border: 0px solid #BFBFBF;">

- **美颜：**

    支持美颜，打开 **设置** > **美颜与虚拟背景** > **美颜** 可打开美颜设置页，如下图所示，支持美颜等级的调整。

    <img alt="image36.png" src="https://yx-web-nosdn.netease.im/common/129ea96d71a681e14c21c00592ddb55c/image36.png" style="width:80%;border: 0px solid #BFBFBF;">

- **虚拟背景：**

    支持虚拟背景，打开 **设置** > **美颜与虚拟背景** > **虚拟背景** 可打开虚拟背景设置页，如下图所示，支持虚拟背景开关、背景切换、背景上传。

    需要注意的是，虚拟背景受限于手机性能，部分性能未达标的电脑暂不支持开启虚拟背景。

    WEB 端暂不支持美颜与虚拟背景

- **聊天：**

    支持聊天，打开 **聊天** 可打开聊天页面，如下图所示，支持文字聊天、发送图片、发送文档。聊天支持撤回消息、查看历史消息。

### 会控功能

当用户身份为主持人时，**参会者列表** 按钮为管理参会者，单击后可弹出参会者列表。

<img alt="image56.png" src="https://yx-web-nosdn.netease.im/common/492c3e0b1f947deced891528394545e3/image56.png" style="width:80%;border: 0px solid #BFBFBF;">

主持人可以进行以下会控操作：

- **成员选项：**

    单击某个参会者的 **...** 更多按钮，可对其进行静音、停止视频、设为焦点视频、设置联席主持人、移入等候室、移交主持人、移除等操作。其中，设为焦点视频的参会者画面，将呈现在所有参会者的主画面位置。主持人不可对自己进行移交主持人、移除的操作。

    需要注意的是，最多可设置 4 个联席主持人。

- **全体静音/解除全体静音：**

    主持人可对房间内全体成员进行静音操作。

    主持人可进行解除全体静音操作，但需要成员同意后才可开启成员麦克风。

- **全体关闭视频/全体打开视频：**

    单击参会者列表底部 **...** 按钮

    主持人可对房间内全体成员进行全体关闭视频操作。

    主持人可对房间内全体成员进行全体打开视频操作，但需成员同意后才可开启成员视频。

- **录制：**

    主持人和联席主持人可以开启和关闭云录制，在会议结束后，会议的创建者可在历史会议详情查看录制文件链接。

- **会议邀请：**

    网易会议支持邀请信息、通讯录邀请、电话呼叫三种会议邀请方式。

    - 邀请信息：通过复制邀请链接，用户可复制并粘贴到其他 IM 社交工具，发送给他人，被邀请人可通过会议 ID、入会链接的方式加入会议。

    - 通讯录邀请：支持通过搜索企业组织通讯录，可批量邀请他人入会。通讯录邀请功能仅支持 4.5.0 及以上版本。体验版暂不支持该功能。

    - 电话呼叫：支持通过拨号/批量呼叫的方式，呼叫他人电话，他人通过电话直接入会。电话呼叫功能为增值功能，可在线咨询商务经理以获得相应的产品报价。

        <img alt="image57.png" src="https://yx-web-nosdn.netease.im/common/923aaa0dd9cf4fd27969c04849156481/image57.png" style="width:80%;border: 0px solid #BFBFBF;">

- **会议安全：**

    主持人和联席主持人可以对会议安全进行设置：

    - 锁定会议：开启后会议锁定。

    - 等候室：开启后，成员入会先进入等候室等候。

    - 会议黑名单：开启后，被标记 **不再允许加入** 的用户将无法加入会议。

    - 会议水印：开启后，会议中会有水印。水印可配置是否强制开启。

    - 访客入会：开启后，外部用户将可以通过 Web 端或 H5 端网页版加入会议。

    - 会中聊天：开启后，会议成员可进行文字互动聊天。

    - 互动批注：开启后，会议成员可以对共享屏幕进行批注。

        <img alt="image38.png" src="https://yx-web-nosdn.netease.im/common/97da7bde35dbb3de063ae1aeb6cf6774/image38.png" style="width:35%;border: 0px solid #BFBFBF;">

- **结束会议：**

当用户身份为主持人时，有 **结束会议** 按钮，单击后可选择结束会议或离开会议。结束会议功能即为将主持人和其他参会者全部移除会议。离开会议功能即仅主持人离开会议，其他参会者仍在会议房间中。

### **会议设置**

- 会议设置界面如下图所示，用户可以进行多种设置。

- 常规：可对入会状态、文件保存位置、显示语言等进行设置。

- 音频：支持切换音频设备、检测音频设备，支持开关音频降噪、开关音乐模式与专业模式。

- 视频：支持切换视频设备，支持开关视频镜像、开关高清模式。

- 美颜与虚拟背景：支持设置美颜与虚拟背景。（虚拟背景受设备性能限制，部分电脑可能不支持虚拟背景）
