**企业代码** 是网易会议解决方案会议租户下的唯一身份凭证，是一个企业或者团队的识别码。在登录用户账号之前，您需要先填写企业代码，用来定位到您所在的企业或团队，并获得访问授权。

## 普通用户

普通用户在 [下载客户端](https://doc.yunxin.163.com/meeting/quick-start?platform=client) 后，登录网易会议会被提示输入企业代码。

企业代码是由管理员创建的。普通用户获取企业代码，请向企业或者团队中网易会议的管理员索取企业代码，然后在客户端登录界面上填写即可。

## 管理员

一般流程中，管理员需要先设置企业代码，随后可在网易会议 [管理后台](https://meeting.yunxin.163.com/manage/user/list) 查看企业代码。

### 查看企业代码

任何首次通过账号登录 [网易会议](https://meeting.163.com/) 的用户（一般是企业内管理会议软件的管理员），会被提示设置企业代码。管理员可以按照以下步骤查看企业代码：

1. 登录 [会议管理后台](https://meeting.yunxin.163.com/manage/user/list)。
2. 在左侧导航栏，选择 **企业设置** > **SSO 登录**。
3. 在 **SSO 登录** 的 **基本配置** 区域，查看 **企业代码**。

    <img alt="image.png" src="https://yx-web-nosdn.netease.im/common/ed8890288ad26c5506afad762b526edd/image.png">

### 设置企业代码

以下流程适用于一个全新注册企业账号的用户，如果您已经有网易云信账号，则可以参考大体步骤前往 [会议管理后台](https://meeting.yunxin.163.com/manage/user/list) 完成企业代号的创建。

:::note note
企业代码一旦设置后，不允许修改。
:::

1. 进入 [网易会议官网](https://meeting.163.com/)。
2. 在页面右上角，单击 [会议管理后台](https://meeting.yunxin.163.com/manage/user/list)，跳转到网易云信账号登录和注册页面。
3. 按照页面提示登录，或者注册一个账号。

    如果您是第一次注册网易云信账号，您可以按照提示前往云信控制台，进行账号信息完善。

    ```mermaid
    flowchart LR
    创建应用 --> 免费试用网易会议 --> 实名认证
    ```

5. 完成上述步骤后，进入 [会议管理后台](https://meeting.yunxin.163.com/manage/user/list)。
6. 根据 **会议初始化配置** 页面的新账号引导提示，完成账号初始化设置。

    :::note notice
    您设置的企业代码会在下文登录客户端使用到，请牢记该配置。
    :::
