## 方案优势

### **支持品牌定制**

网易视频会议解决方案从产品界面到品牌位管理，内外视频协作交互流程等方面都提供丰富的入口。例如视频会议背景、等候室布局、会议虚拟背景、Rooms 会议室大屏屏保、人员签到板等。

- **企业文化**：通过品牌定制，让会议平台成为您的专属视频会议产品，在会议协作中宣传企业的品牌、企业文化。
- **业务价值**：同时，也能为企业技术（IT）和人力资源（HR）等关键部门提供工具价值之外，更能提供业务价值。

### **业务高效集成**

网易视频会议解决方案基于核心产品模块之上再提供开放性，主要实现 **账号集成**、**OpenAPI 开放** 和 **应用机制** 三种能力。

- **账号集成**：打通 OAuth2.0 标准认证方式，实现员工身份在入、转、离、调场景下做到实时同步，轻松解决重复登录和找人困难问题。详情请参考 [设置 SSO 登录](https://doc.yunxin.163.com/meeting/concept/jE0MjEwNzc?platform=client)。
- **开放集成**：支持二次开发被集成能力，业务系统可以通过 [客户端 SDK](https://doc.yunxin.163.com/meeting/concept/zUzOTE4NTU?platform=client) 以及 [服务端 OpenAPI](https://doc.yunxin.163.com/meeting/server-apis?platform=server) 来实现会议管理、会议配置、获取会议数据等，将数据整合到其他自有的业务平台中。
- **应用机制**：提供集成三方能力，在产品中实现应用管理机制。应用以 HTML5 标准方式实现，可以基于场景模板配置不同的应用列表。例如，为面试场景配置 **题库**、为年会场景配置 **抽奖** 应用和抢红包应用、为远程医疗配置 **症状快速筛选** 应用。实现在会议协作过程中完成个性化业务扩展需求。

## 产品特性

### **超大会议规格**

网易会议依托于网易云信自研的 [WE-CAN 全球智能路由网络](https://doc.yunxin.163.com/general/concept/TkzOTcxMzk?platform=others)，满足大规模分布式传输网络需求，保障超大会议规格、充沛的会议资源。最大支持 2000 方双向视频会议。

### **多种终端接入**

支持网易会议软硬件终端、第三方硬件终端多种会议终端接入，满足各类会议场景，如下所示：

<div style="width: 150px">终端类型</div> | 支持的终端型号 |
---- | ---- |
**软件终端** | - 提供 PC 桌面端、移动客户端、Web 浏览器 |\
 | - 适配 Windows 系统、Mac 系统、安卓系统、iOS 系统 |
**硬件终端** | - Windows 系统 |\
 | - Mac 系统 |
**第三方硬件终端** | Polycom、思科等厂商支持 SIP（Session Initiation Protocol，会话发起协议）或 H.323 协议的终端产品<note type="note">如果您有特殊类型的第三方硬件终端，同样支持评估对接，您可在网易云信官网的在线联系区域找到联系方式，通过网易云信商务经理联系网易云信。</note> |

### **灵活入会方式**

会议发起人可以通过微信、短信、邮件、链接多种方式分享会议。为满足各种场景，不同与会者的需求，提供多种方式灵活入会。

<div style="width: 150px">入会方式</div> | 适用场景 |
---- | ---- |
**受邀入会** | 当软终端处于登录状态，在会议开始时，软终端会接收到来电邀请入会。 |
**会议列表入会** | 如果会议发起人在预约会议时添加了与会者，相应的与会者可以从个人会议列表中提前进入会议。 |
**会议 ID 入会** | 已知会议 ID 和密码，可以直接输入会议 ID 和密码入会。 |
**会议链接入会** | 当收到其他与会者分享的会议链接、微信链接、短信链接时，单击会议链接可以立即入会（需安装客户端）。 |
**访客入会** | 与会者没有网易会议账号时，下载软件终端，从会议发起人或其他与会者获取会议 ID 后，输入会议 ID（和会议密码），即可作为外部访客入会。 |

### **会议互动白板**

会议过程中可以通过互动白板完成协作、同步共享数据，还可以软硬件终端实现多方会议用户通过白板完成同时标注，实时互动。

- **数据共享**：支持桌面屏幕共享（多个屏幕时，可选择任一屏幕进行共享），应用程序共享和白板共享。应用程序可以是计算机中的本地文档或表格、音视频文件及其他可在桌面显示的内容，或者正在运行的某个应用程序界面。

- **白板标注**：支持协同在白板上进行标注，观看共享桌面、白板、应用程序时，可进行标注，实时双向互动。

### **会议中聊天**

支持会中主持人和会议成员通过聊天工具进行快速交流，支持文字、图片、文件等多种内容格式，主持人可操作消息内容发送给全员或指定单个会议成员，并可分别针对会议中聊天权限和等候室聊天权限进行控制，例如对会议中聊天可控制会议成员是否允许自由聊天、私聊主持人。

### **会议直播录播**

发起一场会议，可选择是否需要会议直播，开启会议直播后，支持企业主持万人规模的直播会议。详情请参考 [会议直播](https://doc.yunxin.163.com/meeting/concept/DEwMDkxNTQ?platform=client)。

- **直播服务**

    - 通过会议直播链接，支持不同平台访问链接后直接观看，零门槛接入。
    - 直播过程中，可在直播聊天室中进行文字互动，直播管理员可设置禁言。
    - 多维数据统计，业务结果量化。

- **录播服务**

    - 1080P 高清录制，高效转码。
    - 视频、音频、混流多流录制，会议细节不遗漏。
    - 会议录制文件云端存储，支持随时随地下载。

### **Rooms 会议室**

[Rooms 会议室](https://doc.yunxin.163.com/meeting/concept/DMxNjM5MDc?platform=client) 拥有专业音视频能力和软硬件加持，为您打造更高效的会议体验。

- **端云协同会议**：会议室与云端视频会议完美融合，内外部会议无缝衔接。手机、电脑，多端随时接入会议，顺畅沟通。
- **专业音视频**：硬件加速、高清视频，性能更优。
- **远程协作**：屏幕/白板共享，多方同时标注，远程协作如同本地。

### **会议管理后台**

[会议管理后台](https://meeting.yunxin.163.com/manage/user/list) 面向会议租户的企业管理员，提供会议初始化配置、会议管理、Rooms 会议管理、数据统计与资源使用情况为一体的平台管理能力：

- **会议初始化配置**：企业代码设置、部门及用户管理、企业 SSO（Sing Sign-On）登录配置、会议租户账号信息管理。
- **会议管理**：正在召开、即将召开、已结束历史会议、会议录制文件、会议管控设置。
- **Rooms 会议室管理**：Rooms 会议室管理、主机设备管理、控制器 Controller 管理。
- **数据仪表盘**：会议资源使用统计、会议室资源利用情况、员工开会情况。

通过会议管理后台，企业管理员可轻松设置会议接入人员，以及掌控企业会议资源使用情况。
