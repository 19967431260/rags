网易会议提供了会议室连接器，使专业视频会议终端设备能够快速接入会议。会议室连接器是 SIP 或 H.323 协议设备的网关，让 SIP 或 H.323 协议终端设备能够通过会议室连接器加入网易会议。您可以轻松地将传统的视频会议设备接入到网易会议中，实现无缝连接。

本文指导您如何使用会议室连接器将您的 SIP 或 H.323 协议设备接入网易会议。

## 入会方式

支持的入会方式包括：

- 安装了 SIP 或 H.323 协议设备的会议室，通过呼叫会议室连接器的接入地址入会。

- 在会议中邀请安装了 SIP 或 H.323 协议设备的会议室入会。

## **使用条件**

### **版本要求**

- 4.9.0 版本及以上网易会议应用支持该功能。您可以 [下载](https://meeting.163.com/) 或更新网易会议应用体验功能。
- 会议室连接器为增值功能，如需开通，可在线咨询网易云信商务经理以获得相应的产品报价。

### **系统要求**

该功能在以下版本的网易会议客户端上支持：

- Windows、macOS 客户端 4.9.0 版本及以上。

- iOS、Android 客户端 4.9.0 版本及以上。

## **如何加入会议**

安装了 SIP 或 H.323 协议设备的会议室，呼叫会议室连接器接入地址，可加入会议。

### **方法一：拨打会议号 IP 地址**

1. 在设备主页选择 **呼叫/拨打**，进入拨号界面。

2. 在拨号界面输入 **会议号@IP 地址** 然后拨打。

### **方法二：输入会议号码**

通过拨打 IP 地址进入欢迎页面，以交互方式输入会议号码加入会议的步骤如下：

1. 在设备主页选择 **呼叫/拨打**，进入拨号界面。（具体操作请参阅各厂商设备的使用手册）

2. 在拨号界面输入 **IP 地址**，进入网易会议欢迎页面，根据提示输入会议号和会议密码即可加入会议。

## **如何邀请入会**

邀请入会指在会议中邀请安装了 SIP 或 H.323 协议设备的会议室入会。

### **前置条件**

- 当前会议创建者所在的企业必须具备会议室连接器的能力，才能允许邀请 SIP 或 H.323 协议设备入会，并显示邀请呼叫页面。

- 仅支持会议主持人或联席主持人拥有邀请权限。

### **邀请方式**

1. 在已加入会议的网易会议 App 中，单击 **会议邀请**。
2. 选择 **会议室** > **会议室呼叫地址**，以邀请安装了 SIP 或 H.323 协议设备的会议室入会。

    :::note note
    默认的呼叫信令协议为 SIP，但您可以切换为 H.323。同时，还可以设置该设备的入会名称。单击 **呼叫** 即可呼叫对应的设备。
    :::

## **会议中如何操作**

<!-- **使用设备遥控器进行会中操作** -->

### 前置条件

会议室连接器支持使用本地硬件终端的设备遥控器来进行会议中的各项操作。需要注意的是，硬件终端需要支持 SIP 协议。

### **第一步：打开 DTMF 菜单**

在控制器上调出数字小键盘，按 **\*** 键即可显示菜单。

:::note note
不同品牌的会议终端调出数字小键盘的方式有所不同，请查阅对应终端的官方操作说明。
:::

### **第二步：选择要操作的功能**

- **功能 1：切换布局**

    在屏幕上显示菜单的情况下，短按遥控器上的 **1** 键可直接切换网易会议的主要视频布局。布局将在画廊宫格视图、单画面全屏视图和 1+N（N = 5）演讲布局之间循环切换。

- **功能 2：开启麦克风**

    在屏幕上显示菜单的情况下，短按遥控器上的 **2** 键可直接为会议终端开启麦克风。

    :::note note
    该按键仅能解除静音，如果您需要重新静音，可操作终端上的静音按钮或主持人进行静音。
    :::

- **功能 3：开启摄像头**

    在屏幕上显示菜单的情况下，短按遥控器上的 **3** 键可开启会议终端的视频传输。

    :::note note
    该操作让网易会议中的远端成员看到本地视频，并未物理控制本地摄像头。
    您还可以物理关闭摄像头来关闭摄像头。
    :::

- **功能 4：举手**

    在屏幕上显示菜单的情况下，短按遥控器上的 **4** 键可设置会议终端的举手状态。

- **功能 5：显示/隐藏非视频与会者**

    在屏幕上显示菜单的情况下，短按遥控器上的 **5** 键可显示或隐藏关闭视频的与会者画面。

### **第三步：退出菜单**

在屏幕上显示菜单的情况下，短按遥控器上的 **\*** 键即可退出数字小键盘。

如果在显示菜单后 15 秒钟内没有进行任何操作，菜单也会自动消失。

## **注意事项**

- **会议室连接器支持本地部署**

    会议室连接器支持私有化的本地部署，将其部署在企业内部网络中，通过拨打会议室连接器本地部署的接入地址即可加入会议。

- **会议室连接器支持对接呼叫中心和 PSTN 电话**

    会议室连接器还支持对接呼叫中心和 PSTN 电话，具体实施细节可 [提交工单](https://app.yunxin.163.com/global/service/ticket/create) 联系网易云信技术支持工程师。

- **第三方视频会议终端的网络适应性**

    需要注意的是，第三方视频会议终端可能不支持抗丢包和抗抖动功能，因此视频效果可能会受到网络条件的影响。

- **DTMF 模式仅支持 SIP 设备终端**

    当前会中 DTMF 模式操作，仅支持 SIP 协议的第三方设备终端。

- **当在 DTMF 模式下切换视图布局时，其他参会的第三方设备同时切换**

    当前会中 DTMF 模式进行布局切换时，会议中所有参会的第三方设备布局将同时切换。

## **常见问题**

### **第三方友商终端视频画面黑屏**

**视频画面黑屏的可能原因及解决措施**：

- 终端未连接摄像头或摄像头未打开，或连接线存在问题时，应检查并排除终端摄像头的连接、启用状态及连接线的问题。

- 网络限制导致音视频媒体传输不畅时，应检查是否存在防火墙限制的情况。

### **第三方友商终端异常掉会**

**异常掉会的可能原因及解决措施**：

- 因断网、断电或误操作导致掉会时，应检查并排除终端的电源、网络连接及操作情况导致的掉会。

- 网络问题导致第三方设备无码流掉会时：

    - 建议用户检查网络状况后重新入会。

    - 建议用户进行网络优化，例如取消终端连接 IP 的限流，增加终端连接网络 IP 的带宽等。

### **第三方友商终端辅流接收/发送失败**

**辅流接收/发送失败的可能原因及解决措施**：

- 如果是第三方友商终端版本问题，建议将终端升级到互通测试通过的版本。
- 网络限制导致辅流传输不畅时，应检查是否存在防火墙限制的情况。
