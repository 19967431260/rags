虚拟背景是一种基于人像分割技术的运用。在进行视频会议时开启虚拟背景，网易会议将自动识别参会人，并将参会人周围的环境替换为指定图像或视频。网易会议提供了系统预设图片作为会议虚拟背景，也支持上传自定义图片，并直接在预览窗口中查看效果。

## 适用场景

虚拟背景的主要应用优势：

- **个人隐私保护**：有效隐藏真实环境，保护参会人员的隐私空间。
- **企业品牌宣传**：替代传统布展背景墙，展示企业 logo 或主题元素。
- **会议主题呈现**：统一会议背景，突出活动主题，提升专业形象。
- **降低会议成本**：无需实体背景墙，节省场地布置费用和时间。

## 适用条件

使用虚拟背景前，请确保您的应用满足以下条件：

- **登录用户**：所有网易会议用户。

- **版本要求**：要求 4.5.0 及以上版本的网易会议。

## 背景素材

网易会议支持上传背景素材，以实现自定义的虚拟背景风格。对上传的背景素材有以下内容要求：

| 内容类型 | 要求项 | 说明 |
| :---- | :---- | :--- |
| **图片** | 格式 | jpg、jpeg、png |
| ^^ | 分辨率 | 建议 1920x1080 以内 |
| ^^ | 大小 | 建议 5M 以内 |
| ^^ | 长宽比 | 建议 16:9 或 4:3 |
| **视频** | 格式 | mp4、mov |
| ^^ | 分辨率 | 建议 1280×720 |
| ^^ | 大小 | 建议 100M 以内 |
| ^^ | 设备 | 仅桌面端支持 |

## 启用虚拟背景

您可以选择在登录后会议前，或者会议中启用虚拟背景。在会议前设置了虚拟背景后，会自动保存偏好，加入会议后开启摄像头，虚拟背景自动生效。

### Windows/macOS/Linux

- **会议前**：登录后在应用首页，单击主界面左下角 **设置** 图标，选择 **美颜与虚拟背景 > 虚拟背景**。

    <img alt="image.png" src="https://yx-web-nosdn.netease.im/common/bdaf990a4f646bf768f7d64fbdcbb57d/image.png" style="width:60%;border: 1px solid #BFBFBF;border-radius:0px;">

- **会议中**：单击底部工具栏 **更多 > 设置 > 美颜与虚拟背景**，选择 **美颜与虚拟背景 > 虚拟背景**。

    <img alt="image.png" src="https://yx-web-nosdn.netease.im/common/62bb74e7926dc1ec9014105147624e8c/image.png" style="width:60%;border: 1px solid #BFBFBF;border-radius:0px;">

### Android/iOS/HarmonyOS

- **会议前**：登录后在应用首页，单击右上角 **用户头像** 进入 **个人中心** 页，选择 **虚拟背景**。

    <img alt="image.png" src="https://yx-web-nosdn.netease.im/common/f73103f9c40b284bd94380bb964ea2c6/image.png" style="width:60%;border: 1px solid #BFBFBF;border-radius:0px;">

- **会议中**：单击底部工具栏 **更多 > 虚拟背景**。

## 停用虚拟背景

如果您需要停用虚拟背景，请在虚拟背景的开启页面选择 **无** 选项。

## 场景建议

- **为了获得更佳的虚拟背景体验，网易会议推荐用户在以下场景使用该功能**：

    - 场景中含有较少的物件，背景颜色单一，推荐使用绿幕。
    - 使用高清摄像设备进行视频会议。
    - 参会场景中的光照均匀充足。
    - 请避免着装颜色与背景色相似。

- **使用视频作为虚拟背景时，请注意**：不要遮挡摄像头，以免影响采集帧率，导致虚拟背景效果不佳。
