网易会议集成同声传译，灵活配置多语频道，简化翻译流程，智能调节音量，用户自主控制，实现无障碍、沉浸式多语言交流体验。本文介绍如何使用同声传译功能。

## 功能介绍

网易会议支持同声传译，专为提升多语言在线会议的互动流畅性而设计。此特性允许会议组织者在预定会议阶段或会议进程中灵活指派翻译人员，并为不同的语言配置专属频道。一旦会议启动，主持人享有更大的灵活性，可随时调配翻译资源及调整语言通道配置，确保每位参与者都能跨越语言障碍，无缝交流信息。

在会议进程中，译员仅需简便操作，根据所译语言切换至相应频道进行即时翻译。与此同时，参会者可通过专门的同声传译界面，轻松选取他们偏好的语言频道来收听翻译内容。为了平衡听取体验，原始音频会被智能调节至背景级别。此外，系统赋予用户更多控制权，参会者在聆听翻译时可自主选择静音源声音，以全心沉浸在翻译内容之中。

## 使用条件

### 开通方式

同声传译功能为正式版功能。如需正式开通，您可在线咨询网易云信商务经理以获得相应的产品报价。

### 版本支持情况

该功能在以下版本的网易会议客户端上支持，您可以 [下载](https://meeting.163.com/) 或更新网易会议应用体验功能：

- Windows、macOS 客户端 4.6.0 版本及以上。

- iOS、Android 客户端 4.6.0 版本及以上。

## 注意事项

- 同声传译功能仅支持人工同传，暂不支持机器翻译，所以需要在人工传译员的支持下才可以体验该功能。

- 同声传译功能支持最多设置 10 个译员。

- 企业超级管理员可以在会议管理后台 > 会议管理 > 会议设置中控制企业创建的所有会议是否可以添加自定义翻译语言，默认支持 17 种语言（中文、英语、日语、韩语、德语、法语、俄语、葡萄牙语、西班牙语、意大利语、阿拉伯语、印尼语、印地语、泰语、越南语、马来语、土耳其语）。

## **如何进行同声传译**

:::::: div linked-codes
::: code Windows/macOS/Web

1. 当主持人单击 **开始同声传译** 时，被指定传译员的画面会出现提示弹窗，单击 **确定** 即可开始同声传译。同时会出现 **同声传译面板**。

    <img alt="image46.png" src="https://yx-web-nosdn.netease.im/common/bae1afa7260c3732fd6e7a11e9c16c5a/image46.png" style="width:100%;border: 1px solid #BFBFBF;">

2. 单击右下角 **同声传译** 按钮，即可唤出 **同声传译面板**，选择切换收听语言和传译语言。单击 **X** 即可收起 **同声传译面板**。

    <img alt="image47.png" src="https://yx-web-nosdn.netease.im/common/7add2e13d33d4dce81c1b3b09387acf5/image47.png" style="width:100%;border: 1px solid #BFBFBF;">

3. 当收听语言为原声之外的其他语言时，支持选择 **同时收听原声**。收听原声时，支持给原声静音、调整原声音量。

    <img alt="image48.png" src="https://yx-web-nosdn.netease.im/common/4d259633cf5aa397d154ba1ef3abb61f/image48.png" style="width:80%;border: 0px solid #BFBFBF;">

4. 支持长按 **同声设置面板**，将其拖拽至桌面任意位置。

:::
::: code 安卓/iOS

1. 开始同声传译时，被指定传译员的画面会出现提示弹窗单击 **确定** 即可开始同声传译。

2. 译员页面常显传译语言、收听语言、主频道，译员可单击进行快速切换。

3. 译员可以在 **更多** > **同声传译** 中进行语言切换，当收听语言为原声之外的其他语言时，支持选择 **同时收听原声**。收听原声时，支持给原声静音、调整原声音量。

    <img alt="image49.png" src="https://yx-web-nosdn.netease.im/common/197216c180a52e1ee88a61d9da30ab4d/image49.png" style="width:80%;border: 1px solid #BFBFBF;">
