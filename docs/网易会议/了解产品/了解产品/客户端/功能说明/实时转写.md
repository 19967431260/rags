网易会议支持主持人在会议中开启实时转写功能，可以将会议中的语音对话实时转换为文字，方便所有成员实时查看和回顾。会议结束后，创建者还可以导出这些文字记录进行后续参考。

## 推荐场景

- **商务会谈**：通过实时转写会议内容，省去了会后的整理工作。
- **头脑风暴**：实时转写讨论内容，记录下每一个灵感瞬间。
- **线上培训**：实时转写培训材料，轻松记录关键点。
- **项目竞标**：实时转写记录客户反馈，提高沟通效率和后续跟进的准确性。

## 使用条件

### 开通方式

实时转写功能为增值功能，如需开通，可在线咨询网易云信商务经理以获得相应的产品报价。

### 版本支持情况

- 该功能支持客户端包括竖屏移动端 Android/iOS 和横屏桌面端 Windows/Mac/Web。
- 4.7.0 版本及以上支持转写语音内容，支持导出转写文件。
- 4.8.0 版本及以上支持语音内容翻译。

    有关各版本支持的应用功能详情，请参考 [更新日志](https://doc.yunxin.163.com/meeting/concept/jY3Njk2NTQ?platform=client)。

## 注意事项

- 实时转写仅支持主持人/联席主持人开启，所有成员可查看。并告知会议中所有成员当前已开始转写。
- 所有成员仅可以查看自己在会议期间接收到的实时转写记录，无法查看自己未入会前的记录。
- 当前转写仅支持中文普通话、英文识别，暂不支持方言识别，机器识别仅供参考。
- 移动端暂不支持导出转写记录文件。
- 历史会议详情中的转写记录，仅显示原始记录，无翻译语言。
- 如需更多翻译语言，可在线咨询网易云信商务经理。

## 如何使用实时转写

### 主持人/联席主持人

:::::: div linked-codes
::: code Windows/macOS/Web

1. 会中单击底部工具栏 **更多** > **实时转写** 即可开启实时转写。

2. 开启 **实时转写** 后，向会中所有成员发送 **主持人已开启实时转写** 提示，并均可开启 **实时转写**。

3. 主持人和联席主持人可单击转写界面的 **设置** 即可设置 **目标翻译语言**、**同时显示双语。** 仅 4.8.0 版本及以上版本支持语音转写内容翻译。

    - **目标翻译语言**：支持将语音内容翻译显示为中文/英文/日语。

    - **同时显示双语**：开启翻译后，可以选择同时显示双语，则在转写区域会通过分行同时显示源语言与翻译语言。

## 会前设置

:::::: div linked-codes
::: code Windows/macOS

单击客户端主界面 **设置** > **字幕和翻译**，即可在 **翻译设置** 中选择 **目标翻译语言**（不翻译、中文、英文、日语）和 **转写同时显示双语**。完成设置即自动保存，会中默认生效。

:::
::: code Android/iOS

单击客户端 **头像** > **会议设置** > **字幕和翻译**，即可在 **翻译设置** 中选择 **目标翻译语言**（不翻译、中文、英文、日语）和 **转写同时显示双语**。完成设置即自动保存，会中默认生效。
