网易会议支持访客入会功能，便于临时参会者加入会议。无论是预约会议还是即刻召开的会议，主持人均能灵活开启这一特性。

通过分享专属加入链接，受邀嘉宾可直接借助网页端或移动设备的 H5 界面无缝接入会议，全程免去登录步骤，极大地简化了参会流程，确保了会议的高效与包容性，无论是计划内还是紧急会议，都能迅速集合所有必要参与者。

## 使用条件

### 开通方式

访客入会功能为正式版功能。如需正式开通，可在线咨询网易云信商务经理以获得相应的产品报价。

### 版本支持情况

该功能在以下版本的网易会议客户端上支持，您可以 [下载](https://meeting.163.com/) 或更新网易会议应用体验功能：

- Windows、macOS 客户端 4.6.0 版本及以上。

- iOS、Android 客户端 4.6.0 版本及以上。

## 注意事项

- 如果不开启访客入会，非本企业的用户无法加入该会议。

- 暂时不支持已通过 Web 端登录账号后，再通过访客入会链接加入其他企业的会议。

- 如果您希望对观众进行短信验证码校验，需要开通短信功能并配置短信模板。如需开通，请联系网易云信商务经理。

## **访客加入会议**

会议创建者或主持人可以通过 **邀请** > **复制邀请**，复制邀请信息，并将入会链接通过其他方式（第三方通讯工具、邮件等）给到访客，访客使用桌面电脑浏览器或手机浏览器打开链接，选择 **网页版入会**，进入访客入会页面。

:::note note
桌面端（Windows、macOS）和移动端（Android、iOS）在已登录状态下，不支持加入其他企业或组织创建的会议，如需加入，请先退出登录，通过未登录界面的 **加入会议** 入口加入。
:::
