本文描述了一个从认识网易会议、注册账号、登录应用、参与或主持一个会议的简单流程，适用于已开通网易会议解决方案或开通试用的企业管理员以及普通参会成员，以帮助您快速体验视频会议。

## 流程说明

**网易视频会议解决方案** 面向企业和组织提供企业级安全保障、软硬件一体化、高性能音视频体验的会议产品和方案。为了让企业内不同角色用户高效地协作，轻松地沟通，会议接入流程涉及两类用户：

- 管理员：负责前期注册企业组织账号，配置普通用户的认证登录方式（SSO 单点登录或者账号密码登录）。然后，管理员将认证登录方式同步给普通用户。
- 普通用户：自行下载网易会议客户端（下载的时间节点不严格固定，可在获取到登录方式前或后），通过管理员配置的认证登录方式，登录客户端。进入客户端后，根据需求，参加会议或者主持一场会议。

    ```mermaid
    sequenceDiagram
    autonumber
    participant 网易云信
    actor 管理员
    participant 网易会议
    actor 普通用户
    普通用户 -->> 网易会议 : 搜索下载客户端
    管理员 ->> 网易云信 : 注册企业组织账号
    管理员 ->> 网易会议 : 开通网易会议

    管理员 ->> 普通用户 : 告知用户登录方式
    普通用户 -->> 网易会议 : 搜索下载客户端
    普通用户 ->> 网易会议 : 登录会议客户端
    普通用户 ->> 普通用户 : 组织或参加会议
    ```

## 第一步：注册企业组织账号

如您或者您的企业、团队首次使用视频会议，则需要先注册一个企业组织账号，用来进行视频会议解决方案的开通和会议功能管理，可通过网易会议官网进入进行注册。

```mermaid
flowchart LR
classDef default fill:#337EFF,stroke:#337EFF,stroke-width:0px,color:#FFFFFF;
注册或登录云信企业账号 --> 开通会议解决方案 --> 设置企业代码
```

以下流程适用于一个全新注册企业账号的用户，如果您已经有网易云信账号，则可以参考大体步骤前往 [会议管理后台](https://meeting.yunxin.163.com/manage/user/list) 完成企业代号的创建和配置。

1. 进入 [网易会议官网](https://meeting.163.com/)。
2. 在页面右上角，单击 [会议管理后台](https://meeting.yunxin.163.com/manage/user/list)，跳转到网易云信账号登录和注册页面。
3. 按照页面提示登录，或者注册一个账号。
4. 如果您是第一次注册网易云信账号，您可以按照提示前往云信控制台，进行账号信息完善。

    ```mermaid
    flowchart LR
    classDef default fill:#337EFF,stroke:#337EFF,stroke-width:0px,color:#FFFFFF;
    创建应用 --> 免费试用网易会议 --> 实名认证
    ```
    
5. 完成上述步骤后，进入 [会议管理后台](https://meeting.yunxin.163.com/manage/user/list)。
6. 根据 **会议初始化配置** 页面的新账号引导提示，完成账号初始化设置。

    您设置的企业代码会在下文登录客户端使用到，请牢记该配置。

    ![image.png](https://yx-web-nosdn.netease.im/common/1ffba6fcde5ccb3250ffdf9f26cdf2b1/image.png)

    引导过程分为以下三部分，将在下文 **第二步：配置登录方式** 继续描述：

    ```mermaid

    设置企业代码 --> 设置部门及用户 --> 开始体验会议
    ```

## 第二步：配置登录方式

网易会议支持 SSO（单点登录）和账号密码两种登录方式的配置：

* 方式一：SSO（Single Sign-On，单点登录）是一种认证机制，允许用户在多个系统间只需登录一次，即可访问所有相互信任的应用，无需重复验证，提高便捷性和安全性。您可以前往 [**企业设置** > **SSO 登录**](https://meeting.yunxin.163.com/manage/setting/sso) 进行配置。详细步骤，请参考 [设置 SSO 登录](https://doc.yunxin.163.com/meeting/guide/jE0MjEwNzc?platform=console)。

    :::note note
    SSO 适用于企业已有内部的一套 OA（Office Automation，办公自动化）系统或账号协议认证源。此时，您可通过网易会议和企业自有的账号协议及认证源对接，让企业内用户使用统一账号访问会议客户端进行开会，而无需在会议管理后台手动创建部门和用户账号。
    :::

* 方式二：用户账号密码搭配企业代号的登录方式适用于企业没有（或不需要）对接内部的 OA 系统或账号协议认证源的场景。企业内用户的会议账号通过管理员手动创建或导入生成，用户使用分配的账号和初始密码（后续可进行密码修改）登录会议客户端，使用账号入会或主持会议。您可以前往 [**用户管理**](https://meeting.yunxin.163.com/manage/user/department) 进行配置。

因 SSO 认证方式涉及到内部系统对接，因此本文选用 **方式二：账号密码登录** 作为登录方式配置来演示。

1. 前往 [**用户管理 > 部门**](https://meeting.yunxin.163.com/manage/user/department)。

    ![image.png](https://yx-web-nosdn.netease.im/common/38799b5a8ff36a78d0b6551146232930/image.png)
2. 在页面右上角，选择 **新建部门**。
3. 在 **新增部门** 弹窗上，设置一个 **部门名称**，例如 **产品部门**，然后单击 **确定**。
4. 前往 [**用户管理 > 用户**](https://meeting.yunxin.163.com/manage/user/department) 页面。
5. 在页面右上角，选择 **新建用户** > **单个新建**。

    ![image.png](https://yx-web-nosdn.netease.im/common/93953cb8ad780d34737e9fbd016f99f5/image.png)
6. 在 **新建用户** 弹窗上，设置用户的姓名、账号、密码、手机号、邮箱、部门，然后单击 **确定**。如下图所示：

    ![image.png](https://yx-web-nosdn.netease.im/common/32188f1b66abf5cf8c67c638ed2db0b1/image.png)

## 第三步：体验会议

```mermaid
graph LR
classDef default fill:#337EFF,stroke:#337EFF,stroke-width:0px,color:#FFFFFF;
下载会议客户端 --> 登录会议客户端 --> 参加会议或者主持会议
```

### 1: 下载会议客户端

请前往 [网易会议](https://meeting.163.com/#toDown) 下载专区，根据实际系统下载对应的会议客户端。本文后续以 Windows 端为例进行演示。但各个客户端之间的交互差距较小，如果您下载的是其他客户端，您也选择性参考下文相关描述。

### 2：登录会议客户端

根据上文的流程描述，您已经获取了企业代码、账号、密码的三个认证要素。本文的示例配置为：

- 企业代码：testtech
- 用户账号：DocTest
- 用户密码：DocTest1

    ![image.png](https://yx-web-nosdn.netease.im/common/24e5bb1a5e3f8cf788f1b1b5166f6c24/image.png)

    在登录会议客户端时，用户可以将管理员设置的初始密码更新为自定义密码，以增加账号安全性。

### 3：参加或主持会议

用户登录客户端后，在参加会议时，可以选择：

- 发起会议。
- 加入一个即将开始或正在进行中的会议。
- 预约一个待主持召开的会议。
- 会议中投屏。

以发起为例，用户可以：

1. 单击 **发起会议**。
2. 然后设置入会条件，例如是否使用个人会议号、是否开启会议密码、是否静音、是否开启摄像头。
3. 设置完成后，单击 **发起会议** 进入会议界面。
4. 进入会议界面，在底部菜单栏，进行会议功能操作，例如单击 **会议邀请** 邀请更多人参加会议。

    ![image.png](https://yx-web-nosdn.netease.im/common/375577aefdc02887b9a84c3486ff5af5/image.png)
