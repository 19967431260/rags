网易会议提供客户端开放集成和服务端开放集成能力，您可以根据业务需求使用对应的接口进行集成开发。

## 开放形式

### 客户端开放集成

[网易会议组件](https://doc.yunxin.163.com/meeting/concept/zUzOTE4NTU?platform=client)（简称 NEMeetingKit）提供了一套简单易用的接口，允许开发者通过调用组件（SDK）提供的 API 快速 [集成音视频会议功能](https://doc.yunxin.163.com/meeting/guide/Dg4Njg3NDU?platform=android) 至自己的应用中，实现创会、入会、会控等核心能力。

网易会议组件中已内置了入会后的 UI，第三方 App 仅需调用少量接口即可在其应用中集成网易会议的功能。

### 服务端开放集成

网易会议提供 [服务端 RESTful API](https://doc.yunxin.163.com/meeting/server-apis/DkzMzk3NTg?platform=server) 供您调用，涵盖会议账号管理、会议管理、巡查者角色管理、会议直播管理等能力。

第三方业务系统（包括业务后台或终端 App）可通过调用这些接口实现定制化的业务逻辑。
