## 3.4.2 (2022-10-09)

**问题修复**

修复切换音频输出方法引发的问题。

**新增特性**

支持 <a href="/docs/TE5OTgwODE/Tg3MjQ0MTQ">私有云配置</a>。

**API 变更**

| <div style="width:200px">API</div> | API 说明 |

## 3.2.0 (2022-06-30)

底层重构，基于 NERoom 的会议组件 NEMeetingKit 首次发布。

::: note note
- 推荐您使用 3.2.0 版本及其之后的会议组件。同时，更低版本的 NEMeetingKit 仍在维护，但不再提供较大功能更新，相关 **开发指南** 请访问 GitHub 参考 [旧版会议组件](https://github.com/netease-kit/documents/tree/main/%E4%B8%9A%E5%8A%A1%E7%BB%84%E4%BB%B6/%E4%BC%9A%E8%AE%AE%E7%BB%84%E4%BB%B6)。
- 若您需要将会议组件从低版本升级为新版本，请参考 [迁移指南](https://doc.yunxin.163.com/meetingkit/guide/DQ0NzA3MDI)。
:::
