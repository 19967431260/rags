本文介绍结束会议直播。

## 请求
  
### URI
```
POST https://{endpoint}/apps/v2/live-end
```
- {endpoint} 为 会议组件接入地址的域名，默认为 `roomkit.yunxinapi.com`。
- Content-Type：application/json;charset=utf-8

### Header

请求 Header 的参数说明请参考 <a href="https://doc.yunxin.163.com/meeting/server-apis/TM0MzUxOTU?platform=server" target="_blank">请求结构</a>。


### 请求参数



 <div style="width:100px">参数名称</div>     | 类型   | 是否必选 | 示例 | 描述                                                 |
|---|---|---|---|---|
| roomArchiveId | Long | 是 |110119120  | 待结束直播间的 roomArchiveId， 即归档到历史记录中的房间 ID，全局唯一 <br> 您可以通过[创建会议](https://doc.yunxin.163.com/meeting/server-apis/TUwMzc3NzY?platform=server)的返回参数中获取该参数的值|


### 请求体示例
```java
{
    "roomArchiveId":110119120
}
```

  
## 响应
### 响应参数
| <div style="width:100px">参数名称</div>      | 类型   |  <div style="width:100px">示例</div>  | 描述 |
|-------------|--------|-------|--------------|
| code | Integer |  0 |  状态码，0 表示成功，具体请参考<a href="https://doc.yunxin.163.com/meeting/server-apis/DEwNTU4MTg?platform=server" target="_blank">错误码</a>           |
|   msg      | String   |  Success   |    业务结果描述，Success表示成功        |
|    ts     | Long  | 1648021056815  | 会议服务器处理该请求的完成时间。该时间为 Unix 时间戳，即从 1970 年 1 月 1 日 0 点 0 分 0 秒开始到现在的毫秒数 |
|   requestId     | String   |  7c4b6d9c3e9d42*****cc6e3a4d995   |    请求的唯一标识        |
|   cost      | String   |  48ms   |    处理该请求所消耗的时间        |


### 响应体示例
```java
{
    "code": 0,
    "msg": "Success",
    "ts": 1619068087795,
    "cost": "66ms",
    "requestId": "6e507107d1f4447ea731f651dc6d2432",
    "data": null
}
```
