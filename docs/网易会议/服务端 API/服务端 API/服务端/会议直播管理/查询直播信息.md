查询指定直播的详细信息，包括推流地址、布局模式等。

::: note notice
- 若该会议已被回收，则无法查询到对应的会议直播信息。会议状态说明请参考 [创建会议](https://doc.yunxin.163.com/meeting/server-apis/TUwMzc3NzY?platform=server)。
- 只有当该会议未被回收时，您能查询到对应的会议直播信息，包括正在进行的或已结束的。
:::

## 请求信息

### 请求 URI

```
GET https://{endpoint}/apps/v2/live-info/?roomArchiveId={roomArchiveId}
```

{endpoint} 为 网易会议组件接入地址的域名，默认为 `roomkit.yunxinapi.com`。

### 请求头参数

请求 Header 的参数说明请参考 <a href="https://doc.yunxin.163.com/meeting/server-apis/TM0MzUxOTU?platform=server" target="_blank">请求结构</a>。

### 请求查询参数

 <div style="width:100px">参数名称</div> | 类型 | 是否必选 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| roomArchiveId | Long | 是 | 110119120 | 待查询直播间的 roomArchiveId，即归档到历史记录中的房间 ID，全局唯一。您可以通过 [创建会议](https://doc.yunxin.163.com/meeting/server-apis/TUwMzc3NzY?platform=server) 的返回参数中获取该参数的值。 |

### 请求示例

```
https://roomkit.yunxinapi.com/apps/v2/live-info?roomArchiveId=123456
```
