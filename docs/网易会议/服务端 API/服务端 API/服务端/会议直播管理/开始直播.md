本文介绍通过会议服务端接口开启会议直播。会议直播适用于大型会议场景，只有几个主播需要发言，大部分观众只需要拉流观看。会议直播的应用场景包括但不限于企业内部会议、学术会议、在线培训和公共演讲等领域。通过会议直播，不仅可以提高信息传达效率，还可以节省成本。

:::note note
**会议直播** 为增值功能，如需开通，您可在线咨询网易云信商务经理以获得相应的产品报价。
:::

## 请求信息

- 请求方法：POST
- 请求 URL：https://roomkit.yunxinapi.com/apps/v2/live-start
- Content-Type：application/json;charset=utf-8
- AppKey：应用的 AppKey。详情请参考 [创建应用并获取 AppKey](https://doc.yunxin.163.com/console/concept/TIzMDE4NTA?platform=console)。
- 鉴权头：接口鉴权。详情请参考 [请求结构](https://doc.yunxin.163.com/meeting/server-apis/TM0MzUxOTU?platform=server#header)。

### 请求头参数

请求头（Header）的参数说明，请参考 [请求结构](https://doc.yunxin.163.com/meeting/server-apis/TM0MzUxOTU?platform=server#header)。

### 请求体参数

 <div style="width:100px">参数名称</div> | 类型 | 是否必选 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| roomArchiveId | Long | 是 | 110119120 | 归档到历史记录中的房间 ID，全局唯一。您可以通过 [创建会议](https://doc.yunxin.163.com/meeting/server-apis/TUwMzc3NzY?platform=server) 的返回参数中获取该参数的值。 |
| title | String | 是 | 年终总结大会 | 直播标题，最大长度 64 位字符。支持中文、英文、数字和下划线。 |
| liveLayout | Integer | 是 | 1 | 直播布局，包括：<ul><li>1：画廊布局<li>2：聚焦布局<li>5：自定义布局 |
| liveChatRoomIndependent | boolean | 是 | true | 是否使用独立的直播聊天室 <ul><li>false：不使用独立的直播聊天室，使用会议的聊天室作为直播聊天室，此时直播间的聊天消息和会议的聊天消息互通。<li>true：使用独立的直播聊天室，此时直播间的聊天消息和会议的聊天消息不互通。 |
| liveUserUuids | Array of String | 是 | - | 主播的用户 ID，您可以通过 [创建会议账号](https://doc.yunxin.163.com/meeting/server-apis/DU3NTczMTg?platform=server) 的返回参数中获取用户 ID 的值。<note type="note"><ul><li>如果是画廊模式和聚焦模式，主播最多为 4 个。<li>如果是自定义模式，主播最多为 7 个。</note> |
| password | String | 否 | 123456 | 直播密码，格式为 6 位数字的字符串： <ul><li>如果密码为空，用户无需密码即可进入直播。<li>如果设置了密码，用户需要输入密码才能观看直播。 |
| extensionConfig | String | 否 | - | 扩展字段，JSON 格式，用于透传消息给客户端。最大 500 个字符。 |
| layoutCoordinateList | Array of Json | 否 | - | 布局相关参数配置请参考 [自定义直播布局](https://doc.yunxin.163.com/meeting/server-apis/jE3NTg0NDc?platform=server)。 |
