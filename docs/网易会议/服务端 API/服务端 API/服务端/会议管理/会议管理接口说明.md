## 权限说明

- 会议创建者权限：拥有所有操作权限。
- 主持人权限：除转让主持人外的所有操作权限。
- 联席主持人权限：部分管理权限，包括成员管理、静音控制等。
- 普通成员权限：仅查看权限。
