通过该接口修改某一会议的信息。

## 请求信息

### 请求 URL

```
POST https://{endpoint}/scene/meeting/api/{appId}/v2/edit/{meetingId}
```

- {endpoint} 为 网易会议组件接入地址的域名，默认为 `roomkit.yunxinapi.com`。

- Content-Type：application/json;charset=utf-8

### 请求头参数

请求中 Header 的设置请参考 [请求结构](/meeting/server-apis/TM0MzUxOTU) 描述。

### 请求路径参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| ---- | ---- | ---- | ---- | ---- |
| appId | String | 是 | abcdefg****| 应用的 App Key。 |
| meetingId | Long | 是 | 110120119 | 会议唯一 ID。
