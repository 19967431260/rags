<!--keywords: 会议属性设置,会议控制,全体静音,等候室,会议锁定,参会权限,会议管理-->

设置会议中的会议属性，例如：全体静音、是否允许参会者聊天设置、锁定会议、隐藏会议号和密码、是否开启等候室等。


## 请求信息

### 请求 URL

```http
POST https://{endpoint}/scene/meeting/api/v2/{meetingId}/real-control-status
```

- {endpoint} 为 网易会议组件接入地址的域名，默认为 `roomkit.yunxinapi.com`。

- Content-Type：application/json;charset=utf-8。

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](https://doc.yunxin.163.com/meeting/server-apis/TM0MzUxOTU) 描述。

### 请求路径参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
|---------|------|----------|------|------|
| meetingId | Long | 是 | 123456789 | 会议 ID。 |


### 请求体参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
|---------|------|----------|------|------|
| operatorUserUuid | String | 否 | user123 | 操作者 ID。不填时默认不做权限判断；填写时必须是会议创建者、主持人或联席主持人。 |
| muteVideoAll | Boolean | 否 | true | 是否全体关闭视频。 |
| allowUnmuteVideoBySelf | Boolean | 否 | false | 是否允许用户自己开启视频。 |
| muteAudioAll | Boolean | 否 | true | 是否全体静音。 |
| allowUnmuteAudioBySelf | Boolean | 否 | false | 是否允许用户自己开启音频。 |
| waitingRoomEnabled | Boolean | 否 | true | 是否开启等候室。 |
| blacklistEnabled | Boolean | 否 | false | 是否开启黑名单。 |
| guestJoinEnabled | Boolean | 否 | true | 是否开启访客入会。 |
| hideAvatar | Boolean | 否 | false | 是否隐藏头像。 |
| meetingLocked | Boolean | 否 | false | 是否锁定会议。 |
| autoChat | Integer | 否 | 1 | 允许参会者聊天设置：1-自由聊天，2-仅公开聊天，3-仅私聊主持人，4-禁用聊天。 |
| allowShareScreen | Boolean | 否 | true | 是否允许参会者发起屏幕共享。 |
| allowRename | Boolean | 否 | true | 是否允许参会者自己改名。 |
| allowWhiteboard | Boolean | 否 | true | 是否允许参会者共享白板。 |
| allowAnnotation | Boolean | 否 | true | 是否允许参会者互动批注。 |
| allowReaction | Boolean | 否 | true | 是否允许参会者表情回应。 |
| allowSubtitle | Boolean | 否 | true | 是否允许参会者使用字幕。 |
| recordingPermission | Integer | 否 | 1 | 本地录制权限：1-仅主持人，2-部分成员，3-全体成员。 | 

### 请求体示例

```JSON
{
  "operatorUserUuid": "user123",
  "muteVideoAll": true,
  "allowUnmuteVideoBySelf": false,
  "muteAudioAll": true,
  "allowUnmuteAudioBySelf": false,
  "waitingRoomEnabled": true,
  "blacklistEnabled": false,
  "guestJoinEnabled": true,
  "hideAvatar": false,
  "meetingLocked": false,
  "autoChat": 1,
  "allowShareScreen": true,
  "allowRename": true,
  "allowWhiteboard": true,
  "allowAnnotation": true,
  "allowReaction": true,
  "allowSubtitle": true,
  "recordingPermission": 1
}
```

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
|---------|------|------|------------|
| code | Integer | 状态码，0 表示成功。 | 是 |
| msg | String | 响应消息。 | 是 |
| data | Object | 响应数据。 | 是 |

### 响应体示例

```JSON
{
  "code": 0,
  "msg": "success",
  "data": {}
}
```
