设置或撤销会议中指定参会者的联席主持人身份。


## 请求信息

### 请求 URL

<style>
table th:first-of-type {
    width: 20%;
}
</style>

```
POST https://{endpoint}/scene/meeting/api/v2/{meetingId}/co-host
```

- {endpoint} 为 网易会议组件接入地址的域名，默认为 `roomkit.yunxinapi.com`。

- Content-Type：application/json;charset=utf-8。

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](https://doc.yunxin.163.com/meeting/server-apis/TM0MzUxOTU) 描述。

### 请求路径参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| meetingId | Long | 是 | 1234567890 | 会议 ID。 |

### 请求体参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| operatorUserUuid | String | 否 | "user123456" | 操作者 ID。不填时默认不做权限判断；填写时必须是主持人、联席主持人或会议创建者。 |
| userUuid | String | 是 | "user789012" | 用户 UUID，要设置为联席主持人的成员 ID。 |
| action | String | 是 | "set" | 操作类型。可选值：<br/>• set：设置为联席主持人。<br/>• remove：撤销联席主持人身份。 |

### 请求体示例

```JSON
{
  "operatorUserUuid": "user123456",
  "userUuid": "user789012",
  "action": "set"
}
```

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
| --- | --- | --- | --- |
| code | Integer | 状态码，0 表示成功。 | 是 |
| msg | String | 状态描述。 | 是 |
| data | Object | 响应数据。 | 是 |

### 响应体示例

```JSON
{
  "code": 0,
  "msg": "success",
  "data": {}
}
```
