设置或取消会议中的焦点视频。

## 请求信息

### 请求 URL

```
POST https://{endpoint}/scene/meeting/api/v2/{meetingId}/focus-video
```

- {endpoint} 为 网易会议组件接入地址的域名，默认为 `roomkit.yunxinapi.com`。

- Content-Type：application/json;charset=utf-8。

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](https://doc.yunxin.163.com/meeting/server-apis/TM0MzUxOTU) 描述。

### 请求路径参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
|---------|------|----------|------|------|
| meetingId | Long | 是 | 123456789 | 会议 ID。 |

### 请求体参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
|---------|------|----------|------|------|
| operatorUserUuid | String | 否 | "user123456" | 操作者 ID，不填写时默认不做权限判断，填写时必须是主持人、联席主持人或会议创建者。 |
| userUuid | String | 是 | "target_user123" | 目标用户 UUID，要设置或取消焦点视频的用户。 |
| action | String | 是 | "set" | 操作类型，set：设置焦点视频，remove：取消焦点视频。 |

### 请求体示例

```JSON
{
  "operatorUserUuid": "user123456",
  "userUuid": "target_user123",
  "action": "set"
}
```

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
|---------|------|------|------------|
| code | Integer | 错误码，0 表示成功。 | 是 |
| msg | String | 错误信息描述。 | 是 |
| data | Object | 响应数据。 | 是 |

### 响应体示例

```JSON
{
  "code": 0,
  "msg": "success",
  "data": {}
}
```
