<!--keywords: 取消会议, DELETE请求, 会议管理, 周期会议, API接口-->

取消已创建的会议。

:::note note
- 只能取消未开始的会议。
- 取消周期会议时，可以选择是否取消整个周期。
:::

## 请求信息

### 请求 URI

```
DELETE https://roomkit.yunxinapi.com/scene/meeting/api/{appId}/v1/cancel/{meetingId}
```

### 请求头参数

请求 Header 的参数说明请参考 [请求结构](/meeting/server-apis/zIwOTY0NDk)。

### 请求路径参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
|---------|-----|----------|------|------|
| appId | String | 必选 | abcdefg**** | 应用的 App Key |
| meetingId | Long | 必选 | 110120119 | 会议唯一 ID |

### 请求查询参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
|---------|-----|----------|------|------|
| cancelRecurringMeeting | Boolean | 否 | true | 是否取消周期会议，默认为 false。设为 true 时取消整个周期会议，设为 false 时仅取消当前这一场会议。 |

## 响应信息

### 响应参数

| 参数名称 | 类型 | 说明 | 是否必返回 |
|---------|-----|------|-----------|
| code | Integer | 返回码，0 表示成功 | 是 |
| msg | String | 错误信息，成功时为空 | 否 |

### 响应体示例

```JSON
{
  "code": 0
}
```
