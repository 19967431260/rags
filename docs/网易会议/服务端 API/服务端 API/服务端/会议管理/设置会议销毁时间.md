为了更好地控制会议成本，您可以设置会议的销毁时间。当会议时间超出预约的会议结束时间后，系统将自动销毁会议。

设置会议销毁时间有两种方式：

- 可以在 [创建会议](https://doc.yunxin.163.com/meeting/server-apis/TUwMzc3NzY?platform=server) 时，通过 `destroyTime` 参数设置会议销毁时间。
- 可以在会议前或会议中，通过本接口设置会议销毁时间。

设置会议销毁时间后，在到达对应时刻的前 10 分钟、5 分钟、1 分钟会各提醒一次，提醒时间持续 1 分钟。

## 请求信息

- 请求方法：PUT
- 请求 URL：https://roomkit.yunxinapi.com/scene/meeting/api/{appId}/v1/meeting/{meetingId}/destroyTime/{destroyTime}
- AppKey：应用的 AppKey。详情请参考 [创建应用并获取 AppKey](https://doc.yunxin.163.com/console/concept/TIzMDE4NTA?platform=console)。
- 鉴权头：接口鉴权。详情请参考 [请求结构](https://doc.yunxin.163.com/meeting/server-apis/TM0MzUxOTU?platform=server#header)。
