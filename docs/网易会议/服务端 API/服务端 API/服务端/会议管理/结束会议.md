会议开始后，主持人使用会议唯一 ID、会议号或会议短号结束一个会议。

::: note note
当会议内所有人退出会议后，会议会自动结束，无需调用此接口。
:::

## 使用会议唯一 ID 结束会议

### 接口请求信息

- 请求方法：DELETE
- URL：https://roomkit.yunxinapi.com/scene/meeting/api/{appId}/v1/close/meetingId/{meetingId}/recycle/{recycle}

URL 中参数说明：
| <div style="width:60px">参数名称</div>    | <div style="width:50px">类型</div>   | <div style="width:80px">是否必选</div> | <div style="width:100px">示例</div> | <div style="width:200px">描述</div> |
|-------------|--------|----------|------|--------------|
| meetingId | Long |必选|11011**** | 会议唯一 ID。 |
| recycle | Boolean |必选|true | 是否回收。<li>true：回收。当会议在**预约结束时间**之前结束时，与会者不可以再进入该会议。<li>false：不回收。当会议在**预约结束时间**之前结束时，与会者还可以再进入该会议。 |

### 请求参数

请求中 Header 的设置请参考[请求结构](/meeting/server-apis/TM0MzUxOTU)描述。

### 返回参数

此接口的返回结果中无 `data` 属性内包含的参数。具体返回参数请参考[返回结果](/meeting/server-apis/TczMTMyMjg)。

### 示例

#### 请求 Body 示例

无

#### 返回示例

```
//正常返回示例
"Content-Type": "application/json; charset=utf-8"
{
  "code": 0
}

//异常返回示例
{
    "code":501,
    "msg":"服务器内部错误"
}
```
