通过该接口为某个房间的巡检者角色解绑用户。

## 请求信息

### 请求 URL

```
POST https://roomkit.yunxinapi.com/apps/v2/room/unbind/observer
```

### 请求头参数

请求中 Header 的设置请参考 [请求结构](/meeting/server-apis/TM0MzUxOTU) 描述。

### 请求体参数

| 参数名称 | 类型 | 是否必选 | 示例 | 说明 |
| ---- | ---- | ---- | ---- | ---- |
| roomArchiveId | Long | 是 | 1234567 | 请填写目标房间的 roomArchiveId，您可以通过 [创建会议](https://doc.yunxin.163.com/meeting/server-apis/TUwMzc3NzY?platform=server) 或会议抄送中的返回参数中获取该参数的值。 |
| userUuid | String | 是 | abc | 待解绑的巡检者对应的用户 ID。您可以通过 [创建会议账号](https://doc.yunxin.163.com/meeting/server-apis/jU0MDAzNTg?platform=server) 的返回参数中获取该参数的值。 |

### 请求体示例

```JSON
{
    "roomArchiveId": 1234567,
    "userUuid": "abc"
}
```

## 响应信息

### 响应体参数

以下是响应结果中 `data` 属性内包含的参数。其他统一响应参数的相关说明请参考 [返回结果](/meeting/server-apis/TczMTMyMjg)。

| 参数名称 | 类型 | 说明 |
| --- | --- | --- |
| code | Integer | 状态码。0 表示成功。非 0 表示失败。 |
| msg | String | 状态描述。 |
| ts | Long | 当前时间戳，单位为毫秒。 |
| cost | String | 该请求耗费的时间。 |
| requestId | String | 请求 ID。 |

### 响应体示例

```JSON
{
  "code": 0,
  "msg": "Success",
  "ts": 1619068087795,
  "requestId": "6e507107d1f441234731abcdec6d2432",
  "cost": "66ms"
}
```
