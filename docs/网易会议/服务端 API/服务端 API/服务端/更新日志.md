本文介绍网易云信视频会议（原会议组件）在服务端 API 层面的更新记录。

## 2024-12-13

- [创建会议](https://doc.yunxin.163.com/meeting/server-apis/TUwMzc3NzY?platform=server) 接口新增功能，支持时区设置（`timezoneId`），联席主持人数量限制（`coHostLimit`），并支持预约周期性会议（`recurringRule`）。
- 新增新版 [修改会议信息](https://doc.yunxin.163.com/meeting/server-apis/TY0Mzk3MTc?platform=server) 接口，支持编辑周期性会议（`recurringRule`）。

    :::note note
    旧版 [修改会议信息](https://doc.yunxin.163.com/meeting/server-apis/jY3ODE3NDg?platform=server) 继续为您服务。接口会逐渐废弃，所以建议您及时迁移到新接口，以体验更好的功能效果。
    :::

## 2024-12-10

新增 [获取纪要文件](https://doc.yunxin.163.com/meeting/server-apis/zczODA1NDE?platform=server) 接口。会议中开启了智能录制，会在会议结束时，生成对应的 **会议 AI 纪要** 的文件，该接口用于获取指定会议的纪要文件信息。

## 2024-08-14

新增 [获取转写文件](https://doc.yunxin.163.com/meeting/server-apis/zY3NDkyMjI?platform=server) 接口。会议中开启了实时转写功能后，会在会议结束时，生成功能开启期间对应的 **语音转文本** 的转写文件。您可以调用该接口获取转写文件。

## 2024-06-13

新增一套完整的视频会议账号管理接口，包括：

- [创建会议账号](https://doc.yunxin.163.com/meeting/server-apis/jU0MDAzNTg?platform=server)
- [更新会议账号](https://doc.yunxin.163.com/meeting/server-apis/zIxNjcwMDg?platform=server)
- [批量查询会议账号](https://doc.yunxin.163.com/meeting/server-apis/jI1MTgwNTE?platform=server)
- [封禁会议账号](https://doc.yunxin.163.com/meeting/server-apis/zU5OTQ2OTQ?platform=server)
- [解禁会议账号](https://doc.yunxin.163.com/meeting/server-apis/jkzMjgxOTA?platform=server)
- [注销会议账号](https://doc.yunxin.163.com/meeting/server-apis/jU4NDUwNjc?platform=server)
- [重置用户令牌](https://doc.yunxin.163.com/meeting/server-apis/zc3MTAyODU?platform=server)

:::note note
原有的 [创建会议账号](https://doc.yunxin.163.com/meeting/server-apis/DU3NTczMTg?platform=server)、[查询会议账号](https://doc.yunxin.163.com/meeting/server-apis/zM1NDQ1NDU?platform=server)、[更新会议账号](https://doc.yunxin.163.com/meeting/server-apis/TUxNDU5MjA?platform=server) 继续为您服务。接口会逐渐废弃，所以建议您及时迁移到新接口，以体验更好的功能效果。
:::

## 2023-08-14

新增和修改的接口如下表所示。

 <div style="width:120px">功能</div> | 描述
---- | ----
[创建会议](https://doc.yunxin.163.com/meeting/server-apis/TUwMzc3NzY?platform=server) | 增加 `destroyTime` 参数，用于设置会议销毁时间，以便更好地控制会议成本。当会议时间超出预约的会议结束时间后，系统将自动销毁会议，避免不必要的费用支出。
[修改会议销毁时间](https://doc.yunxin.163.com/meeting/server-apis/DgyNjU0OTI?platform=server) | 修改会议销毁时间。
[开始直播](https://doc.yunxin.163.com/meeting/server-apis/Tg1OTgxNjc?platform=server) | 开启会议直播。会议直播适用于大型会议场景，只有几个主播需要发言，大部分观众只需要拉流观看的场景，包括但不限于企业内部会议、学术会议、在线培训和公共演讲等领域。
[修改直播](https://doc.yunxin.163.com/meeting/server-apis/jU3NDMyODk?platform=server) | 主播上线时、修改画面布局等场景中需要调用此接口更新直播。
[查询直播信息](https://doc.yunxin.163.com/meeting/server-apis/DY5MTgwMTg?platform=server) | 查询指定直播的详细信息，包括推流地址、布局模式等。
[结束直播](https://doc.yunxin.163.com/meeting/server-apis/zI2NDkwMjc?platform=server) | 结束会议直播。

## 2023-07-12

新增巡检者角色管理，包括：

功能 | 描述
---- | ----
[增加巡检者](https://doc.yunxin.163.com/meeting/server-apis/jM2NDU1NTI) | 为某个房间的巡检者角色绑定用户，巡检者加入会议对会议内容进行巡检，保障会议服务的安全。
[解绑巡检者角色](https://doc.yunxin.163.com/meeting/server-apis/DcwNDM3NjA) | 巡检者离开会议后，解绑该房间的巡检者角色。

## 2023-06-30

增加 [会议抄送事件](https://doc.yunxin.163.com/meeting/server-apis/Dc2Mjk1MDE?platform=server)。

功能 | 接口变更

会议账号管理 | 增加 [创建会议账号](/meeting/server-apis/DU3NTczMTg)、[查询会议账号](/meeting/server-apis/zM1NDQ1NDU) 等服务端接口。
会议管理 | 增加 [创建会议](/meeting/server-apis/TUwMzc3NzY)、[查询会议信息](/meeting/server-apis/TU4NjQxODU)、[修改会议信息](/meeting/server-apis/jY3ODE3NDg) 等服务端接口。
