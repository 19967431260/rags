## 通用

| `code` 错误码 | `msg` 错误信息 | 场景说明 |
| --- | --- | --- |
| 0 | Success | 操作成功。 |
| 102 | need login | 请先登录 NERoom。 |
| 200 | success | 请求成功。 |
| 304 | Not Modified | 删除麦位时，麦位不存在。 |
| 400 | Bad Request | 客户端传参有问题。 |
| 401 | Unauthorized | 鉴权失败。 |
| 402 | token not correct | Token 不正确。 |
| 403 | Forbidden | 该用户无操作权限。 |
| 404 | Not Found | 未找到该有的数据。例如进行房间操作时，会校验房间是否存在。 |
| 405 | Method Not Allowed | 不支持该方法。 |
| 406 | token expired | Token 已过期。 |
| 409 | Target Already Exists | 创建房间时，房间号已经存在。 |
| 415 | Unsupported Media Type | 不支持的 MediaType，例如非 JSON 的请求体（Body）。 |
| 500 | Internal Server Error | 系统内部错误。 |
| 503 | Service Busy | 服务忙碌中。 |
| 600420 | - | 应用功能未开通，请开通后重试。 |
