会议中开启了智能录制，会在会议结束时，生成对应的 **会议 AI 纪要** 的文件，该接口用于获取指定会议的纪要文件信息。

:::note note
**AI 纪要** 为增值功能，如需开通，您可在线咨询网易云信商务经理以获得相应的产品报价。
:::

## 请求信息

- 请求方法：GET
- 请求 URL：https://roomkit.yunxinapi.com/scene/meeting/api/v1/summary-file?meetingId={meetingId}&ttl={ttl}&type={type}
- AppKey：应用的 AppKey。详情请参考 [创建应用并获取 AppKey](https://doc.yunxin.163.com/console/concept/TIzMDE4NTA?platform=console)。
- 鉴权头：接口鉴权。详情请参考 [请求结构](https://doc.yunxin.163.com/meeting/server-apis/TM0MzUxOTU?platform=server#header)。

### 请求头参数

请求头（Header）的参数说明，请参考 [请求结构](https://doc.yunxin.163.com/meeting/server-apis/TM0MzUxOTU?platform=server#header)。

### 请求 URL 参数

| <div style="width:80px">参数名称</div> | 类型 | 是否必选 | 示例 | 说明 |
| ---- | ---- | ---- | ---- | ---- |
| meetingId | Long | 是 | 110120119 | 会议唯一 ID。 |
| ttl | Long | 是 | 30 | 撰写文件的有效时间（Time To Live），单位为秒。默认 300 秒，最大 604800（7 天），超过时间后，下载链接将过期。 |
| type | String | 否 | txt | 导出的纪要文件的格式。取值范围：<ul><li>`txt`（默认值）</li><li>`docx`</li><li>`pdf`</li></ul> |

## 响应信息

### 响应参数

以下是返回结果中 `data` 属性内包含的参数。其他统一返回参数的相关说明请参考 [返回结果](https://doc.yunxin.163.com/meeting/server-apis/TczMTMyMjg)。

| 参数名称 | 类型 | 示例 | 说明 |
| ---- | ---- | ---- | ---- |
| type | String | txt | 文件类型。 |
| downloadUrl | String | www.downloadexample.com | 下载链接，有效性跟随参数 `ttl` 设置对应的时效。 |
| size | Long | 102400 | 文件大小，单位：byte。 |

### 响应体示例

```JSON
[
    {
        "type": "txt",
        "downloadUrl":"www.downloadexample1.com",
        "size":102400
    },
    {
        "type": "txt",
        "downloadUrl":"www.downloadexample2.com",
        "size":204800
    }
]
```
