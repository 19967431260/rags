调用 NEMeetingKit 客户端 API 时，可能返回的错误码如下：

|错误码|说明|
|--|--|
|0|成功。|
|-1|通用错误。|
|-3|网络连接错误。|
|-4|会议已经存在。|
|-5|参数错误。|
|-6|当前已在会议中。|
|-7|鉴权失败。|
|-9|操作取消。|
|-100002|SDK 未初始化。|
|100004|IM 复用失败，未登录 IM SDK。|
|100005|IM 复用失败，IM 账号不匹配。|
|112001|IM 复用失败，不支持匿名入会。|

::: note note
更多服务端错误码请参考<a href="https://doc.yunxin.163.com/meeting/server-apis/DEwNTU4MTg?platform=server" target="_blank">错误码</a>。
:::
