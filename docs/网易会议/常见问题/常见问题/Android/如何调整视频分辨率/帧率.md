
## 问题描述

会议组件如何调整视频分辨率/帧率？

## 问题原因

相较于 RTC SDK，会议组件目前无法通过客户端接口设定视频分辨率，但是可以借助控制台的房间组件模板（NERoom）设置。

## 实现方法

:::note notice
使用 NERoom 房间组件的前提条件如下：
- 已创建应用。
- 已开通 IM 套餐包、聊天室和音视频通话 2.0。
:::

1. 在 [云信控制台](https://app.yunxin.163.com/global/home) 首页 **应用管理** 列表下，选择目标应用。

2. 在左侧导航中单击 **房间组件模块**，然后开启房间组件。

    ![开通NERoom.png](https://yx-web-nosdn.netease.im/common/00134a9c58238e8c59ef54b2dd0c76d9/开通NERoom.png)

3. 开启后，在 **组件模块管理** 区域，符合生效条件的场景方案模板的状态会从 **未开启** 变为 **已开启**。

4. 单击 **线上会议** 的 **角色管理**。


    ![线上会议角色管理.png](https://yx-web-nosdn.netease.im/common/6bb7fc9573392fe3074fbe4f1a726088/线上会议角色管理.png)


5. 在 **参数配置** 下的 **视频效果** 中进行配置。


    ![配置分辨率.png](https://yx-web-nosdn.netease.im/common/ca07765d1f2d1137a3a1e37f8f9d7401/配置分辨率.png)


    根据您的业务需求，设置分辨率和帧率。

    例如设置 “host”、“cohost”、“member”和自定义角色的分辨率，相同角色的用户使用同一个分辨率模式。
