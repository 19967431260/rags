## **Windows 版本 Chrome 浏览器如何恢复麦克风权限？**

当您打开麦克风或者在会议界面中看到 **没有摄像头/麦克风权限** 提示语时，请按如下操作恢复麦克风权限：

1. 先在底部操作栏单击打开麦克风。
2. 弹出提示后，单击浏览器右上方的摄像头小图标。
3. 选择允许该网址访问您的麦克风，如下图所示。

    Windows Chrome 恢复麦克风权限：

## **MacOS 版本 Chrome 浏览器如何设置共享屏幕权限？**

在会中单击 **共享** 后，可以选择您需要共享的内容：标签页、窗口、整个屏幕。

选择 Chrome 标签页或者 Chrome 浏览器窗口时不需要设置权限。

若您选择了其他窗口或者整个屏幕时没有分享成功，并提示无法共享，可以进行如下操作：

1. 在 **系统设置 > 隐私安全 > 屏幕录制** 打开 Chrome 浏览器的屏幕录制权限，如图所示

2. 重新启动 Chrome 浏览器，进入会议后再次发起共享。
