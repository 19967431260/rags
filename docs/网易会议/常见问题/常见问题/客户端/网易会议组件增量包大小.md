## 在 Android 平台上，网易会议组件会给我的应用增加多少大小？

在 Android 平台上，大部分情况下您可以只保留 arm64-v8a 架构，这将带来约 41MB 的增量大小。如果需要支持 armeabi-v7a 和 arm64-v8a 双架构，增量大小将达到约 92MB。

**详细数据：**
| 打包架构 | armeabi-v7a 和 arm64-v8a 双架构 | arm64-v8a |
| :---- | :---- | :---- |
| Demo APK 大小 | 91.9MB | 41.4MB |
| 空工程 APK 大小 | 63KB | 63KB |
| 增量 | 91.837MB | 41.337MB |

## 在 iOS 平台上，网易会议组件会增加多少包体积？

在 iOS 平台上，您只需保留 arm64 架构，这将带来约 55MB 的增量大小。下载大小约为 35.4MB，安装后占用空间约为 91.4MB。

**详细数据：**
| 打包架构 | arm64 单架构 |
| :---- | :---- |
| Demo IPA 大小 | 55.6MB |
| 空工程 IPA 大小 | 100KB |
| IPA 增量 | 55.6MB |
| 下载大小 | 35.4MB |
| 安装大小 | 91.4MB |

## 在 macOS 的 Electron 应用中，网易会议组件会增加多少体积？

在 macOS 平台上，为了支持 Intel 和 ARM 芯片，通常需要打包 Universal 双架构，这将带来约 99MB 的增量大小。如果只支持 Intel(x64)架构，增量大小同样约为 99MB。

**详细数据：**
| Electron 打包架构 | Intel(x64) | Universal(x64 + arm64) |
| :---- | :---- | :---- |
| 空工程大小（dmg 产物） | 95MB | 164MB |
| Demo 大小（dmg 产物） | 194MB | 263MB |
| 增量 | 99MB | 99MB |

## 在 Windows 的 Electron 应用中，网易会议组件会增加多少体积？

在 Windows 平台上，您只需要支持 x64 架构，这将带来约 40MB 的增量大小。

**详细数据：**
| 打包架构 | x64 位 |
| :---- | :---- |
| Electron 空工程大小(exe 产物) | 63.5MB |
| Electron Sample 大小（exe 产物） | 103MB |
| 增量 | 39.5MB |
