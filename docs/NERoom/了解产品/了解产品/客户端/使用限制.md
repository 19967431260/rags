在使用 NERoom SDK之前，请先了解使用限制和各平台的开发环境要求。

<table>
    <tr>
        <th width="20%"><b>平台</b></th>

        <td>Android SDK</td>
        <td>
            <ul>
                <li>Android 系统 5.0 及以上版本的移动设备
                <li>Android SDK API 等级 21 及以上、Android Studio 3.0 或以上版本
                <li>支持 armeabi-v7a、arm64-v8a、x86 架构
        </td>
    </tr>
    <tr>
        <td>iOS SDK</td>
        <td>
            <ul>
                <li>Xcode 9.0 及以上版本
                <li>支持 iOS 10.0 及以上版本的 iPhone 或 iPad 真机
                <li>项目已配置有效的开发者签名
        </td>
    </tr>
    <tr>
        <td>Windows SDK</td>
        <td>
            <ul>
                <li>系统版本：Win 7 及以上版本
                <li>架构：x86、x64
        </td>
    </tr>
    <tr>
        <td>macOS SDK</td>
        <td>
            <ul>
                <li>Xcode 9.0 及以上版本
                <li>OS X 10.14 及以上版本的 Mac 真机
                <li>项目已配置有效的开发者签名
        </td>
    </tr>
    <tr>
        <td>Web SDK</td>
        <td>桌面端推荐使用：<ul>
                <li>Chrome 72及以上版本</li>
                <li>Safari 12 及以上版本</li>
             </ul>
        </td>
    </tr>
    <tr>
        <td>Flutter SDK</td>
        <td>
            <ul>
                <li>Flutter 1.22.0 及以上版本</li>
                <li>Android 端开发：<ul>
                        <li>Android Studio 4.1 及以上版本</li>
                        <li>App 要求 Android 5.0 及以上版本设备</li>
                    </ul>
                </li>
                <li>iOS 端开发：<ul>
                        <li>Xcode 11.0 及以上版本</li>
                        <li>请确保您的项目已设置有效的开发者签名</li>
                    </ul>
                </li>
        </td>
