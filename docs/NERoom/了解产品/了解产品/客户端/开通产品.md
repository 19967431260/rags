本文介绍如何在网易云信控制台开通和配置房间组件。

## <span id="如何开通房间组件">开通房间组件</span>

您可以在网易云信控制台开通房间组件。在 [网易云信控制台](https://app.yunxin.163.com/global/home) 开通 IM 即时通讯及聊天室、音视频通话 2.0 后，即能启用房间组件。开通步骤请参考 [开通或试用服务](https://doc.yunxin.163.com/console/concept/zc3NDYzNzc?platform=console)。

## 配置功能

### <span id="如何获取组件模板 ID">获取组件模板 ID</span>

NERoom 提供多种典型业务场景模板，每个模板对应唯一模板 ID。创建房间时指定模板 ID 即可应用对应场景配置，例如 `template_id` 为 `9496` 表示创建语聊房类型房间。更多详情，请参考 [创建房间](https://doc.yunxin.163.com/neroom/guide/zk1ODc5MjE?platform=android)。

您可以在网易云信控制台获取对应的模板 ID。具体步骤如下：

1. 登录 <a href="https://app.yunxin.163.com/index#/">网易云信控制台</a>。
2. 在首页选择对应的应用。
3. 在左侧导航栏选择 **产品功能** > **房间组件模板**，在 **组件模板管理** 区域，单击对应模板的 **角色管理**。
4. 查看模板 ID 的值。

    <img alt="组件模板" src="https://yx-web-nosdn.netease.im/common/b18e0a93d117dd24c1f878ed8381e22f/image.png" style="width:80%;border: 1px solid #BFBFBF;">

### <span id="如何获取角色 ID">获取角色 ID</span>

每个 NERoom 应用会绑定一个组件模板，组件模板中可以设置角色。角色 ID 就是您的 NERoom 应用绑定的模板中的角色的 ID。

您可以在网易云信控制台获取创建房间时所需的角色 ID。具体步骤如下：

1. 登录 <a href="https://app.yunxin.163.com/index#/">网易云信控制台</a>。
2. 在首页选择对应的应用。
3. 在左侧导航栏选择 **产品功能** > **房间组件模板**，在 **组件模板管理** 区域，选择某个模板，单击 **角色管理**。

    <img alt="角色管理.png" src="https://yx-web-nosdn.netease.im/common/39e59d0200b65c0d2d45c9a837f1976e/角色管理.png" style="width:80%;border: 0px solid #BFBFBF;">

4. 查看各个角色 ID 的值。

    <img alt="角色 ID.png" src="https://yx-web-nosdn.netease.im/common/67382d6d1894a492a61b4e2ee14bffdf/角色ID.png" style="width:50%;border: 0px solid #BFBFBF;">

### <span id="如何修改角色的权限">修改角色权限</span>

角色权限是在您的 NERoom 应用绑定的组件模板中定义的。

- 如果您的 NERoom 应用绑定的是系统自带的组件模板，例如互动直播模板，则无法修改角色的权限。
- 如果您的 NERoom 应用绑定的是自定义组件模板，您可以修改角色的权限。具体步骤如下：
    1. 登录 <a href="https://app.yunxin.163.com/index#/">网易云信控制台</a>。
    2. 在首页选择对应的应用。
    3. 在左侧导航栏选择 **产品功能** > **房间组件模板**，在 **组件模板管理** 区域，单击 **自定义方案** 中的 **角色管理**。

        <img alt="角色管理.png" src="https://yx-web-nosdn.netease.im/common/39e59d0200b65c0d2d45c9a837f1976e/角色管理.png" style="width:80%;border: 0px solid #BFBFBF;">

    4. 在 **角色管理** 区域，单击对应的角色，修改角色的权限。单击 **保存**。

        <img alt="修改角色.png" src="https://yx-web-nosdn.netease.im/common/d1a15e606aa7cdade595f84b08be18fe/修改角色.png" style="width:80%;border: 0px solid #BFBFBF;">

    :::note note
    在 <a href="https://app.yunxin.163.com/index#/">网易云信控制台</a> 修改角色权限后，对已经创建的房间不生效，只对新创建的房间生效。
    :::
