本文介绍网易云信房间组件（NeteaseRoom，简称 NERoom） uni-app 平台的更新日志。

## <span id="1.41.0 (2026-03-23">1.41.0 (2026-03-23)</span>

网易云信 NERoom uni-app SDK 正式发布！

该版本支持 Android 和 iOS 平台，功能对齐 v1.41.0 版本的 Android SDK 和 iOS SDK。具体功能详情请参考 [Android 更新日志](https://doc.yunxin.163.com/neroom/concept/zgzMDM3MTM?platform=client#1410-2025-09-25) 和 [iOS 更新日志](https://doc.yunxin.163.com/neroom/concept/TMxODUyOTc?platform=client#1410-2025-09-25)。
 
NERoom uni-app SDK 底层兼容以下产品：

- 兼容 `NIMSDK` 10.8.30 版本
- 兼容 `NERtcSDK` 5.8.33 版本
- 兼容 `NEWhiteboard` 3.9.15 版本
