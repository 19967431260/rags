网易云信房间组件（下文简称 NERoom）以房间为基础，提供网易云信全系列能力，包括即时消息、音视频通话、直播、互动白板等，是一个易接入、强扩展、高效部署且覆盖多场景的服务。

## 适用客群

NERoom 适用于需要快速构建多场景互动应用的企业和开发者。特别适合在线教育机构、远程办公团队、社交娱乐平台、游戏开发商和内容创作社区。这些场景通常需要融合即时通讯、音视频互动、直播、互动白板的技术资源。NERoom 通过提供成熟的场景化模板（如语聊房、游戏房、K 歌房等）和可组合的功能模块，帮助您大幅降低开发成本和上线时间，同时保持灵活的定制化能力，满足各行业特定的业务需求和用户体验要求。

## 产品架构

NERoom 整合了网易云信的核心能力，包括 [即时通讯 IM](https://doc.yunxin.163.com/messaging2/concept?platform=client)、[音视频通话 2.0 RTC](https://doc.yunxin.163.com/nertc/concept?platform=client)、[直播](https://doc.yunxin.163.com/live-streaming/concept?platform=client)、[点播](https://doc.yunxin.163.com/vod/concept?platform=client) 和 [互动白板](https://doc.yunxin.163.com/whiteboard/concept?platform=web)，方便您快速搭建业务场景，极速上线业务 App。

### 能力层

底层采用网易云信 PaaS 能力（IM、RTC、直播、点播、互动白板）和通用能力（日志、信令、抄送、长连接通道、透传等）。

### 框架层

- 通过房间（`roomkit`）框架和配套项结合，生成在线 K 歌、语聊房、游戏房、线上会议、直播房、权限管理组件等业务组件。
- 根据不同业务场景提供通用场景模板，一键生成对应 SDK。
- 业务组件和场景模板包括业务逻辑和 UIKit。

### 业务层

您可以根据场景模板生成的 Demo 快速搭建业务 App，或通过修改逻辑、UI 和组件进行个性化定制。

## 产品优势

- **专业沉淀**：基于网易云信在 IM 即时通讯、音视频通话等领域多年的技术积累与业务沉淀，提供高效、低成本的能力接入。

- **丰富模板**：提供在线 K 歌、语聊房、游戏房、线上会议、直播房等典型业务场景模板，助您快速搭建业务场景。支持自定义模板，可适用于在线办公协同、教学互动、内训等多种场景。

- **灵活构建**：提供即时消息、音视频通话、互动白板、麦位管理、角色管理等模块，实现积木式功能组装，通过可视化界面配置降低集成难度。

- **扩展性**：开放各类标准接口和扩展能力，支持对接第三方厂商或自有业务系统，满足业务定制化需求，实现业务全面发展。

## 产品功能

在 [网易云信控制台](https://app.yunxin.163.com/global/home) 开通 IM 即时通讯及聊天室、音视频通话 2.0 后，即能启用房间组件。开通步骤请参考 [开通或试用服务](https://doc.yunxin.163.com/console/concept/zc3NDYzNzc?platform=console)。

### 组件模板

NERoom 提供多种典型业务场景模板，每个模板对应唯一模板 ID。创建房间时指定模板 ID 即可应用对应场景配置，例如 `template_id` 为 `9496` 表示创建语聊房类型房间。更多详情，请参考 [创建房间](https://doc.yunxin.163.com/neroom/guide/zk1ODc5MjE?platform=android)。

<img alt="组件模板" src="https://yx-web-nosdn.netease.im/common/b18e0a93d117dd24c1f878ed8381e22f/image.png" style="width:80%;border: 1px solid #BFBFBF;">

### 角色管理

NERoom 通过角色系统实现精细化配置和权限管理。模板默认提供主播（`host`）和观众（`audience`）角色，您也可以添加自定义角色满足特定需求。

每个角色可配置的参数和权限包括：

- **音频效果**：音频场景、音频模式
- **视频效果**：分辨率、帧率
- **房间场景**：通信模式、直播模式
- **权限控制**：细粒度功能权限开关设置

:::note note
可以根据业务需求，在网易云信控制台上，为具体组件模板设置开关来打开或者关闭相应的权限。修改后的配置仅对新创建的房间生效，不影响已创建的房间。
:::

您可以根据业务需求，选择基于 NERoom 或者基于上层业务组件开发实现功能。

您可以参考上文 [产品功能](#feature)，先在网易云信控制台上开通和配置 NERoom，然后使用 SDK 进行功能开发。以安卓端为例，集成步骤包括：

- [集成 NERoom SDK](https://doc.yunxin.163.com/neroom/guide/Dc4MTA4Mjk?platform=android)
- [账号集成与登录](https://doc.yunxin.163.com/neroom/guide/TQxNDU0NjA?platform=android)
- [房间管理](https://doc.yunxin.163.com/neroom/guide/zk1ODc5MjE?platform=android)
- [成员管理](https://doc.yunxin.163.com/neroom/guide/jc3MzU5MzY?platform=android)
- [即时消息](https://doc.yunxin.163.com/neroom/guide/zgyNTc5OTY?platform=android)

### 基于场景化业务组件

NERoom 是网易云信 [多人会议（RoomKit）](https://doc.yunxin.163.com/meeting/guide?platform=android)和 [在线直播（NELiveKit）](https://doc.yunxin.163.com/pk/concept/jg2MTU0Nzc?platform=client)  & [语音聊天室（VoiceRoomKit）](https://doc.yunxin.163.com/group-voice-room/guide/DQyODA0MDM?platform=android) 共同使用的开发工具包，为这些上层业务组件提供了关键技术优势：

- **全面功能**：包含房间管理、成员管理、麦位控制、美颜等完整能力。
- **高清画质**：支持 720P 和 1080P，在丢包率高的弱网环境下仍保持高质量传输。
- **专业音效**：采用 48kHz 采样率，结合 3A 处理算法消除回声和啸叫。
- **极致体验**：全链路 128kbps 高音质立体声，打造沉浸式互动环境。
