本文介绍 NERoom SDK 的产品优势。

## 超低的接入成本

NERoom SDK 底层提供房间式的一体化接口，一个 SDK 即可实现即时消息、音视频通话、直播、互动白板的全系列功能。


## 超简单的接入方式

NERoom SDK 提供互动直播和语聊房的在线模板，您可以将模板一键集成至应用，助您快速搭建业务应用。

NERoom SDK 也支持积木式的组件搭配，您可以按需选择高频模块和上百个接口，实现按需选择，按需组装。

## 超灵活的接入能力

NERoom SDK 提供直播、白板、共享桌面、即时消息、连麦、录制回放、美颜等丰富的功能，支持按业务需求自由组装。

## 超强大的适配能力

NERoom SDK 开放各类标准接口，房间功能轻松扩展。可基于组件，修改业务逻辑、UI展示，同时支持基于标准接口对接第三方厂商或自有业务系统，扩展自有业务所需功能。

## 超丰富的场景支持

覆盖娱乐社交、企业协同、教育等领域的多人语聊房、电商直播、企业直播、娱乐直播等场景，同时也可通过自定义接口进行二次开发，满足个性化定制的需求。

## 高可靠高并发

NERoom SDK 提供消息必达的即时消息能力、安全稳定的实时音视频互动以及直播推流能力，可支撑百万级或更高级别的并发观看及互动能力。
