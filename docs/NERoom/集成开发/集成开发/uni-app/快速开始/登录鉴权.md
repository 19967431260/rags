您需要先登录 NERoom，才能进行创建房间、进入房间、发消息、共享白板等操作。


## 功能原理


![NERoom登录鉴权.png](https://yx-web-nosdn.netease.im/common/f5319cf141b15f8246296ef4e5c34f6d/NERoom登录鉴权.png)


1. App 客户端向 App Server（App自己的服务器）请求 Token。
2. App Server 通过网易云信 NERoom Server 提供的<a href="https://doc.yunxin.163.com/neroom/docs/TY1NzM5MjQ?platform=server" target="_blank">`创建账号`</a> 接口，获取 Token。
3. 网易云信 NERoom Server 返回 Token 给 App Server。
4. App Server 将 Token 返回给 App 客户端，App 客户端负责保存 Token。
5. App 客户端向 NERoom SDK 请求登录 NERoom，传入 userUuid 和 Token。
6. NERoom SDK 向 NERoom Server 请求校验该 Token 是否正确。
7. NERoom Server 校验 Token 成功，返回成功的回调给 NERoom SDK。
8. NERoom SDK 返回成功的回调给 App。



## 账号登录
### 配置步骤

1. 初始化 RoomKit SDK。
2. `AuthService` 实例。
3. 通过 `AuthService` 实例中的 `login` 接口进行账号登录。

::: note note
登录时的账号信息，请通过服务端的`getToken`接口获取。
:::

### 示例代码

  
实现账号登录的示例代码如下：
```
   /**
    * @param account 登录的账号
    * @param token 登录的token
    */
    const auth = new AuthService()
    auth.login('userId', 'token', (code, message) => {
    if (code !== 0) return console.error('登录失败', message)
        console.log('登录成功')
    })
```

## 监听登录状态

调用 `on` 接口进行监听登录状态。

```
/**
 * 监听登录状态
 */
    // NEAuthEvent: kickOut / forbidden / accountTokenError / loggedIn / loggedOut / incorrectToken / tokenExpired / reconnected
    const authEvents = ['kickOut','forbidden','accountTokenError','loggedIn','loggedOut','incorrectToken','tokenExpired','reconnected'];
    authEvents.forEach((evt) => {
      authService.on(evt, (data: any) => {
        console.error(`[AuthEvent] ${evt}, data:`, JSON.stringify(data));
      });
    });

/**
 * 取消监听
 */
authService.off('loggedIn')

```

## 查询登录状态

通过authService的 `isLoggedIn` 字段，查询登录状态。

- true：表示登录成功。
- false：表示未登录。

示例代码如下：

``` 
const loginStatus = authService.isLoggedIn
```

## 退出登录

调用 `logout` 接口实现退出登录。

示例代码如下：

```
authService.logout((code, message, data) => {
  console.log(
    `authService logout callback`,
    code,
    message,
    data,
  );
  if (code !== 0) {
    reject(new Error(message || "退出登录失败"));
  }
});
```
