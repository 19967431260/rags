将用户从房间移出。

使用场景：在内容安全审核等场景中，如果发现某个成员涉嫌违规，您需要将该成员强制移出房间，您可以调用该接口实现。服务端使用 checkSum 鉴权后，调用该接口具有超级管理员权限，对本产品下的所有房间成员具有移出权限。

## 接口请求地址

- 请求方法：DELETE
- URL：https://roomkit.yunxinapi.com/apps/v2/room-user?roomArchiveId={roomArchiveId}&userUuid={userUuid}
