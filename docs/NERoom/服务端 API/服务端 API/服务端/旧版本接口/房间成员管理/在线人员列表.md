查询房间的当前在线人员列表。

使用场景：如果您的业务系统中需要展示各个房间的实时在线人员，您可以调用该接口查询对应房间的在线人员情况。
::: note note
已结束的房间无法通过该接口查询在线人员列表。
:::
  

## 接口请求地址

- 请求方法：GET
- URL：https://roomkit.yunxinapi.com/apps/v2/online-user-list?roomArchiveId={roomArchiveId}&pageNumber={pageNumber}&pageSize={pageSize}



## URL 中参数说明

请求 Header 的设置请参见 <a href="https://doc.yunxin.163.com/neroom/docs/TMyMDg2Mzg?platform=server" target="_blank">请求结构</a>。

| 参数名称    | 类型   | 是否必选 | 示例 | 描述                                                 |
|-------------|--------|----------|------|--------------|
|   roomArchiveId      | String   | 是  |  4***251 |创建房间时，NERoom 服务端自动生成的一个用于标识房间的 ID，全局唯一，最大长度36个字符。<br>获取方法请参见<a href="https://doc.yunxin.163.com/neroom/docs/TM0MTUyNjc?platform=server" target="_blank">创建房间</a> 中的返回参数。        |
|   pageNumber      | Integer   | 否 |  1 |   查询页码，从 1 开始。如果参数为空，默认查询第 1 页。   |
|   pageSize      | Integer   | 否  |  10 |每页显示个数，最大支持 50 个。如果参数为空，默认显示 10 个。      |
