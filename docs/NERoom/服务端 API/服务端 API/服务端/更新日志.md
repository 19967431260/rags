## <span id="2022-10-24">2022-10-24</span>

**新增 API**

| API | 说明 |
| -- | -- |
| [更新昵称和头像](https://doc.yunxin.163.com/neroom/server-apis/jQ3MTcxMjY?platform=server) | 更新用户的昵称和头像。
| [创建房间](https://doc.yunxin.163.com/neroom/server-apis/jQ3MDk5ODc?platform=server) | 如果您需要在您的业务系统中保存创建房间后的相关房间信息，您可以使用该服务端创建房间的接口。 |
| [关闭房间](https://doc.yunxin.163.com/neroom/server-apis/zU5NDM1NDk?platform=server) | 对接您的业务系统，实现对所有房间进行管理的需要。 |
| [查询在线人员列表](https://doc.yunxin.163.com/neroom/server-apis/jQ1OTgyNDk?platform=server) | 如果您的业务系统中需要展示各个房间的实时在线人员，您可以调用该接口查询对应房间的在线人员情况。 |
| [移出房间成员](https://doc.yunxin.163.com/neroom/server-apis/TIxNjQ5NTA?platform=server) | 在内容安全审核等场景中，如果发现某个成员涉嫌违规，您需要将该成员强制移出房间，您可以调用该接口实现。 |
| [查询房间录制记录](https://doc.yunxin.163.com/neroom/server-apis/DA1MjA3NjI?platform=server) | 如果您的业务系统需要获取房间的录制记录以便进行回放，您可以通过该接口查询录制信息。 |
| [查询房间直播流名称](https://doc.yunxin.163.com/neroom/server-apis/DcwMTYyNDE?platform=server) | 直播、点播结束后，如果您需要查询对应的直播、点播用量数据，您可以通过该方法查询直播流名称，再通过 [数据统计](https://doc.yunxin.163.com/live-streaming/server-apis/jk1NTM3MTg?platform=server) 查询数据信息。 |
