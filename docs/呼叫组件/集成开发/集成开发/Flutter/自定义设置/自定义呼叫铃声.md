本文介绍在呼叫组件中如何自定义呼叫铃声。

## 功能介绍

呼叫组件的铃声分为 **呼叫铃声** 和 **被叫铃声**。

- 如果您通过源码依赖 NECallKit 组件，您可以替换铃声资源文件（呼叫铃声、被叫铃声）自定义铃声。
- 如果您通过非源码依赖 NECallKit 组件，目前支持自定义被叫铃声，呼叫铃声暂时不支持自定义。

## 实现方法

### 源码依赖

如果您通过源码依赖 NECallKit 组件，那么您可以直接替换源码中的铃声资源文件（呼叫铃声、被叫铃声）进行自定义铃声。

目前铃声资源文件的路径如下：

[呼叫铃声](https://github.com/netease-kit/NECallKit/blob/main/Flutter/assets/audios/avchat_connecting.mp3)：**NECallKit/Flutter/assets/audios/avchat_connecting.mp3**

[被叫铃声](https://github.com/netease-kit/NECallKit/blob/main/Flutter/assets/audios/avchat_ring.mp3)：**NECallKit/Flutter/assets/audios/avchat_ring.mp3**

### 非源码依赖

如果您通过非源码依赖 NECallKit 组件，请参考以下步骤自定义被叫铃声。

1. 将您的自定义铃声文件添加到 Flutter 项目的 `assets` 目录中。

    **推荐格式**：
    - **格式**：MP3、WAV、M4A（推荐 MP3）
    - **时长**：建议不超过 30 秒

    **目录结构**：
    ```
    your_project/
    └── assets/
        └── audios/
            └── custom_ringtone.mp3  # 您的自定义铃声
    ```

2. 在 `pubspec.yaml` 文件中声明音频资源：

    ```yaml
    flutter:
    assets:
        - assets/audios/custom_ringtone.mp3
    ```

3. 在完成初始化后，调用 `setCallingBell` 方法设置自定义铃声。

    ```dart
    import 'package:netease_callkit_ui/ne_callkit_ui.dart';

    // 设置自定义铃声
    await NECallKitUI.instance.setCallingBell('assets/audios/custom_ringtone.mp3');
    ```

    ::: note notice
    - `assetName` 参数应该是完整的 assets 路径，例如：`assets/audios/custom_ringtone.mp3`。
    - 如果只传入文件名，需要确保文件在默认的 assets 目录中。
    :::
