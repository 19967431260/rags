本文介绍在呼叫组件中如何自定义通话的超时时间。

## 功能介绍

网易云信呼叫组件支持设置通话的超时时间，单位为秒，默认为 30s。

设置后呼叫/接听的超时时间同时生效。

## 示例代码

```dart
import 'package:netease_callkit/netease_callkit.dart';

// 设置超时时间为 20 秒
final result = await NECallEngine.instance.setTimeout(20);
if (result.code == 0) {
  print('超时时间设置成功');
} else {
  print('设置失败: ${result.message}');
}
```
