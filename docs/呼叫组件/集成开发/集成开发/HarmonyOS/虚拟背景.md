网易云信音视频呼叫组件支持虚拟背景（模糊背景）功能。

## 功能概述

云信音视频组件允许用户在通话时使用虚拟背景功能，将视频通话画面的背景进行模糊处理。

::: note note
如果使用呼叫组件自带的虚拟背景，默认使用 **模糊背景（Blur）** 中的 **高模糊度** 类型。若需要使用其他类型，只能通过自定义虚拟背景功能，通过 RTC SDK 进行自定义。
:::

## 实现虚拟背景

如果您使用 `callkit_ui` 模块进行开发，可以通过 `NECallUI` API 来控制虚拟背景。

您可以通过调用 `enableVirtualBackground` 方法进行开启：

```
import { NECallUI } from 'callkit_ui';

// 启用虚拟背景
NECallUI.getInstance().enableVirtualBackground(true);

// 禁用虚拟背景
NECallUI.getInstance().enableVirtualBackground(false);
```

**完整的示例代码如下：**

```
import { NECallUI } from 'callkit_ui';
import { CallState } from 'callkit_ui/src/main/ets/impl/CallState';

/**
 * 通话页面组件
 */
@Entry
@Component
export struct CallPage {
  @State isVirtualBackgroundEnabled: boolean = false;

  aboutToAppear(): void {
    // 从设置中读取虚拟背景状态
    this.loadVirtualBackgroundSetting();
  }

  /**
   * 从设置中加载虚拟背景配置
   */
  private loadVirtualBackgroundSetting(): void {
    // 从 CallState 或设置中读取
    this.isVirtualBackgroundEnabled = CallState.instance.enableBlurBackground;
    
    // 应用设置
    if (this.isVirtualBackgroundEnabled) {
      NECallUI.getInstance().enableVirtualBackground(true);
    }
  }

  /**
   * 切换虚拟背景
   */
  toggleVirtualBackground(): void {
    const enable = !this.isVirtualBackgroundEnabled;
    
    // 调用 UI API 启用/禁用虚拟背景
    NECallUI.getInstance().enableVirtualBackground(enable);
    
    // 更新状态
    this.isVirtualBackgroundEnabled = enable;
    CallState.instance.enableBlurBackground = enable;
    
    console.info(`虚拟背景已${enable ? '启用' : '禁用'}`);
  }

  build() {
    Column() {
      // 通话界面...
      
      // 虚拟背景控制按钮
      Button(this.isVirtualBackgroundEnabled ? '关闭虚化' : '开启虚化')
        .onClick(() => {
          this.toggleVirtualBackground();
        })
    }
  }
}
```

## 自定义虚拟背景配置

如果需要自定义虚拟背景参数（如模糊程度），需要直接使用 RTC SDK 接口进行配置。

```
import { NERtcSDK, NERtcConstants } from '@nertc/nertc_sdk';

/**
 * 自定义虚拟背景配置
 */
export class VirtualBackgroundManager {
  /**
   * 启用模糊背景（高模糊度）
   */
  enableBlurBackgroundHigh(): number {
    const rtcService = NERtcSDK.getInstance();
    const source = new NERtcConstants.NERtcVirtualBackgroundSource();
    source.backgroundSourceType = NERtcConstants.NERtcVirtualBackgroundSourceType.kNERtcVirtualBackgroundBlur;
    source.blur_degree = NERtcConstants.NERtcBlurDegree.kNERtcBlurHigh;
    
    return rtcService.enableVirtualBackground(true, source, true);
  }

  /**
   * 启用模糊背景（中模糊度）
   */
  enableBlurBackgroundMedium(): number {
    const rtcService = NERtcSDK.getInstance();
    const source = new NERtcConstants.NERtcVirtualBackgroundSource();
    source.backgroundSourceType = NERtcConstants.NERtcVirtualBackgroundSourceType.kNERtcVirtualBackgroundBlur;
    source.blur_degree = NERtcConstants.NERtcBlurDegree.kNERtcBlurMedium;
    
    return rtcService.enableVirtualBackground(true, source, true);
  }

  /**
   * 启用模糊背景（低模糊度）
   */
  enableBlurBackgroundLow(): number {
    const rtcService = NERtcSDK.getInstance();
    const source = new NERtcConstants.NERtcVirtualBackgroundSource();
    source.backgroundSourceType = NERtcConstants.NERtcVirtualBackgroundSourceType.kNERtcVirtualBackgroundBlur;
    source.blur_degree = NERtcConstants.NERtcBlurDegree.kNERtcBlurLow;
    
    return rtcService.enableVirtualBackground(true, source, true);
  }

  /**
   * 使用图片作为背景（需要图片路径）
   */

    const rtcService = NERtcSDK.getInstance();
    const source = new NERtcConstants.NERtcVirtualBackgroundSource();
    source.backgroundSourceType = NERtcConstants.NERtcVirtualBackgroundSourceType.kNERtcVirtualBackgroundImg;
    source.source = imagePath; // 图片路径
    
    return rtcService.enableVirtualBackground(true, source, true);
  }

  /**
   * 禁用虚拟背景
   */
  disableVirtualBackground(): number {
    const rtcService = NERtcSDK.getInstance();
    return rtcService.enableVirtualBackground(false, null, true);
  }
}
```
