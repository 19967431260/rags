本文介绍如何快速运行 CallKit 的 HarmonyOS 示例项目，体验一对一音视频呼叫能力。

## 开发环境

示例项目对开发环境的要求如下所示：

环境要求 | 说明
--- | ---
DevEco Studio | 5.0 及以上版本
HarmonyOS SDK | API 12 及以上版本
HarmonyOS 设备 | HarmonyOS 5.0.0.102 及以上版本的设备
网络 | 稳定的网络连接

## 功能列表

模块 | 功能
--- | ---
音频通话 | <ul><li>支持一对一音频通话</li><li>支持扬声器/听筒切换</li><li>支持静音功能</li></ul>
视频通话 | <ul><li>支持一对一视频通话</li><li>支持前后摄像头切换</li><li>支持虚拟背景（需配置）</li><li>支持本地/远程画面切换</li></ul>
通话管理 | <ul><li>呼叫超时处理</li><li>通话状态管理</li><li>通话时长显示</li></ul>
UI 功能 | <ul><li>全屏通话界面</li><li>通话状态提示</li><li>Toast 提示</li></ul>
美颜功能(FaceUnity) | 需要配置 FaceUnity 相关资源且在设置中启用。
特效功能(ByteEffect) | 需要配置 ByteEffect 相关资源且在设置中启用。

## 前提条件

在开始运行示例项目之前，请确保您已完成以下操作：

- 已 [创建应用并获取应用的 App Key](https://doc.yunxin.163.com/console/concept/TIzMDE4NTA?platform=console)。
- 已开通以下服务，若未开通，请参考 [开通服务](https://doc.yunxin.163.com/console/concept/zc3NDYzNzc?platform=console) 进行开通。
    - IM 即时通讯。当使用呼叫组件自带的话单功能时，需开通 IM。
    - 信令。用于实现点对点呼叫邀请以及音视频通话。
    - 音视频通话 2.0。用于实现实时音视频通话。

        ::: note note
        建议开通音视频通话的 **调试模式** （[鉴权方式](https://doc.yunxin.163.com/nertc/server-apis/TcxNDAxMTI?platform=server)），调试模式建议只在集成开发阶段使用，请在应用正式上线前改回安全模式。
        :::

    - 如需要抄送，请提前开通消息抄送中的 **话单** 抄送服务，实现在一通通话结束后，发送事件通知消息，标记此次通话是否接通以及通话时间、类型等数据。

## 操作步骤

:::note note
- 示例源码仅供开发者接入参考，实际应用开发场景中，请结合具体业务需求修改使用。
- 若您计划将源码用于生产环境，请确保应用正式上线前已经过全面测试，以免因兼容性等问题造成损失。
- 示例代码中包含业务登录逻辑，如果您的业务中需要使用登录相关功能，请修改登录相关的逻辑。
:::

1. 前往 GitHub 克隆 [NECallKit 示例项目源码](https://github.com/netease-kit/NECallKit/tree/main/HarmonyOS) 仓库至您本地。

    示例项目结构如下：

    ```
    HarmonyOS/
    ├── entry/                    # 入口模块
    │   ├── src/main/ets/
    │   │   ├── config/          # 配置文件
    │   │   │   └── AppConfig.ets  # APP_KEY 配置
    │   │   ├── pages/           # 页面
    │   │   │   ├── Index.ets    # 首页
    │   │   │   ├── LoginPage.ets # 登录页
    │   │   │   └── SingleCallPage.ets # 通话页
    │   │   ├── viewmodel/       # 视图模型
    │   │   └── model/           # 数据模型
    │   └── src/main/module.json5
    │
    │
    ├── callkit_ui/               # UI 模块
    │   ├── src/main/ets/
    │   │   ├── pages/           # UI 页面
    │   │   ├── widget/          # UI 组件
    │   │   ├── impl/            # UI 实现
    │   │   └── utils/           # UI 工具
    │   └── oh-package.json5
    │
    ├── ByteEffect/                # 字节跳动特效模块（可选）
    ├── FaceUnity/                 # FaceUnity 美颜模块（可选）
    └── oh-package.json5          # 项目依赖配置
    ```

2. 找到 **entry/src/main/ets/config/AppConfig.ets** 文件替换为您的 AppKey。

    ```Groovy
    export class AppConfig {
    private static readonly APP_KEY: string = "your app key";  // 替换为您的 APP_KEY
    
    public static getAppKey(): string {
        return AppConfig.APP_KEY;
    }
    }
    ```

    ::: note note 
    AppKey 是应用在云信平台的唯一标识，需要在云信控制台获取，具体方法请参见 [获取 AppKey](https://doc.yunxin.163.com/console/concept/TIzMDE4NTA?platform=console)。
    :::

3. 在项目根目录执行 `ohpm install` 命令安装依赖。

4. 运行项目。

    1. 连接 HarmonyOS 设备或启动模拟器。
    2. 点击 Run 按钮。
    3. 等待应用安装和启动。

## 权限配置

实现音视频功能需要以下各种权限，云信提供的示例项目已在 ***entry/src/main/module.json5** 中配置应用权限。

- 摄像头权限（视频通话必需）：ohos.permission.CAMERA
- 麦克风权限（音频/视频通话必需）：ohos.permission.MICROPHONE
- 后台运行权限：ohos.permission.KEEP_BACKGROUND_RUNNING
- 网络信息权限：ohos.permission.GET_NETWORK_INFO

## 依赖配置

实现音视频功能需要添加以下依赖，云信提供的示例项目已在各模块的 **oh-package.json5** 中配置应用权限。

- 网易云信 RTC SDK：@nertc/nertc_sdk
- 网易云信 IM SDK 相关模块：@nimsdk/*
- 云信基础库：@yunxin/xkitbase

## 体验音视频

1. 登录应用。

    启动应用后，进入登录页面，输入账号 ID 和 Token，点击登录按钮。

    ::: note note 
    - 账号 ID 和 Token 需要在您的业务服务器生成。
    - 测试环境可以使用云信控制台生成的测试账号。
    :::

2. 发起音频/视频通话。

    输入对方账号 ID 后选择音频/视频通话，然后点击呼叫。

3. 接听通话。

    点击接听按钮接听或挂断按钮拒绝。

4. （可选）在通话过程中使用扬声器、虚拟背景等功能，具体请参见具体的功能文档。
5. （可选）在通话页面点击设置按钮，可配置呼叫超时时间、虚化、推送、话单等功能，具体请参见具体的功能文档。
