网易云信音视频呼叫（呼叫组件 NERtcCallKit）是网易云信推出的含 UI 的音视频通话组件，您只需要几行代码，就可以快速实现音视频呼叫，在 App 中添加包括呼叫、接通、拒接、以及通话中音频和视频的开关控制功能，并包含呼叫的 UI 界面。本文将介绍如何快速实现单聊通话。

## 功能概述

网易云信呼叫组件（NERTCCallkit）是基于网易云信信令、音视频通话 2.0 和 IM 即时通讯产品封装的融合性场景组件，简化了呼叫流程，将呼叫功能以 UI 组件化的形式提供给客户，提高接入效率、降低使用成本。支持直播、视频聊天、语音聊天、多人连麦、网络会议等社交办公场景。

呼叫组件（NERtcCallKit）支持的功能如下表所示。

功能 | 描述
--- | ---
自定义 UI | 您可以自行设计通话和呼叫页面 UI。
音视频呼叫 | App 通过此功能通知被叫用户呼叫请求，呼叫请求类型包括音频和视频呼叫。
音视频通话 | 接通后可依照呼叫类型进行实时通话。
音视频控制 | 通话过程中可以控制本端音频或视频的开关，以及摄像头方向等。
话单 | 每次通话结束后都会收到对应的话单消息，标记本次通话是否接通以及通话时间、类型等数据。

## 适用场景

- **在线教育**：一对一辅导、小班课教学

- **在线问诊**：远程医疗咨询、健康管理
- **远程办公**：视频会议、远程协作

## 开发环境

- **HbuilderX 版本要求**：建议使用最新版本

    HBuilder X 4.64 和 4.65 版本对 uts 插件打包存在兼容性问题，请勿使用
- **iOS 设备**：iOS 13.0 及以上，支持音视频功能，不支持模拟器
- **Android 设备**：Android 5.0 及以上，支持音视频功能，不支持模拟器（使用真机请开启调试选项）
- **网络环境**：设备已连接到互联网

### 步骤一：创建项目

若您已经有自己的项目，可忽略此步骤。

打开 HBuilderX 开发工具，点击新建 uni-app 项目，填入项目名称，选择默认模板以及 Vue 版本并创建项目。

### 步骤二：导入组件

打开 [netease-CallKit 插件](https://ext.dcloud.net.cn/plugin?id=26107)，单击下载插件并导入 HBuilderX，选择需要集成的项目并单击确定。

![下载并导入.png](https://yx-web-nosdn.netease.im/common/03ddff36b4234a8c99a1c97ffccdd504/下载并导入.png)

### 步骤三：工程配置

1. 安装 nim-web-sdk-ng。

    打开命令行窗口，在项目根目录下执行如下命令。

    ``` bash
    npm install nim-web-sdk-ng
    ```

2. 注册通话页面。

    在项目的 pages.json 文件中注册通话页面。

    ``` javascript
    {
        "path": "uni_modules/Netease-Call/callkit/callPage",
        "style": {
          "navigationBarTitleText": "",
          "navigationStyle": "custom"
        }
    }
    ```

### 步骤四：登录

登录是使用 CallKit 所有功能的前提，请确保参数配置正确。

1. 全局注册实例。

    在 main.js 中引入 nim-web-sdk-ng、NECallKit 和 NEStore ，将其注册到全局变量 uni 上，确保 App 内所有组件共享一个实例。

    ``` javascript
    import V2NIM from "nim-web-sdk-ng/dist/v2/NIM_MINIAPP_SDK";
    import { NECallKit } from "@/uni_modules/Netease-Call/callkit/callServices/services/index";
    import { NEStore } from "@/uni_modules/Netease-Call/callkit/callServices/NEStore/index";

    const appkey = "Your AppKey";

    uni.$UIKitNIM = V2NIM.getInstance({appkey, apiVersion: 'v2'});
    uni.$NECallKit = NECallKit;
    uni.$NEStore = NEStore;
    ```

2. 调用登录接口。

    在您需要使用通话功能的页面，调用 `uni.$UIKitNIM.V2NIMLoginService.login、uni.$NECallKit.login` 方法进行登录。

    ``` javascript
    const accountId = "Your AccountId";
    const token = "Your Token";

    await uni.$UIKitNIM.V2NIMLoginService.login(accountId, token);

    uni.$NECallKit.init({appKey});

    uni.$NECallKit.login({
      accountId,
      token,
      success: () => {
        console.log("login success");
      },
      fail: (errCode, errMsg) => {
        console.error(`login failed, errCode: ${errCode}, errMsg: ${errMsg}`)
      },
    });
    ```

### 步骤五：发起通话

登录成功后，即可调用 `uni.$NECallKit.call` 方法向其他用户发起通话。

``` javascript
uni.$NECallKit.call({
  accountId: 'jenny',
  callType: 2, // 媒体类型：1: 语音通话 2: 视频通话
});
```

### 步骤六：接听通话

CallKit 插件实现了完整的接听界面和来电提醒逻辑。当收到其他用户的呼叫时，会自动弹出通话界面，无需编写额外的接听逻辑。

如果应用处于后台或离线状态，需要额外的离线唤醒配置，具体请参考 [离线推送](https://doc.yunxin.163.com/nertccallkit/guide/TI0NDU1NTA?platform=uniapp) 文档。

1. 引入离线插件。

    在 main.js 中 uni.$UIKitNIM 初始化后，执行离线推送配置。

    ``` javascript
    uni.$NIMPushPlugin = uni.requireNativePlugin("NIMUniPlugin-PluginModule");

    uni.$UIKitNIM.V2NIMSettingService.setOfflinePushConfig(uni.$NIMPushPlugin, {
      miPush: {
          appId: 'xxx',
          appKey: 'yyy',
          certificateName: 'XIAOMI_PUSH_CERT'
      },
      vivoPush: {
          certificateName: 'VIVO_PUSH_CERT'
      },
      oppoPush: {
          appId: 'xxx',
          appKey: 'yyy',
          secret: 'zzz',
          certificateName: 'OPPO_PUSH_CERT'
      },
      hwPush: {
          appId: 'xxx',
          certificateName: 'HUAWEI_PUSH_CERT'
      },
      fcmPush: {
          certificateName: 'FCM_PUSH_CERT'
      },
      mzPush: {
          appId: 'xxx',
          appKey: 'yyy',
          certificateName: 'MEIZU_PUSH_CERT'
      },
      honorPush: {
          certificateName: 'HONOR_PUSH_CERT'
      },
      apns: {
        certificateName: 'APPLE_PUSH_CERT',
      },
    });
    ```

2. 注册通知点击事件。

    在 App.vue 中 onShow 声明周期中处理通知点击跳转逻辑。

    ``` javascript
    onShow: function () {
      console.log("App Show");
      uni?.$UIKitNIM?.V2NIMSettingService?.setAppBackground(false);
      
      uni?.$NIMPushPlugin?.addOpenNotificationListener((result) => {
        if (typeof result === 'object') {
          console.log('=====addOpenNotificationListener success:', JSON.stringify(result))
          
          //TODO 处理推送点击跳转逻辑
        } else {
          console.log('=====addOpenNotificationListener unexpected:', result)
        }
      });
    },

    onHide: function () {
      console.log("App Hide");
      uni?.$UIKitNIM?.V2NIMSettingService?.setAppBackground(true);
    },
    ```
