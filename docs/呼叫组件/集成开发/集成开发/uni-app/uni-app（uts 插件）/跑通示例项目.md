本文将介绍如何在您的 uni-app 项目中快速集成 NERtcCallKit UTS 插件，跑通音视频呼叫的示例项目，体验音视频通话功能。

## 开发环境

示例项目对开发环境的要求如下所示：

- HBuilder X：建议使用最新版本

    HBuilder X 4.64 和 4.65 版本对 uts 插件打包存在兼容性问题，建议使用最新版本。
    
- Android 设备 5.0 及以上
- iOS 设备 13.0 及以上

## 前提条件

在开始运行示例项目之前，请确保您已完成以下操作：

- 已 [创建应用并获取应用的 App Key](https://doc.yunxin.163.com/console/concept/TIzMDE4NTA?platform=console)。
- 已获取本插件使用中需要的 account 和 token，具体请参考 [注册 IM 账号](https://doc.yunxin.163.com/messaging2/guide/jU0Mzg0MTU?platform=client#第二步注册-im-账号)。
- 使用音视频呼叫（呼叫组件）之前，需要开通以下功能。如何开通请参考 [开通服务](https://doc.yunxin.163.com/console/concept/zc3NDYzNzc?platform=console)。
    - IM 即时通讯。当使用呼叫组件自带的话单功能时，需开通 IM。
    - 信令。用于实现点对点呼叫邀请以及音视频通话。
    - 音视频通话 2.0。用于实现实时音视频通话。

        ::: note note
        建议开通音视频通话的 **调试模式** （[鉴权方式](https://doc.yunxin.163.com/nertc/server-apis/TcxNDAxMTI?platform=server)），调试模式建议只在集成开发阶段使用，请在应用正式上线前改回安全模式。
        :::

    - 如需要抄送，请提前开通消息抄送中的 **话单** 抄送服务，实现在一通通话结束后，发送事件通知消息，标记此次通话是否接通以及通话时间、类型等数据。

::: note notice
如果您同时接入了 IM 即时通讯和呼叫组件，那么需要提前在网易云信控制台开启 **多端登录**，具体请参考 [多端登录与互踢](https://doc.yunxin.163.com/messaging2/guide/DUyNjI2MjQ?platform=client)。
:::

## 操作步骤

:::note note
- 本项目使用 uni-app UTS插件，具体请参考 [UTS 插件](https://doc.dcloud.net.cn/uni-app-x/plugin/uts-plugin.html) 。
- 示例源码仅供开发者接入参考，实际应用开发场景中，请结合具体业务需求修改使用。
- 若您计划将源码用于生产环境，请确保应用正式上线前已经过全面测试，以免因兼容性等问题造成损失。
- 示例代码中已包含业务登录逻辑，如果您的业务中需要使用登录相关功能，请按需修改。
:::

### 步骤一：下载 Demo

1. 前往 GitHub 下载 [NECallkit-app-uts](https://github.com/netease-kit/NECallKit/tree/main/uni-app/NECallkit-app-uts) 源码至您的仓库，或者直接在命令行运行以下命令。

   ``` bash
   git clone https://github.com/netease-kit//NECallKit.git
   ```
   
2. 安装依赖。

   ``` bash
   cd ./NECallKit/uni-app/NECallkit-app-uts
   npm install
   ``` 

3. 通过 HbuiderX 打开 **NECallKit/uni-app/NECallkit-app-uts** 项目。

   ![](https://yx-web-nosdn.netease.im/common/d3bad4fbfc8bcd57b465b40ac3ac575b/uni-app-uts_img01.png)

4. 在 `NECallKit/uni-app/NECallKit-app-uts/config.js` 文件中配置您的 AppKey。

   ![](https://yx-web-nosdn.netease.im/common/943264022f8684bce1ca0a2d3a906dc1/uni-app-uts_img05.png)

    ```javascript
    // 应用Key - - 请替换为你的实际appKey
    appKey: "your_app_key",
    ```

### 步骤三：拨打音视频通话

1. 分别在两台设备上登录两个用户，一方作为主叫，一方作为被叫。

2. 在主叫方进入通话界面，输入被叫方的 accoudId，发起通话。

3. 被叫方收到通话的请求，点击接听后进行通话。
