本文介绍呼叫组件的错误码及相关说明。

## 组件内部错误码

| 错误码 |                   说明                   |
|------|----------------------------------------|
| 100    | 内部错误                                 |
| 101    | IM 连接已断开                             |
| 102    | IM 实例不存在                             |
| 103    | 房间信息不存在或已销毁                   |
| 104    | RTC 的 client 实例不存在                    |
| 105    | RTC 的音视频 stream 不存在                  |
| 106    | 呼叫类型不存在                           |
| 108    | 接口超时                                 |
| 110    | 取消呼叫部分失败，可忽略                 |
| 111    | 信令离开失败，可能是已离开导致的，可忽略 |
| 112    | RTC 离开失败，可能是已离开导致的，可忽略 |
| 113    | 通话时长不存在                           |
| 114    | 设备 ID 不存在                             |
| 115    | 邀请者房间信息不存在                     |
| 116    | 呼叫正忙                                 |
| 117    | 被叫正忙                                 |
| 118    | 群呼叫只有通个中和呼叫中才能邀请其他人  |
| 119    | 没有群呼叫                                 |
| 120    | 房间 ID 不存在                             |
| 121    | RTC 实例不存在                            |
