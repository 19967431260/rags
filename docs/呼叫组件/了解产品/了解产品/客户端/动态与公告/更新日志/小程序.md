本文介绍呼叫组件（NERtcCallKit）小程序的更新日志和升级说明。

## 4.2.0 (2026-03-14)

**新增特性**

- 新增微信小程序端独立构建产物（`miniprogram_dist`），支持通过 `@xkit-yx/call-kit/miniprogram_dist/index` 引入。
- 新增通话铃声功能，呼叫中/来电中自动播放，接通/挂断时停止。
- 新增本地视频小窗支持拖动（movable-area + movable-view）。
- 新增通话结束时按 `reasonCode` 展示提示语（多端接听/拒绝/超时等场景）。
- 新增鸿蒙系统自适应 live-player mode（鸿蒙使用 live，其他平台使用 RTC）。

**兼容版本**

- 适配 NIM SDK 10.9.70 版本。
- 适配 NERtc SDK 5.6.41 版本。
- 兼容微信小程序基础库 live-pusher/live-player 相关 API。
