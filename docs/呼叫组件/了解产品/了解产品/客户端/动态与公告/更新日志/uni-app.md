本文记录了网易云信呼叫组件（NERtcCallKit）适配 uni-app 框架的版本迭代历史，包括功能更新、问题修复及适配优化等信息。您可以前往《DCloud 官方网站》下载集成 [呼叫组件 NERtcCallKit](https://ext.dcloud.net.cn/plugin?id=14739)。

## 4.2.0 (2026-03-14)

**新增特性**

底层兼容 NIM V10 SDK（nim-web-sdk-ng）。信令层全面迁移至 `V2NIMSignallingService` API。

**问题修复**

修复通话正常挂断后未生成话单的问题（status=1 completed 话单，包含通话时长）。

**兼容版本**

- 适配 NIM SDK 10.3.0 版本。
- 适配 NERtc SDK 5.6.41 版本。

## 3.8.0 (2025-11-6)

**新增特性**

- **音质优化：** 引入 AI 降噪模式，显著提升通话音质。
- **群呼体验升级：**
   - 增加推送配置选项，优化离线通知体验。
   - 新增自定义配置频道名称。

**问题修复**

更新相关 framework，修复 iOS 可能出现的上架问题。

**兼容版本**

- 适配 CallKit Android 3.8.0 版本。
- 适配 CallKit iOS 3.8.0 版本。

## 3.7.0 (2025-09-26)

**新增特性**

- **多人群呼功能**：支持多人通话场景，满足群组音视频交流需求。
- **小窗拖拽交互**：一对一通话支持小窗模式自由拖拽，提升多任务操作体验。

**兼容版本**

- 适配 CallKit Android 3.7.0 版本
- 适配 CallKit iOS 3.7.0 版本

## 2.6.0 (2025-01-18)

**新增功能**

- **语音转视频询问机制**：通话过程中进行语音转视频时，新增询问确认弹窗。

**平台适配**

- **Android**：升级呼叫组件内核，完成 Target API 34 适配。
- **iOS**：升级呼叫组件内核，解决 iOS 18 系统上 `maskView` 冲突导致的崩溃问题。

**问题修复**

- 修复首次呼叫时对方账号昵称无法正确展示的问题。

## 2.4.5 (2025-01-06)

**问题修复**

- 修复 iOS 平台全局监听事件不生效的问题。
- 修复 Android 平台用户昵称展示错误的问题。

## 2.4.4 (2024-12-21)

**优化调整**

- 关闭 Android 平台的自启动功能。

## 2.4.3 (2024-11-01)

**新增功能**

- 支持外部传递 RTC channel name 和 RTC UID 参数。

## 2.4.2 (2024-06-20)

**问题修复**

- 修复通话过程中偶现铃声从听筒播放的问题。

## 2.4.1 (2024-06-17)

**新增功能**

## 2.4.0 (2024-06-05)

**新增功能**

- **音视频转换配置**：新增 `initConfig` 配置项 `enableAudio2Video` 和 `enableVideo2Audio`，支持开启或隐藏音视频通话转换功能。
- **呼叫回调**：`toCallPage` 接口新增成功/失败回调，便于开发者获取呼叫结果。
