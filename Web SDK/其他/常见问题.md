## Q: Web SDK 登录后调用 getLocalMsgs 查询本地消息为空是什么原因？

**答案**：Web SDK 的本地消息存储在浏览器的 IndexedDB 中。如果查询为空，可能是：1) 首次登录，本地还没有消息；2) 浏览器清理了 IndexedDB 数据；3) 使用了无痕模式或隐私模式。Web SDK 不像移动端有持久化存储，依赖浏览器的本地存储机制。
