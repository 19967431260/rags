## Q: Flutter SDK 登录报错 code=102422 是什么原因？

**答案**：错误码 102422 表示账户被封禁（account banned）。可以通过服务端 API 解除封禁：https://doc.yunxin.163.com/messaging2/server-apis/TIxMzI4MjE?platform=server
