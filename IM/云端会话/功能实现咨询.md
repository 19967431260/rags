## Q: Web SDK V9升级到C++ SDK V10历史会话迁移方案

**问题描述**：客户从Web SDK V9升级到C++ SDK V10，需要迁移历史会话数据。V9使用本地会话，V10需要云端会话，担心历史会话丢失。

**答案**：**迁移方案**：
1. 开启云端会话功能（修改device.properties配置）
2. 本地会话通过服务端接口批量创建云端会话入口
3. 历史消息通过云端历史消息接口查询（可开到1年）
4. 双写一段时间（1-2个月），等云端数据覆盖足够久后切换
5. V9的sessionId格式（p2p-account2）需转换为V10格式（account1|1|account2）

**配置开启**：
```
# 进入容器
docker exec -it aio bash
# 修改两个配置文件
/opt/nim/webapps/nimserver/WEB-INF/classes/device.properties
/etc/nim/device/device.properties
ConversationEnable=false->true
ConversationGroupEnable=false->true
# 10分钟后自动生效，验证nim_conversation表有数据
```
