## Q: 服务端 API 查询多个群聊时，如果某个 teamId 不存在就返回 414 错误，能否支持匹配多少返回多少？

**答案**：V9 接口不支持部分匹配返回。建议使用 V10 的群组查询接口（https://doc.yunxin.163.com/messaging2/server-apis/DAyODY4ODE?platform=server），该接口做了优化。V9 和 V10 接口可以同时使用，但返回字段可能有区别，需要调整代码适配。
