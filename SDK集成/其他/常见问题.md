## Q: iOS SDK 登录失败报错 code=191001 是什么原因？

**答案**：错误码 191001 通常是因为 SDK 未初始化。需要先调用初始化方法再进行登录操作。

## Q: iOS SDK 是否支持 SPM (Swift Package Manager) URL 方式集成？

**答案**：当前版本暂不支持 SPM URL 方式集成，只支持 Pod 集成和本地手动集成。该需求已排期，预计下周会快速安排版本支持。

## Q: iOS SDK 海外版本需要集成哪些 xcframework？

**答案**：海外版本必须集成：NIMSDK.xcframework 和 NIMFCS.xcframework。如果使用 AWS 存储方案（NIMAWSCore + NIMAWSS3）替代 NIMFCS，则需要集成：NIMSDK.xcframework、NIMAWSCore.xcframework、NIMAWSS3.xcframework 和 NIMNOS.xcframework（四个库缺一不可）。
