## Q: Android 端为什么收不到特定用户的消息推送？

**答案**：问题原因是该用户的头像使用了 HTTP 地址。当消息 payload 中包含 HTTP 头像地址时，推送到华为厂商通道会返回报错，要求必须使用 HTTPS 地址。解决方案：确保用户头像和消息中的所有图片资源都使用 HTTPS 协议。
