## 问题：Web端呼叫组件仅支持React框架，Vue需使用无UI版本

## 问题详情

**现象**：
客户在Uniapp平台购买v9版本，需要在Web端聊天界面与App端进行语音通话。查看文档发现呼叫组件示例代码是React的，询问Vue框架如何集成。

## 排查过程



## 问题原因

云信呼叫组件Web端带UI版本仅支持React框架

## 解决方案

1. 带UI的呼叫组件Web端仅支持React框架
2. Vue项目可使用不带UI的呼叫组件进行集成
3. 参考文档：https://doc.yunxin.163.com/nertccallkit/guide/jU5NjA0MzI?platform=web#集成呼叫组件不含ui
